# 61. 삼분 탐색 (Ternary Search)

볼록(convex) 또는 오목(concave) 함수에서 극값(최솟값/최댓값)을 O(log N)에 찾는 탐색 알고리즘이다. 구간을 3등분하여 두 내부 점의 함수값을 비교함으로써 극값이 존재하지 않는 구간을 매 반복마다 1/3씩 제거한다. BOJ 골드 상위~플래티넘 문제에서 최적화 문제로 자주 등장하는 상위권 차별화 유형이다.
