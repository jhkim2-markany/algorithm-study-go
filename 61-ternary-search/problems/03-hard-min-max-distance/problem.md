# 최소 최대 거리

**난이도:** 상

## 문제 설명

2차원 평면 위에 N개의 점이 주어진다. x축 위의 한 점 P(t, 0)을 선택하여, N개의 점까지의 거리 중 최댓값을 최소화하려고 한다.

즉, 다음 값을 최소화하는 t를 구하시오:

```
f(t) = max(dist(P, Pᵢ)) = max(√((t - xᵢ)² + yᵢ²))  (i = 1, ..., N)
```

여기서 dist(P, Pᵢ)는 점 P(t, 0)과 점 Pᵢ(xᵢ, yᵢ) 사이의 유클리드 거리이다.

## 입력 형식

- 첫째 줄에 점의 개수 N이 주어진다 (1 ≤ N ≤ 100,000)
- 다음 N개의 줄에 각 점의 좌표 xᵢ, yᵢ가 공백으로 구분되어 주어진다

## 출력 형식

- 첫째 줄에 최적의 t 값을 소수점 아래 6자리까지 출력한다
- 둘째 줄에 최소화된 최대 거리를 소수점 아래 6자리까지 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- -10⁶ ≤ xᵢ, yᵢ ≤ 10⁶
- xᵢ, yᵢ는 정수로 주어진다
- 시간 제한: 2초

## 예제

### 예제 입력 1

```
3
0 3
4 0
8 3
```

### 예제 출력 1

```
4.000000
5.000000
```

### 예제 입력 2

```
2
0 1
6 1
```

### 예제 출력 2

```
3.000000
3.162278
```
