# 최소 최대 거리 - 해설

## 접근 방식

1. N개의 점 좌표를 입력받는다
2. x축 위의 점 P(t, 0)에서 모든 점까지의 최대 거리를 함수 f(t)로 정의한다
3. f(t)가 볼록 함수임을 이용하여 삼분 탐색으로 최솟값을 찾는다
4. 탐색 구간은 x 좌표의 범위에 여유를 두고 설정한다
5. 200회 반복으로 충분한 정밀도를 확보한다

## 핵심 아이디어

이 문제의 핵심은 f(t) = max(dist(P(t,0), Pᵢ))가 **볼록 함수**임을 파악하는 것이다.

각 점 Pᵢ(xᵢ, yᵢ)에 대해 거리 함수:

```
dᵢ(t) = √((t - xᵢ)² + yᵢ²)
```

는 t에 대한 볼록 함수이다 (이차 함수의 제곱근이며, 합성 함수의 볼록성 보존).

볼록 함수의 최댓값(pointwise maximum)도 볼록 함수이므로:

```
f(t) = max(d₁(t), d₂(t), ..., dₙ(t))
```

도 볼록 함수이다. 따라서 삼분 탐색을 적용할 수 있다.

기하학적 직관으로 보면, 이 문제는 "N개의 점을 모두 포함하는 가장 작은 원의 중심이 x축 위에 있을 때, 그 원의 반지름을 최소화하라"는 문제와 동일하다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N × 200) = O(N), 매 반복마다 N개 점 순회 |
| 공간 복잡도 | O(N) |

## 대안적 접근

- 이진 탐색 + 판별: "최대 거리가 D 이하인 t가 존재하는가?"를 판별 함수로 만들어 이진 탐색. 각 점에서 거리 D 이내인 x축 구간의 교집합을 구한다
- 기하학적 접근: 최소 포함 원(Minimum Enclosing Circle) 알고리즘을 x축 제약 조건과 결합
