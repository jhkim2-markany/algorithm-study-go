# 최대 넓이 구하기 - 해설

## 접근 방식

1. 철사 길이 L을 입력받는다
2. 정사각형에 사용하는 길이를 x라 하면, 넓이 합 함수 f(x)를 정의한다:
   - 정사각형 넓이: (x/4)² = x²/16
   - 원 넓이: π × ((L-x)/(2π))² = (L-x)²/(4π)
   - f(x) = x²/16 + (L-x)²/(4π)
3. f(x)는 이차 함수이며 x²의 계수가 양수이므로 아래로 볼록하다
4. 볼록 함수의 최댓값은 구간의 양 끝점에서 발생한다
5. 삼분 탐색으로 최솟값 위치를 찾고, 양 끝점의 함수값을 비교하여 최댓값을 결정한다

## 핵심 아이디어

넓이 합 함수를 미분하면:

```
f(x) = x²/16 + (L-x)²/(4π)
f'(x) = x/8 - (L-x)/(2π)
```

f'(x) = 0을 풀면 최솟값 위치를 구할 수 있다. 하지만 이 문제의 핵심은 삼분 탐색의 활용이다.

볼록 함수에서:
- 삼분 탐색은 최솟값을 직접 찾는다
- 최댓값은 구간의 양 끝점 중 하나에서 발생한다

이 문제에서 f(0) = L²/(4π), f(L) = L²/16이다. π > 4이므로 4π > 16이고, 따라서 L²/(4π) < L²/16이다. 즉 x = L (전부 정사각형)일 때 넓이가 더 크다.

하지만 실제로는 1/(4π) ≈ 0.0796이고 1/16 = 0.0625이므로 f(0) > f(L)이다. 따라서 x = 0 (전부 원)일 때 넓이 합이 최대이다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(200) = O(1), 고정 반복 횟수 |
| 공간 복잡도 | O(1) |

## 대안적 접근

- 미분을 이용한 해석적 풀이: f'(x) = 0을 직접 풀어 최솟값 위치를 구하고, 양 끝점과 비교
- 볼록 함수의 성질을 이용: 최댓값이 반드시 구간 끝점에서 발생함을 증명하고 직접 비교
