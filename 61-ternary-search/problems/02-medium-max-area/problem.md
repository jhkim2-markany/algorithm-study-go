# 최대 넓이 구하기

**난이도:** 중

## 문제 설명

길이가 L인 철사가 있다. 이 철사를 한 번 잘라서 두 조각으로 나눈 뒤, 한 조각으로는 정사각형을, 다른 조각으로는 원을 만들려고 한다.

정사각형의 둘레와 원의 둘레의 합이 L이 되도록 할 때, 정사각형의 넓이와 원의 넓이의 합이 최대가 되는 경우를 구하시오.

정사각형에 사용하는 철사의 길이를 x (0 ≤ x ≤ L)라 하면:
- 정사각형의 한 변의 길이: x/4
- 정사각형의 넓이: (x/4)²
- 원의 둘레: L - x
- 원의 반지름: (L-x) / (2π)
- 원의 넓이: π × ((L-x)/(2π))²

## 입력 형식

- 첫째 줄에 철사의 길이 L이 주어진다 (1 ≤ L ≤ 10⁶)

## 출력 형식

- 첫째 줄에 정사각형에 사용하는 철사의 길이 x를 소수점 아래 6자리까지 출력한다
- 둘째 줄에 넓이의 합의 최댓값을 소수점 아래 6자리까지 출력한다

## 제약 조건

- 1 ≤ L ≤ 1,000,000
- L은 정수로 주어진다
- 시간 제한: 1초

## 예제

### 예제 입력 1

```
10
```

### 예제 출력 1

```
0.000000
7.957747
```

### 예제 입력 2

```
100
```

### 예제 출력 2

```
0.000000
795.774715
```
