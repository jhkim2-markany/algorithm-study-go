# 볼록 함수 최솟값 찾기

**난이도:** 하

## 문제 설명

이차 함수 f(x) = ax² + bx + c가 주어진다 (a > 0). 구간 [lo, hi]에서 f(x)의 최솟값을 구하시오.

a > 0이므로 f(x)는 아래로 볼록한 함수이며, 구간 내에서 유일한 최솟값이 존재한다.

## 입력 형식

- 첫째 줄에 정수 a, b, c가 공백으로 구분되어 주어진다 (1 ≤ a ≤ 100, -1000 ≤ b, c ≤ 1000)
- 둘째 줄에 실수 lo, hi가 공백으로 구분되어 주어진다 (-10⁶ ≤ lo < hi ≤ 10⁶)

## 출력 형식

- 첫째 줄에 f(x)가 최솟값을 가지는 x 좌표를 소수점 아래 6자리까지 출력한다
- 둘째 줄에 f(x)의 최솟값을 소수점 아래 6자리까지 출력한다

## 제약 조건

- 1 ≤ a ≤ 100
- -1,000 ≤ b, c ≤ 1,000
- -10⁶ ≤ lo < hi ≤ 10⁶
- 최솟값의 x 좌표는 구간 [lo, hi] 안에 존재함이 보장된다
- 시간 제한: 1초

## 예제

### 예제 입력 1

```
1 -6 10
0 10
```

### 예제 출력 1

```
3.000000
1.000000
```

### 예제 입력 2

```
2 4 1
-5 5
```

### 예제 출력 2

```
-1.000000
-1.000000
```
