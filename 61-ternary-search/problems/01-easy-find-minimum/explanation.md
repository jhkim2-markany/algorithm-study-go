# 볼록 함수 최솟값 찾기 - 해설

## 접근 방식

1. 이차 함수 f(x) = ax² + bx + c의 계수와 탐색 구간 [lo, hi]를 입력받는다
2. a > 0이므로 f(x)는 아래로 볼록한 유니모달 함수이다
3. 삼분 탐색을 적용하여 최솟값의 x 좌표를 찾는다
4. 구간을 3등분하는 두 점 m1, m2에서 함수값을 비교하여 구간을 축소한다
5. 200회 반복 후 구간의 중점을 최솟값 위치로 출력한다

## 핵심 아이디어

이차 함수 f(x) = ax² + bx + c (a > 0)는 꼭짓점 x = -b/(2a)에서 최솟값을 가진다. 수학적으로는 공식으로 바로 구할 수 있지만, 이 문제는 삼분 탐색의 기본 적용을 연습하는 것이 목적이다.

삼분 탐색의 핵심은:
- 구간 [lo, hi]를 3등분하는 두 점 m1, m2를 잡는다
- f(m1) < f(m2)이면 최솟값은 [lo, m2] 구간에 존재한다
- f(m1) ≥ f(m2)이면 최솟값은 [m1, hi] 구간에 존재한다
- 매 반복마다 구간이 2/3로 줄어든다

반복 횟수를 200회로 고정하면 구간이 (2/3)^200 ≈ 10^-35 배로 줄어들어, 어떤 초기 구간에서도 충분한 정밀도를 얻는다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(200) = O(1), 고정 반복 횟수 |
| 공간 복잡도 | O(1) |

## 대안적 접근

- 수학 공식: x = -b/(2a)로 직접 계산 가능 (O(1))
- 이진 탐색: 도함수 f'(x) = 2ax + b가 0이 되는 점을 이진 탐색으로 찾을 수도 있다
