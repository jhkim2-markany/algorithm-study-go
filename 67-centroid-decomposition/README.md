# 67. 센트로이드 분할 (Centroid Decomposition)

트리에서 센트로이드(무게 중심)를 반복적으로 찾아 제거하면서 분할 정복을 수행하는 기법이다. 센트로이드를 기준으로 트리를 분할하면 재귀 깊이가 O(log N)으로 보장되므로, 트리 위의 경로 관련 문제를 효율적으로 해결할 수 있다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이

## 문제 목록

| 번호 | 문제 | 난이도 |
| --- | --- | --- |
| 01 | 거리가 K 이하인 노드 쌍 세기 | 하 |
| 02 | 특정 거리 경로 수 세기 | 중 |
| 03 | 색칠된 경로 쿼리 | 상 |
