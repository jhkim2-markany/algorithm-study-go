# 색칠된 경로 쿼리

**난이도:** 상

## 문제 설명

N개의 노드로 이루어진 트리가 주어진다. 각 노드에는 색상이 칠해져 있으며, 색상은 1 이상의 정수로 표현된다. 각 간선의 가중치는 1이다.

Q개의 쿼리가 주어진다. 각 쿼리는 정수 K로, 트리에서 거리가 정확히 K이면서 양 끝 노드의 색상이 같은 순서쌍 (u, v) (u < v)의 수를 구해야 한다.

## 입력 형식

- 첫째 줄에 노드의 수 N과 쿼리의 수 Q가 주어진다.
- 둘째 줄에 각 노드의 색상 c₁, c₂, ..., cₙ이 공백으로 구분되어 주어진다.
- 셋째 줄부터 N-1개의 줄에 간선 정보 u, v가 주어진다.
- 그 다음 Q개의 줄에 각 쿼리의 정수 K가 주어진다.

## 출력 형식

- 각 쿼리에 대해 조건을 만족하는 노드 쌍의 수를 한 줄에 하나씩 출력한다.

## 제약 조건

- 1 ≤ N ≤ 50,000
- 1 ≤ Q ≤ 100
- 1 ≤ cᵢ ≤ N
- 1 ≤ K ≤ N - 1
- 1 ≤ u, v ≤ N
- 주어지는 그래프는 항상 트리이다
