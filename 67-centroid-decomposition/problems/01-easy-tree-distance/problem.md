# 거리가 K 이하인 노드 쌍 세기

**난이도:** 하

## 문제 설명

N개의 노드로 이루어진 트리가 주어진다. 각 간선의 가중치는 1이다.

정수 K가 주어질 때, 트리에서 두 노드 사이의 거리가 K 이하인 순서쌍 (u, v) (u < v)의 수를 구하라.

두 노드 사이의 거리는 두 노드를 잇는 경로 위의 간선 수이다.

## 입력 형식

- 첫째 줄에 노드의 수 N과 정수 K가 주어진다.
- 둘째 줄부터 N-1개의 줄에 간선 정보 u, v가 주어진다. 이는 노드 u와 노드 v가 간선으로 연결되어 있음을 의미한다.

## 출력 형식

- 거리가 K 이하인 노드 쌍의 수를 출력한다.

## 제약 조건

- 1 ≤ N ≤ 100,000
- 1 ≤ K ≤ N - 1
- 1 ≤ u, v ≤ N
- 주어지는 그래프는 항상 트리이다

## 예제

### 예제 입력 1

```
5 2
1 2
2 3
3 4
4 5
```

### 예제 출력 1

```
7
```

### 예제 입력 2

```
4 1
1 2
1 3
1 4
```

### 예제 출력 2

```
3
```

### 예제 입력 3

```
1 1
```

### 예제 출력 3

```
0
```
