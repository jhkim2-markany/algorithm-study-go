# 거리가 K 이하인 노드 쌍 세기 - 해설

## 접근 방식

1. 센트로이드 분할을 수행한다
2. 각 센트로이드에서 서브트리 내 모든 노드까지의 거리를 수집한다
3. 센트로이드를 지나는 경로 중 거리가 K 이하인 쌍을 정렬 + 투 포인터로 센다
4. 같은 서브트리 내의 쌍은 포함-배제로 빼준다
5. 각 서브트리에 대해 재귀적으로 반복한다

## 핵심 아이디어

센트로이드 분할의 가장 기본적인 활용이다. 센트로이드 c를 지나는 경로의 길이는 "c에서 노드 u까지의 거리 + c에서 노드 v까지의 거리"이다.

```
센트로이드 c를 지나는 경로:
  dist(u, v) = dist(u, c) + dist(c, v)
  조건: dist(u, c) + dist(c, v) ≤ K
```

거리 배열을 정렬한 뒤 투 포인터로 합이 K 이하인 쌍을 O(N log N)에 셀 수 있다.

같은 서브트리 내의 두 노드는 센트로이드를 지나지 않으므로, 포함-배제 원리로 빼준다:
- 전체 거리 배열에서 K 이하인 쌍 수를 구한다 (포함)
- 각 서브트리의 거리 배열에서 K 이하인 쌍 수를 빼준다 (배제)

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N log² N) |
| 공간 복잡도 | O(N) |

각 재귀 레벨에서 정렬에 O(N log N)이 소요되고, 재귀 깊이가 O(log N)이므로 전체 O(N log² N)이다.
