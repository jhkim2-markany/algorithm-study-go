# 30. 게임 이론

두 플레이어가 번갈아 최적의 수를 두는 조합 게임에서 승패를 판별하는 알고리즘 유형이다. Nim 게임, Sprague-Grundy 정리, 승리/패배 포지션 분석 등이 핵심이다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
