# 님 게임

**난이도:** 하

## 문제 설명

두 플레이어가 N개의 돌 더미로 님(Nim) 게임을 한다. 각 플레이어는 자기 차례에 하나의 더미를 선택하여 1개 이상의 돌을 가져간다. 마지막 돌을 가져가는 플레이어가 승리한다. 두 플레이어 모두 최적의 전략으로 플레이할 때, 선수(먼저 시작하는 플레이어)가 승리하는지 판별하시오.

## 입력 형식

- 첫째 줄에 돌 더미의 수 N이 주어진다
- 둘째 줄에 각 더미의 돌 개수 a₁, a₂, ..., aₙ이 공백으로 구분되어 주어진다

## 출력 형식

- 선수가 승리하면 "First"를, 후수가 승리하면 "Second"를 출력한다

## 제약 조건

- 1 ≤ N ≤ 100
- 1 ≤ aᵢ ≤ 1,000,000
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
3
3 4 5
```

### 예제 출력 1

```
First
```

### 예제 입력 2

```
3
1 2 3
```

### 예제 출력 2

```
Second
```
