# 님 게임 - 해설

## 접근 방식

1. Nim 게임의 승패는 모든 돌 더미의 XOR 합으로 결정된다
2. 각 더미의 돌 개수를 입력받으면서 XOR 연산을 누적한다
3. 최종 XOR 합이 0이 아니면 선수 승리, 0이면 후수 승리를 출력한다

## 핵심 아이디어

Nim 게임의 핵심은 Bouton의 정리이다. 모든 돌 더미의 XOR 합이 0이 아닌 상태(N-position)에서는 항상 XOR 합을 0으로 만드는 수가 존재한다. 반대로 XOR 합이 0인 상태(P-position)에서는 어떤 수를 두어도 XOR 합이 0이 아닌 상태가 된다.

따라서 선수가 XOR 합이 0이 아닌 상태에서 시작하면, 매번 XOR 합을 0으로 만드는 최적의 수를 두어 상대를 항상 패배 포지션에 놓을 수 있다.

예제 1에서 3 ⊕ 4 ⊕ 5 = 2 ≠ 0이므로 선수 승리이다.
예제 2에서 1 ⊕ 2 ⊕ 3 = 0이므로 후수 승리이다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N) |
| 공간 복잡도 | O(1) |

N개의 돌 더미를 한 번 순회하며 XOR 연산을 수행하므로 O(N)이다.

## 대안적 접근

- 돌 더미가 1개인 경우 돌이 1개 이상이면 무조건 선수 승리이다. 더미가 2개인 경우 두 더미의 돌 수가 같으면 후수 승리, 다르면 선수 승리이다. 이러한 특수 케이스를 일반화한 것이 XOR 판별법이다.
