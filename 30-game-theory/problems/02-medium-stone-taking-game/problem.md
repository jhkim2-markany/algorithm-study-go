# 돌 가져가기 게임

**난이도:** 중

## 문제 설명

돌이 N개 있다. 두 플레이어가 번갈아 돌을 가져가며, 자기 차례에 가져갈 수 있는 돌의 개수는 주어진 집합 S에 포함된 수 중 하나이다. 마지막 돌을 가져가는 플레이어가 승리한다. 두 플레이어 모두 최적의 전략으로 플레이할 때, 선수가 승리하는지 판별하시오.

## 입력 형식

- 첫째 줄에 돌의 개수 N과 가져갈 수 있는 경우의 수 K가 공백으로 구분되어 주어진다
- 둘째 줄에 가져갈 수 있는 돌의 개수 s₁, s₂, ..., sₖ가 공백으로 구분되어 주어진다

## 출력 형식

- 선수가 승리하면 "First"를, 후수가 승리하면 "Second"를 출력한다

## 제약 조건

- 1 ≤ N ≤ 10,000
- 1 ≤ K ≤ 20
- 1 ≤ sᵢ ≤ N
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
10 3
1 3 4
```

### 예제 출력 1

```
First
```

### 예제 입력 2

```
8 2
1 3
```

### 예제 출력 2

```
Second
```
