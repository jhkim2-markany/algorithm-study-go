# 돌 가져가기 게임 - 해설

## 접근 방식

1. 돌 0개인 상태를 패배 포지션(L)으로 초기화한다
2. 돌 1개부터 N개까지 순서대로 각 상태의 승패를 판별한다
3. 현재 상태에서 가능한 이동 중 상대를 패배 포지션으로 보낼 수 있으면 승리 포지션이다
4. 모든 이동이 상대를 승리 포지션으로 보내면 패배 포지션이다
5. dp[N]의 값으로 최종 승패를 출력한다

## 핵심 아이디어

게임 이론의 기본 원리인 승리/패배 포지션 분석을 DP로 구현한다. 핵심 규칙은 다음과 같다:

- 패배 포지션(L): 현재 플레이어가 어떤 수를 두어도 상대가 승리 포지션에 놓인다
- 승리 포지션(W): 현재 플레이어가 상대를 패배 포지션으로 보내는 수가 존재한다

이 규칙을 바텀업 DP로 구현하면, 돌 0개(기저 상태)부터 시작하여 N개까지 모든 상태의 승패를 O(N × K)에 결정할 수 있다.

예제 2에서 가져갈 수 있는 수가 {1, 3}일 때, dp 배열은 [L, W, L, W, L, W, L, W, L]이 된다. 돌 8개인 상태는 패배 포지션이므로 후수가 승리한다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N × K) |
| 공간 복잡도 | O(N) |

N개의 상태 각각에 대해 최대 K개의 이동을 확인하므로 O(N × K)이다.

## 대안적 접근

- 가져갈 수 있는 수가 {1, 2, ..., K}처럼 연속적인 경우, 돌 개수를 (K+1)로 나눈 나머지가 0이면 후수 승리라는 패턴이 존재한다. 하지만 일반적인 경우에는 DP가 필요하다.
- Sprague-Grundy 정리를 적용하여 그런디 수를 계산하는 방법도 있다. 단일 게임에서는 그런디 수가 0이면 후수 승리, 0이 아니면 선수 승리이므로 결과는 동일하다.
