# 다중 파일 그런디 게임

**난이도:** 상

## 문제 설명

M개의 독립적인 돌 게임이 동시에 진행된다. 각 게임에는 돌이 nᵢ개 있으며, 한 번에 가져갈 수 있는 돌의 개수는 모든 게임에서 동일한 집합 S로 주어진다. 각 플레이어는 자기 차례에 하나의 게임을 선택하여 S에 포함된 수만큼 돌을 가져간다. 모든 게임에서 더 이상 돌을 가져갈 수 없는 플레이어가 패배한다. 두 플레이어 모두 최적의 전략으로 플레이할 때, 선수가 승리하는지 판별하시오.

## 입력 형식

- 첫째 줄에 게임의 수 M과 가져갈 수 있는 경우의 수 K가 공백으로 구분되어 주어진다
- 둘째 줄에 가져갈 수 있는 돌의 개수 s₁, s₂, ..., sₖ가 공백으로 구분되어 주어진다
- 셋째 줄에 각 게임의 돌 개수 n₁, n₂, ..., nₘ이 공백으로 구분되어 주어진다

## 출력 형식

- 선수가 승리하면 "First"를, 후수가 승리하면 "Second"를 출력한다

## 제약 조건

- 1 ≤ M ≤ 100
- 1 ≤ K ≤ 20
- 1 ≤ sᵢ ≤ 100
- 0 ≤ nᵢ ≤ 10,000
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
3 3
1 3 4
5 3 7
```

### 예제 출력 1

```
First
```

### 예제 입력 2

```
2 2
1 2
4 4
```

### 예제 출력 2

```
Second
```
