# 두 노드의 최소 공통 조상

**난이도:** 하

## 문제 설명

N개의 노드로 이루어진 루트가 1인 트리가 주어진다. M개의 쿼리가 주어지며, 각 쿼리는 두 노드 u, v에 대해 최소 공통 조상(LCA)을 구하는 것이다.

## 입력 형식

- 첫째 줄에 노드의 수 N이 주어진다
- 다음 N-1개의 줄에 트리의 간선 정보가 두 정수 a, b로 주어진다 (a와 b가 연결됨)
- 다음 줄에 쿼리의 수 M이 주어진다
- 다음 M개의 줄에 각 쿼리의 두 노드 u, v가 주어진다

## 출력 형식

- 각 쿼리에 대해 두 노드의 최소 공통 조상을 한 줄에 하나씩 출력한다

## 제약 조건

- 2 ≤ N ≤ 100,000
- 1 ≤ M ≤ 100,000
- 1 ≤ u, v ≤ N
- 트리의 루트는 1번 노드이다
- 시간 제한: 2초
- 메모리 제한: 256MB
