# 트리에서 두 노드 사이의 거리 - 해설

## 접근 방식

1. 가중치 트리를 인접 리스트로 구성한다
2. 루트(1번 노드)에서 DFS를 수행하여 각 노드의 깊이, 루트까지의 거리, Binary Lifting 희소 배열을 구성한다
3. 두 노드 u, v 사이의 거리는 dist[u] + dist[v] - 2 * dist[LCA(u, v)]로 계산한다

## 핵심 아이디어

트리에서 두 노드 사이의 경로는 유일하며, 반드시 LCA를 경유한다. 따라서 u에서 v까지의 거리는 다음과 같이 분해할 수 있다:

- dist(u, v) = dist(root, u) + dist(root, v) - 2 × dist(root, LCA(u, v))

루트에서 각 노드까지의 거리를 DFS로 한 번만 계산해두면, LCA를 구한 뒤 O(1)에 두 노드 사이의 거리를 구할 수 있다. 이 기법은 가중치 트리의 경로 문제에서 매우 자주 사용되는 패턴이다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N log N + M log N) |
| 공간 복잡도 | O(N log N) |

전처리에 O(N log N), 각 쿼리에 O(log N)이 소요된다. 거리 계산 자체는 O(1)이지만 LCA를 구하는 데 O(log N)이 필요하다.

## 대안적 접근

- 오프라인 처리: 쿼리를 미리 모아서 DFS 한 번으로 처리하는 Tarjan의 오프라인 LCA 알고리즘을 사용할 수 있다. 유니온 파인드를 활용하며 전체 O(N α(N) + M) 시간에 동작한다.
