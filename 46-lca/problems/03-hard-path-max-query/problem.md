# 트리 경로 위의 최댓값 쿼리

**난이도:** 상

## 문제 설명

N개의 노드로 이루어진 가중치 트리가 주어진다. 각 간선에는 양의 정수 가중치가 있다. M개의 쿼리가 주어지며, 각 쿼리는 두 노드 u, v 사이의 경로에 있는 간선 가중치 중 최댓값을 구하는 것이다.

## 입력 형식

- 첫째 줄에 노드의 수 N이 주어진다
- 다음 N-1개의 줄에 세 정수 a, b, w가 주어진다 (a와 b가 가중치 w인 간선으로 연결됨)
- 다음 줄에 쿼리의 수 M이 주어진다
- 다음 M개의 줄에 각 쿼리의 두 노드 u, v가 주어진다

## 출력 형식

- 각 쿼리에 대해 경로 위 간선 가중치의 최댓값을 한 줄에 하나씩 출력한다

## 제약 조건

- 2 ≤ N ≤ 100,000
- 1 ≤ M ≤ 100,000
- 1 ≤ a, b ≤ N
- 1 ≤ w ≤ 1,000,000
- 트리의 루트는 1번 노드이다
- u ≠ v
- 시간 제한: 2초
- 메모리 제한: 256MB
