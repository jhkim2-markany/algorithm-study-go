# 46. LCA (최소 공통 조상)

트리에서 두 노드의 최소 공통 조상(Lowest Common Ancestor)을 효율적으로 구하는 알고리즘이다. 나이브 방법, 희소 배열(Binary Lifting), 오일러 투어 + RMQ 등이 핵심이다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이

## 문제 목록

| 번호 | 문제 | 난이도 |
| --- | --- | --- |
| 01 | 두 노드의 최소 공통 조상 | 하 |
| 02 | 트리에서 두 노드 사이의 거리 | 중 |
| 03 | 트리 경로 위의 최댓값 쿼리 | 상 |
