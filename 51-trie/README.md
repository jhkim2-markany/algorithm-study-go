# 51. 트라이

문자열 집합을 효율적으로 저장하고 검색하기 위한 트리 기반 자료구조이다. 접두사(prefix) 기반 검색, 자동완성, 사전 구현 등에 핵심적으로 사용된다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
