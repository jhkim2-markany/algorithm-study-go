# XOR 최댓값

**난이도:** 상

## 문제 설명

N개의 음이 아닌 정수로 이루어진 배열 A가 주어진다. 배열에서 두 수를 골라 XOR 연산한 결과의 최댓값을 구하라.

XOR(배타적 논리합)은 두 수의 이진 표현에서 같은 자리의 비트가 다르면 1, 같으면 0이 되는 연산이다. 예를 들어 5 XOR 3 = 101₂ XOR 011₂ = 110₂ = 6이다.

## 입력 형식

- 첫째 줄에 배열의 크기 N이 주어진다 (2 ≤ N ≤ 100,000)
- 둘째 줄에 N개의 음이 아닌 정수가 공백으로 구분되어 주어진다 (0 ≤ Aᵢ ≤ 10⁹)

## 출력 형식

- 배열에서 두 수를 골라 XOR한 결과의 최댓값을 출력한다

## 제약 조건

- 2 ≤ N ≤ 100,000
- 0 ≤ Aᵢ ≤ 10⁹
- 같은 인덱스의 수를 두 번 고를 수 없다 (i ≠ j)
- 시간 제한: 1초

## 예제

### 예제 입력 1

```
5
3 10 5 25 2
```

### 예제 출력 1

```
28
```

### 예제 입력 2

```
4
1 2 3 4
```

### 예제 출력 2

```
7
```
