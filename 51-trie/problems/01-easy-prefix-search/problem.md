# 접두사 검색

**난이도:** 하

## 문제 설명

N개의 단어가 주어진다. 이후 M개의 질의가 주어지며, 각 질의는 하나의 문자열이다. 각 질의 문자열이 N개의 단어 중 하나의 접두사인지 판별하라.

접두사란 어떤 단어의 앞부분에 해당하는 부분 문자열을 말한다. 예를 들어 "app"은 "apple"의 접두사이고, "apple"은 "apple" 자기 자신의 접두사이기도 하다.

## 입력 형식

- 첫째 줄에 단어의 수 N과 질의의 수 M이 주어진다 (1 ≤ N, M ≤ 10,000)
- 다음 N줄에 각 단어가 주어진다 (영소문자, 길이 1 이상 100 이하)
- 다음 M줄에 각 질의 문자열이 주어진다 (영소문자, 길이 1 이상 100 이하)

## 출력 형식

- 각 질의에 대해 해당 문자열이 N개의 단어 중 하나 이상의 접두사이면 "YES", 아니면 "NO"를 출력한다

## 제약 조건

- 1 ≤ N, M ≤ 10,000
- 각 단어와 질의 문자열의 길이는 1 이상 100 이하
- 모든 문자열은 영소문자로만 구성된다
- 시간 제한: 1초
