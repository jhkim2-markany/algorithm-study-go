# 자동완성 시스템

**난이도:** 중

## 문제 설명

사전에 N개의 단어가 등록되어 있다. 사용자가 접두사를 입력하면, 해당 접두사로 시작하는 단어 중 사전 순으로 가장 앞에 오는 단어를 최대 K개까지 추천하는 자동완성 시스템을 구현하라.

M개의 질의가 주어지며, 각 질의는 사용자가 입력한 접두사이다. 각 질의에 대해 추천 단어를 사전 순으로 출력하라. 해당 접두사로 시작하는 단어가 없으면 -1을 출력한다.

## 입력 형식

- 첫째 줄에 단어의 수 N, 질의의 수 M, 최대 추천 개수 K가 주어진다 (1 ≤ N ≤ 10,000, 1 ≤ M ≤ 5,000, 1 ≤ K ≤ 5)
- 다음 N줄에 각 단어가 주어진다 (영소문자, 길이 1 이상 50 이하, 중복 없음)
- 다음 M줄에 각 질의 접두사가 주어진다 (영소문자, 길이 1 이상 50 이하)

## 출력 형식

- 각 질의에 대해 한 줄에 추천 단어를 공백으로 구분하여 출력한다
- 추천 단어는 사전 순으로 최대 K개까지 출력한다
- 해당 접두사로 시작하는 단어가 없으면 -1을 출력한다

## 제약 조건

- 1 ≤ N ≤ 10,000
- 1 ≤ M ≤ 5,000
- 1 ≤ K ≤ 5
- 각 단어의 길이는 1 이상 50 이하
- 모든 문자열은 영소문자로만 구성된다
- 단어에 중복은 없다
- 시간 제한: 2초
