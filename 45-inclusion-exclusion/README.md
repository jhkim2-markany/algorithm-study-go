# 45. 포함 배제의 원리

여러 집합의 합집합 크기를 구할 때, 각 집합의 크기를 더하고 교집합을 빼는 과정을 반복하여 정확한 크기를 계산하는 원리이다. 비트마스크를 이용한 구현, 오일러 피 함수와의 관계 등이 핵심이다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
