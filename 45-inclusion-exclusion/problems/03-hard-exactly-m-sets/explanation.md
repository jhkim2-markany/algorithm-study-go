# K개 집합의 정확히 M개에 속하는 원소 수 - 해설

## 접근 방식

1. 각 소수 pᵢ에 대해 Aᵢ = {1~N 중 pᵢ의 배수} 집합을 정의한다
2. S[j] = 크기가 j인 모든 부분집합에 대해 교집합 크기의 합을 구한다
3. 비트마스크로 모든 부분집합을 열거하여 S[j]를 계산한다
4. 뫼비우스 반전 공식을 적용하여 정확히 M개에 속하는 수의 개수를 구한다
5. exactly[M] = Σ(j=M to K) (-1)^(j-M) × C(j, M) × S[j]

## 핵심 아이디어

이 문제의 핵심은 "최소 j개에 속하는 수"에서 "정확히 M개에 속하는 수"로 변환하는 것이다. S[j]는 크기 j인 모든 부분집합의 교집합 크기 합이므로, 정확히 t개의 집합에 속하는 원소는 S[j]에서 C(t, j)번 세어진다 (t개 중 j개를 선택하는 경우의 수). 이 관계를 역변환하면:

```
exactly[M] = Σ(j=M to K) (-1)^(j-M) × C(j, M) × S[j]
```

이 공식은 이항 반전(Binomial Inversion)이라 불리며, 포함 배제의 원리를 일반화한 것이다. M=0일 때는 어떤 집합에도 속하지 않는 수의 개수가 되어 기본 포함 배제와 동일하다.

S[0] = N으로 설정하는 것이 중요하다. 공집합의 교집합은 전체 집합이므로 크기가 N이다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(2^K + K^2) |
| 공간 복잡도 | O(K^2) |

비트마스크 열거에 O(2^K), 이항 계수 테이블 구성에 O(K^2)이 소요된다. K ≤ 15이므로 2^15 = 32,768번의 연산으로 충분히 빠르다.

## 대안적 접근

- 각 수를 직접 순회하며 몇 개의 집합에 속하는지 세는 방법도 있지만, N이 최대 10^9이므로 시간 초과가 발생한다.
- 뫼비우스 함수를 이용한 접근도 가능하지만, 비트마스크 + 이항 반전이 구현이 더 직관적이다.
