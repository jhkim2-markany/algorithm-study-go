# K개 집합의 정확히 M개에 속하는 원소 수

**난이도:** 상

## 문제 설명

1부터 N까지의 자연수가 있다. K개의 소수 p₁, p₂, ..., pₖ가 주어질 때, 각 소수 pᵢ에 대해 "pᵢ의 배수인 수의 집합" Aᵢ를 정의한다. 1 이상 N 이하의 자연수 중에서 정확히 M개의 집합에 속하는 수의 개수를 구하시오.

예를 들어 N=30, 소수가 {2, 3, 5}이고 M=2이면, 정확히 2개의 소수의 배수이면서 나머지 1개의 소수의 배수가 아닌 수의 개수를 구한다.

## 입력 형식

- 첫째 줄에 자연수 N, 소수의 개수 K, 정확히 속해야 하는 집합의 수 M이 공백으로 구분되어 주어진다
- 둘째 줄에 K개의 서로 다른 소수가 공백으로 구분되어 주어진다

## 출력 형식

- 1 이상 N 이하의 자연수 중 정확히 M개의 집합에 속하는 수의 개수를 출력한다

## 제약 조건

- 1 ≤ N ≤ 10^9
- 1 ≤ K ≤ 15
- 0 ≤ M ≤ K
- 주어지는 수는 모두 소수이며 서로 다르다
- 각 소수는 2 이상 100 이하이다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
30 3 0
2 3 5
```

### 예제 출력 1

```
8
```

### 예제 입력 2

```
30 3 1
2 3 5
```

### 예제 출력 2

```
14
```

### 예제 입력 3

```
30 3 2
2 3 5
```

### 예제 출력 3

```
7
```
