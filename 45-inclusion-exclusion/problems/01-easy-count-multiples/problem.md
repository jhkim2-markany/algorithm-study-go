# 배수의 개수 세기

**난이도:** 하

## 문제 설명

자연수 N과 서로 다른 소수 K개가 주어진다. 1 이상 N 이하의 자연수 중에서 주어진 소수들 중 하나 이상의 배수인 수의 개수를 구하시오.

## 입력 형식

- 첫째 줄에 자연수 N과 소수의 개수 K가 공백으로 구분되어 주어진다
- 둘째 줄에 K개의 서로 다른 소수가 공백으로 구분되어 주어진다

## 출력 형식

- 1 이상 N 이하의 자연수 중 주어진 소수들 중 하나 이상의 배수인 수의 개수를 출력한다

## 제약 조건

- 1 ≤ N ≤ 10^9
- 1 ≤ K ≤ 15
- 주어지는 수는 모두 소수이며 서로 다르다
- 각 소수는 2 이상 100 이하이다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
30 3
2 3 5
```

### 예제 출력 1

```
22
```

### 예제 입력 2

```
10 2
2 3
```

### 예제 출력 2

```
7
```
