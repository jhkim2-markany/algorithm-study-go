# 배수의 개수 세기 - 해설

## 접근 방식

1. 각 소수 pᵢ에 대해 Aᵢ = {1~N 중 pᵢ의 배수} 집합을 정의한다
2. 구하려는 값은 |A₁ ∪ A₂ ∪ ... ∪ Aₖ|이다
3. 비트마스크로 1부터 2^K - 1까지 모든 비어있지 않은 부분집합을 열거한다
4. 각 부분집합에 대해 선택된 소수들의 곱 product를 구한다
5. N/product가 해당 교집합의 크기이다 (소수들이 서로소이므로 곱이 LCM)
6. 선택된 소수의 개수가 홀수이면 더하고, 짝수이면 뺀다

## 핵심 아이디어

포함 배제의 원리에서 핵심은 중복 계산을 보정하는 것이다. 단순히 각 소수의 배수 개수를 더하면 여러 소수의 공배수가 중복 계산된다. 예를 들어 6은 2의 배수이면서 3의 배수이므로 두 번 세어진다. 포함 배제를 적용하면 이런 중복을 정확히 보정할 수 있다.

주어진 수가 모두 소수이므로 임의의 부분집합의 LCM은 단순히 곱과 같다. 이 성질 덕분에 교집합 크기 계산이 간단해진다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(2^K) |
| 공간 복잡도 | O(K) |

K ≤ 15이므로 2^15 = 32,768번의 연산으로 충분히 빠르게 동작한다.

## 대안적 접근

- K가 매우 작은 경우(2~3개) 직접 공식을 전개하여 계산할 수도 있다. 예를 들어 K=2일 때 |A∪B| = N/a + N/b - N/(a*b)로 바로 구할 수 있다. 하지만 비트마스크 방식이 K에 관계없이 일반적으로 적용 가능하다.
