# 서로소인 수의 개수

**난이도:** 중

## 문제 설명

자연수 N이 주어질 때, 1 이상 N 이하의 자연수 중에서 N과 서로소인 수의 개수를 구하시오. 두 수가 서로소라 함은 최대공약수가 1인 것을 의미한다.

## 입력 형식

- 첫째 줄에 테스트 케이스의 수 T가 주어진다
- 다음 T개의 줄에 각각 자연수 N이 주어진다

## 출력 형식

- 각 테스트 케이스에 대해 1 이상 N 이하의 자연수 중 N과 서로소인 수의 개수를 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ T ≤ 100
- 1 ≤ N ≤ 10^12
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
4
1
6
12
97
```

### 예제 출력 1

```
1
2
4
96
```

### 예제 입력 2

```
3
30
100
360
```

### 예제 출력 2

```
8
40
96
```
