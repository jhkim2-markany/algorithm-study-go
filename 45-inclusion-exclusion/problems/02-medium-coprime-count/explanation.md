# 서로소인 수의 개수 - 해설

## 접근 방식

1. N과 서로소인 수의 개수는 오일러 피 함수 φ(N)과 같다
2. N을 소인수분해한다: N = p₁^a₁ × p₂^a₂ × ... × pₖ^aₖ
3. 포함 배제의 원리를 적용하면 φ(N) = N × (1 - 1/p₁) × (1 - 1/p₂) × ... × (1 - 1/pₖ)
4. 소인수를 하나씩 찾으면서 result = result - result/p 연산을 반복한다
5. √N까지 시행 나눗셈으로 소인수를 찾고, 남은 수가 1보다 크면 마지막 소인수로 처리한다

## 핵심 아이디어

오일러 피 함수는 포함 배제의 원리를 직접 적용한 결과이다. N 이하에서 소인수 pᵢ의 배수 집합을 Aᵢ라 하면, N과 서로소인 수의 개수는 N - |A₁ ∪ A₂ ∪ ... ∪ Aₖ|이다. 포함 배제를 전개하면 곱 형태의 공식 φ(N) = N × Π(1 - 1/pᵢ)가 유도된다.

구현에서는 정수 나눗셈을 사용하여 부동소수점 오차를 피한다. result -= result / p 연산은 result × (1 - 1/p)와 동일하며, 정수 범위에서 정확하게 계산된다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(T × √N) |
| 공간 복잡도 | O(1) |

각 테스트 케이스마다 √N까지 시행 나눗셈을 수행한다. N ≤ 10^12이므로 √N ≤ 10^6이며, T ≤ 100이므로 총 연산 횟수는 충분히 빠르다.

## 대안적 접근

- N의 소인수를 먼저 모두 구한 뒤 비트마스크 포함 배제를 적용할 수도 있다. 하지만 소인수의 개수가 최대 약 15개이므로 비트마스크 방식은 불필요하게 복잡하다. 곱 형태의 공식이 더 간결하고 효율적이다.
- 에라토스테네스의 체를 이용하여 √N 이하의 소수를 미리 구해두면 소인수분해를 더 빠르게 할 수 있다.
