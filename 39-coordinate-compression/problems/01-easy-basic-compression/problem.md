# 기본 좌표 압축

**난이도:** 하

## 문제 설명

N개의 정수로 이루어진 수열이 주어진다. 각 수에 대해 해당 수보다 작은 서로 다른 수의 개수를 구하시오.

즉, 수열의 각 원소를 좌표 압축하여 0부터 시작하는 순위(rank)로 변환하시오.

## 입력 형식

- 첫째 줄에 수열의 크기 N이 주어진다.
- 둘째 줄에 N개의 정수가 공백으로 구분되어 주어진다.

## 출력 형식

- 첫째 줄에 N개의 정수를 공백으로 구분하여 출력한다.
- 각 정수는 해당 위치의 원래 값보다 작은 서로 다른 수의 개수이다.

## 제약 조건

- 1 ≤ N ≤ 1,000,000
- -10⁹ ≤ 각 정수 ≤ 10⁹
- 시간 제한: 2초

## 예제

### 예제 입력 1

```
5
2 4 -10 4 -9
```

### 예제 출력 1

```
2 3 0 3 1
```

### 예제 입력 2

```
6
1000 999 1000 999 1000 999
```

### 예제 출력 2

```
1 0 1 0 1 0
```
