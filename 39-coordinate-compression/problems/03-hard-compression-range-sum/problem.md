# 좌표 압축 + 구간 합

**난이도:** 상

## 문제 설명

수직선 위에 N개의 점이 있다. 각 점은 좌표 x와 가중치 w를 가진다. 이후 Q개의 쿼리가 주어지며, 각 쿼리는 구간 [L, R]에 속하는 점들의 가중치 합을 구하는 것이다.

좌표 범위가 매우 크므로 좌표 압축을 적용한 뒤 누적합(Prefix Sum)을 이용하여 효율적으로 답을 구하시오.

## 입력 형식

- 첫째 줄에 점의 수 N과 쿼리의 수 Q가 주어진다.
- 다음 N개의 줄에 각 점의 좌표 x와 가중치 w가 주어진다.
- 다음 Q개의 줄에 각 쿼리의 구간 L, R이 주어진다.

## 출력 형식

- Q개의 줄에 걸쳐 각 쿼리의 답을 출력한다.

## 제약 조건

- 1 ≤ N ≤ 200,000
- 1 ≤ Q ≤ 200,000
- -10⁹ ≤ x ≤ 10⁹
- 1 ≤ w ≤ 10⁹
- -10⁹ ≤ L ≤ R ≤ 10⁹
- 같은 좌표에 여러 점이 있을 수 있으며, 이 경우 가중치를 합산한다
- 시간 제한: 2초
