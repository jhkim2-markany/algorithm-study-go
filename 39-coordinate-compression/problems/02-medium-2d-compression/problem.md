# 2D 좌표 압축 활용

**난이도:** 중

## 문제 설명

2차원 평면 위에 N개의 직사각형이 주어진다. 각 직사각형은 왼쪽 아래 꼭짓점 (x1, y1)과 오른쪽 위 꼭짓점 (x2, y2)로 정의된다.

직사각형들이 덮는 서로 다른 영역의 총 넓이를 구하시오. 겹치는 부분은 한 번만 계산한다.

## 입력 형식

- 첫째 줄에 직사각형의 수 N이 주어진다.
- 다음 N개의 줄에 각 직사각형의 x1, y1, x2, y2가 주어진다.

## 출력 형식

- 직사각형들이 덮는 총 넓이를 출력한다.

## 제약 조건

- 1 ≤ N ≤ 1,000
- -10⁹ ≤ x1 < x2 ≤ 10⁹
- -10⁹ ≤ y1 < y2 ≤ 10⁹
- 시간 제한: 2초

## 예제

### 예제 입력 1

```
2
1 1 4 4
2 2 5 5
```

### 예제 출력 1

```
14
```

### 예제 입력 2

```
3
0 0 2 2
1 1 3 3
2 0 4 2
```

### 예제 출력 2

```
10
```
