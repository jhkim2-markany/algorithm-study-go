# 39. 좌표 압축

큰 범위의 좌표 값을 작은 범위의 연속된 인덱스로 변환하는 기법이다. 정렬과 이진 탐색을 기반으로 중복을 제거한 뒤 인덱스를 매핑하는 것이 핵심이다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
