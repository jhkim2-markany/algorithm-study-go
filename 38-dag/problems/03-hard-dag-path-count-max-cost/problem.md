# DAG 경로 수와 최대 비용

**난이도:** 상

## 문제 설명

N개의 정점과 M개의 방향 간선으로 이루어진 DAG가 주어진다. 각 간선에는 가중치가 있다. 시작 정점 S에서 도착 정점 T까지에 대해 다음 세 가지를 구하시오:

1. S에서 T까지의 서로 다른 경로의 수 (1,000,000,007로 나눈 나머지)
2. S에서 T까지의 최장 경로 길이 (가중치 합의 최댓값)
3. 최장 경로의 길이를 달성하는 서로 다른 경로의 수 (1,000,000,007로 나눈 나머지)

S에서 T로 도달할 수 없는 경우 세 값 모두 0을 출력한다.

## 입력 형식

- 첫째 줄에 정점의 수 N, 간선의 수 M, 시작 정점 S, 도착 정점 T가 주어진다.
- 다음 M개의 줄에 각 간선의 시작 정점 u, 도착 정점 v, 가중치 w가 주어진다.

## 출력 형식

- 첫째 줄에 S에서 T까지의 경로 수를 1,000,000,007로 나눈 나머지를 출력한다.
- 둘째 줄에 S에서 T까지의 최장 경로 길이를 출력한다.
- 셋째 줄에 최장 경로를 달성하는 경로 수를 1,000,000,007로 나눈 나머지를 출력한다.

## 제약 조건

- 2 ≤ N ≤ 100,000
- 1 ≤ M ≤ 300,000
- 0 ≤ S, T ≤ N - 1
- S ≠ T
- 0 ≤ u, v ≤ N - 1
- 1 ≤ w ≤ 1,000
- 그래프는 DAG임이 보장된다

## 예제

### 예제 입력 1

```
5 6 0 4
0 1 3
0 2 5
1 3 4
2 3 2
1 4 10
3 4 6
```

### 예제 출력 1

```
3
13
1
```

### 예제 입력 2

```
4 4 0 3
0 1 5
0 2 5
1 3 3
2 3 3
```

### 예제 출력 2

```
2
8
2
```
