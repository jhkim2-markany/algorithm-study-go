# DAG 최장 경로 - 해설

## 접근 방식

1. 그래프를 인접 리스트로 구성하면서 진입 차수를 계산한다
2. Kahn's Algorithm으로 위상 정렬을 수행한다
3. 시작 정점의 거리를 0, 나머지를 음의 무한대로 초기화한다
4. 위상 정렬 순서대로 각 정점에서 나가는 간선을 확인하며 최장 거리를 갱신한다
5. 도착 정점의 거리가 음의 무한대이면 도달 불가능하므로 "IMPOSSIBLE"을 출력한다

## 핵심 아이디어

일반 그래프에서 최장 경로 문제는 NP-Hard이지만, DAG에서는 위상 정렬을 활용하여 O(V + E)에 해결할 수 있다. 최단 경로와 동일한 구조에서 min을 max로 바꾸기만 하면 된다.

위상 정렬 순서대로 처리하면 현재 정점에 도달하는 모든 경로가 이미 고려된 상태이므로, 최장 거리가 정확하게 계산된다. 이는 DAG의 사이클 없음 성질 덕분에 가능하다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(V + E) |
| 공간 복잡도 | O(V + E) |

위상 정렬과 간선 완화 모두 O(V + E)이므로 전체 시간 복잡도는 O(V + E)이다.

## 대안적 접근

DFS + 메모이제이션으로도 풀 수 있다. 각 정점에서 도착 정점까지의 최장 거리를 재귀적으로 계산하고 결과를 캐싱하면 동일한 O(V + E) 시간 복잡도를 달성한다. 다만 재귀 깊이가 깊어질 수 있으므로 위상 정렬 기반 방법이 더 안정적이다.
