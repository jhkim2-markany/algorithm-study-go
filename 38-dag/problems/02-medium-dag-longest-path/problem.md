# DAG 최장 경로

**난이도:** 중

## 문제 설명

N개의 정점과 M개의 방향 간선으로 이루어진 DAG가 주어진다. 각 간선에는 가중치가 있다. 이 DAG에서 시작 정점 S부터 도착 정점 T까지의 최장 경로(가중치 합이 최대인 경로)의 길이를 구하시오.

S에서 T로 도달할 수 없는 경우 "IMPOSSIBLE"을 출력한다.

## 입력 형식

- 첫째 줄에 정점의 수 N, 간선의 수 M, 시작 정점 S, 도착 정점 T가 주어진다.
- 다음 M개의 줄에 각 간선의 시작 정점 u, 도착 정점 v, 가중치 w가 주어진다.

## 출력 형식

- S에서 T까지의 최장 경로 길이를 출력한다.
- 도달할 수 없는 경우 "IMPOSSIBLE"을 출력한다.

## 제약 조건

- 2 ≤ N ≤ 10,000
- 1 ≤ M ≤ 100,000
- 0 ≤ S, T ≤ N - 1
- S ≠ T
- 0 ≤ u, v ≤ N - 1
- 1 ≤ w ≤ 1,000
- 그래프는 DAG임이 보장된다

## 예제

### 예제 입력 1

```
6 7 0 5
0 1 3
0 2 6
1 3 4
1 4 11
2 3 8
3 5 2
4 5 7
```

### 예제 출력 1

```
21
```

### 예제 입력 2

```
4 2 0 3
0 1 5
2 3 3
```

### 예제 출력 2

```
IMPOSSIBLE
```
