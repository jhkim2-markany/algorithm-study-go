# DAG 최단 경로 - 해설

## 접근 방식

1. 그래프를 인접 리스트로 구성하면서 각 정점의 진입 차수를 계산한다
2. Kahn's Algorithm으로 위상 정렬을 수행한다
3. 시작 정점의 거리를 0, 나머지를 무한대로 초기화한다
4. 위상 정렬 순서대로 각 정점에서 나가는 간선을 확인하며 최단 거리를 갱신한다
5. 도달 불가능한 정점은 "INF"로 출력한다

## 핵심 아이디어

DAG에서는 위상 정렬 순서가 곧 DP의 처리 순서가 된다. 위상 정렬 순서대로 정점을 처리하면, 현재 정점을 처리할 때 이전의 모든 선행 정점이 이미 처리된 상태이므로 최단 거리가 확정되어 있다.

이 방법은 Dijkstra와 달리 음수 가중치가 있어도 정확하게 동작하며, 우선순위 큐가 필요 없어 구현이 간단하다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(V + E) |
| 공간 복잡도 | O(V + E) |

위상 정렬에 O(V + E), 간선 완화에 O(V + E)가 소요되어 전체 O(V + E)이다.

## 대안적 접근

Dijkstra 알고리즘으로도 풀 수 있지만, 음수 가중치를 처리할 수 없고 시간 복잡도가 O((V + E) log V)로 더 느리다. Bellman-Ford는 음수 가중치를 처리할 수 있지만 O(VE)로 훨씬 느리다. DAG라는 조건이 주어지면 위상 정렬 기반 방법이 가장 효율적이다.
