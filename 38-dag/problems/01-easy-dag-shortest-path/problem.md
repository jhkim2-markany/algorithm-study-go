# DAG 최단 경로

**난이도:** 하

## 문제 설명

N개의 정점과 M개의 방향 간선으로 이루어진 DAG(방향 비순환 그래프)가 주어진다. 각 간선에는 가중치가 있다. 시작 정점 S에서 다른 모든 정점까지의 최단 거리를 구하시오.

도달할 수 없는 정점에 대해서는 "INF"를 출력한다.

## 입력 형식

- 첫째 줄에 정점의 수 N과 간선의 수 M, 시작 정점 S가 주어진다.
- 다음 M개의 줄에 각 간선의 시작 정점 u, 도착 정점 v, 가중치 w가 주어진다.

## 출력 형식

- N개의 줄에 걸쳐 시작 정점 S에서 각 정점(0번부터 N-1번)까지의 최단 거리를 출력한다.
- 도달할 수 없는 경우 "INF"를 출력한다.

## 제약 조건

- 2 ≤ N ≤ 10,000
- 1 ≤ M ≤ 100,000
- 0 ≤ S ≤ N - 1
- 0 ≤ u, v ≤ N - 1
- -1,000 ≤ w ≤ 1,000
- 그래프는 DAG임이 보장된다

## 예제

### 예제 입력 1

```
5 7 0
0 1 5
0 2 3
1 3 6
1 2 2
2 4 4
2 3 7
3 4 1
```

### 예제 출력 1

```
0
5
3
10
7
```
