# 덱 기본 연산

**난이도:** 하

## 문제 설명

정수를 저장하는 덱(Deque)을 구현하고, 주어진 명령을 처리하는 프로그램을 작성하라.

명령은 다음과 같다:

- `push_front X`: 정수 X를 덱의 앞에 넣는다
- `push_back X`: 정수 X를 덱의 뒤에 넣는다
- `pop_front`: 덱의 앞에서 정수를 빼고 출력한다. 덱이 비어 있으면 -1을 출력한다
- `pop_back`: 덱의 뒤에서 정수를 빼고 출력한다. 덱이 비어 있으면 -1을 출력한다
- `size`: 덱에 들어있는 정수의 개수를 출력한다
- `empty`: 덱이 비어 있으면 1, 아니면 0을 출력한다
- `front`: 덱의 앞에 있는 정수를 출력한다. 덱이 비어 있으면 -1을 출력한다
- `back`: 덱의 뒤에 있는 정수를 출력한다. 덱이 비어 있으면 -1을 출력한다

## 입력 형식

- 첫째 줄에 명령의 수 N이 주어진다 (1 ≤ N ≤ 10,000)
- 둘째 줄부터 N개의 줄에 명령이 하나씩 주어진다

## 출력 형식

- 출력이 필요한 명령이 주어질 때마다 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ N ≤ 10,000
- push_front, push_back에서 X는 1 이상 100,000 이하의 정수이다
- 시간 제한: 1초
