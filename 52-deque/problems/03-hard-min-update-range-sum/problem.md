# 최솟값 갱신 구간 합

**난이도:** 상

## 문제 설명

N개의 정수로 이루어진 배열 A가 주어진다. 배열 B를 다음과 같이 정의한다:

B[i] = 배열 A에서 인덱스 i를 포함하는 모든 크기 K 윈도우의 최솟값들 중 최댓값

즉, i번째 원소가 포함되는 모든 연속 부분 배열(길이 K) 중에서 각 부분 배열의 최솟값을 구하고, 그 최솟값들 중 가장 큰 값이 B[i]이다.

배열 B의 모든 원소의 합을 구하라.

단, 인덱스는 0부터 시작하며, i번째 원소가 포함되는 크기 K 윈도우는 max(0, i-K+1)번째부터 min(N-K, i)번째 윈도우까지이다.

## 입력 형식

- 첫째 줄에 배열의 크기 N과 윈도우 크기 K가 주어진다 (1 ≤ K ≤ N ≤ 500,000)
- 둘째 줄에 N개의 정수가 공백으로 구분되어 주어진다 (1 ≤ 각 정수 ≤ 1,000,000)

## 출력 형식

- 배열 B의 모든 원소의 합을 출력한다

## 제약 조건

- 1 ≤ K ≤ N ≤ 500,000
- 1 ≤ A[i] ≤ 1,000,000
- 시간 제한: 2초

## 예제

### 예제 입력 1

```
5 3
2 1 4 3 5
```

### 예제 출력 1

```
13
```

### 예제 입력 2

```
4 2
3 1 2 4
```

### 예제 출력 2

```
10
```

### 예제 입력 3

```
3 1
5 3 7
```

### 예제 출력 3

```
15
```
