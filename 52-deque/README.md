# 52. 덱 (Deque)

양쪽 끝에서 삽입과 삭제가 모두 가능한 자료구조이다. 스택과 큐의 기능을 결합한 형태로, 슬라이딩 윈도우 최솟값/최댓값, 회문 검사, 작업 스케줄링 등 다양한 문제에 활용된다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이

## 문제 목록

| 번호 | 문제 | 난이도 |
| --- | --- | --- |
| 01 | 덱 기본 연산 | 하 |
| 02 | 슬라이딩 윈도우 최댓값 | 중 |
| 03 | 최솟값 갱신 구간 합 | 상 |
