# N 이하의 소수 구하기 - 해설

## 접근 방식

1. 크기 N+1의 불리언 배열을 생성하고 2부터 N까지 true로 초기화한다
2. 2부터 √N까지 순회하며, 현재 수가 소수(true)이면 그 배수를 모두 false로 표시한다
3. 배수 제거는 i*i부터 시작한다 (i*i 미만의 배수는 더 작은 소수에 의해 이미 제거됨)
4. 순회가 끝나면 배열에서 true인 인덱스를 순서대로 출력한다

## 핵심 아이디어

에라토스테네스의 체의 핵심은 "소수의 배수는 합성수"라는 성질이다. 작은 소수부터 시작하여 배수를 제거하면, 남는 수는 모두 소수가 된다. 외부 루프를 √N까지만 돌리는 이유는, N 이하의 합성수는 반드시 √N 이하의 소인수를 가지기 때문이다. 또한 배수 제거를 i*i부터 시작하면 중복 제거를 줄여 실행 시간을 단축할 수 있다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N log log N) |
| 공간 복잡도 | O(N) |

에라토스테네스의 체의 시간 복잡도는 O(N log log N)으로, N = 10^6일 때 매우 빠르게 동작한다. log log N은 극히 느리게 증가하므로 사실상 O(N)에 가깝다.

## 대안적 접근

- 짝수를 제외하고 홀수만 체크하면 메모리와 시간을 약 절반으로 줄일 수 있다. 다만 이 문제의 제약 조건에서는 기본 체로 충분하다.
