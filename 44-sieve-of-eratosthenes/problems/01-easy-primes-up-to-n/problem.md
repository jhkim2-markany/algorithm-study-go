# N 이하의 소수 구하기

**난이도:** 하

## 문제 설명

자연수 N이 주어질 때, 2 이상 N 이하의 모든 소수를 오름차순으로 출력하시오.

## 입력 형식

- 첫째 줄에 자연수 N이 주어진다

## 출력 형식

- 2 이상 N 이하의 소수를 한 줄에 공백으로 구분하여 오름차순으로 출력한다

## 제약 조건

- 2 ≤ N ≤ 1,000,000
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
20
```

### 예제 출력 1

```
2 3 5 7 11 13 17 19
```

### 예제 입력 2

```
10
```

### 예제 출력 2

```
2 3 5 7
```
