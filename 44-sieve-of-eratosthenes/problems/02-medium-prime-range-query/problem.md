# 구간 소수 개수 쿼리

**난이도:** 중

## 문제 설명

자연수 N과 Q개의 쿼리가 주어진다. 각 쿼리는 구간 [L, R]로 이루어져 있으며, L 이상 R 이하의 소수 개수를 구해야 한다.

에라토스테네스의 체와 누적합을 결합하여 각 쿼리를 효율적으로 처리하시오.

## 입력 형식

- 첫째 줄에 자연수 N과 쿼리의 수 Q가 공백으로 구분되어 주어진다
- 다음 Q개의 줄에 각각 정수 L과 R이 공백으로 구분되어 주어진다

## 출력 형식

- 각 쿼리에 대해 [L, R] 구간의 소수 개수를 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ N ≤ 1,000,000
- 1 ≤ Q ≤ 100,000
- 1 ≤ L ≤ R ≤ N
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
20 3
1 10
5 15
11 20
```

### 예제 출력 1

```
4
3
4
```

### 예제 입력 2

```
50 2
1 50
25 30
```

### 예제 출력 2

```
15
1
```
