# 구간 소수 세기 (세그먼트 체)

**난이도:** 상

## 문제 설명

두 정수 L과 R이 주어질 때, L 이상 R 이하의 소수의 개수를 구하시오.

L과 R의 값이 매우 클 수 있으므로, 일반적인 에라토스테네스의 체로는 메모리가 부족할 수 있다. 세그먼트 체(Segmented Sieve)를 활용하여 효율적으로 구간 내 소수를 구해야 한다.

## 입력 형식

- 첫째 줄에 두 정수 L과 R이 공백으로 구분되어 주어진다

## 출력 형식

- L 이상 R 이하의 소수의 개수를 출력한다

## 제약 조건

- 1 ≤ L ≤ R ≤ 10^12
- R - L ≤ 1,000,000
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
2 20
```

### 예제 출력 1

```
8
```

### 예제 입력 2

```
100 200
```

### 예제 출력 2

```
21
```

### 예제 입력 3

```
999999999900 1000000000000
```

### 예제 출력 3

```
3
```
