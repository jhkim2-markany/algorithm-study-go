# 구간 소수 세기 (세그먼트 체) - 해설

## 접근 방식

1. R이 최대 10^12이므로 전체 범위에 대한 배열을 만들 수 없다. 세그먼트 체를 사용한다
2. 먼저 √R까지의 소수를 기본 에라토스테네스의 체로 구한다 (√(10^12) = 10^6)
3. [L, R] 구간에 대한 불리언 배열(최대 10^6+1 크기)을 생성한다
4. 1단계에서 구한 각 소수 p에 대해, 구간 내 p의 배수를 합성수로 표시한다
5. p의 배수 시작점은 ceil(L/p) × p로 계산하되, p 자체는 소수이므로 건너뛴다
6. 배열에서 소수로 남은 수의 개수를 센다

## 핵심 아이디어

세그먼트 체의 핵심은 "N 이하의 합성수는 반드시 √N 이하의 소인수를 가진다"는 성질이다. 따라서 √R까지의 소수만 알면 [L, R] 구간의 모든 합성수를 걸러낼 수 있다.

R이 10^12으로 매우 크지만, R - L ≤ 10^6이라는 제약 덕분에 구간 크기만큼의 배열만 필요하다. √R ≈ 10^6이므로 기본 체에 필요한 메모리도 적다. 이 두 가지를 결합하면 메모리 O(√R + (R-L+1))로 문제를 해결할 수 있다.

배수 시작점 계산에서 ceil(L/p) × p를 사용하는 것이 구현의 핵심이다. 이는 L 이상인 p의 최소 배수를 구하는 공식이다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(√R log log √R + (R-L+1) log log R) |
| 공간 복잡도 | O(√R + (R-L+1)) |

기본 체에 O(√R log log √R), 구간 체에 O((R-L+1) × π(√R)/√R × log)이 소요된다. R = 10^12, R-L = 10^6일 때 충분히 빠르다.

## 대안적 접근

- 구간이 작고 L이 충분히 작다면 일반 에라토스테네스의 체로도 해결 가능하지만, L이 10^12까지 갈 수 있으므로 세그먼트 체가 필수적이다.
- 밀러-라빈 소수 판정법으로 구간 내 각 수를 개별 판정할 수도 있지만, 구간 크기가 10^6이면 세그먼트 체가 더 효율적이다.
