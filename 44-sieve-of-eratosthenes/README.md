# 44. 에라토스테네스의 체

주어진 범위 내의 모든 소수를 효율적으로 찾아내는 알고리즘이다. 기본 체, 최적화 체(짝수 제외, √N까지만 탐색), 세그먼트 체, 소인수분해 전처리 등이 핵심이다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이

## 문제 목록

| 번호 | 문제 | 난이도 |
| --- | --- | --- |
| 01 | N 이하의 소수 구하기 | 하 |
| 02 | 구간 소수 개수 쿼리 | 중 |
| 03 | 구간 소수 세기 (세그먼트 체) | 상 |
