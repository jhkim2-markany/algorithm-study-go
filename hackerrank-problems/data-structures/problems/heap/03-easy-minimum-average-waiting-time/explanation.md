# 최소 평균 대기 시간 - 해설

## 접근 방식

1. 고객을 도착 시간 기준으로 정렬한다
2. 현재 시간에 도착한 모든 고객을 조리 시간 기준 최소 힙에 넣는다
3. 힙에서 조리 시간이 가장 짧은 고객을 꺼내 처리한다
4. 현재 시간을 갱신하고 대기 시간을 누적한다
5. 힙이 비어있으면 다음 고객의 도착 시간으로 점프한다
6. 총 대기 시간을 고객 수로 나눈다

## 핵심 아이디어

SJF(Shortest Job First) 스케줄링 알고리즘을 적용한다. 현재 시점에서 이미 도착한 고객 중 조리 시간이 가장 짧은 고객을 먼저 처리하면 전체 평균 대기 시간이 최소화된다. 최소 힙을 사용하여 조리 시간이 가장 짧은 고객을 효율적으로 선택한다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N log N) |
| 공간 복잡도 | O(N) |

정렬에 O(N log N), 각 고객을 힙에 삽입/삭제하는 데 O(log N)이 걸린다. 전체 O(N log N)이다.

## 대안적 접근

- **완전 탐색**: 모든 순서를 시도하면 O(N!)으로 비현실적이다.
- **우선순위 큐 없이 정렬만 사용**: 도착 시간을 고려하지 않으면 최적해를 보장할 수 없다.
