# 최소 평균 대기 시간

**난이도:** 하
**출처:** [Minimum Average Waiting Time](https://www.hackerrank.com/challenges/minimum-average-waiting-time)

## 문제 설명

피자 가게에 N명의 고객이 방문한다. 각 고객은 도착 시간과 피자 조리에 필요한 시간이 주어진다. 피자 가게에는 오븐이 하나뿐이므로 한 번에 하나의 피자만 조리할 수 있다. 고객의 대기 시간은 피자가 완성되는 시간에서 도착 시간을 뺀 값이다.

모든 고객의 평균 대기 시간을 최소화하는 순서로 피자를 조리할 때, 최소 평균 대기 시간을 구하시오.

## 입력 형식

- 첫째 줄에 고객의 수 N이 주어진다
- 다음 N개의 줄에 각 고객의 도착 시간과 조리 시간이 공백으로 구분되어 주어진다

## 출력 형식

- 최소 평균 대기 시간을 정수로 출력한다 (소수점 이하 버림)

## 제약 조건

- 1 ≤ N ≤ 10^5
- 0 ≤ 도착 시간 ≤ 10^9
- 1 ≤ 조리 시간 ≤ 10^9

## 예제

### 예제 입력 1

```text
3
0 3
1 9
2 6
```

### 예제 출력 1

```text
9
```
