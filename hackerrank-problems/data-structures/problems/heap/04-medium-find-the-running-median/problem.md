# 실시간 중앙값 찾기

**난이도:** 중
**출처:** [Find the Running Median](https://www.hackerrank.com/challenges/find-the-running-median)

## 문제 설명

정수가 하나씩 주어진다. 각 정수를 읽을 때마다 지금까지 읽은 모든 정수의 중앙값을 출력하시오.

중앙값은 정렬된 수열의 가운데 값이다. 수열의 길이가 홀수이면 가운데 값, 짝수이면 가운데 두 값의 평균이다.

## 입력 형식

- 첫째 줄에 정수의 개수 N이 주어진다
- 다음 N개의 줄에 각 정수가 하나씩 주어진다

## 출력 형식

- 각 정수를 읽은 후의 중앙값을 소수점 첫째 자리까지 출력한다

## 제약 조건

- 1 ≤ N ≤ 10^5
- 0 ≤ 각 정수 ≤ 10^5

## 예제

### 예제 입력 1

```text
6
12
4
5
3
8
7
```

### 예제 출력 1

```text
12.0
8.0
5.0
4.5
5.0
6.0
```
