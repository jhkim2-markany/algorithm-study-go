# 힙 정렬

**난이도:** 중
**출처:** [Heap Sort — Full Sort](https://www.hackerrank.com/challenges/heapsort)

## 문제 설명

N개의 정수가 주어진다. 힙 정렬(Heap Sort) 알고리즘을 사용하여 배열을 오름차순으로 정렬하시오. 힙 정렬의 각 단계에서 배열의 상태를 출력한다.

힙 정렬은 다음과 같이 동작한다:
1. 배열을 최대 힙으로 구성한다 (Build Max Heap)
2. 힙의 루트(최댓값)를 배열의 마지막 원소와 교환한다
3. 힙 크기를 1 줄이고 루트에서 Heapify를 수행한다
4. 힙 크기가 1이 될 때까지 2-3을 반복한다

## 입력 형식

- 첫째 줄에 배열의 크기 N이 주어진다
- 둘째 줄에 N개의 정수가 공백으로 구분되어 주어진다

## 출력 형식

- 각 교환 후 배열의 상태를 한 줄에 출력한다

## 제약 조건

- 1 ≤ N ≤ 10^5
- -10^9 ≤ 각 원소 ≤ 10^9

## 예제

### 예제 입력 1

```text
5
1 2 3 4 5
```

### 예제 출력 1

```text
4 2 3 1 5
3 2 1 4 5
2 1 3 4 5
1 2 3 4 5
```
