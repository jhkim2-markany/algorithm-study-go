# 최소 연산 - 해설

## 접근 방식

1. 각 상자에 하나의 색상만 남기도록 배정해야 한다
2. 상자에 색상을 배정하는 모든 조합 중 이동 횟수가 최소인 것을 찾는다
3. 상자에 색상 c를 배정하면, 해당 상자에서 c가 아닌 공의 수만큼 이동이 필요하다
4. 힙을 사용하여 각 색상별로 가장 많은 공을 가진 상자를 우선 배정한다

## 핵심 아이디어

각 상자에 하나의 색상을 배정하는 문제이다. 상자 i에 색상 c를 배정하면 비용은 (총 공 수 - c색 공 수)이다. 즉, c색 공이 많을수록 비용이 적다.

그리디 + 힙 접근: 각 색상별로 "해당 색상의 공 개수"를 최대 힙에 넣는다. 각 상자에서 가장 많은 색상의 공을 남기면 이동 횟수가 최소화된다.

단순한 경우(N ≤ 3): 각 상자에 빨강/초록/파랑 중 하나를 배정하는 순열을 모두 시도한다.
일반적인 경우: 각 상자에서 가장 많은 색상의 공을 남기는 것이 최적이다. 총 공 수에서 각 상자의 최대 색상 공 수를 빼면 된다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N) |
| 공간 복잡도 | O(N) |

각 상자에서 세 색상 중 최댓값을 구하는 것은 O(1)이므로 전체 O(N)이다.

## 대안적 접근

- **힙 기반 배정**: 각 색상별 공 개수를 힙에 넣어 최적 배정을 찾을 수 있다.
- **헝가리안 알고리즘**: 상자-색상 배정을 이분 매칭으로 풀 수 있지만 과도하다.
- **완전 탐색**: 3^N 가지 배정을 모두 시도하면 N이 클 때 불가능하다.
