# 중앙값 갱신

**난이도:** 상
**출처:** [Median Updates](https://www.hackerrank.com/challenges/median)

## 문제 설명

빈 리스트에서 시작하여 다음 두 가지 연산을 처리하시오:

- `a x`: 리스트에 정수 x를 추가한다
- `r x`: 리스트에서 정수 x를 하나 제거한다 (존재하지 않으면 "Wrong!" 출력)

각 연산 후 리스트의 중앙값을 출력한다. 리스트가 비어있으면 "Wrong!"을 출력한다. 중앙값이 정수이면 정수로, 아니면 소수점 첫째 자리까지 출력한다.

## 입력 형식

- 첫째 줄에 연산의 개수 N이 주어진다
- 다음 N개의 줄에 각 연산이 주어진다 (형식: `a x` 또는 `r x`)

## 출력 형식

- 각 연산 후 중앙값을 출력한다
- 리스트가 비어있거나 삭제할 원소가 없으면 "Wrong!"을 출력한다

## 제약 조건

- 1 ≤ N ≤ 10^5
- 0 ≤ x ≤ 10^9

## 예제

### 예제 입력 1

```text
7
a 1
a 2
a 3
r 1
r 2
r 3
r 4
```

### 예제 출력 1

```text
1
1.5
2
2.5
3
Wrong!
Wrong!
```
