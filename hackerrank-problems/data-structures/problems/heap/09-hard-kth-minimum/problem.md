# 구간 K번째 최솟값

**난이도:** 상
**출처:** [Kth Minimum in Range](https://www.hackerrank.com/challenges/kth-minimum-in-range)

## 문제 설명

N개의 정수로 이루어진 배열이 주어진다. Q개의 쿼리가 주어지며, 각 쿼리는 구간 [l, r]에서 k번째로 작은 원소를 찾는 것이다.

## 입력 형식

- 첫째 줄에 배열의 크기 N과 쿼리의 개수 Q가 주어진다
- 둘째 줄에 N개의 정수가 공백으로 구분되어 주어진다
- 다음 Q개의 줄에 각 쿼리 l, r, k가 주어진다 (0-indexed)

## 출력 형식

- 각 쿼리에 대해 구간 [l, r]에서 k번째로 작은 원소를 출력한다

## 제약 조건

- 1 ≤ N ≤ 10^5
- 1 ≤ Q ≤ 10^5
- 0 ≤ l ≤ r < N
- 1 ≤ k ≤ r - l + 1
- -10^9 ≤ 각 원소 ≤ 10^9

## 예제

### 예제 입력 1

```text
7 3
1 5 2 6 3 7 4
2 5 3
0 6 4
0 3 1
```

### 예제 출력 1

```text
6
4
1
```
