# 고정 길이 쿼리

**난이도:** 중
**출처:** [Queries with Fixed Length](https://www.hackerrank.com/challenges/queries-with-fixed-length)

## 문제 설명

N개의 정수로 이루어진 수열 A가 주어진다. 정수 d를 사용하여 다음 값을 계산한다:

min(max(A[i], A[i+1], ..., A[i+d-1])) (0 ≤ i ≤ N-d)

즉, 길이 d인 모든 연속 부분 수열의 최댓값을 구한 뒤, 그 최댓값들 중 최솟값을 구한다.

Q개의 쿼리가 주어지며, 각 쿼리마다 d 값이 주어진다. 각 쿼리에 대한 답을 출력하시오.

## 입력 형식

- 첫째 줄에 수열의 길이 N과 쿼리의 개수 Q가 공백으로 구분되어 주어진다
- 둘째 줄에 N개의 정수가 공백으로 구분되어 주어진다
- 다음 Q개의 줄에 각 쿼리의 d 값이 주어진다

## 출력 형식

- 각 쿼리에 대한 답을 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ N ≤ 10^5
- 0 ≤ A[i] ≤ 10^6
- 1 ≤ Q ≤ 100
- 1 ≤ d ≤ N

## 예제

### 예제 입력 1

```text
5 5
33 11 44 11 55
1
2
3
4
5
```

### 예제 출력 1

```text
11
33
44
44
55
```
