# 이진 트리의 높이

**난이도:** 하
**출처:** [Tree: Height of a Binary Tree](https://www.hackerrank.com/challenges/tree-height-of-a-binary-tree)

## 문제 설명

이진 트리의 루트 노드가 주어진다. 트리의 높이를 구하시오. 트리의 높이는 루트에서 가장 먼 리프 노드까지의 간선(edge) 수로 정의한다. 노드가 하나뿐인 트리의 높이는 0이다.

## 입력 형식

- 첫째 줄에 노드의 개수 N이 주어진다
- 다음 N개의 줄에 각 노드의 데이터가 주어진다 (BST 삽입 순서대로)

## 출력 형식

- 트리의 높이를 한 줄에 출력한다

## 제약 조건

- 1 ≤ N ≤ 20
- 1 ≤ 각 노드의 데이터 ≤ 1000

## 예제

### 예제 입력 1

```text
7
3
5
2
1
4
6
7
```

### 예제 출력 1

```text
3
```
