# 이진 트리 전위 순회 - 해설

## 접근 방식

1. 루트 노드부터 시작한다
2. 현재 노드의 데이터를 출력한다
3. 왼쪽 서브트리를 재귀적으로 전위 순회한다
4. 오른쪽 서브트리를 재귀적으로 전위 순회한다
5. 노드가 nil이면 반환한다

## 핵심 아이디어

전위 순회(Preorder)는 "루트 → 왼쪽 → 오른쪽" 순서로 방문하는 트리 순회 방법이다. 재귀 함수를 사용하면 자연스럽게 구현할 수 있다. 현재 노드를 먼저 처리한 후 왼쪽, 오른쪽 자식을 순서대로 방문한다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N) |
| 공간 복잡도 | O(H) |

N개의 노드를 한 번씩 방문하므로 시간 복잡도는 O(N)이다. 재귀 호출 스택의 깊이는 트리의 높이 H에 비례하므로 공간 복잡도는 O(H)이다. 최악의 경우(편향 트리) H = N이 된다.

## 대안적 접근

- **스택을 이용한 반복적 순회**: 명시적 스택을 사용하여 재귀 없이 전위 순회를 구현할 수 있다. 스택에 오른쪽 자식을 먼저 넣고 왼쪽 자식을 나중에 넣으면 전위 순회 순서가 된다.
- **Morris 순회**: O(1) 공간으로 전위 순회를 수행할 수 있으나 구현이 복잡하다.
