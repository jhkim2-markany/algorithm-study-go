# 이진 트리 탑 뷰

**난이도:** 중
**출처:** [Tree: Top View](https://www.hackerrank.com/challenges/tree-top-view)

## 문제 설명

이진 트리의 루트 노드가 주어진다. 트리를 위에서 내려다보았을 때 보이는 노드들을 왼쪽에서 오른쪽 순서로 출력하시오.

각 노드에 수평 거리(horizontal distance)를 부여한다. 루트의 수평 거리는 0이고, 왼쪽 자식은 부모보다 1 작고, 오른쪽 자식은 부모보다 1 크다. 같은 수평 거리에 여러 노드가 있으면 가장 위에 있는(먼저 방문되는) 노드만 보인다.

## 입력 형식

- 첫째 줄에 노드의 개수 N이 주어진다
- 다음 N개의 줄에 각 노드의 데이터가 주어진다 (BST 삽입 순서대로)

## 출력 형식

- 탑 뷰에서 보이는 노드의 데이터를 왼쪽에서 오른쪽 순서로 공백 구분하여 출력한다

## 제약 조건

- 1 ≤ N ≤ 10000
- 1 ≤ 각 노드의 데이터 ≤ 1000

## 예제

### 예제 입력 1

```text
6
1
2
5
3
6
4
```

### 예제 출력 1

```text
1 2 5 6
```
