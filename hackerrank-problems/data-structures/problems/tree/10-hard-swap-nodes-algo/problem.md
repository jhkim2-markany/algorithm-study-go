# 노드 교환 알고리즘

**난이도:** 상
**출처:** [Swap Nodes [Algo]](https://www.hackerrank.com/challenges/swap-nodes-algo)

## 문제 설명

이진 트리가 주어진다. 깊이가 K의 배수인 모든 노드에서 왼쪽과 오른쪽 자식을 교환하는 연산을 수행한다. 각 교환 연산 후 트리의 중위 순회 결과를 출력하시오.

트리는 인덱스 기반으로 주어진다. 루트의 인덱스는 1이고, 각 노드의 왼쪽/오른쪽 자식 인덱스가 주어진다. -1은 자식이 없음을 의미한다. 루트의 깊이는 1이다.

## 입력 형식

- 첫째 줄에 노드의 개수 N이 주어진다
- 다음 N개의 줄에 각 노드의 왼쪽 자식 인덱스와 오른쪽 자식 인덱스가 공백으로 구분되어 주어진다 (i번째 줄은 인덱스 i 노드의 자식)
- 다음 줄에 교환 쿼리의 개수 T가 주어진다
- 다음 T개의 줄에 각 쿼리의 K 값이 주어진다

## 출력 형식

- 각 교환 연산 후 중위 순회 결과를 공백으로 구분하여 한 줄에 출력한다

## 제약 조건

- 1 ≤ N ≤ 1024
- 1 ≤ T ≤ 100
- 1 ≤ K ≤ N
- 인덱스는 1부터 N까지이거나 -1(자식 없음)

## 예제

### 예제 입력 1

```text
3
2 3
-1 -1
-1 -1
2
1
1
```

### 예제 출력 1

```text
3 1 2
2 1 3
```

### 예제 입력 2

```text
5
2 3
-1 4
-1 5
-1 -1
-1 -1
1
2
```

### 예제 출력 2

```text
4 2 1 5 3
```
