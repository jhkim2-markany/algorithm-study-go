# 이진 트리 중위 순회 - 해설

## 접근 방식

1. 루트 노드부터 시작한다
2. 왼쪽 서브트리를 재귀적으로 중위 순회한다
3. 현재 노드의 데이터를 출력한다
4. 오른쪽 서브트리를 재귀적으로 중위 순회한다
5. 노드가 nil이면 반환한다

## 핵심 아이디어

중위 순회(Inorder)는 "왼쪽 → 루트 → 오른쪽" 순서로 방문하는 트리 순회 방법이다. BST에서 중위 순회를 수행하면 데이터가 오름차순으로 정렬된 결과를 얻을 수 있다. 이 성질은 BST의 유효성 검증에도 활용된다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N) |
| 공간 복잡도 | O(H) |

N개의 노드를 한 번씩 방문하므로 시간 복잡도는 O(N)이다. 재귀 호출 스택의 깊이는 트리의 높이 H에 비례하므로 공간 복잡도는 O(H)이다.

## 대안적 접근

- **스택을 이용한 반복적 순회**: 명시적 스택을 사용하여 왼쪽 자식을 계속 스택에 넣다가, 더 이상 왼쪽이 없으면 팝하여 출력하고 오른쪽으로 이동한다.
- **Morris 순회**: 스레디드 트리 개념을 활용하여 O(1) 공간으로 중위 순회를 수행할 수 있다.
