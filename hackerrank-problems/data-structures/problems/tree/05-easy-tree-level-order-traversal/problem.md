# 이진 트리 레벨 순서 순회

**난이도:** 하
**출처:** [Tree: Level Order Traversal](https://www.hackerrank.com/challenges/tree-level-order-traversal)

## 문제 설명

이진 트리의 루트 노드가 주어진다. 트리를 레벨 순서(Level Order)로 순회하여 각 노드의 데이터를 공백으로 구분하여 한 줄에 출력하시오.

레벨 순서 순회는 루트부터 시작하여 같은 레벨의 노드를 왼쪽에서 오른쪽으로 방문한 후, 다음 레벨로 이동한다.

## 입력 형식

- 첫째 줄에 노드의 개수 N이 주어진다
- 다음 N개의 줄에 각 노드의 데이터가 주어진다 (BST 삽입 순서대로)

## 출력 형식

- 레벨 순서 순회 결과를 공백으로 구분하여 한 줄에 출력한다

## 제약 조건

- 1 ≤ N ≤ 500
- 1 ≤ 각 노드의 데이터 ≤ 1000

## 예제

### 예제 입력 1

```text
6
1
2
5
3
6
4
```

### 예제 출력 1

```text
1 2 5 3 6 4
```
