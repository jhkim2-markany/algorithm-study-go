# 이진 트리 레벨 순서 순회 - 해설

## 접근 방식

1. 큐를 생성하고 루트 노드를 큐에 넣는다
2. 큐가 비어 있지 않은 동안 반복한다
3. 큐에서 노드를 꺼내 데이터를 출력한다
4. 왼쪽 자식이 있으면 큐에 넣는다
5. 오른쪽 자식이 있으면 큐에 넣는다

## 핵심 아이디어

레벨 순서 순회(BFS)는 큐를 사용하여 구현한다. 큐의 FIFO 특성 덕분에 같은 레벨의 노드가 먼저 처리되고, 그 다음 레벨의 노드가 순서대로 처리된다. 이는 그래프의 너비 우선 탐색(BFS)과 동일한 원리이다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N) |
| 공간 복잡도 | O(N) |

N개의 노드를 한 번씩 방문하므로 시간 복잡도는 O(N)이다. 큐에 최대 한 레벨의 노드가 저장되며, 완전 이진 트리의 경우 마지막 레벨에 약 N/2개의 노드가 있으므로 공간 복잡도는 O(N)이다.

## 대안적 접근

- **재귀적 레벨 순회**: 각 레벨을 별도로 순회하는 재귀 함수를 작성할 수 있으나, 매 레벨마다 트리를 처음부터 탐색해야 하므로 O(N²) 시간이 소요된다.
- **레벨별 구분 출력**: 큐의 크기를 이용하여 레벨별로 구분하여 출력할 수도 있다.
