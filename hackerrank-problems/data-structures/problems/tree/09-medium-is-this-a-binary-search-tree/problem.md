# 이것은 이진 탐색 트리인가?

**난이도:** 중
**출처:** [Is This a Binary Search Tree?](https://www.hackerrank.com/challenges/is-binary-search-tree)

## 문제 설명

이진 트리의 루트 노드가 주어진다. 이 트리가 유효한 이진 탐색 트리(BST)인지 판별하시오.

유효한 BST의 조건:
- 왼쪽 서브트리의 모든 노드 값은 현재 노드 값보다 작다
- 오른쪽 서브트리의 모든 노드 값은 현재 노드 값보다 크다
- 왼쪽과 오른쪽 서브트리도 각각 유효한 BST이다
- 중복된 값은 없다

## 입력 형식

- 첫째 줄에 노드의 개수 N이 주어진다
- 다음 N개의 줄에 각 노드의 데이터가 주어진다 (BST 삽입 순서대로)

## 출력 형식

- 유효한 BST이면 "Yes", 아니면 "No"를 출력한다

## 제약 조건

- 0 ≤ N ≤ 20
- 0 ≤ 각 노드의 데이터 ≤ 10000

## 예제

### 예제 입력 1

```text
7
4
2
6
1
3
5
7
```

### 예제 출력 1

```text
Yes
```
