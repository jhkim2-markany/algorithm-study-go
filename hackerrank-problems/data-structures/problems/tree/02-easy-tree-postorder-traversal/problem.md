# 이진 트리 후위 순회

**난이도:** 하
**출처:** [Tree: Postorder Traversal](https://www.hackerrank.com/challenges/tree-postorder-traversal)

## 문제 설명

이진 트리의 루트 노드가 주어진다. 트리를 후위 순회(Postorder Traversal)하여 각 노드의 데이터를 공백으로 구분하여 한 줄에 출력하시오.

후위 순회는 다음 순서로 노드를 방문한다:
1. 왼쪽 서브트리를 후위 순회한다
2. 오른쪽 서브트리를 후위 순회한다
3. 현재 노드를 방문한다

## 입력 형식

- 첫째 줄에 노드의 개수 N이 주어진다
- 다음 N개의 줄에 각 노드의 데이터가 주어진다 (BST 삽입 순서대로)

## 출력 형식

- 후위 순회 결과를 공백으로 구분하여 한 줄에 출력한다

## 제약 조건

- 1 ≤ N ≤ 500
- 1 ≤ 각 노드의 데이터 ≤ 1000

## 예제

### 예제 입력 1

```text
6
1
2
5
3
6
4
```

### 예제 출력 1

```text
4 3 6 5 2 1
```
