# 이진 탐색 트리 삽입

**난이도:** 중
**출처:** [Binary Search Tree: Insertion](https://www.hackerrank.com/challenges/binary-search-tree-insertion)

## 문제 설명

이진 탐색 트리(BST)의 루트 노드와 삽입할 값이 주어진다. 값을 BST에 삽입한 후 루트 노드를 반환하시오.

BST의 삽입 규칙:
- 삽입할 값이 현재 노드보다 작으면 왼쪽 서브트리로 이동한다
- 삽입할 값이 현재 노드보다 크면 오른쪽 서브트리로 이동한다
- 빈 위치를 찾으면 새 노드를 생성하여 삽입한다

## 입력 형식

- 첫째 줄에 기존 트리의 노드 개수 N이 주어진다
- 다음 N개의 줄에 각 노드의 데이터가 주어진다 (BST 삽입 순서대로)
- 마지막 줄에 새로 삽입할 값이 주어진다

## 출력 형식

- 삽입 후 트리의 레벨 순서 순회 결과를 공백으로 구분하여 출력한다

## 제약 조건

- 1 ≤ N ≤ 500
- 1 ≤ 각 노드의 데이터 ≤ 10000
- 삽입할 값은 기존 트리에 존재하지 않는다

## 예제

### 예제 입력 1

```text
6
4
2
7
1
3
6
5
```

### 예제 출력 1

```text
4 2 7 1 3 6 5
```
