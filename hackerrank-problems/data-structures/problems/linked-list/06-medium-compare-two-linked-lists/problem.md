# 두 연결 리스트 비교

**난이도:** 중
**출처:** [Compare two linked lists](https://www.hackerrank.com/challenges/compare-two-linked-lists)

## 문제 설명

두 개의 단일 연결 리스트의 헤드 포인터가 주어진다. 두 리스트가 동일한지 비교하시오. 두 리스트의 길이가 같고, 대응하는 모든 노드의 데이터가 같으면 동일한 리스트이다.

## 입력 형식

- 첫째 줄에 테스트 케이스의 수 T가 주어진다
- 각 테스트 케이스마다:
  - 첫째 줄에 첫 번째 리스트의 노드 수 N이 주어진다
  - 다음 N개의 줄에 첫 번째 리스트의 데이터가 주어진다
  - 다음 줄에 두 번째 리스트의 노드 수 M이 주어진다
  - 다음 M개의 줄에 두 번째 리스트의 데이터가 주어진다

## 출력 형식

- 각 테스트 케이스마다 두 리스트가 동일하면 1, 다르면 0을 출력한다

## 제약 조건

- 1 ≤ T ≤ 10
- 1 ≤ N, M ≤ 1000
- 1 ≤ 각 노드의 데이터 ≤ 1000

## 예제

### 예제 입력 1

```text
2
2
1
2
2
1
2
2
1
2
1
3
```

### 예제 출력 1

```text
1
0
```
