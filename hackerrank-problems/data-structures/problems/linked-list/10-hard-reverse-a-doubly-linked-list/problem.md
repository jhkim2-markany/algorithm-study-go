# 이중 연결 리스트 뒤집기

**난이도:** 상
**출처:** [Reverse a doubly linked list](https://www.hackerrank.com/challenges/reverse-a-doubly-linked-list)

## 문제 설명

이중 연결 리스트의 헤드 포인터가 주어진다. 리스트의 순서를 뒤집고, 뒤집힌 리스트의 헤드 포인터를 반환하시오. 이중 연결 리스트의 각 노드는 데이터, 다음 노드 포인터(Next), 이전 노드 포인터(Prev)를 가진다.

## 입력 형식

- 첫째 줄에 테스트 케이스의 수 T가 주어진다
- 각 테스트 케이스마다:
  - 첫째 줄에 노드의 개수 N이 주어진다
  - 다음 N개의 줄에 각 노드의 데이터 값이 주어진다

## 출력 형식

- 각 테스트 케이스마다 뒤집힌 리스트의 각 노드 데이터를 공백으로 구분하여 한 줄에 출력한다

## 제약 조건

- 1 ≤ T ≤ 10
- 1 ≤ N ≤ 1000
- 0 ≤ 각 노드의 데이터 ≤ 1000

## 예제

### 예제 입력 1

```text
1
4
1
2
3
4
```

### 예제 출력 1

```text
4 3 2 1
```
