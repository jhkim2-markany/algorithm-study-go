# 끝에서 N번째 노드 값 구하기

**난이도:** 중
**출처:** [Get Node Value](https://www.hackerrank.com/challenges/get-the-value-of-the-node-at-a-specific-position-from-the-tail)

## 문제 설명

단일 연결 리스트의 헤드 포인터와 끝에서부터의 위치(0-based)가 주어진다. 끝에서 해당 위치에 있는 노드의 데이터 값을 반환하시오. 끝에서 0번째는 마지막 노드를 의미한다.

## 입력 형식

- 첫째 줄에 테스트 케이스의 수 T가 주어진다
- 각 테스트 케이스마다:
  - 첫째 줄에 노드의 개수 N이 주어진다
  - 다음 N개의 줄에 각 노드의 데이터 값이 주어진다
  - 마지막 줄에 끝에서부터의 위치 positionFromTail이 주어진다

## 출력 형식

- 각 테스트 케이스마다 끝에서 해당 위치의 노드 데이터를 출력한다

## 제약 조건

- 1 ≤ T ≤ 10
- 1 ≤ N ≤ 1000
- 0 ≤ positionFromTail < N
- 1 ≤ 각 노드의 데이터 ≤ 1000

## 예제

### 예제 입력 1

```text
1
3
1
3
5
0
```

### 예제 출력 1

```text
5
```

### 예제 입력 2

```text
1
3
1
3
5
2
```

### 예제 출력 2

```text
1
```
