# 정렬된 두 연결 리스트 병합

**난이도:** 중
**출처:** [Merge two sorted linked lists](https://www.hackerrank.com/challenges/merge-two-sorted-linked-lists)

## 문제 설명

오름차순으로 정렬된 두 개의 단일 연결 리스트의 헤드 포인터가 주어진다. 두 리스트를 하나의 오름차순 정렬된 연결 리스트로 병합하고, 병합된 리스트의 헤드 포인터를 반환하시오.

## 입력 형식

- 첫째 줄에 테스트 케이스의 수 T가 주어진다
- 각 테스트 케이스마다:
  - 첫째 줄에 첫 번째 리스트의 노드 수 N이 주어진다
  - 다음 N개의 줄에 첫 번째 리스트의 데이터가 오름차순으로 주어진다
  - 다음 줄에 두 번째 리스트의 노드 수 M이 주어진다
  - 다음 M개의 줄에 두 번째 리스트의 데이터가 오름차순으로 주어진다

## 출력 형식

- 각 테스트 케이스마다 병합된 리스트의 각 노드 데이터를 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ T ≤ 10
- 1 ≤ N, M ≤ 1000
- 1 ≤ 각 노드의 데이터 ≤ 1000

## 예제

### 예제 입력 1

```text
1
3
1
3
5
2
2
4
```

### 예제 출력 1

```text
1
2
3
4
5
```
