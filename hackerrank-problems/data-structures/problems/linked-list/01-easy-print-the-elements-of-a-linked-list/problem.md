# 연결 리스트의 원소 출력

**난이도:** 하
**출처:** [Print the Elements of a Linked List](https://www.hackerrank.com/challenges/print-the-elements-of-a-linked-list)

## 문제 설명

단일 연결 리스트의 헤드 포인터가 주어진다. 리스트의 각 노드에 저장된 데이터 값을 순서대로 출력하시오. 각 값은 한 줄에 하나씩 출력한다. 리스트가 비어 있으면 아무것도 출력하지 않는다.

## 입력 형식

- 첫째 줄에 노드의 개수 N이 주어진다
- 다음 N개의 줄에 각 노드의 데이터 값이 주어진다 (삽입 순서대로)

## 출력 형식

- 연결 리스트의 각 노드 데이터를 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ N ≤ 1000
- 1 ≤ 각 노드의 데이터 ≤ 1000

## 예제

### 예제 입력 1

```text
4
2
3
4
1
```

### 예제 출력 1

```text
2
3
4
1
```
