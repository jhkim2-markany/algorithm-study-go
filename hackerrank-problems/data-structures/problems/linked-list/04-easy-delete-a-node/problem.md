# 노드 삭제

**난이도:** 하
**출처:** [Delete a Node](https://www.hackerrank.com/challenges/delete-a-node-from-a-linked-list)

## 문제 설명

단일 연결 리스트의 헤드 포인터와 삭제할 노드의 위치(0-based index)가 주어진다. 해당 위치의 노드를 삭제하고, 갱신된 리스트의 헤드 포인터를 반환하시오.

## 입력 형식

- 첫째 줄에 노드의 개수 N이 주어진다
- 다음 N개의 줄에 각 노드의 데이터 값이 주어진다
- 마지막 줄에 삭제할 노드의 위치 position이 주어진다 (0-based)

## 출력 형식

- 삭제 후 연결 리스트의 각 노드 데이터를 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ N ≤ 1000
- 0 ≤ position < N
- 1 ≤ 각 노드의 데이터 ≤ 1000

## 예제

### 예제 입력 1

```text
4
20
6
2
19
1
```

### 예제 출력 1

```text
20
2
19
```
