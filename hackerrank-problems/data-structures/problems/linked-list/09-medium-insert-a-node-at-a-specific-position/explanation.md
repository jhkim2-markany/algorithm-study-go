# 특정 위치에 노드 삽입 - 해설

## 접근 방식

1. 새 노드를 생성한다
2. 삽입 위치가 0이면 새 노드를 헤드로 설정한다
3. 그렇지 않으면 (position-1)번째 노드까지 순회한다
4. 새 노드의 Next를 현재 노드의 Next로 설정한다
5. 현재 노드의 Next를 새 노드로 설정한다

## 핵심 아이디어

연결 리스트의 특정 위치에 삽입하려면 삽입 위치의 이전 노드를 찾아야 한다. 이전 노드의 `Next`를 새 노드로, 새 노드의 `Next`를 원래 이전 노드의 `Next`로 설정하면 삽입이 완료된다. 위치가 0인 경우(헤드 앞 삽입)는 별도로 처리한다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N) |
| 공간 복잡도 | O(1) |

삽입 위치까지 순회해야 하므로 최악의 경우 시간 복잡도는 O(N)이다. 새 노드 하나만 생성하므로 공간 복잡도는 O(1)이다.

## 대안적 접근

- **더미 헤드 사용**: 더미 노드를 사용하면 위치 0 삽입을 별도로 처리하지 않아도 된다.
- **배열 기반 리스트**: 배열 기반이면 인덱스로 바로 접근 가능하지만, 삽입 시 원소 이동이 필요하여 O(N)이다.
