# 연결 리스트 앞에 노드 삽입 - 해설

## 접근 방식

1. 새로운 노드를 생성한다
2. 새 노드의 Next를 기존 헤드로 설정한다
3. 새 노드를 새로운 헤드로 반환한다

## 핵심 아이디어

연결 리스트의 맨 앞에 삽입하는 것은 가장 간단한 연산이다. 새 노드를 만들고 그 노드의 `Next`를 기존 헤드로 연결하면 된다. 리스트가 비어 있어도(헤드가 nil) 동일한 로직이 적용된다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(1) |
| 공간 복잡도 | O(1) |

리스트를 순회할 필요 없이 바로 삽입하므로 시간 복잡도는 O(1)이다. 새 노드 하나만 생성하므로 공간 복잡도도 O(1)이다.

## 대안적 접근

- **더미 헤드 사용**: 더미(sentinel) 노드를 사용하면 빈 리스트 처리를 별도로 하지 않아도 된다. 다만 이 문제에서는 불필요한 복잡성이 추가된다.
