# 연결 리스트 앞에 노드 삽입

**난이도:** 하
**출처:** [Insert a node at the head of a linked list](https://www.hackerrank.com/challenges/insert-a-node-at-the-head-of-a-linked-list)

## 문제 설명

단일 연결 리스트의 헤드 포인터와 삽입할 데이터 값이 주어진다. 새로운 노드를 리스트의 맨 앞(head)에 삽입하고, 갱신된 리스트의 헤드 포인터를 반환하시오.

## 입력 형식

- 첫째 줄에 노드의 개수 N이 주어진다
- 다음 N개의 줄에 각 노드의 데이터 값이 주어진다 (순서대로 head에 삽입)

## 출력 형식

- 완성된 연결 리스트의 각 노드 데이터를 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ N ≤ 1000
- 1 ≤ 각 노드의 데이터 ≤ 1000

## 예제

### 예제 입력 1

```text
5
383
484
392
975
321
```

### 예제 출력 1

```text
321
975
392
484
383
```
