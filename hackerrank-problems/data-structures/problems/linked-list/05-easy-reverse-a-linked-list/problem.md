# 연결 리스트 뒤집기

**난이도:** 하
**출처:** [Reverse a linked list](https://www.hackerrank.com/challenges/reverse-a-linked-list)

## 문제 설명

단일 연결 리스트의 헤드 포인터가 주어진다. 리스트의 순서를 뒤집고, 뒤집힌 리스트의 헤드 포인터를 반환하시오.

## 입력 형식

- 첫째 줄에 노드의 개수 N이 주어진다
- 다음 N개의 줄에 각 노드의 데이터 값이 주어진다

## 출력 형식

- 뒤집힌 연결 리스트의 각 노드 데이터를 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ N ≤ 1000
- 1 ≤ 각 노드의 데이터 ≤ 1000

## 예제

### 예제 입력 1

```text
5
1
2
3
4
5
```

### 예제 출력 1

```text
5
4
3
2
1
```
