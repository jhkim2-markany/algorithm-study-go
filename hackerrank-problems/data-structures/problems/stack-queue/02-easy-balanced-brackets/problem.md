# 균형 잡힌 괄호

**난이도:** 하
**출처:** [Balanced Brackets](https://www.hackerrank.com/challenges/balanced-brackets)

## 문제 설명

괄호 문자열이 주어진다. 문자열이 균형 잡혀 있는지 판별하시오. 괄호 문자열은 `(`, `)`, `[`, `]`, `{`, `}`로만 구성된다.

균형 잡힌 문자열의 조건:
- 빈 문자열은 균형 잡혀 있다
- 문자열 s가 균형 잡혀 있으면 `(s)`, `[s]`, `{s}`도 균형 잡혀 있다
- 문자열 s와 t가 균형 잡혀 있으면 `st`도 균형 잡혀 있다

## 입력 형식

- 첫째 줄에 테스트 케이스의 개수 T가 주어진다
- 다음 T개의 줄에 각 괄호 문자열이 주어진다

## 출력 형식

- 각 테스트 케이스에 대해 균형 잡혀 있으면 `YES`, 아니면 `NO`를 출력한다

## 제약 조건

- 1 ≤ T ≤ 10^3
- 1 ≤ 문자열 길이 ≤ 10^3
- 문자열은 `(`, `)`, `[`, `]`, `{`, `}`로만 구성된다

## 예제

### 예제 입력 1

```text
3
{[()]}
{[(])}
{{[[(())]]}}
```

### 예제 출력 1

```text
YES
NO
YES
```
