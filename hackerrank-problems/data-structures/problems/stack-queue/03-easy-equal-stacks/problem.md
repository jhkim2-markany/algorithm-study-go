# 같은 높이의 스택

**난이도:** 하
**출처:** [Equal Stacks](https://www.hackerrank.com/challenges/equal-stacks)

## 문제 설명

세 개의 스택이 주어진다. 각 스택은 원통형 블록으로 구성되어 있으며, 각 블록의 높이가 주어진다. 세 스택의 높이가 같아지도록 블록을 제거할 때, 가능한 최대 높이를 구하시오. 블록은 스택의 최상위에서만 제거할 수 있다.

## 입력 형식

- 첫째 줄에 세 스택의 블록 개수 n1, n2, n3이 주어진다
- 둘째 줄에 첫 번째 스택의 블록 높이가 위에서 아래 순서로 주어진다
- 셋째 줄에 두 번째 스택의 블록 높이가 위에서 아래 순서로 주어진다
- 넷째 줄에 세 번째 스택의 블록 높이가 위에서 아래 순서로 주어진다

## 출력 형식

- 세 스택의 높이가 같아질 때의 최대 높이를 출력한다

## 제약 조건

- 0 < n1, n2, n3 ≤ 10^5
- 0 < 각 블록의 높이 ≤ 100

## 예제

### 예제 입력 1

```text
5 3 4
3 2 1 1 1
4 3 2
1 1 4 1
```

### 예제 출력 1

```text
5
```
