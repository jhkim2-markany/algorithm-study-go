# 최대 원소

**난이도:** 하
**출처:** [Maximum Element](https://www.hackerrank.com/challenges/maximum-element)

## 문제 설명

스택에 대해 다음 세 가지 쿼리를 처리하시오:

- `1 x`: 원소 x를 스택에 푸시한다
- `2`: 스택의 최상위 원소를 팝한다
- `3`: 스택에서 현재 최대 원소를 출력한다

## 입력 형식

- 첫째 줄에 쿼리의 개수 N이 주어진다
- 다음 N개의 줄에 각 쿼리가 주어진다

## 출력 형식

- 쿼리 타입 3이 주어질 때마다 스택의 최대 원소를 한 줄에 출력한다

## 제약 조건

- 1 ≤ N ≤ 10^5
- 1 ≤ x ≤ 10^9
- 쿼리 타입 2와 3이 호출될 때 스택은 비어있지 않다

## 예제

### 예제 입력 1

```text
10
1 97
2
1 20
2
1 26
1 20
2
3
1 91
3
```

### 예제 출력 1

```text
26
91
```
