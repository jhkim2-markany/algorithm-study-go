# 웨이터

**난이도:** 중
**출처:** [Waiter](https://www.hackerrank.com/challenges/waiter)

## 문제 설명

N개의 접시가 쌓여 있는 스택이 있다. 각 접시에는 번호가 적혀 있다. 웨이터는 Q번의 반복을 수행한다.

i번째 반복에서 웨이터는 현재 스택의 모든 접시를 위에서부터 하나씩 확인한다:
- 접시 번호가 i번째 소수로 나누어지면 스택 B_i에 넣는다
- 나누어지지 않으면 임시 스택 A_i에 넣는다

각 반복이 끝나면 A_i가 다음 반복의 입력 스택이 된다.

모든 반복이 끝난 후, B_1, B_2, ..., B_Q의 최상위부터 순서대로, 그리고 마지막 A_Q의 최상위부터 순서대로 출력한다.

## 입력 형식

- 첫째 줄에 접시 개수 N과 반복 횟수 Q가 주어진다
- 둘째 줄에 N개의 접시 번호가 주어진다 (스택의 아래에서 위 순서)

## 출력 형식

- 결과 접시 번호를 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ N ≤ 5 × 10^4
- 2 ≤ 접시 번호 ≤ 10^4
- 1 ≤ Q ≤ 1200

## 예제

### 예제 입력 1

```text
5 1
3 4 7 6 5
```

### 예제 출력 1

```text
4
6
3
7
5
```
