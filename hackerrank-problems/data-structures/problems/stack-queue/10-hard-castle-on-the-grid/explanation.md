# 격자 위의 성 - 해설

## 접근 방식

1. BFS(너비 우선 탐색)를 사용하여 최소 이동 횟수를 구한다
2. 각 위치에서 상하좌우 네 방향으로 벽이나 경계까지 직선 이동한다
3. 직선 경로 위의 모든 칸을 방문 처리하고 큐에 추가한다
4. 목표 위치에 도달하면 이동 횟수를 반환한다

## 핵심 아이디어

일반적인 BFS와 달리, 한 번의 이동으로 한 방향의 여러 칸을 지나갈 수 있다. 따라서 각 위치에서 네 방향으로 벽이나 경계까지 확장하며 모든 도달 가능한 칸을 큐에 넣는다. BFS의 특성상 먼저 도달한 경로가 최단 경로이다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N³) |
| 공간 복잡도 | O(N²) |

각 칸은 최대 한 번 방문되지만, 각 칸에서 최대 N칸까지 확장할 수 있으므로 시간 복잡도는 O(N³)이다. 방문 배열과 큐에 O(N²) 공간이 필요하다.

## 대안적 접근

- **다익스트라**: 모든 이동 비용이 1이므로 BFS가 더 효율적이다. 다익스트라는 불필요한 오버헤드가 발생한다.
- **A* 탐색**: 맨해튼 거리를 휴리스틱으로 사용하여 탐색 공간을 줄일 수 있으나, 이 문제의 이동 규칙에서는 효과가 제한적이다.
