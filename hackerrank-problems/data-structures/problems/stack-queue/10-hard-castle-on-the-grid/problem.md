# 격자 위의 성

**난이도:** 상
**출처:** [Castle on the Grid](https://www.hackerrank.com/challenges/castle-on-the-grid)

## 문제 설명

N × N 격자가 주어진다. 격자의 각 칸은 빈 칸(`.`) 또는 벽(`X`)이다. 시작 위치에서 목표 위치까지 이동해야 한다.

이동 규칙:
- 상하좌우 한 방향으로 벽이나 격자 경계에 닿을 때까지 직선으로 이동할 수 있다
- 한 번의 이동으로 같은 방향의 여러 칸을 지나갈 수 있다

시작 위치에서 목표 위치까지의 최소 이동 횟수를 구하시오.

## 입력 형식

- 첫째 줄에 격자의 크기 N이 주어진다
- 다음 N개의 줄에 격자가 주어진다
- 마지막 줄에 시작 행, 시작 열, 목표 행, 목표 열이 주어진다

## 출력 형식

- 최소 이동 횟수를 출력한다

## 제약 조건

- 1 ≤ N ≤ 100
- 시작 위치와 목표 위치는 항상 빈 칸이다

## 예제

### 예제 입력 1

```text
3
.X.
.X.
...
0 0 0 2
```

### 예제 출력 1

```text
3
```
