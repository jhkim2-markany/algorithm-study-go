# 간단한 텍스트 편집기

**난이도:** 중
**출처:** [Simple Text Editor](https://www.hackerrank.com/challenges/simple-text-editor)

## 문제 설명

빈 문자열 S에 대해 다음 네 가지 연산을 수행하는 간단한 텍스트 편집기를 구현하시오:

- `1 W`: 문자열 W를 S의 끝에 추가한다 (append)
- `2 k`: S의 마지막 k개 문자를 삭제한다 (delete)
- `3 k`: S의 k번째 문자를 출력한다 (1-indexed)
- `4`: 마지막 연산(타입 1 또는 2)을 취소한다 (undo)

## 입력 형식

- 첫째 줄에 연산의 개수 Q가 주어진다
- 다음 Q개의 줄에 각 연산이 주어진다

## 출력 형식

- 연산 타입 3이 주어질 때마다 해당 문자를 한 줄에 출력한다

## 제약 조건

- 1 ≤ Q ≤ 10^6
- 1 ≤ |W| ≤ 10^6
- 1 ≤ k ≤ |S|
- 모든 입력 문자는 영어 소문자이다
- 모든 연산의 입력 문자 총합은 10^6을 넘지 않는다

## 예제

### 예제 입력 1

```text
8
1 abc
3 3
2 3
1 xy
3 2
4
4
3 1
```

### 예제 출력 1

```text
c
y
a
```
