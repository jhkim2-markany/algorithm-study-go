# 독성 식물

**난이도:** 중
**출처:** [Poisonous Plants](https://www.hackerrank.com/challenges/poisonous-plants)

## 문제 설명

N개의 식물이 일렬로 놓여 있다. 각 식물에는 살충제 양이 주어진다. 매일 왼쪽 이웃보다 살충제 양이 많은 식물은 죽는다. 가장 왼쪽 식물은 왼쪽 이웃이 없으므로 절대 죽지 않는다.

더 이상 식물이 죽지 않을 때까지 며칠이 걸리는지 구하시오.

## 입력 형식

- 첫째 줄에 식물의 개수 N이 주어진다
- 둘째 줄에 N개의 살충제 양이 주어진다

## 출력 형식

- 더 이상 식물이 죽지 않을 때까지의 일수를 출력한다

## 제약 조건

- 1 ≤ N ≤ 10^5
- 0 ≤ 살충제 양 ≤ 10^9

## 예제

### 예제 입력 1

```text
7
6 5 8 4 7 10 9
```

### 예제 출력 1

```text
2
```
