# 두 개의 스택으로 큐 구현

**난이도:** 하
**출처:** [Queue using Two Stacks](https://www.hackerrank.com/challenges/queue-using-two-stacks)

## 문제 설명

두 개의 스택을 사용하여 큐를 구현하시오. 다음 세 가지 쿼리를 처리해야 한다:

- `1 x`: 원소 x를 큐의 뒤에 삽입한다 (enqueue)
- `2`: 큐의 앞에서 원소를 제거한다 (dequeue)
- `3`: 큐의 앞에 있는 원소를 출력한다 (peek)

## 입력 형식

- 첫째 줄에 쿼리의 개수 Q가 주어진다
- 다음 Q개의 줄에 각 쿼리가 주어진다

## 출력 형식

- 쿼리 타입 3이 주어질 때마다 큐의 앞에 있는 원소를 한 줄에 출력한다

## 제약 조건

- 1 ≤ Q ≤ 10^6
- 1 ≤ x ≤ 10^9
- 쿼리 타입 2와 3이 호출될 때 큐는 비어있지 않다

## 예제

### 예제 입력 1

```text
10
1 42
2
1 14
3
1 28
3
1 60
1 78
2
2
```

### 예제 출력 1

```text
14
14
```
