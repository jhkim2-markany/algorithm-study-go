# 자료구조

자료구조(Data Structures)는 데이터를 효율적으로 저장하고 관리하기 위한 구조이다. 적절한 자료구조를 선택하면 데이터의 삽입, 삭제, 탐색 등의 연산을 효율적으로 수행할 수 있으며, 알고리즘의 성능을 크게 향상시킬 수 있다. 이 카테고리에서는 코딩 테스트에서 자주 출제되는 핵심 자료구조를 다룬다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
