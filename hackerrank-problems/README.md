# HackerRank 문제 풀이

HackerRank에서 선별한 자료구조와 알고리즘 문제 학습 가이드이다.
백준 실버3~골드4 수준에 해당하는 문제를 난이도별로 구성하였다.

## 구성

| 대분류 | 세부 주제 | 문제 수 |
| --- | --- | --- |
| 자료구조 | 연결 리스트, 트리, 힙, 스택/큐 | 40문제 |
| 알고리즘 | 그리디, 동적 프로그래밍, 그래프, 탐색 | 40문제 |

## 난이도 분포

- 하(Easy): 약 40%
- 중(Medium): 약 40%
- 상(Hard): 약 20%
