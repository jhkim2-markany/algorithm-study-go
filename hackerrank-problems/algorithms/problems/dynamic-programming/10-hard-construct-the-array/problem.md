# Construct the Array

**난이도:** 상
**출처:** [Construct the Array](https://www.hackerrank.com/challenges/construct-the-array)

## 문제 설명

길이 N인 배열을 구성하되, 다음 조건을 만족해야 한다:

1. 각 원소는 1 이상 K 이하의 정수이다
2. 첫 번째 원소는 1이다
3. 마지막 원소는 x이다
4. 인접한 두 원소는 서로 달라야 한다

조건을 만족하는 배열의 개수를 10^9 + 7로 나눈 나머지를 구하시오.

## 입력 형식

- 한 줄에 세 개의 정수 N, K, x가 공백으로 구분되어 주어진다

## 출력 형식

- 조건을 만족하는 배열의 개수를 10^9 + 7로 나눈 나머지를 출력한다

## 제약 조건

- 3 ≤ N ≤ 10^5
- 2 ≤ K ≤ 10^5
- 1 ≤ x ≤ K

## 예제

### 예제 입력 1

```text
4 3 2
```

### 예제 출력 1

```text
3
```
