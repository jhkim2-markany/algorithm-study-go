# Construct the Array - 해설

## 접근 방식

1. dp를 두 가지 상태로 정의한다:
   - one[i]: i번째 위치의 값이 1인 배열의 수
   - notOne[i]: i번째 위치의 값이 1이 아닌 배열의 수
2. 전이식:
   - one[i] = notOne[i-1] (이전이 1이 아닌 경우에만 현재 1 가능)
   - notOne[i] = one[i-1] × (K-1) + notOne[i-1] × (K-2)
3. 마지막 위치에서 x == 1이면 one[N], 아니면 notOne[N] / (K-1)

## 핵심 아이디어

인접 원소가 달라야 하므로, 현재 위치의 값이 1인 경우와 아닌 경우를 분리하여 추적한다. 1이 아닌 값은 K-1개 중 하나이므로, 대칭성을 이용하면 "1이 아닌 특정 값 x"의 개수는 notOne[N] / (K-1)로 구할 수 있다.

전이 근거:
- one[i]: 이전 값이 1이 아닌 모든 경우에서 현재 1을 선택
- notOne[i]: 이전이 1이면 K-1개 선택 가능, 이전이 1이 아니면 자기 자신 제외 K-2개 선택 가능

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N) |
| 공간 복잡도 | O(1) |

N번 반복하므로 O(N)이다. 이전 상태 두 개만 유지하면 되므로 O(1)이다.

## 대안적 접근

- **행렬 거듭제곱**: 전이 행렬을 사용하면 O(log N)으로 최적화할 수 있다.
- **조합론**: 포함-배제 원리를 사용하여 닫힌 형태의 공식을 유도할 수도 있다.
