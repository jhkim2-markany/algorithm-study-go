# Equal

**난이도:** 하
**출처:** [Equal](https://www.hackerrank.com/challenges/equal)

## 문제 설명

회사에 N명의 동료가 있고, 각 동료가 가진 초콜릿 수가 주어진다. 한 번의 연산으로 한 명을 제외한 나머지 모든 동료에게 1개, 2개, 또는 5개의 초콜릿을 줄 수 있다. 모든 동료가 같은 수의 초콜릿을 갖도록 하기 위한 최소 연산 횟수를 구하시오.

한 명을 제외한 나머지에게 주는 것은, 한 명에게서 빼는 것과 동일하다. 따라서 문제를 "한 명에게서 1, 2, 또는 5를 빼서 모두 같게 만들기"로 변환할 수 있다.

## 입력 형식

- 첫째 줄에 테스트 케이스의 수 T가 주어진다
- 각 테스트 케이스마다:
  - 첫째 줄에 동료의 수 N이 주어진다
  - 둘째 줄에 N개의 정수(각 동료의 초콜릿 수)가 공백으로 구분되어 주어진다

## 출력 형식

- 각 테스트 케이스마다 최소 연산 횟수를 한 줄에 출력한다

## 제약 조건

- 1 ≤ T ≤ 100
- 1 ≤ N ≤ 10000
- 0 ≤ 각 초콜릿 수 ≤ 1000

## 예제

### 예제 입력 1

```text
1
4
2 2 3 7
```

### 예제 출력 1

```text
2
```
