# Sam and Substrings

**난이도:** 중
**출처:** [Sam and substrings](https://www.hackerrank.com/challenges/sam-and-substrings)

## 문제 설명

숫자로 이루어진 문자열 n이 주어진다. 이 문자열의 모든 부분 문자열이 나타내는 수의 합을 구하시오. 결과가 매우 클 수 있으므로 10^9 + 7로 나눈 나머지를 출력한다.

예를 들어, "16"의 부분 문자열은 "1", "6", "16"이고, 합은 1 + 6 + 16 = 23이다.

## 입력 형식

- 한 줄에 숫자 문자열 n이 주어진다

## 출력 형식

- 모든 부분 문자열의 합을 10^9 + 7로 나눈 나머지를 출력한다

## 제약 조건

- 1 ≤ |n| ≤ 2 × 10^5

## 예제

### 예제 입력 1

```text
16
```

### 예제 출력 1

```text
23
```
