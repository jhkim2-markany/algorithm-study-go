# Sam and Substrings - 해설

## 접근 방식

1. 각 자릿수가 전체 합에 기여하는 값을 계산한다
2. i번째 자릿수(0-indexed)는 i+1개의 부분 문자열에서 마지막 자리로 등장한다
3. f(i)를 i번째 자릿수로 끝나는 모든 부분 문자열의 합이라 하면:
   - f(i) = f(i-1) × 10 + digit[i] × (i+1)
4. 전체 합은 f(0) + f(1) + ... + f(n-1)이다

## 핵심 아이디어

i번째 자릿수로 끝나는 부분 문자열들을 생각하면, 이전 자릿수로 끝나는 모든 부분 문자열에 현재 자릿수를 붙인 것(×10 + digit)과 현재 자릿수 하나만으로 이루어진 부분 문자열이 있다. 이를 점화식으로 표현하면 f(i) = f(i-1) × 10 + digit[i] × (i+1)이 된다.

모든 연산에서 10^9 + 7로 모듈러 연산을 수행한다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N) |
| 공간 복잡도 | O(1) |

문자열을 한 번 순회하므로 O(N)이다. 이전 값과 누적 합만 유지하면 되므로 O(1)이다.

## 대안적 접근

- **브루트포스**: 모든 부분 문자열을 생성하고 값을 계산하면 O(N²) 또는 O(N³)이 소요된다.
- **자릿수별 기여도 계산**: 각 자릿수가 전체 합에 기여하는 횟수와 자릿값을 직접 계산할 수도 있다.
