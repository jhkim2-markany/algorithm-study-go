# Sherlock and Cost - 해설

## 접근 방식

1. 각 위치에서 A[i]는 1 또는 B[i]를 선택하는 것이 최적이다
2. dp를 두 가지 상태로 정의한다:
   - low[i]: i번째 위치에서 A[i] = 1을 선택했을 때의 최대 S
   - high[i]: i번째 위치에서 A[i] = B[i]를 선택했을 때의 최대 S
3. 각 위치에서 이전 위치의 두 상태로부터 전이한다
4. max(low[N-1], high[N-1])이 최종 답이다

## 핵심 아이디어

인접 원소 간 차이를 최대화하려면, 각 원소를 가능한 한 극단적인 값(1 또는 B[i])으로 설정해야 한다. 중간값은 절대 최적이 아니다. 따라서 각 위치에서 1과 B[i] 두 가지 선택만 고려하면 된다.

전이식:
- low[i] = max(low[i-1], high[i-1] + |1 - B[i-1]|)
- high[i] = max(low[i-1] + |B[i] - 1|, high[i-1] + |B[i] - B[i-1]|)

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N) |
| 공간 복잡도 | O(1) |

각 위치를 한 번씩 순회하므로 O(N)이다. 이전 상태 두 개만 유지하면 되므로 O(1)이다.

## 대안적 접근

- **완전 탐색**: 각 위치에서 1~B[i] 중 하나를 선택하는 모든 경우를 탐색할 수 있으나, 지수적 시간이 소요된다.
- **그리디**: 단순 그리디로는 최적해를 보장할 수 없다. 이전 선택이 이후에 영향을 미치기 때문이다.
