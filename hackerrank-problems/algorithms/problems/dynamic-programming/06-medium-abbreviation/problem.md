# Abbreviation

**난이도:** 중
**출처:** [Abbreviation](https://www.hackerrank.com/challenges/abbr)

## 문제 설명

문자열 a와 b가 주어진다. 문자열 a에 대해 다음 두 가지 연산을 수행할 수 있다:

1. a의 소문자를 대문자로 변환한다
2. a의 소문자를 삭제한다

이 연산들을 적절히 수행하여 a를 b와 같게 만들 수 있는지 판별하시오.

## 입력 형식

- 첫째 줄에 쿼리의 수 q가 주어진다
- 각 쿼리마다:
  - 첫째 줄에 문자열 a가 주어진다
  - 둘째 줄에 문자열 b가 주어진다

## 출력 형식

- 각 쿼리마다 변환 가능하면 "YES", 불가능하면 "NO"를 출력한다

## 제약 조건

- 1 ≤ q ≤ 10
- 1 ≤ |a| ≤ 1000
- 1 ≤ |b| ≤ 1000
- a는 대소문자 영문자로 구성된다
- b는 대문자 영문자로만 구성된다

## 예제

### 예제 입력 1

```text
1
daBcd
ABC
```

### 예제 출력 1

```text
YES
```
