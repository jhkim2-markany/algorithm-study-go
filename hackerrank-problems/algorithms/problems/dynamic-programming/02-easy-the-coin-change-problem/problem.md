# 동전 교환 문제

**난이도:** 하
**출처:** [The Coin Change Problem](https://www.hackerrank.com/challenges/coin-change)

## 문제 설명

주어진 금액 n과 m종류의 동전이 있을 때, 동전을 사용하여 금액 n을 만들 수 있는 방법의 수를 구하시오. 각 동전은 무한히 사용할 수 있다.

## 입력 형식

- 첫째 줄에 금액 n과 동전의 종류 수 m이 공백으로 구분되어 주어진다
- 둘째 줄에 m개의 동전 액면가가 공백으로 구분되어 주어진다

## 출력 형식

- 금액 n을 만들 수 있는 방법의 수를 출력한다

## 제약 조건

- 1 ≤ n ≤ 250
- 1 ≤ m ≤ 50
- 1 ≤ 각 동전의 액면가 ≤ 50

## 예제

### 예제 입력 1

```text
4 3
1 2 3
```

### 예제 출력 1

```text
4
```
