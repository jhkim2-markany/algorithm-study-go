# 변형 피보나치

**난이도:** 하
**출처:** [Fibonacci Modified](https://www.hackerrank.com/challenges/fibonacci-modified)

## 문제 설명

피보나치 수열의 변형 버전을 구현한다. 일반 피보나치 수열에서는 다음 항이 이전 두 항의 합이지만, 이 변형에서는 다음과 같은 점화식을 따른다:

t(i+2) = t(i) + t(i+1)²

첫 번째 항 t1, 두 번째 항 t2, 그리고 구하고자 하는 항의 번호 n이 주어질 때, n번째 항의 값을 출력하시오.

## 입력 형식

- 한 줄에 공백으로 구분된 세 개의 정수 t1, t2, n이 주어진다

## 출력 형식

- n번째 항의 값을 출력한다

## 제약 조건

- 0 ≤ t1, t2 ≤ 2
- 3 ≤ n ≤ 20
- 결과값이 매우 커질 수 있으므로 큰 수 처리가 필요하다

## 예제

### 예제 입력 1

```text
0 1 5
```

### 예제 출력 1

```text
5
```
