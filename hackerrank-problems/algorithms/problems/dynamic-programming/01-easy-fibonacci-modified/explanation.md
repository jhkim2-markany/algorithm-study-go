# 변형 피보나치 - 해설

## 접근 방식

1. 첫 번째 항 t1과 두 번째 항 t2를 초기값으로 설정한다
2. 점화식 t(i+2) = t(i) + t(i+1)²을 반복 적용한다
3. n번째 항까지 계산하여 결과를 출력한다
4. 값이 매우 커질 수 있으므로 Go의 `math/big` 패키지를 사용한다

## 핵심 아이디어

일반 피보나치와 달리 제곱 연산이 포함되어 값이 기하급수적으로 증가한다. n이 최대 20이므로 반복 횟수는 적지만, 중간 결과값이 int64 범위를 초과할 수 있다. Go의 `math/big.Int`를 사용하여 임의 정밀도 정수 연산을 수행해야 한다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N) |
| 공간 복잡도 | O(1) |

N번째 항까지 한 번의 반복으로 계산하므로 시간 복잡도는 O(N)이다. 이전 두 항만 유지하면 되므로 공간 복잡도는 O(1)이다. 단, 큰 수 연산 자체의 비용은 별도이다.

## 대안적 접근

- **재귀 + 메모이제이션**: 재귀적으로 구현할 수 있으나, N이 작으므로 반복문이 더 간단하다.
- **행렬 거듭제곱**: 일반 피보나치에서는 O(log N)으로 최적화할 수 있지만, 제곱 연산이 포함된 변형에서는 적용이 어렵다.
