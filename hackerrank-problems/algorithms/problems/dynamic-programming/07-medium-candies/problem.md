# Candies

**난이도:** 중
**출처:** [Candies](https://www.hackerrank.com/challenges/candies)

## 문제 설명

선생님이 N명의 학생에게 사탕을 나누어 주려고 한다. 각 학생에게는 최소 1개의 사탕을 주어야 하며, 인접한 학생보다 성적이 높은 학생은 더 많은 사탕을 받아야 한다. 필요한 사탕의 최소 총 개수를 구하시오.

## 입력 형식

- 첫째 줄에 학생 수 N이 주어진다
- 다음 N개의 줄에 각 학생의 성적이 주어진다

## 출력 형식

- 필요한 사탕의 최소 총 개수를 출력한다

## 제약 조건

- 1 ≤ N ≤ 10^5
- 1 ≤ 각 성적 ≤ 10^5

## 예제

### 예제 입력 1

```text
3
1
2
2
```

### 예제 출력 1

```text
4
```
