# Candies - 해설

## 접근 방식

1. 모든 학생에게 1개씩 사탕을 배정한다
2. 왼쪽에서 오른쪽으로 순회하며, 왼쪽 학생보다 성적이 높으면 왼쪽 학생보다 1개 더 준다
3. 오른쪽에서 왼쪽으로 순회하며, 오른쪽 학생보다 성적이 높으면 오른쪽 학생보다 1개 더 준다
4. 각 위치에서 두 패스의 최댓값을 취한다
5. 전체 합을 구한다

## 핵심 아이디어

두 번의 선형 스캔으로 양쪽 이웃 조건을 모두 만족시킨다. 왼쪽 패스는 왼쪽 이웃 조건을, 오른쪽 패스는 오른쪽 이웃 조건을 보장한다. 각 위치에서 두 값의 최댓값을 취하면 양쪽 조건을 동시에 만족한다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N) |
| 공간 복잡도 | O(N) |

두 번의 선형 스캔으로 O(N)이다. 사탕 배열을 저장하므로 O(N) 공간이 필요하다.

## 대안적 접근

- **단일 패스**: 증가/감소 구간의 길이를 추적하여 한 번의 순회로 해결할 수 있으나, 구현이 복잡하다.
- **우선순위 큐**: 성적이 낮은 학생부터 처리하는 방법도 있으나, 정렬 비용이 추가된다.
