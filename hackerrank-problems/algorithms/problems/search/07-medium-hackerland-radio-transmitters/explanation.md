# 해커랜드 라디오 송신기 - 해설

## 접근 방식

1. 집 위치를 오름차순으로 정렬한다
2. 가장 왼쪽 집부터 시작하여 그리디하게 송신기를 배치한다
3. 현재 커버되지 않은 가장 왼쪽 집에서 K 이내인 가장 오른쪽 집에 송신기를 설치한다
4. 송신기가 커버하는 범위(설치 위치 + K) 밖의 첫 번째 집으로 이동하여 반복한다

## 핵심 아이디어

그리디 알고리즘으로 풀 수 있다. 정렬 후 왼쪽부터 순회하면서, 커버되지 않은 첫 번째 집을 기준으로 가능한 한 오른쪽에 송신기를 설치한다. 이렇게 하면 송신기 하나로 최대한 많은 집을 커버할 수 있다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N log N) |
| 공간 복잡도 | O(1) |

정렬에 O(N log N), 그리디 탐색에 O(N)이므로 전체 O(N log N)이다. 추가 공간은 상수만 사용한다.

## 대안적 접근

- **이진 탐색**: 정렬 후 각 단계에서 이진 탐색으로 송신기 설치 위치와 커버 범위 끝을 찾을 수 있다. 시간 복잡도는 동일하지만 구현이 더 복잡하다.
- **브루트포스**: 모든 가능한 송신기 배치를 시도한다. N이 작을 때만 적합하다.
