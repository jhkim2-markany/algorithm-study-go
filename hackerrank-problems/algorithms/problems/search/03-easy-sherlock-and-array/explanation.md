# 셜록과 배열 - 해설

## 접근 방식

1. 배열 전체의 합을 구한다
2. 왼쪽부터 순회하면서 왼쪽 합(leftSum)을 누적한다
3. 각 위치에서 오른쪽 합 = 전체 합 - 왼쪽 합 - 현재 원소
4. 왼쪽 합과 오른쪽 합이 같으면 "YES"를 반환한다

## 핵심 아이디어

접두사 합(prefix sum)을 활용하는 문제이다. 전체 합을 미리 구해두면, 각 위치에서 왼쪽 합과 오른쪽 합을 O(1)에 계산할 수 있다. 오른쪽 합은 `전체 합 - 왼쪽 합 - 현재 원소`로 구한다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N) |
| 공간 복잡도 | O(1) |

배열을 두 번 순회(합 계산 + 탐색)하므로 O(N)이다. 추가 공간은 상수만 사용한다.

## 대안적 접근

- **접두사 합 배열**: 접두사 합 배열을 미리 만들어 각 위치의 왼쪽/오른쪽 합을 조회할 수 있다. 시간 O(N), 공간 O(N).
- **브루트포스**: 각 위치마다 왼쪽/오른쪽 합을 직접 계산한다. 시간 O(N²), 공간 O(1).
