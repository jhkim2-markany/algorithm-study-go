# 셜록과 배열

**난이도:** 하
**출처:** [Sherlock and Array](https://www.hackerrank.com/challenges/sherlock-and-array)

## 문제 설명

배열이 주어질 때, 왼쪽 원소들의 합과 오른쪽 원소들의 합이 같은 원소가 존재하는지 판별하시오. 해당 원소 자체는 양쪽 합에 포함하지 않는다. 왼쪽이나 오른쪽에 원소가 없으면 그 쪽의 합은 0이다.

## 입력 형식

- 첫째 줄에 테스트 케이스 수 T가 주어진다
- 각 테스트 케이스마다:
  - 첫째 줄에 배열의 크기 N이 주어진다
  - 둘째 줄에 N개의 정수가 공백으로 구분되어 주어진다

## 출력 형식

- 각 테스트 케이스마다 조건을 만족하는 원소가 있으면 "YES", 없으면 "NO"를 출력한다

## 제약 조건

- 1 ≤ T ≤ 10
- 1 ≤ N ≤ 10^5
- 1 ≤ 각 원소 ≤ 2 × 10^4

## 예제

### 예제 입력 1

```text
2
3
1 2 3
4
1 2 3 3
```

### 예제 출력 1

```text
NO
YES
```
