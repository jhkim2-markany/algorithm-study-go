# 쌍 찾기

**난이도:** 하
**출처:** [Pairs](https://www.hackerrank.com/challenges/pairs)

## 문제 설명

N개의 정수로 이루어진 배열과 목표 차이값 K가 주어진다. 배열에서 차이가 정확히 K인 쌍의 개수를 구하시오. 배열의 모든 원소는 서로 다르다.

## 입력 형식

- 첫째 줄에 배열의 크기 N과 목표 차이값 K가 공백으로 구분되어 주어진다
- 둘째 줄에 N개의 정수가 공백으로 구분되어 주어진다

## 출력 형식

- 차이가 K인 쌍의 개수를 출력한다

## 제약 조건

- 2 ≤ N ≤ 10^5
- 0 < K < 10^9
- 각 원소는 서로 다르다
- 0 < 각 원소 < 2^31 - 1

## 예제

### 예제 입력 1

```text
5 2
1 5 3 4 2
```

### 예제 출력 1

```text
3
```
