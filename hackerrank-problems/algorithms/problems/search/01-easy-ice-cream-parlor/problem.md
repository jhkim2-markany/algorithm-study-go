# 아이스크림 가게

**난이도:** 하
**출처:** [Ice Cream Parlor](https://www.hackerrank.com/challenges/icecream-parlor)

## 문제 설명

아이스크림 가게에서 두 가지 맛의 아이스크림을 사려고 한다. 가게에는 여러 가지 맛의 아이스크림이 있고, 각각 가격이 다르다. 가진 돈으로 정확히 두 가지 맛을 사야 하며, 두 맛의 가격 합이 가진 돈과 같아야 한다. 각 맛은 한 번만 선택할 수 있다.

여러 번의 방문(trip)이 주어지며, 각 방문마다 가진 돈과 아이스크림 가격 목록이 주어진다. 가격 합이 가진 돈과 같은 두 아이스크림의 인덱스(1-indexed)를 오름차순으로 출력하시오.

## 입력 형식

- 첫째 줄에 방문 횟수 T가 주어진다
- 각 방문마다:
  - 첫째 줄에 가진 돈 M이 주어진다
  - 둘째 줄에 아이스크림 맛의 수 N이 주어진다
  - 셋째 줄에 N개의 아이스크림 가격이 공백으로 구분되어 주어진다

## 출력 형식

- 각 방문마다 가격 합이 M인 두 아이스크림의 1-indexed 인덱스를 공백으로 구분하여 출력한다

## 제약 조건

- 1 ≤ T ≤ 50
- 2 ≤ M ≤ 10^4
- 2 ≤ N ≤ 10^4
- 1 ≤ 각 가격 ≤ 10^4
- 항상 유일한 해가 존재한다

```text
4
4

```

### 예제 출력 1

```text
1 4
```
