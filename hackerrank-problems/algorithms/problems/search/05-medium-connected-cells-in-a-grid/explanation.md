# 격자에서 연결된 셀 - 해설

## 접근 방식

1. 격자의 모든 셀을 순회한다
2. 값이 1이고 아직 방문하지 않은 셀을 발견하면 DFS/BFS를 시작한다
3. DFS/BFS로 연결된 모든 1인 셀을 방문하며 영역 크기를 센다
4. 모든 영역 중 최대 크기를 반환한다

## 핵심 아이디어

전형적인 격자 탐색(flood fill) 문제이다. 8방향(상하좌우 + 대각선 4방향)으로 인접한 1인 셀들을 하나의 영역으로 묶는다. DFS를 사용하면 재귀적으로 연결된 셀을 모두 방문할 수 있고, 방문한 셀은 0으로 바꿔 재방문을 방지한다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N × M) |
| 공간 복잡도 | O(N × M) |

모든 셀을 한 번씩 방문하므로 O(N × M)이다. DFS 재귀 스택의 최대 깊이는 O(N × M)이다.

## 대안적 접근

- **BFS**: 큐를 사용한 BFS로도 동일하게 풀 수 있다. 재귀 깊이 제한이 있는 환경에서 유리하다.
- **Union-Find**: 인접한 1인 셀들을 합집합으로 묶고 가장 큰 집합의 크기를 구할 수 있다. 구현이 더 복잡하다.
