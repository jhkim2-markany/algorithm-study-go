# 격자에서 연결된 셀

**난이도:** 중
**출처:** [Connected Cells in a Grid](https://www.hackerrank.com/challenges/connected-cell-in-a-grid)

## 문제 설명

N×M 크기의 이진 행렬이 주어진다. 값이 1인 셀들이 상하좌우 및 대각선으로 인접하면 하나의 영역(region)을 이룬다. 가장 큰 영역의 셀 수를 구하시오.

## 입력 형식

- 첫째 줄에 행의 수 N이 주어진다
- 둘째 줄에 열의 수 M이 주어진다
- 다음 N개의 줄에 M개의 정수(0 또는 1)가 공백으로 구분되어 주어진다

## 출력 형식

- 가장 큰 영역의 셀 수를 출력한다

## 제약 조건

- 0 < N, M < 10
- 각 셀의 값은 0 또는 1

## 예제

### 예제 입력 1

```text
4
4
1 1 0 0
0 1 1 0
0 0 1 0
1 0 0 0
```

### 예제 출력 1

```text
5
```
