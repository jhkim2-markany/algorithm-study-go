# 빠진 숫자

**난이도:** 하
**출처:** [Missing Numbers](https://www.hackerrank.com/challenges/missing-numbers)

## 문제 설명

두 개의 배열이 주어진다. 두 번째 배열은 원본이고, 첫 번째 배열은 원본에서 일부 숫자가 빠진 배열이다. 원본에는 있지만 첫 번째 배열에는 빠진 숫자들을 오름차순으로 출력하시오. 같은 숫자가 여러 번 빠질 수 있으므로 빈도 차이를 비교해야 한다.

## 입력 형식

- 첫째 줄에 첫 번째 배열의 크기 N이 주어진다
- 둘째 줄에 N개의 정수가 공백으로 구분되어 주어진다
- 셋째 줄에 두 번째 배열(원본)의 크기 M이 주어진다
- 넷째 줄에 M개의 정수가 공백으로 구분되어 주어진다

## 출력 형식

- 빠진 숫자들을 오름차순으로 공백으로 구분하여 출력한다

## 제약 조건

- 1 ≤ N, M ≤ 2 × 10^5
- N ≤ M
- 각 숫자의 최댓값과 최솟값의 차이는 100 이하이다

## 예제

### 예제 입력 1

```text
10
203 204 205 206 207 208 203 204 205 206
13
203 204 204 205 206 207 205 208 203 206 205 206 204
```

### 예제 출력 1

```text
204 205 206
```
