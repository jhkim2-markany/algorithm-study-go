# 최소 손실

**난이도:** 중
**출처:** [Minimum Loss](https://www.hackerrank.com/challenges/minimum-loss)

## 문제 설명

집을 사고 나중에 되팔려고 한다. 향후 N년간의 예상 집값이 주어진다. 한 해에 집을 사고 그 이후의 해에 되팔아야 한다. 손해를 보더라도 팔아야 하지만, 손실을 최소화하고 싶다. 즉, 매입가보다 매도가가 낮을 때 그 차이(매입가 - 매도가)를 최소화하시오.

## 입력 형식

- 첫째 줄에 연도 수 N이 주어진다
- 둘째 줄에 N개의 집값이 공백으로 구분되어 주어진다 (각 값은 서로 다름)

## 출력 형식

- 최소 손실(매입가 - 매도가의 최솟값, 양수)을 출력한다

## 제약 조건

- 2 ≤ N ≤ 2 × 10^5
- 1 ≤ 각 집값 ≤ 10^16
- 모든 집값은 서로 다르다
- 손실이 발생하는 경우가 항상 존재한다

## 예제

### 예제 입력 1

```text
3
5 10 3
```

### 예제 출력 1

```text
2
```

### 예제 입력 2

```text
5
20 7 8 2 5
```

### 예제 출력 2

```text
2
```
