# 행운 세기

**난이도:** 중
**출처:** [Count Luck](https://www.hackerrank.com/challenges/count-luck)

## 문제 설명

론과 헤르미온느가 마법의 숲에서 탈출하려 한다. 숲은 N×M 격자로 표현되며, '.'은 이동 가능한 셀, 'X'는 나무(이동 불가), 'M'은 시작 위치, '*'은 포트키(목적지)이다.

헤르미온느는 갈림길(이동 가능한 방향이 2개 이상인 지점)에서 마법 지팡이를 흔든다. 시작 위치에서 포트키까지의 유일한 경로에서 지팡이를 흔든 횟수가 주어진 K와 같은지 판별하시오.

## 입력 형식

- 첫째 줄에 테스트 케이스 수 T가 주어진다
- 각 테스트 케이스마다:
  - 첫째 줄에 N과 M이 주어진다
  - 다음 N개의 줄에 격자가 주어진다
  - 마지막 줄에 정수 K가 주어진다

## 출력 형식

- 각 테스트 케이스마다 지팡이를 흔든 횟수가 K와 같으면 "Impressed", 다르면 "Oops!"를 출력한다

## 제약 조건

- 1 ≤ T ≤ 10
- 1 ≤ N, M ≤ 100
- 0 ≤ K ≤ 10000
- 시작 위치에서 포트키까지의 경로는 항상 존재한다

## 예제

### 예제 입력 1

```text
3

1

.X*.X.XXX.X
.XX.X.XM...
......XXXX.
4
```

### 예제 출력 1

```text
Impressed
Impressed
Oops!
```
