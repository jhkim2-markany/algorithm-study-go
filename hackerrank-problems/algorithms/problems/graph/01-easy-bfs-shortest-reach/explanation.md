# BFS: 최단 거리 - 해설

## 접근 방식

1. 인접 리스트로 그래프를 구성한다
2. 시작 노드에서 BFS를 수행한다
3. 각 노드까지의 거리를 간선 수 × 6으로 계산한다
4. 방문하지 못한 노드는 -1로 처리한다

## 핵심 아이디어

BFS는 비가중 그래프에서 최단 경로를 보장한다. 큐에서 노드를 꺼낼 때마다 해당 노드까지의 최단 거리가 확정된다. 모든 간선의 가중치가 6으로 동일하므로, BFS 레벨 × 6이 최단 거리가 된다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N + M) |
| 공간 복잡도 | O(N + M) |

BFS는 모든 노드와 간선을 한 번씩 방문하므로 O(N + M)이다. 인접 리스트와 방문 배열에 O(N + M) 공간이 필요하다.

## 대안적 접근

- **다익스트라 알고리즘**: 가중 그래프에서도 동작하지만, 비가중 그래프에서는 BFS가 더 효율적이다.
- **DFS**: 최단 경로를 보장하지 않으므로 이 문제에는 적합하지 않다.
