# 프림 MST: 특별한 부분 트리 - 해설

## 접근 방식

1. 시작 노드를 MST에 포함시킨다
2. MST에 포함된 노드와 연결된 간선 중 가중치가 최소인 간선을 선택한다
3. 선택한 간선으로 새 노드를 MST에 추가한다
4. 모든 노드가 MST에 포함될 때까지 반복한다

## 핵심 아이디어

프림 알고리즘은 노드 기반으로 MST를 확장한다. 우선순위 큐(최소 힙)를 사용하여 현재 MST와 연결된 간선 중 최소 가중치 간선을 효율적으로 선택한다. 이미 MST에 포함된 노드로의 간선은 무시한다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O((N + M) log N) |
| 공간 복잡도 | O(N + M) |

우선순위 큐 연산에 O(log N), 모든 간선을 처리하므로 O((N + M) log N)이다.

## 대안적 접근

- **크루스칼 알고리즘**: 간선 기반으로 MST를 구성한다. 희소 그래프에서 유리하다.
- **인접 행렬 + 단순 프림**: 우선순위 큐 없이 O(N²)에 동작한다. 밀집 그래프에서 유리하다.
