# 다익스트라: 최단 거리 2 - 해설

## 접근 방식

1. 인접 리스트로 가중 그래프를 구성한다
2. 시작 노드에서 다익스트라 알고리즘을 수행한다
3. 우선순위 큐(최소 힙)를 사용하여 최단 거리를 갱신한다
4. 도달 불가능한 노드는 -1로 처리한다

## 핵심 아이디어

다익스트라 알고리즘은 양의 가중치 그래프에서 단일 출발점 최단 경로를 구한다. 우선순위 큐에서 가장 가까운 노드를 꺼내 인접 노드의 거리를 갱신하는 과정을 반복한다. 중복 간선이 있을 수 있으므로 최소 가중치만 사용한다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O((N + M) log N) |
| 공간 복잡도 | O(N + M) |

우선순위 큐 연산에 O(log N), 모든 간선을 처리하므로 O((N + M) log N)이다.

## 대안적 접근

- **벨만-포드**: 음의 가중치도 처리 가능하지만 O(N × M)으로 느리다.
- **플로이드-워셜**: 모든 쌍 최단 경로를 구하지만 O(N³)으로 이 문제에는 과도하다.
- **BFS**: 가중치가 동일한 경우에만 사용 가능하다.
