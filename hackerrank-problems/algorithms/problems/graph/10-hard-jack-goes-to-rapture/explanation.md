# Jack의 랩처 여행 - 해설

## 접근 방식

1. 변형된 다익스트라 알고리즘을 사용한다
2. 거리 대신 "경로 상 최대 간선 가중치"를 최소화한다
3. 우선순위 큐에서 최대 가중치가 가장 작은 경로를 우선 탐색한다
4. N번 노드에 도달하면 해당 값이 답이다

## 핵심 아이디어

미니맥스 경로(Minimax Path) 문제이다. 일반 다익스트라에서 거리 합산 대신 max 연산을 사용한다. dist[v] = min(dist[v], max(dist[u], w(u,v)))로 갱신한다. 이는 MST의 성질과도 관련이 있다: MST 위의 경로가 미니맥스 경로이다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O((N + M) log N) |
| 공간 복잡도 | O(N + M) |

변형 다익스트라의 복잡도와 동일하다. 우선순위 큐 연산에 O(log N)이 소요된다.

## 대안적 접근

- **크루스칼 MST**: MST를 구성한 뒤 1번에서 N번까지의 경로에서 최대 간선을 찾는다. MST 위의 경로가 미니맥스 경로임이 보장된다.
- **이분 탐색 + BFS**: 최대 가중치를 이분 탐색하고, 해당 가중치 이하의 간선만으로 1번에서 N번까지 도달 가능한지 BFS로 확인한다.
