# 뱀과 사다리 - 해설

## 접근 방식

1. 게임판을 그래프로 모델링한다 (각 칸이 노드)
2. 사다리와 뱀 정보를 매핑 테이블에 저장한다
3. 1번 칸에서 BFS를 수행하여 100번 칸까지의 최단 거리를 구한다
4. 각 칸에서 주사위 1~6을 굴려 이동 가능한 칸으로 간선을 연결한다

## 핵심 아이디어

BFS는 비가중 그래프에서 최단 경로를 보장한다. 각 칸에서 주사위를 굴려 이동할 수 있는 6개의 칸이 인접 노드가 된다. 사다리나 뱀이 있는 칸에 도착하면 즉시 이동하므로, 해당 칸의 도착지를 실제 이동 위치로 사용한다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(100) = O(1) |
| 공간 복잡도 | O(100) = O(1) |

게임판 크기가 100으로 고정이므로 상수 시간/공간이다.

## 대안적 접근

- **다익스트라**: 가중 그래프로 모델링할 수도 있지만, 모든 간선 가중치가 1이므로 BFS가 더 효율적이다.
- **DP**: 뒤에서부터 최소 횟수를 계산하는 동적 프로그래밍 접근도 가능하다.
