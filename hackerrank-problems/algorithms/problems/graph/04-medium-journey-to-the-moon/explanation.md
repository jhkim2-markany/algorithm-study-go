# 달로의 여행 - 해설

## 접근 방식

1. Union-Find로 같은 나라 출신 우주비행사를 그룹화한다
2. 각 그룹(나라)의 크기를 구한다
3. 전체 조합 C(N, 2)에서 같은 나라 내 조합을 빼면 답이다

## 핵심 아이디어

같은 나라 관계는 전이적이므로 Union-Find로 연결 요소를 구한다. 서로 다른 나라에서 2명을 뽑는 경우의 수는 전체 조합에서 같은 나라 내 조합을 빼는 것이 효율적이다. 즉, 답 = N×(N-1)/2 - Σ(size_i × (size_i - 1) / 2)이다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N + P × α(N)) |
| 공간 복잡도 | O(N) |

Union-Find 연산은 경로 압축과 랭크 기반 합치기로 거의 O(1)이다. 전체적으로 O(N + P)에 가깝다.

## 대안적 접근

- **BFS/DFS**: 인접 리스트를 구성하고 연결 요소를 탐색할 수 있다. 시간 복잡도는 동일하지만 Union-Find가 더 간결하다.
- **누적 곱 방식**: 그룹 크기를 순회하며 이전 그룹들의 크기 합과 현재 그룹 크기를 곱하여 누적하는 방법도 있다.
