# 달로의 여행

**난이도:** 중
**출처:** [Journey to the Moon](https://www.hackerrank.com/challenges/journey-to-the-moon)

## 문제 설명

N명의 우주비행사가 있고, 같은 나라 출신인 쌍의 정보가 주어진다. 달에 보낼 2명의 우주비행사를 서로 다른 나라에서 선발해야 한다. 가능한 조합의 수를 구하시오.

## 입력 형식

- 첫째 줄에 우주비행사 수 N과 쌍의 수 P가 주어진다
- 다음 P개의 줄에 같은 나라 출신인 우주비행사 쌍 (a, b)가 주어진다

## 출력 형식

- 서로 다른 나라에서 2명을 선발하는 경우의 수를 출력한다

## 제약 조건

- 1 ≤ N ≤ 10^5
- 0 ≤ P ≤ 10^4

## 예제

### 예제 입력 1

```text
5 3
0 1
2 3
0 4
```

### 예제 출력 1

```text
6
```
