# 도로와 도서관

**난이도:** 하
**출처:** [Roads and Libraries](https://www.hackerrank.com/challenges/torque-and-development)

## 문제 설명

N개의 도시와 M개의 양방향 도로가 있다. 도서관 건설 비용은 cLib, 도로 수리 비용은 cRoad이다. 모든 도시에서 도서관에 접근할 수 있도록 하는 최소 비용을 구하시오. 도시에서 도서관에 접근하려면 해당 도시에 도서관이 있거나, 도로를 통해 도서관이 있는 도시에 도달할 수 있어야 한다.

## 입력 형식

- 첫째 줄에 쿼리의 수 Q가 주어진다
- 각 쿼리마다:
  - 첫째 줄에 도시 수 N, 도로 수 M, 도서관 비용 cLib, 도로 비용 cRoad가 주어진다
  - 다음 M개의 줄에 도로 정보 (u, v)가 주어진다

## 출력 형식

- 각 쿼리마다 최소 비용을 출력한다

## 제약 조건

- 1 ≤ Q ≤ 10
- 1 ≤ N ≤ 10^5
- 0 ≤ M ≤ min(10^5, N×(N-1)/2)
- 1 ≤ cLib, cRoad ≤ 10^5
