# 크루스칼 MST: 정말 특별한 부분 트리 - 해설

## 접근 방식

1. 모든 간선을 가중치 오름차순으로 정렬한다 (같은 가중치면 노드 합 오름차순)
2. Union-Find를 초기화한다
3. 정렬된 간선을 순회하며, 사이클을 형성하지 않는 간선을 MST에 추가한다
4. N-1개의 간선이 추가되면 종료한다

## 핵심 아이디어

크루스칼 알고리즘은 그리디 전략으로 MST를 구성한다. 가중치가 작은 간선부터 선택하되, 사이클을 형성하는 간선은 건너뛴다. Union-Find로 사이클 여부를 O(α(N))에 판별한다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(M log M) |
| 공간 복잡도 | O(N + M) |

간선 정렬에 O(M log M), Union-Find 연산에 O(M × α(N))이 소요된다. 전체적으로 정렬이 지배적이다.

## 대안적 접근

- **프림 알고리즘**: 노드 기반으로 MST를 구성한다. 밀집 그래프에서는 프림이, 희소 그래프에서는 크루스칼이 유리하다.
- **보루프카 알고리즘**: 각 연결 요소에서 최소 간선을 동시에 선택하는 방법이다.
