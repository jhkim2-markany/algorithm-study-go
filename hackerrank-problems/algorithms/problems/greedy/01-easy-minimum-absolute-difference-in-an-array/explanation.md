# 배열의 최소 절대 차이 - 해설

## 접근 방식

1. 배열을 오름차순으로 정렬한다
2. 인접한 두 원소의 차이를 계산한다
3. 모든 인접 쌍의 차이 중 최솟값을 반환한다

## 핵심 아이디어

배열을 정렬하면 최소 절대 차이는 반드시 인접한 두 원소 사이에서 발생한다. 정렬되지 않은 상태에서 모든 쌍을 비교하면 O(N²)이지만, 정렬 후 인접 쌍만 비교하면 O(N log N)으로 해결할 수 있다. 이것이 그리디 접근의 핵심이다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N log N) |
| 공간 복잡도 | O(1) |

정렬에 O(N log N), 인접 쌍 비교에 O(N)이 소요되므로 전체 시간 복잡도는 O(N log N)이다. 추가 메모리는 정렬 알고리즘에 따라 다르지만 일반적으로 O(1) 또는 O(N)이다.

## 대안적 접근

- **브루트포스**: 모든 쌍을 비교하여 최솟값을 구한다. 시간 복잡도 O(N²)으로 N이 클 때 비효율적이다.
- **해시 기반**: 원소를 해시셋에 넣고 각 원소에 대해 가까운 값을 탐색할 수 있으나, 정렬 방식이 더 간단하고 효율적이다.
