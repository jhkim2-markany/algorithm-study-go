# 배열의 최소 절대 차이

**난이도:** 하
**출처:** [Minimum Absolute Difference in an Array](https://www.hackerrank.com/challenges/minimum-absolute-difference-in-an-array)

## 문제 설명

정수 배열이 주어진다. 배열에서 서로 다른 두 원소를 골라 그 차이의 절댓값이 최소가 되는 값을 구하시오.

## 입력 형식

- 첫째 줄에 배열의 크기 N이 주어진다
- 둘째 줄에 공백으로 구분된 N개의 정수가 주어진다

## 출력 형식

- 배열에서 서로 다른 두 원소의 차이의 절댓값 중 최솟값을 출력한다

## 제약 조건

- 2 ≤ N ≤ 10^5
- -10^9 ≤ arr[i] ≤ 10^9

## 예제

### 예제 입력 1

```text
3
3 -7 0
```

### 예제 출력 1

```text
3
```
