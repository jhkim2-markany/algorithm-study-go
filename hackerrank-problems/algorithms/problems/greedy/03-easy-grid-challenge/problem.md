# 그리드 챌린지

**난이도:** 하
**출처:** [Grid Challenge](https://www.hackerrank.com/challenges/grid-challenge)

## 문제 설명

N×N 크기의 문자 그리드가 주어진다. 각 행의 문자를 재배열하여 모든 열이 알파벳 순서(비내림차순)가 되도록 할 수 있는지 판별하시오.

## 입력 형식

- 첫째 줄에 테스트 케이스의 수 T가 주어진다
- 각 테스트 케이스의 첫째 줄에 그리드의 크기 N이 주어진다
- 다음 N개의 줄에 길이 N인 문자열이 주어진다

## 출력 형식

- 각 테스트 케이스에 대해 가능하면 "YES", 불가능하면 "NO"를 출력한다

## 제약 조건

- 1 ≤ T ≤ 10
- 1 ≤ N ≤ 100
- 각 문자열은 소문자 영어 알파벳으로만 구성된다

## 예제

### 예제 입력 1

```text
1
5
ebacd
fghij
olmkn
trpqs
xywuv
```

### 예제 출력 1

```text
YES
```
