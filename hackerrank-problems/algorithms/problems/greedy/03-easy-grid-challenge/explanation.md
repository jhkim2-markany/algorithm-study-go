# 그리드 챌린지 - 해설

## 접근 방식

1. 각 행의 문자를 알파벳 순서로 정렬한다
2. 정렬된 그리드에서 각 열이 비내림차순인지 확인한다
3. 모든 열이 조건을 만족하면 "YES", 아니면 "NO"를 출력한다

## 핵심 아이디어

각 행을 독립적으로 정렬하는 것이 그리디 전략이다. 행 내에서 문자를 자유롭게 재배열할 수 있으므로, 각 행을 정렬하면 열 방향으로도 최대한 정렬된 상태가 된다. 정렬 후 열 조건을 확인하면 된다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N² log N) |
| 공간 복잡도 | O(N²) |

N개의 행을 각각 O(N log N)에 정렬하므로 O(N² log N)이다. 열 확인은 O(N²)이다.

## 대안적 접근

- **열 우선 접근**: 열별로 가능한 문자 조합을 확인하는 방법도 있으나, 행 정렬 후 열 확인이 가장 간단하고 효율적이다.
