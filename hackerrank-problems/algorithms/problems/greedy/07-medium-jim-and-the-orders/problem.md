# Jim과 주문

**난이도:** 중
**출처:** [Jim and the Orders](https://www.hackerrank.com/challenges/jim-and-the-orders)

## 문제 설명

Jim의 버거 가게에 N명의 고객이 있다. 각 고객 i는 주문 시각 order[i]와 준비 시간 prep[i]가 있다. 고객 i의 버거는 시각 order[i] + prep[i]에 완성된다. 버거가 완성되는 순서대로 고객 번호를 출력하시오. 같은 시각에 완성되면 번호가 작은 고객이 먼저 온다.

## 입력 형식

- 첫째 줄에 고객의 수 N이 주어진다
- 다음 N개의 줄에 각 고객의 주문 시각과 준비 시간이 주어진다

## 출력 형식

- 버거가 완성되는 순서대로 고객 번호(1-indexed)를 공백으로 구분하여 출력한다

## 제약 조건

- 1 ≤ N ≤ 10^3
- 1 ≤ order[i], prep[i] ≤ 10^6

## 예제

### 예제 입력 1

```text
3
1 3
2 3
3 3
```

### 예제 출력 1

```text
1 2 3
```
