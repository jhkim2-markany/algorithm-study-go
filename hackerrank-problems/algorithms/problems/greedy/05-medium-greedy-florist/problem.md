# 그리디 꽃집

**난이도:** 중
**출처:** [Greedy Florist](https://www.hackerrank.com/challenges/greedy-florist)

## 문제 설명

꽃집에 N개의 꽃이 있고, K명의 친구가 모든 꽃을 사려 한다. 꽃집 주인은 이전에 꽃을 산 적이 있는 고객에게 추가 비용을 부과한다. 어떤 사람이 이전에 p개의 꽃을 샀다면, 다음 꽃의 가격은 (p+1) × 원래가격이 된다.

K명의 친구가 모든 꽃을 구매할 때 최소 총 비용을 구하시오.

## 입력 형식

- 첫째 줄에 꽃의 수 N과 친구의 수 K가 주어진다
- 둘째 줄에 공백으로 구분된 N개의 꽃 가격이 주어진다

## 출력 형식

- 최소 총 비용을 출력한다

## 제약 조건

- 1 ≤ N ≤ 100
- 1 ≤ K ≤ N
- 1 ≤ c[i] ≤ 10^6

## 예제

### 예제 입력 1

```text
3 3
2 5 6
```

### 예제 출력 1

```text
13
```

### 예제 입력 2

```text
3 2
2 5 6
```

### 예제 출력 2

```text
15
```
