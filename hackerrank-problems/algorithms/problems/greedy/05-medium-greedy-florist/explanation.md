# 그리디 꽃집 - 해설

## 접근 방식

1. 꽃 가격을 내림차순으로 정렬한다
2. 비싼 꽃부터 K명의 친구에게 라운드 로빈 방식으로 배분한다
3. i번째 꽃의 실제 비용은 (i/K + 1) × 원래가격이다
4. 모든 꽃의 실제 비용을 합산한다

## 핵심 아이디어

배수(multiplier)는 같은 사람이 꽃을 살수록 증가한다. 따라서 비싼 꽃에 낮은 배수를 적용해야 총 비용이 최소화된다. 비싼 꽃부터 K명에게 균등하게 배분하면, 비싼 꽃은 첫 번째 라운드(배수 1)에서 처리되고, 싼 꽃은 나중 라운드(높은 배수)에서 처리된다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N log N) |
| 공간 복잡도 | O(1) |

정렬에 O(N log N), 비용 계산에 O(N)이 소요된다.

## 대안적 접근

- **시뮬레이션**: K명의 친구 각각에 대해 구매 횟수를 추적하며 시뮬레이션할 수 있다. 결과는 동일하지만 코드가 더 복잡해진다.
- **우선순위 큐**: 각 친구의 현재 배수를 최소 힙으로 관리하여 가장 낮은 배수를 가진 친구에게 꽃을 배정할 수도 있다.
