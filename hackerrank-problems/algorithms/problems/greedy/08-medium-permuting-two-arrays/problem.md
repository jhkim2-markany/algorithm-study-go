# 두 배열 순열

**난이도:** 중
**출처:** [Permuting Two Arrays](https://www.hackerrank.com/challenges/two-arrays)

## 문제 설명

길이 N인 두 정수 배열 A와 B, 그리고 정수 K가 주어진다. A와 B를 각각 재배열하여 모든 i(0 ≤ i < N)에 대해 A[i] + B[i] ≥ K가 되도록 할 수 있는지 판별하시오.

## 입력 형식

- 첫째 줄에 쿼리의 수 Q가 주어진다
- 각 쿼리의 첫째 줄에 N과 K가 주어진다
- 둘째 줄에 배열 A의 원소가 주어진다
- 셋째 줄에 배열 B의 원소가 주어진다

## 출력 형식

- 각 쿼리에 대해 가능하면 "YES", 불가능하면 "NO"를 출력한다

## 제약 조건

- 1 ≤ Q ≤ 10
- 1 ≤ N ≤ 1000
- 0 ≤ K ≤ 10^9
- 0 ≤ A[i], B[i] ≤ 10^9

## 예제

### 예제 입력 1

```text
2
3 10
2 1 3
7 8 9
4 5
1 2 2 1
3 3 3 4
```

### 예제 출력 1

```text
YES
NO
```
