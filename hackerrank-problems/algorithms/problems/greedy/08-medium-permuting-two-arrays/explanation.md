# 두 배열 순열 - 해설

## 접근 방식

1. 배열 A를 오름차순으로 정렬한다
2. 배열 B를 내림차순으로 정렬한다
3. 각 인덱스 i에 대해 A[i] + B[i] ≥ K인지 확인한다
4. 모든 쌍이 조건을 만족하면 "YES", 아니면 "NO"를 출력한다

## 핵심 아이디어

A의 가장 작은 원소는 B의 가장 큰 원소와 매칭해야 합이 K 이상이 될 가능성이 높다. 이 그리디 전략은 재배열 부등식에 의해 최적임이 보장된다. 만약 이 매칭으로 불가능하다면, 어떤 다른 매칭으로도 불가능하다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N log N) |
| 공간 복잡도 | O(1) |

두 배열의 정렬에 O(N log N), 쌍 확인에 O(N)이 소요된다.

## 대안적 접근

- **브루트포스**: A의 모든 순열을 시도하여 조건을 만족하는지 확인한다. 시간 복잡도 O(N!)으로 매우 비효율적이다.
- **이분 매칭**: 각 A[i]에 대해 B에서 K - A[i] 이상인 원소를 이분 탐색으로 찾아 매칭할 수도 있다.
