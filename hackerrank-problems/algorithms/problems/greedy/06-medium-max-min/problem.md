# Max Min

**난이도:** 중
**출처:** [Max Min](https://www.hackerrank.com/challenges/angry-children)

## 문제 설명

N개의 정수 배열에서 K개의 원소를 선택하여 부분 배열을 만든다. 이 부분 배열에서 최댓값과 최솟값의 차이를 "불공정도(unfairness)"라 한다. 불공정도가 최소가 되도록 K개의 원소를 선택하시오.

## 입력 형식

- 첫째 줄에 배열의 크기 N이 주어진다
- 둘째 줄에 선택할 원소의 수 K가 주어진다
- 다음 N개의 줄에 각 정수가 주어진다

## 출력 형식

- 최소 불공정도를 출력한다

## 제약 조건

- 2 ≤ N ≤ 10^5
- 2 ≤ K ≤ N
- 0 ≤ arr[i] ≤ 10^9

## 예제

### 예제 입력 1

```text
7
3
10
100
300
200
1000
20
30
```

### 예제 출력 1

```text
20
```
