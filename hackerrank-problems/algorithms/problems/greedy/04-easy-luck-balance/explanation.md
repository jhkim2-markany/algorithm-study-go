# 행운 균형 - 해설

## 접근 방식

1. 중요하지 않은 대회(T=0)는 모두 진다 (행운 추가)
2. 중요한 대회(T=1)의 행운 값을 내림차순으로 정렬한다
3. 상위 K개의 중요한 대회는 지고, 나머지는 이긴다
4. 총 행운 균형을 계산한다

## 핵심 아이디어

행운을 최대화하려면 가능한 한 많은 대회에서 져야 한다. 중요하지 않은 대회는 제한 없이 질 수 있으므로 모두 진다. 중요한 대회는 K번만 질 수 있으므로, 행운 값이 가장 큰 K개의 대회에서 지는 것이 최적이다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N log N) |
| 공간 복잡도 | O(N) |

중요한 대회를 정렬하는 데 O(N log N)이 소요된다. 중요한 대회 목록을 저장하는 데 O(N) 공간이 필요하다.

## 대안적 접근

- **전체 정렬**: 모든 대회를 행운 값 기준으로 정렬하고, 중요한 대회 카운트를 관리하며 처리할 수도 있다.
- **우선순위 큐**: 중요한 대회를 최대 힙에 넣고 K개를 추출하는 방법도 가능하다.
