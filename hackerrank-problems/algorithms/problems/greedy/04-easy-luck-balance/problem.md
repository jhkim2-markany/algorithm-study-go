# 행운 균형

**난이도:** 하
**출처:** [Luck Balance](https://www.hackerrank.com/challenges/luck-balance)

## 문제 설명

Lena는 프로그래밍 대회에 참가한다. 각 대회에는 행운 값 L[i]와 중요도 T[i]가 있다. T[i]=1이면 중요한 대회, T[i]=0이면 중요하지 않은 대회이다.

대회에서 지면 행운 균형에 L[i]가 더해지고, 이기면 L[i]가 빼진다. Lena는 중요한 대회에서 최대 K번까지만 질 수 있다. 중요하지 않은 대회에서는 제한 없이 질 수 있다. 최대 행운 균형을 구하시오.

## 입력 형식

- 첫째 줄에 대회의 수 N과 중요한 대회에서 질 수 있는 최대 횟수 K가 주어진다
- 다음 N개의 줄에 각 대회의 행운 값 L[i]와 중요도 T[i]가 주어진다

## 출력 형식

- 최대 행운 균형을 출력한다

## 제약 조건

- 1 ≤ N ≤ 100
- 0 ≤ K ≤ N
- 1 ≤ L[i] ≤ 10^4
- T[i] ∈ {0, 1}

## 예제

### 예제 입력 1

```text
6 3
5 1
2 1
1 1
8 1
10 0
5 0
```

### 예제 출력 1

```text
29
```
