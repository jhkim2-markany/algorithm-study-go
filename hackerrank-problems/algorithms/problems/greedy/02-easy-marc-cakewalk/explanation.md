# Marc의 케이크워크 - 해설

## 접근 방식

1. 칼로리 배열을 내림차순으로 정렬한다
2. 가장 높은 칼로리의 컵케이크를 먼저 먹는다 (i=0일 때 2^0=1)
3. 각 컵케이크에 대해 2^i × calorie[i]를 합산한다

## 핵심 아이디어

2^i는 i가 커질수록 기하급수적으로 증가한다. 따라서 칼로리가 높은 컵케이크를 먼저 먹어야(작은 i에 배치) 총 마일 수가 최소화된다. 이는 전형적인 그리디 전략으로, 큰 값에 작은 가중치를 곱하는 것이 최적이다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N log N) |
| 공간 복잡도 | O(1) |

정렬에 O(N log N), 합산에 O(N)이 소요된다. N이 최대 40이므로 매우 빠르게 동작한다.

## 대안적 접근

- **브루트포스**: 모든 순열을 시도하여 최솟값을 구한다. 시간 복잡도 O(N!)으로 N이 작을 때만 가능하다.
- **수학적 증명**: 재배열 부등식(Rearrangement Inequality)에 의해 내림차순 정렬이 최적임을 증명할 수 있다.
