# Marc의 케이크워크

**난이도:** 하
**출처:** [Marc's Cakewalk](https://www.hackerrank.com/challenges/marcs-cakewalk)

## 문제 설명

Marc는 컵케이크를 좋아한다. 그는 N개의 컵케이크를 먹으려 한다. 각 컵케이크에는 칼로리 값이 있다. Marc가 i번째로 먹는 컵케이크의 칼로리가 c라면, 그가 걸어야 하는 마일 수는 2^i × c이다 (i는 0부터 시작). 모든 컵케이크를 먹은 후 걸어야 하는 총 마일 수를 최소화하시오.

## 입력 형식

- 첫째 줄에 컵케이크의 개수 N이 주어진다
- 둘째 줄에 공백으로 구분된 N개의 칼로리 값이 주어진다

## 출력 형식

- 걸어야 하는 최소 마일 수를 출력한다

## 제약 조건

- 1 ≤ N ≤ 40
- 1 ≤ calorie[i] ≤ 10^3

## 예제

### 예제 입력 1

```text
3
1 3 2
```

### 예제 출력 1

```text
11
```
