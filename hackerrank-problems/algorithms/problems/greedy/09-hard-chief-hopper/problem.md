# Chief Hopper

**난이도:** 상
**출처:** [Chief Hopper](https://www.hackerrank.com/challenges/chief-hopper)

## 문제 설명

로봇이 N개의 건물을 순서대로 점프한다. 각 건물의 높이가 h[i]로 주어진다. 로봇은 초기 에너지(봇 에너지) E를 가지고 시작한다.

- 현재 에너지 E가 다음 건물 높이 h보다 크거나 같으면: 새 에너지 = E - (E - h) = 2E - h (에너지가 증가)
- 현재 에너지 E가 다음 건물 높이 h보다 작으면: 새 에너지 = E - (h - E) = 2E - h (에너지가 감소)

즉, 어떤 경우든 새 에너지는 2E - h이다. 모든 건물을 통과한 후 에너지가 0 이상이 되도록 하는 최소 초기 에너지 E를 구하시오.

## 입력 형식

- 첫째 줄에 건물의 수 N이 주어진다
- 둘째 줄에 공백으로 구분된 N개의 건물 높이가 주어진다

## 출력 형식

- 최소 초기 에너지를 출력한다

## 제약 조건

- 1 ≤ N ≤ 10^5
- 1 ≤ h[i] ≤ 10^5

## 예제

### 예제 입력 1

```text
5
3 4 3 2 4
```

### 예제 출력 1

```text
4
```

### 예제 입력 2

```text
3
4 4 4
```

### 예제 출력 2

```text
4
```
