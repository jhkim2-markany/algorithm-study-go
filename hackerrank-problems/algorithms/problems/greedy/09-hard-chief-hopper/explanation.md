# Chief Hopper - 해설

## 접근 방식

1. 마지막 건물부터 역순으로 탐색한다
2. 최종 에너지가 0 이상이어야 하므로, 역방향으로 필요한 최소 에너지를 계산한다
3. 각 단계에서 E_prev = ceil((E_next + h[i]) / 2)를 계산한다
4. 첫 번째 건물까지 역추적하면 최소 초기 에너지를 구할 수 있다

## 핵심 아이디어

에너지 변환 공식 E_new = 2E - h에서 역으로 E = (E_new + h) / 2이다. 마지막 건물 이후 에너지가 0 이상이면 되므로, E_final = 0에서 시작하여 역순으로 각 건물에 필요한 최소 에너지를 계산한다. 올림(ceil)을 적용하여 정수 에너지를 보장한다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N) |
| 공간 복잡도 | O(1) |

배열을 한 번 역순으로 순회하므로 O(N)이다. 추가 메모리는 상수만 사용한다.

## 대안적 접근

- **이진 탐색**: 초기 에너지를 이진 탐색하여 시뮬레이션으로 검증한다. 시간 복잡도 O(N log M)으로 역추적보다 느리다.
- **수학적 풀이**: 에너지 공식을 전개하면 E_0 = ceil(sum(h[i] / 2^i))로 표현할 수 있으나, 부동소수점 오차에 주의해야 한다.
