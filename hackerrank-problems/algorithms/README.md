# 알고리즘

알고리즘(Algorithms)은 문제를 효율적으로 해결하기 위한 절차와 방법이다. 주어진 입력에 대해 원하는 출력을 얻기 위한 단계적 과정을 정의하며, 시간과 공간의 효율성을 고려하여 최적의 해법을 찾는 것이 핵심이다. 이 카테고리에서는 코딩 테스트에서 자주 출제되는 핵심 알고리즘 유형을 다룬다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
