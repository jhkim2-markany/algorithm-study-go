# KMP 문자열 검색

**난이도:** 중

## 문제 설명

길이 N인 텍스트 문자열 T와 길이 M인 패턴 문자열 P가 주어진다. KMP 알고리즘을 사용하여 텍스트에서 패턴이 등장하는 모든 시작 위치를 찾아 출력하시오.

시작 위치는 1부터 시작하는 인덱스로 출력한다.

## 입력 형식

- 첫째 줄에 텍스트 문자열 T가 주어진다
- 둘째 줄에 패턴 문자열 P가 주어진다

## 출력 형식

- 첫째 줄에 패턴이 등장하는 횟수를 출력한다
- 둘째 줄에 패턴이 등장하는 시작 위치를 공백으로 구분하여 출력한다
- 패턴이 등장하지 않으면 0만 출력한다

## 제약 조건

- 1 ≤ |P| ≤ |T| ≤ 1,000,000
- T와 P는 영문 대문자로만 구성된다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
ABABABABCABABABABCABABABABC
ABABABABC
```

### 예제 출력 1

```
3
1 9 17
```

### 예제 입력 2

```
AABAABAAB
AABAA
```

### 예제 출력 2

```
2
1 4
```

### 예제 입력 3

```
ABCDE
FGH
```

### 예제 출력 3

```
0
```
