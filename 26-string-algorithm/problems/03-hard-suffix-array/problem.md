# 접미사 배열

**난이도:** 상

## 문제 설명

문자열 S가 주어진다. 문자열의 접미사 배열(Suffix Array)을 구성하고, 인접한 접미사 간의 LCP(Longest Common Prefix) 배열을 계산하시오.

접미사 배열은 문자열의 모든 접미사를 사전순으로 정렬했을 때, 각 접미사의 시작 인덱스를 순서대로 나열한 배열이다. LCP 배열은 접미사 배열에서 인접한 두 접미사의 최장 공통 접두사 길이를 나열한 배열이다.

시작 인덱스는 0부터 시작한다.

## 입력 형식

- 첫째 줄에 문자열 S가 주어진다

## 출력 형식

- 첫째 줄에 접미사 배열을 공백으로 구분하여 출력한다
- 둘째 줄에 LCP 배열을 공백으로 구분하여 출력한다 (첫 번째 값은 항상 0)

## 제약 조건

- 1 ≤ |S| ≤ 500,000
- S는 영문 소문자로만 구성된다
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
banana
```

### 예제 출력 1

```
5 3 1 0 4 2
0 1 3 0 0 2
```

### 예제 입력 2

```
abcabc
```

### 예제 출력 2

```
0 3 1 4 2 5
0 3 0 2 0 1
```

### 예제 입력 3

```
aaa
```

### 예제 출력 3

```
2 1 0
0 1 2
```
