# 26. 문자열 알고리즘

문자열 내에서 패턴을 효율적으로 검색하거나, 문자열의 구조적 특성을 분석하는 알고리즘이다. 브루트포스 매칭부터 KMP, 접미사 배열 등 다양한 기법을 활용하여 문자열 처리 문제를 해결한다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이

## 문제 목록

| 번호 | 문제 | 난이도 |
| --- | --- | --- |
| 01 | 문자열 패턴 매칭 | 하 |
| 02 | KMP 문자열 검색 | 중 |
| 03 | 접미사 배열 | 상 |
