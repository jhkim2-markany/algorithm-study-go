# LCA 쿼리

**난이도:** 상

## 문제 설명

N개의 노드로 이루어진 루트 트리가 주어진다. 루트는 1번 노드이다. M개의 쿼리가 주어지며, 각 쿼리는 두 노드 u, v의 최소 공통 조상(LCA, Lowest Common Ancestor)을 묻는다.

최소 공통 조상이란, 두 노드 u와 v의 공통 조상 중 가장 깊은(루트에서 가장 먼) 노드를 말한다.

## 입력 형식

- 첫째 줄에 노드의 수 N이 주어진다 (1 ≤ N ≤ 100,000)
- 다음 N-1개의 줄에 각각 두 정수 u, v가 주어진다. 이는 노드 u와 노드 v 사이에 간선이 있음을 의미한다 (1 ≤ u, v ≤ N)
- 다음 줄에 쿼리의 수 M이 주어진다 (1 ≤ M ≤ 100,000)
- 다음 M개의 줄에 각각 두 정수 a, b가 주어진다. 이는 노드 a와 노드 b의 LCA를 구하라는 쿼리이다 (1 ≤ a, b ≤ N)

## 출력 형식

- 각 쿼리에 대해 LCA를 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- 1 ≤ M ≤ 100,000
- 주어지는 그래프는 항상 트리이다
- 시간 제한: 2초
- 메모리 제한: 256MB
