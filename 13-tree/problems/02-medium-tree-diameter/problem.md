# 트리의 지름

**난이도:** 중

## 문제 설명

N개의 노드로 이루어진 트리가 주어진다. 각 간선에는 가중치(길이)가 있다. 트리의 지름을 구하시오.

트리의 지름이란, 트리에서 임의의 두 노드 사이의 거리 중 가장 긴 것을 말한다. 두 노드 사이의 거리는 경로에 포함된 간선의 가중치 합이다.

## 입력 형식

- 첫째 줄에 노드의 수 N이 주어진다 (1 ≤ N ≤ 100,000)
- 다음 N-1개의 줄에 각각 세 정수 u, v, w가 주어진다. 이는 노드 u와 노드 v 사이에 가중치 w인 간선이 있음을 의미한다 (1 ≤ u, v ≤ N, 1 ≤ w ≤ 10,000)

## 출력 형식

- 트리의 지름을 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- 1 ≤ w ≤ 10,000
- 주어지는 그래프는 항상 트리이다
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5
1 2 3
1 3 2
2 4 5
3 5 4
```

### 예제 출력 1

```
12
```

### 예제 입력 2

```
3
1 2 1
2 3 1
```

### 예제 출력 2

```
2
```
