# 트리 순회

**난이도:** 하

## 문제 설명

N개의 노드로 이루어진 루트 트리가 주어진다. 루트는 1번 노드이다. 트리의 전위 순회(Preorder) 결과를 출력하시오.

전위 순회는 현재 노드를 먼저 방문한 뒤, 자식 노드를 번호가 작은 순서대로 방문하는 방식이다.

## 입력 형식

- 첫째 줄에 노드의 수 N이 주어진다 (1 ≤ N ≤ 100,000)
- 다음 N-1개의 줄에 각각 두 정수 u, v가 주어진다. 이는 노드 u와 노드 v 사이에 간선이 있음을 의미한다 (1 ≤ u, v ≤ N)

## 출력 형식

- 전위 순회 결과를 공백으로 구분하여 한 줄에 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- 주어지는 그래프는 항상 트리이다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
7
1 2
1 3
2 4
2 5
3 6
4 7
```

### 예제 출력 1

```
1 2 4 7 5 3 6
```

### 예제 입력 2

```
1
```

### 예제 출력 2

```
1
```
