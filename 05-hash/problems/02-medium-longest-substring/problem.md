# 가장 긴 부분 문자열

**난이도:** 중

## 문제 설명

알파벳 소문자로 이루어진 문자열 S가 주어진다. S의 부분 문자열 중에서 같은 문자가 두 번 이상 등장하지 않는 가장 긴 부분 문자열의 길이를 구하시오.

부분 문자열이란 문자열에서 연속된 일부분을 말한다.

## 입력 형식

- 첫째 줄에 문자열 S가 주어진다

## 출력 형식

- 중복 문자가 없는 가장 긴 부분 문자열의 길이를 출력한다

## 제약 조건

- 1 ≤ |S| ≤ 100,000
- S는 알파벳 소문자로만 구성된다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
abcabcbb
```

### 예제 출력 1

```
3
```

### 예제 입력 2

```
bbbbb
```

### 예제 출력 2

```
1
```

### 예제 입력 3

```
pwwkew
```

### 예제 출력 3

```
3
```
