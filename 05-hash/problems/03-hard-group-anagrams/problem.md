# 아나그램 그룹화

**난이도:** 상

## 문제 설명

N개의 문자열이 주어진다. 서로 아나그램 관계인 문자열들을 같은 그룹으로 묶어 출력하시오.

아나그램이란 두 문자열이 같은 문자를 같은 개수만큼 포함하되 순서만 다른 관계를 말한다. 예를 들어 "eat"와 "tea"는 아나그램이다.

각 그룹은 사전순으로 정렬하여 출력하고, 그룹 간의 순서는 각 그룹의 첫 번째 문자열을 기준으로 사전순으로 출력한다.

## 입력 형식

- 첫째 줄에 문자열의 개수 N이 주어진다 (1 ≤ N ≤ 10,000)
- 다음 N개의 줄에 각각 하나의 문자열이 주어진다

## 출력 형식

- 각 아나그램 그룹을 한 줄에 출력한다
- 같은 그룹의 문자열은 사전순으로 정렬하여 공백으로 구분한다
- 그룹 간의 순서는 각 그룹의 첫 번째 문자열 기준 사전순이다

## 제약 조건

- 1 ≤ N ≤ 10,000
- 1 ≤ 문자열 길이 ≤ 100
- 문자열은 알파벳 소문자로만 구성된다
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
6
eat
tea
tan
ate
nat
bat
```

### 예제 출력 1

```
ate eat tea
bat
nat tan
```

### 예제 입력 2

```
3
abc
bca
def
```

### 예제 출력 2

```
abc bca
def
```
