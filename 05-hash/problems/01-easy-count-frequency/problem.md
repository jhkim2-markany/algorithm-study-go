# 빈도수 세기

**난이도:** 하

## 문제 설명

N개의 정수로 이루어진 배열이 주어진다. 배열에 포함된 각 정수의 등장 횟수를 구하여, 등장 횟수가 많은 순서대로 출력하시오. 등장 횟수가 같은 경우에는 값이 작은 순서대로 출력한다.

## 입력 형식

- 첫째 줄에 배열의 크기 N이 주어진다 (1 ≤ N ≤ 100,000)
- 둘째 줄에 N개의 정수가 공백으로 구분되어 주어진다

## 출력 형식

- 각 줄에 정수와 등장 횟수를 공백으로 구분하여 출력한다
- 등장 횟수가 많은 순서대로 출력하며, 같으면 값이 작은 순서대로 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- -1,000,000 ≤ 배열 원소 ≤ 1,000,000
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
10
1 3 2 3 1 3 2 1 1 5
```

### 예제 출력 1

```
1 4
3 3
2 2
5 1
```

### 예제 입력 2

```
5
5 5 5 5 5
```

### 예제 출력 2

```
5 5
```
