# 05. 해시

키-값 쌍을 효율적으로 저장하고 조회하는 해시 자료구조를 활용하는 알고리즘 유형이다. 해시 함수를 통해 데이터를 O(1)에 접근할 수 있으며, 빈도수 세기, 중복 검사, 그룹화 등 다양한 문제에 적용된다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이

## 문제 목록

| 번호 | 문제 | 난이도 |
| --- | --- | --- |
| 01 | 빈도수 세기 | 하 |
| 02 | 가장 긴 부분 문자열 | 중 |
| 03 | 아나그램 그룹화 | 상 |
