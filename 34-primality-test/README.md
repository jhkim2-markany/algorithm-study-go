# 34. 소수 판정

주어진 수가 소수인지 아닌지를 효율적으로 판별하는 알고리즘 유형이다. 시행 나눗셈(Trial Division), 밀러-라빈(Miller-Rabin) 소수 판정법 등이 핵심이다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
