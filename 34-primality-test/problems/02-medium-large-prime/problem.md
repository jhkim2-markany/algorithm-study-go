# 큰 소수 판정

**난이도:** 중

## 문제 설명

매우 큰 자연수 N이 주어질 때, N이 소수인지 아닌지를 판별하시오. N은 최대 10^18까지 주어질 수 있으므로, 효율적인 소수 판정 알고리즘을 사용해야 한다.

T개의 테스트 케이스가 주어지며, 각 테스트 케이스에 대해 소수이면 "YES", 아니면 "NO"를 출력한다.

## 입력 형식

- 첫째 줄에 테스트 케이스의 수 T가 주어진다
- 다음 T개의 줄에 각각 자연수 N이 주어진다

## 출력 형식

- 각 테스트 케이스에 대해 N이 소수이면 "YES", 아니면 "NO"를 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ T ≤ 100
- 2 ≤ N ≤ 10^18
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
4
7
1000000007
1000000006
999999999999999989
```

### 예제 출력 1

```
YES
YES
NO
YES
```
