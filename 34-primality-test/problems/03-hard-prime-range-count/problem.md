# 소수 구간 카운팅

**난이도:** 상

## 문제 설명

두 정수 L과 R이 주어질 때, L 이상 R 이하의 범위에 포함된 소수의 개수를 구하시오. L과 R은 매우 클 수 있지만, 구간의 길이 R - L은 비교적 작다.

## 입력 형식

- 첫째 줄에 두 정수 L과 R이 공백으로 구분되어 주어진다

## 출력 형식

- L 이상 R 이하의 소수의 개수를 출력한다

## 제약 조건

- 1 ≤ L ≤ R ≤ 10^12
- R - L ≤ 1,000,000
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
2 10
```

### 예제 출력 1

```
4
```

### 예제 입력 2

```
100 120
```

### 예제 출력 2

```
5
```
