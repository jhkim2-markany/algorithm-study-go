# 소수 구간 카운팅 - 해설

## 접근 방식

1. R이 최대 10^12이므로 전체 범위에 에라토스테네스의 체를 적용할 수 없다
2. 구간 체(Segmented Sieve) 기법을 사용한다
3. 먼저 √R 이하의 소수를 일반 에라토스테네스의 체로 구한다
4. 구한 소수들의 배수를 [L, R] 구간에서 제거한다
5. 남은 수가 소수이므로 개수를 센다

## 핵심 아이디어

구간 체는 에라토스테네스의 체를 특정 구간에 적용하는 기법이다. 핵심 관찰은 다음과 같다:

N 이하의 합성수는 반드시 √N 이하의 소인수를 가진다. 따라서 √R 이하의 소수 목록만 있으면 [L, R] 구간의 모든 합성수를 걸러낼 수 있다.

구간 [L, R]에 대해 크기 (R - L + 1)의 불리언 배열을 만들고, 각 작은 소수 p에 대해 구간 내 p의 배수를 합성수로 표시한다. p의 배수 중 L 이상인 가장 작은 수는 ⌈L/p⌉ × p로 구할 수 있다.

주의할 점은 p 자체가 구간에 포함될 수 있으므로, p 자체는 제거하지 않아야 한다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(√R × log log √R + (R-L) × log log R) |
| 공간 복잡도 | O(√R + (R-L)) |

√R 이하의 소수를 구하는 데 O(√R × log log √R), 구간 체에서 각 소수의 배수를 제거하는 데 O((R-L) × log log R)이 소요된다. R ≤ 10^12이므로 √R ≈ 10^6이고, R - L ≤ 10^6이므로 시간과 메모리 모두 충분하다.

## 대안적 접근

- 각 수에 대해 밀러-라빈 테스트를 개별적으로 수행할 수도 있으나, 구간 길이가 최대 10^6이므로 구간 체가 더 효율적이다.
- L과 R이 작은 경우(10^7 이하)에는 일반 에라토스테네스의 체로 충분하다.
