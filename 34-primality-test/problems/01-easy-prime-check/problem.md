# 소수 판별

**난이도:** 하

## 문제 설명

자연수 N이 주어질 때, N이 소수인지 아닌지를 판별하시오. 소수란 1과 자기 자신만을 약수로 가지는 1보다 큰 자연수이다.

T개의 테스트 케이스가 주어지며, 각 테스트 케이스에 대해 소수이면 "YES", 아니면 "NO"를 출력한다.

## 입력 형식

- 첫째 줄에 테스트 케이스의 수 T가 주어진다
- 다음 T개의 줄에 각각 자연수 N이 주어진다

## 출력 형식

- 각 테스트 케이스에 대해 N이 소수이면 "YES", 아니면 "NO"를 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ T ≤ 1,000
- 1 ≤ N ≤ 1,000,000
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5
1
2
7
15
97
```

### 예제 출력 1

```
NO
YES
YES
NO
YES
```
