# 음수 사이클이 있는 최단 경로 - 해설

## 접근 방식

1. N×N 거리 행렬을 초기화하고 간선을 입력받는다.
2. 플로이드-워셜 3중 루프를 수행하여 모든 쌍 최단 거리를 구한다.
3. 음수 사이클에 속하는 정점을 식별한다: dist[k][k] < 0인 정점 k는 음수 사이클에 포함된다.
4. 음수 사이클의 영향을 전파한다: 정점 i에서 음수 사이클 정점 k를 거쳐 정점 j에 도달할 수 있으면, (i, j) 쌍의 최단 거리는 -∞이다.
5. 각 질의에 대해 도달 불가능, 음수 사이클 영향, 일반 최단 거리를 구분하여 출력한다.

## 핵심 아이디어

플로이드-워셜 수행 후 dist[k][k] < 0이면 정점 k는 음수 사이클에 포함된다. 이때 i에서 k로 도달 가능하고 k에서 j로 도달 가능하면, i에서 j로 가는 경로에서 음수 사이클을 무한히 반복하여 비용을 -∞로 만들 수 있다.

이를 판별하기 위해 플로이드-워셜 수행 후 추가 루프를 돌며, 음수 사이클 정점을 경유하는 모든 (i, j) 쌍을 -∞로 표시한다. 도달 가능성은 dist[i][k] < INF && dist[k][j] < INF로 확인한다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N³) |
| 공간 복잡도 | O(N²) |

플로이드-워셜 O(N³)에 음수 사이클 전파 O(N³)이 추가되어 총 O(N³)이다. N ≤ 200이므로 충분히 빠르다.

## 대안적 접근

벨만-포드 알고리즘을 각 정점에서 실행하여 음수 사이클을 탐지할 수도 있다. 하지만 모든 쌍에 대한 정보가 필요하므로 플로이드-워셜이 더 적합하다.
