# 음수 사이클이 있는 최단 경로

**난이도:** 상

## 문제 설명

N개의 도시와 M개의 단방향 도로가 있다. 각 도로에는 이동 비용이 주어지며, 비용이 음수인 도로도 존재한다. 음수 사이클이 존재할 수 있다.

Q개의 질의가 주어지며, 각 질의는 출발 도시 s와 도착 도시 e로 구성된다. 각 질의에 대해 다음을 출력하라:

- s에서 e로 도달할 수 없으면 -1을 출력한다
- s에서 e로 가는 경로 중 음수 사이클을 포함하여 비용을 무한히 줄일 수 있으면 -2를 출력한다
- 그 외의 경우 최소 비용을 출력한다

## 입력 형식

- 첫째 줄에 도시의 수 N과 도로의 수 M이 주어진다 (1 ≤ N ≤ 200, 0 ≤ M ≤ N×(N-1))
- 다음 M줄에 각 도로의 출발 도시 u, 도착 도시 v, 비용 w가 주어진다 (1 ≤ u, v ≤ N, -10,000 ≤ w ≤ 10,000)
- 다음 줄에 질의의 수 Q가 주어진다 (1 ≤ Q ≤ 10,000)
- 다음 Q줄에 각 질의의 출발 도시 s와 도착 도시 e가 주어진다 (1 ≤ s, e ≤ N)

## 출력 형식

- 각 질의에 대해 한 줄에 결과를 출력한다
- 도달 불가능: -1
- 음수 사이클로 무한히 줄일 수 있음: -2
- 그 외: 최소 비용

## 제약 조건

- 1 ≤ N ≤ 200
- 0 ≤ M ≤ N×(N-1)
- -10,000 ≤ w ≤ 10,000
- 1 ≤ Q ≤ 10,000
- 시간 제한: 1초
