# 모든 쌍 최단 거리

**난이도:** 하

## 문제 설명

N개의 도시와 M개의 단방향 도로가 있다. 각 도로에는 이동 비용이 주어진다. 모든 도시 쌍 (i, j)에 대해 도시 i에서 도시 j로 이동하는 최소 비용을 구하라.

도달할 수 없는 경우 0을 출력한다.

## 입력 형식

- 첫째 줄에 도시의 수 N과 도로의 수 M이 주어진다 (1 ≤ N ≤ 100, 0 ≤ M ≤ N×(N-1))
- 다음 M줄에 각 도로의 출발 도시 u, 도착 도시 v, 비용 w가 주어진다 (1 ≤ u, v ≤ N, 1 ≤ w ≤ 100,000)
- 같은 출발-도착 쌍에 여러 도로가 있을 수 있으며, 이 경우 비용이 가장 작은 도로만 고려한다

## 출력 형식

- N줄에 걸쳐 N×N 행렬을 출력한다
- i번째 줄의 j번째 값은 도시 i에서 도시 j로의 최소 비용이다
- 도달할 수 없으면 0을 출력한다
- 자기 자신까지의 비용은 0이다

## 제약 조건

- 1 ≤ N ≤ 100
- 0 ≤ M ≤ N×(N-1)
- 1 ≤ w ≤ 100,000
- 시간 제한: 1초
