# 경로 복원

**난이도:** 중

## 문제 설명

N개의 도시와 M개의 단방향 도로가 있다. 각 도로에는 이동 비용이 주어진다. Q개의 질의가 주어지며, 각 질의는 출발 도시 s와 도착 도시 e로 구성된다.

각 질의에 대해 s에서 e까지의 최소 비용과 실제 경로를 출력하라. 도달할 수 없는 경우 -1을 출력한다.

## 입력 형식

- 첫째 줄에 도시의 수 N과 도로의 수 M이 주어진다 (1 ≤ N ≤ 200, 0 ≤ M ≤ N×(N-1))
- 다음 M줄에 각 도로의 출발 도시 u, 도착 도시 v, 비용 w가 주어진다 (1 ≤ u, v ≤ N, 1 ≤ w ≤ 10,000)
- 다음 줄에 질의의 수 Q가 주어진다 (1 ≤ Q ≤ 1,000)
- 다음 Q줄에 각 질의의 출발 도시 s와 도착 도시 e가 주어진다 (1 ≤ s, e ≤ N, s ≠ e)

## 출력 형식

- 각 질의에 대해 두 줄을 출력한다
- 첫째 줄: 최소 비용 (도달 불가능하면 -1)
- 둘째 줄: 경로에 포함된 도시의 수와 경로를 순서대로 출력 (도달 불가능하면 0)

## 제약 조건

- 1 ≤ N ≤ 200
- 0 ≤ M ≤ N×(N-1)
- 1 ≤ w ≤ 10,000
- 1 ≤ Q ≤ 1,000
- 시간 제한: 1초
