# 50. 플로이드-워셜 (Floyd-Warshall)

그래프에서 모든 정점 쌍 사이의 최단 경로를 구하는 동적 프로그래밍 기반 알고리즘이다. 3중 루프로 구현하며, 음수 가중치 처리, 경로 복원, 도달 가능성 판별 등에 활용된다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이

## 문제 목록

| 번호 | 문제 | 난이도 |
| --- | --- | --- |
| 01 | 모든 쌍 최단 거리 | 하 |
| 02 | 경로 복원 | 중 |
| 03 | 음수 사이클이 있는 최단 경로 | 상 |
