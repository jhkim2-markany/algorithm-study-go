# 17. 백트래킹

가능한 모든 경우를 탐색하되, 조건에 맞지 않는 경우를 조기에 가지치기(pruning)하여 탐색 공간을 줄이는 알고리즘이다. 재귀를 기반으로 상태 공간 트리를 탐색하며, 순열, 조합, 부분집합 생성 및 제약 조건 만족 문제(CSP) 등에 활용된다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
