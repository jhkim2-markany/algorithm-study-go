# N과 M (순열)

**난이도:** 하

## 문제 설명

자연수 N과 M이 주어진다. 1부터 N까지의 자연수 중에서 중복 없이 M개를 고른 수열을 모두 구하시오.

수열은 사전 순으로 출력해야 한다.

## 입력 형식

- 첫째 줄에 자연수 N과 M이 주어진다 (1 ≤ M ≤ N ≤ 8)

## 출력 형식

- 한 줄에 하나씩 문제의 조건을 만족하는 수열을 출력한다
- 각 수열의 수는 공백으로 구분한다

## 제약 조건

- 1 ≤ M ≤ N ≤ 8
- 중복되는 수열은 출력하지 않는다
- 수열은 사전 순으로 출력한다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
3 1
```

### 예제 출력 1

```
1
2
3
```

### 예제 입력 2

```
4 2
```

### 예제 출력 2

```
1 2
1 3
1 4
2 1
2 3
2 4
3 1
3 2
3 4
4 1
4 2
4 3
```
