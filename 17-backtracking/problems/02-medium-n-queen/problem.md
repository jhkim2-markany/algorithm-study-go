# N-Queen

**난이도:** 중

## 문제 설명

N×N 크기의 체스판 위에 N개의 퀸을 서로 공격할 수 없도록 배치하는 경우의 수를 구하시오.

퀸은 같은 행, 같은 열, 같은 대각선에 있는 다른 퀸을 공격할 수 있다. 따라서 N개의 퀸이 어떤 두 퀸도 같은 행, 같은 열, 같은 대각선에 놓이지 않도록 배치해야 한다.

## 입력 형식

- 첫째 줄에 N이 주어진다 (1 ≤ N ≤ 15)

## 출력 형식

- N개의 퀸을 서로 공격할 수 없게 배치하는 경우의 수를 출력한다

## 제약 조건

- 1 ≤ N ≤ 15
- 시간 제한: 10초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
8
```

### 예제 출력 1

```
92
```

### 예제 입력 2

```
1
```

### 예제 출력 2

```
1
```
