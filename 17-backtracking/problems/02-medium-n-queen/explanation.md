# N-Queen - 해설

## 접근 방식

1. 각 행에 퀸을 하나씩 배치하는 방식으로 탐색한다 (행 충돌 자동 방지).
2. 0번 행부터 시작하여 각 행에서 0~N-1열을 순서대로 시도한다.
3. 현재 위치에 퀸을 놓을 수 있는지 열 충돌과 대각선 충돌을 검사한다.
4. 충돌이 없으면 퀸을 배치하고 다음 행으로 재귀 호출한다.
5. 충돌이 있으면 해당 열을 건너뛴다 (가지치기).
6. N번째 행까지 모두 배치하면 경우의 수를 1 증가시킨다.

## 핵심 아이디어

N-Queen 문제의 핵심은 효율적인 충돌 검사이다. 각 행에 퀸을 하나씩 배치하므로 행 충돌은 자동으로 방지된다. 열 충돌과 대각선 충돌을 O(1)에 검사하기 위해 세 개의 불리언 배열을 사용한다:

- `col[c]`: c열에 퀸이 있는지 여부
- `diag1[r+c]`: 우상향 대각선(↗)에 퀸이 있는지 여부. 같은 대각선의 칸은 r+c 값이 동일하다.
- `diag2[r-c+n-1]`: 좌상향 대각선(↖)에 퀸이 있는지 여부. 같은 대각선의 칸은 r-c 값이 동일하다. 음수 인덱스를 방지하기 위해 n-1을 더한다.

이 세 배열을 통해 퀸 배치 가능 여부를 O(1)에 판단할 수 있어, 매번 이전에 배치한 모든 퀸과 비교하는 O(N) 방식보다 효율적이다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N!) |
| 공간 복잡도 | O(N) |

최악의 경우 각 행에서 N개의 열을 시도하므로 O(N^N)이지만, 가지치기에 의해 실제로는 O(N!)에 가깝다. 열과 대각선 배열에 O(N), 재귀 깊이에 O(N) 공간이 필요하다.

## 대안적 접근

- 비트마스크를 사용하면 열과 대각선 충돌 검사를 비트 연산으로 더 빠르게 수행할 수 있다.
- 대칭성을 활용하여 탐색 공간을 절반으로 줄일 수 있다 (첫 행의 절반만 탐색 후 2배).
