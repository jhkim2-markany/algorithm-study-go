# 스도쿠 풀기

**난이도:** 상

## 문제 설명

9×9 크기의 스도쿠 퍼즐이 주어진다. 빈 칸(0으로 표시)을 채워서 스도쿠를 완성하시오.

스도쿠의 규칙은 다음과 같다:
- 각 행에 1부터 9까지의 숫자가 중복 없이 들어가야 한다
- 각 열에 1부터 9까지의 숫자가 중복 없이 들어가야 한다
- 각 3×3 박스에 1부터 9까지의 숫자가 중복 없이 들어가야 한다

## 입력 형식

- 9개의 줄에 걸쳐 9개의 정수가 공백으로 구분되어 주어진다
- 빈 칸은 0으로 표시된다

## 출력 형식

- 완성된 스도쿠를 9개의 줄에 걸쳐 출력한다
- 각 줄에 9개의 숫자를 공백으로 구분하여 출력한다

## 제약 조건

- 입력은 유효한 스도쿠 퍼즐이다 (해가 반드시 존재한다)
- 해는 유일하다
- 빈 칸의 수는 최대 60개이다
- 시간 제한: 2초
- 메모리 제한: 256MB
