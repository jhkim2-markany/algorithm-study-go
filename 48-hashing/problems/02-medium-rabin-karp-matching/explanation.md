# 라빈-카프 패턴 매칭 - 해설

## 접근 방식

1. 텍스트의 접두사 해시 배열과 거듭제곱 배열을 전처리한다
2. 패턴의 해시 값을 계산한다
3. 텍스트의 각 위치에서 길이 M인 부분 문자열의 해시를 O(1)에 구한다
4. 이중 해시가 패턴의 해시와 모두 일치하면 매칭으로 판정한다

## 핵심 아이디어

라빈-카프 알고리즘의 핵심은 롤링 해시(접두사 해시)를 이용하여 부분 문자열의 해시를 O(1)에 구하는 것이다. 접두사 해시 H[i]와 거듭제곱 P[i]를 전처리하면, 부분 문자열 S[l..r]의 해시를 다음과 같이 구할 수 있다:

```
hash(S[l..r]) = (H[r+1] - H[l] × P[r-l+1]) mod M
```

이중 해싱을 사용하면 두 해시 값이 동시에 충돌할 확률이 약 1/(M₁×M₂) ≈ 10⁻¹⁸으로 극히 낮아, 실제 문자열 비교 없이도 정확한 매칭이 가능하다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N + M) |
| 공간 복잡도 | O(N) |

N은 텍스트 길이, M은 패턴 길이이다. 전처리 O(N), 패턴 해시 O(M), 각 위치 비교 O(1)이므로 총 O(N+M)이다.

## 대안적 접근

- KMP 알고리즘: 실패 함수를 이용하여 O(N+M)에 패턴 매칭을 수행한다. 해시 충돌 걱정이 없다.
- 나이브 방법: 모든 위치에서 직접 비교하면 O(NM)으로 문자열이 길 때 시간 초과가 발생한다.
- 아호-코라식: 여러 패턴을 동시에 검색할 때 유리하다.
