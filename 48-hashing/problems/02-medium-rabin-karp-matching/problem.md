# 라빈-카프 패턴 매칭

**난이도:** 중

## 문제 설명

텍스트 문자열 T와 패턴 문자열 P가 주어진다. 라빈-카프 알고리즘을 이용하여 텍스트에서 패턴이 출현하는 모든 위치를 찾으시오.

출현 위치는 1-indexed로 출력한다.

## 입력 형식

- 첫째 줄에 텍스트 문자열 T가 주어진다
- 둘째 줄에 패턴 문자열 P가 주어진다
- 두 문자열은 소문자 알파벳으로만 구성된다

## 출력 형식

- 첫째 줄에 패턴이 출현하는 횟수를 출력한다
- 둘째 줄에 출현하는 위치를 공백으로 구분하여 출력한다 (1-indexed)
- 패턴이 출현하지 않으면 0만 출력한다

## 제약 조건

- 1 ≤ |P| ≤ |T| ≤ 1,000,000
- 문자열은 소문자 알파벳으로만 구성된다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
ababababab
abab
```

### 예제 출력 1

```
4
1 3 5 7
```

### 예제 입력 2

```
abcdef
xyz
```

### 예제 출력 2

```
0
```

### 예제 입력 3

```
aaaaaaa
aaa
```

### 예제 출력 3

```
5
1 2 3 4 5
```
