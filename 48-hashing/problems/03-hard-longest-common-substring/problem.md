# 가장 긴 공통 부분 문자열

**난이도:** 상

## 문제 설명

두 문자열 A와 B가 주어진다. 두 문자열에 공통으로 포함된 부분 문자열 중 가장 긴 것의 길이를 구하시오.

부분 문자열이란 문자열에서 연속된 일부분을 의미한다. 예를 들어 "abc"의 부분 문자열은 "a", "b", "c", "ab", "bc", "abc"이다.

## 입력 형식

- 첫째 줄에 문자열 A가 주어진다
- 둘째 줄에 문자열 B가 주어진다
- 두 문자열은 소문자 알파벳으로만 구성된다

## 출력 형식

- 가장 긴 공통 부분 문자열의 길이를 출력한다
- 공통 부분 문자열이 없으면 0을 출력한다

## 제약 조건

- 1 ≤ |A|, |B| ≤ 200,000
- 문자열은 소문자 알파벳으로만 구성된다
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
abcdefg
bcdexy
```

### 예제 출력 1

```
4
```

### 예제 입력 2

```
abcabc
xyzxyz
```

### 예제 출력 2

```
0
```

### 예제 입력 3

```
aababc
babcba
```

### 예제 출력 3

```
4
```
