# 문자열 해시 비교

**난이도:** 하

## 문제 설명

N개의 문자열이 주어진다. 이 중에서 서로 같은 문자열 쌍의 개수를 구하시오. 두 문자열이 같다는 것은 길이가 같고 모든 위치의 문자가 동일하다는 뜻이다.

다항식 해싱을 이용하여 문자열 비교를 효율적으로 수행하시오.

## 입력 형식

- 첫째 줄에 문자열의 개수 N이 주어진다
- 다음 N개의 줄에 각 문자열이 주어진다 (소문자 알파벳으로만 구성)

## 출력 형식

- 서로 같은 문자열 쌍의 개수를 출력한다
- 쌍 (i, j)에서 i < j인 경우만 센다

## 제약 조건

- 1 ≤ N ≤ 100,000
- 각 문자열의 길이는 1 이상 100 이하
- 문자열은 소문자 알파벳으로만 구성된다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5
abc
def
abc
ghi
abc
```

### 예제 출력 1

```
3
```

### 예제 입력 2

```
4
aaa
aaa
aaa
aaa
```

### 예제 출력 2

```
6
```
