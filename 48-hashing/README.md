# 48. 해싱 (Hashing)

문자열이나 데이터를 고정 크기의 정수 값으로 변환하는 기법이다. 다항식 해싱, 라빈-카프 알고리즘, 이중 해싱 등을 활용하여 문자열 비교와 패턴 매칭을 효율적으로 수행한다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이

## 문제 목록

| 번호 | 문제 | 난이도 |
| --- | --- | --- |
| 01 | 문자열 해시 비교 | 하 |
| 02 | 라빈-카프 패턴 매칭 | 중 |
| 03 | 가장 긴 공통 부분 문자열 | 상 |
