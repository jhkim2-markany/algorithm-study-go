## 학습 방법

각 Day 폴더의 README.md를 열고 순서대로 진행:

1. **핵심 개념** 읽기 (5분)
2. 기존 알고리즘 폴더의 **theory.md** 정독 (15분)
3. **examples/** 코드 읽고 실행 (10분)
4. **코드 템플릿** 빈 파일에 직접 타이핑 (10분)
5. **problems/** easy → medium 순서로 풀기 (60분)
6. 막히면 **explanation.md** 확인 후 다시 풀기
