# 행렬 거듭제곱 피보나치 - 해설

## 접근 방식

1. N이 최대 10^18이므로 단순 반복(O(N))이나 메모이제이션으로는 풀 수 없다
2. 피보나치 점화식을 행렬 곱셈으로 변환한다
3. 전이 행렬 {{1,1},{1,0}}을 (N-1)번 거듭제곱하면 F(N)을 구할 수 있다
4. 행렬 거듭제곱에 분할 정복을 적용하여 O(log N)에 계산한다

## 핵심 아이디어

피보나치 수열의 점화식 F(n) = F(n-1) + F(n-2)를 행렬로 표현하면:

```
| F(n+1) |   | 1 1 |   | F(n)   |
| F(n)   | = | 1 0 | × | F(n-1) |
```

이를 반복 적용하면:

```
| F(n+1) |   | 1 1 |^n   | F(1) |   | 1 1 |^n   | 1 |
| F(n)   | = | 1 0 |   × | F(0) | = | 1 0 |   × | 0 |
```

따라서 행렬 {{1,1},{1,0}}을 n번 거듭제곱한 결과의 [0][0] 위치가 F(n+1), [1][0] 위치가 F(n)이 된다. 행렬 거듭제곱에 분할 정복을 적용하면 2×2 행렬 곱셈을 O(log N)번만 수행하면 된다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(log N) |
| 공간 복잡도 | O(1) |

2×2 행렬 곱셈은 상수 시간(8번의 곱셈과 4번의 덧셈)이므로, 행렬 거듭제곱의 전체 시간 복잡도는 O(log N)이다. 행렬 크기가 고정이므로 공간 복잡도는 O(1)이다.

## 대안적 접근

- 피보나치 항등식 활용: F(2k) = F(k) × (2F(k+1) - F(k)), F(2k+1) = F(k+1)² + F(k)²를 이용한 분할 정복도 O(log N)에 가능하다. 행렬 곱셈 없이 구현할 수 있어 상수가 더 작다.
