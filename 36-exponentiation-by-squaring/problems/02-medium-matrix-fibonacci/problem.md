# 행렬 거듭제곱 피보나치

**난이도:** 중

## 문제 설명

피보나치 수열은 다음과 같이 정의된다:
- F(0) = 0
- F(1) = 1
- F(n) = F(n-1) + F(n-2) (n ≥ 2)

정수 N이 주어질 때, F(N) mod 1,000,000,007의 값을 구하시오. N이 매우 클 수 있으므로 행렬 거듭제곱을 사용하여 효율적으로 계산해야 한다.

## 입력 형식

- 첫째 줄에 정수 N이 주어진다

## 출력 형식

- F(N) mod 1,000,000,007의 값을 출력한다

## 제약 조건

- 0 ≤ N ≤ 1,000,000,000,000,000,000 (10^18)
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
10
```

### 예제 출력 1

```
55
```

### 예제 입력 2

```
1000000000000000000
```

### 예제 출력 2

```
209783453
```
