# 모듈러 거듭제곱 계산

**난이도:** 하

## 문제 설명

정수 A, B, M이 주어진다. A^B mod M의 값을 구하시오. 단, B가 매우 클 수 있으므로 단순 반복으로는 시간 내에 풀 수 없다. 분할 정복 거듭제곱을 사용하여 효율적으로 계산하시오.

## 입력 형식

- 첫째 줄에 테스트 케이스의 수 T가 주어진다
- 다음 T개의 줄에 각각 세 정수 A, B, M이 공백으로 구분되어 주어진다

## 출력 형식

- 각 테스트 케이스에 대해 A^B mod M의 값을 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ T ≤ 100
- 1 ≤ A ≤ 1,000,000,000
- 0 ≤ B ≤ 1,000,000,000
- 2 ≤ M ≤ 1,000,000,000
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
3
2 10 1000
3 0 7
7 256 13
```

### 예제 출력 1

```
24
1
9
```
