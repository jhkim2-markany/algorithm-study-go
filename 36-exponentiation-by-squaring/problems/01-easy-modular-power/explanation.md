# 모듈러 거듭제곱 계산 - 해설

## 접근 방식

1. A^B mod M을 구해야 하는데, B가 최대 10^9이므로 단순 반복은 시간 초과가 발생한다
2. 분할 정복 거듭제곱을 사용하여 지수를 반씩 줄여 나간다
3. 지수가 홀수이면 결과에 밑을 한 번 곱하고, 밑을 제곱한 뒤 지수를 반으로 나눈다
4. 매 곱셈마다 mod M을 취하여 오버플로를 방지한다

## 핵심 아이디어

분할 정복 거듭제곱의 핵심은 지수의 이진 표현을 활용하는 것이다. 예를 들어 a^13에서 13 = 1101₂이므로:

- a^13 = a^8 × a^4 × a^1

지수를 이진수로 보고, 비트가 1인 자리에 해당하는 거듭제곱만 결과에 곱하면 된다. 밑을 계속 제곱해 나가면서(a → a² → a⁴ → a⁸) 현재 비트가 1이면 결과에 곱한다.

모듈러 연산의 성질 (a × b) mod m = ((a mod m) × (b mod m)) mod m을 이용하여 중간 결과가 항상 m 미만으로 유지된다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(T × log B) |
| 공간 복잡도 | O(1) |

각 테스트 케이스마다 지수 B를 반씩 줄이므로 O(log B)이고, T개의 테스트 케이스를 처리하므로 전체 O(T × log B)이다.
