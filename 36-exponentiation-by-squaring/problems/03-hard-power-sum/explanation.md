# 거듭제곱 합 구하기 - 해설

## 접근 방식

1. S(N) = A^1 + A^2 + ... + A^N을 구해야 하는데, N이 최대 10^18이므로 단순 반복은 불가능하다
2. 점화식을 세운다: S(n) = A × S(n-1) + A^n
3. 이 점화식을 행렬 곱셈으로 변환한다
4. 행렬 거듭제곱으로 O(log N)에 계산한다

## 핵심 아이디어

S(n) = A^1 + A^2 + ... + A^n이라 하면, S(n) = S(n-1) + A^n이 성립한다. A^n = A × A^(n-1)이므로, 이를 행렬로 표현하면:

```
| S(n) |   | 1  A |   | S(n-1)   |
| A^n  | = | 0  A | × | A^(n-1)  |
```

초기 상태 벡터는 [S(0), A^0] = [0, 1]이다. 전이 행렬 T = {{1, A}, {0, A}}를 n번 거듭제곱한 뒤 초기 벡터를 곱하면:

```
| S(n) |       | 0 |
| A^n  | = T^n | 1 |
```

따라서 T^n의 [0][1] 위치가 S(n)이 된다.

이 방법은 등비급수 합 공식 S = A(A^N - 1)/(A - 1)을 사용하는 것보다 범용적이다. 등비급수 공식은 (A-1)의 모듈러 역원이 필요하여 M이 소수가 아니거나 A ≡ 1 (mod M)인 경우 별도 처리가 필요하지만, 행렬 거듭제곱은 이런 제약이 없다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(log N) |
| 공간 복잡도 | O(1) |

2×2 행렬 곱셈은 상수 시간이므로, 행렬 거듭제곱의 전체 시간 복잡도는 O(log N)이다. 행렬 크기가 고정(2×2)이므로 공간 복잡도는 O(1)이다.

## 대안적 접근

- 분할 정복 재귀: S(n) = S(n/2) + A^(n/2) × S(n/2) (n이 짝수), S(n) = S(n-1) + A^n (n이 홀수)으로 재귀적으로 풀 수 있다. 모듈러 거듭제곱과 결합하면 O(log² N)에 가능하다.
- 등비급수 공식: M이 소수이고 A ≢ 1 (mod M)이면 S = A(A^N - 1) × (A-1)^(-1) mod M으로 O(log N)에 계산할 수 있다. 페르마 소정리로 역원을 구한다.
