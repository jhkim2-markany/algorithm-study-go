# 거듭제곱 합 구하기

**난이도:** 상

## 문제 설명

정수 A, N, M이 주어진다. 다음 값을 구하시오:

S = (A^1 + A^2 + A^3 + ... + A^N) mod M

N이 매우 클 수 있으므로 단순 반복으로는 시간 내에 풀 수 없다. 분할 정복을 활용하여 효율적으로 계산하시오.

## 입력 형식

- 첫째 줄에 세 정수 A, N, M이 공백으로 구분되어 주어진다

## 출력 형식

- S = (A^1 + A^2 + ... + A^N) mod M의 값을 출력한다

## 제약 조건

- 1 ≤ A ≤ 1,000,000
- 1 ≤ N ≤ 1,000,000,000,000,000,000 (10^18)
- 2 ≤ M ≤ 1,000,000,007
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
2 3 1000
```

### 예제 출력 1

```
14
```

### 예제 입력 2

```
3 5 100
```

### 예제 출력 2

```
63
```
