# 36. 분할 정복 거듭제곱

밑(base)의 거듭제곱을 지수를 반씩 나누어 빠르게 계산하는 알고리즘 유형이다. 모듈러 거듭제곱, 행렬 거듭제곱 등이 핵심이다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이

## 문제 목록

| 번호 | 문제 | 난이도 |
| --- | --- | --- |
| 01 | 모듈러 거듭제곱 계산 | 하 |
| 02 | 행렬 거듭제곱 피보나치 | 중 |
| 03 | 거듭제곱 합 구하기 | 상 |
