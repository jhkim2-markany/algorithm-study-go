# 최장 공통 부분 수열 (LCS)

**난이도:** 상

## 문제 설명

두 문자열이 주어질 때, 두 문자열의 최장 공통 부분 수열(Longest Common Subsequence, LCS)의 길이를 구하고, 그 부분 수열을 출력하시오.

부분 수열이란 원래 문자열에서 일부 문자를 삭제하여(삭제하지 않아도 됨) 만들 수 있는 문자열로, 남은 문자들의 상대적 순서는 유지된다.

## 입력 형식

- 첫째 줄에 문자열 A가 주어진다
- 둘째 줄에 문자열 B가 주어진다

## 출력 형식

- 첫째 줄에 LCS의 길이를 출력한다
- 둘째 줄에 LCS를 출력한다 (LCS가 여러 개이면 아무거나 하나를 출력한다)
- LCS의 길이가 0이면 둘째 줄은 출력하지 않는다

## 제약 조건

- 문자열 A, B는 알파벳 대문자로만 구성된다
- 1 ≤ |A|, |B| ≤ 1,000
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
ACAYKP
CAPCAK
```

### 예제 출력 1

```
4
ACAK
```

### 예제 입력 2

```
ABCDE
ACE
```

### 예제 출력 2

```
3
ACE
```
