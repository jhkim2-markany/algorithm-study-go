# 피보나치 수

**난이도:** 하

## 문제 설명

정수 N이 주어질 때, N번째 피보나치 수를 구하시오.

피보나치 수열은 다음과 같이 정의된다:
- F(0) = 0
- F(1) = 1
- F(N) = F(N-1) + F(N-2) (N ≥ 2)

## 입력 형식

- 첫째 줄에 정수 N이 주어진다 (0 ≤ N ≤ 45)

## 출력 형식

- N번째 피보나치 수를 출력한다

## 제약 조건

- 0 ≤ N ≤ 45
- 결과는 32비트 정수 범위 내이다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
0
```

### 예제 출력 1

```
0
```

### 예제 입력 2

```
10
```

### 예제 출력 2

```
55
```

### 예제 입력 3

```
45
```

### 예제 출력 3

```
1134903170
```
