# 배낭 문제 (0/1 Knapsack)

**난이도:** 중

## 문제 설명

N개의 물건이 있고, 각 물건은 무게 W와 가치 V를 가진다. 최대 용량이 K인 배낭에 물건을 넣을 때, 배낭에 담을 수 있는 물건들의 가치 합의 최댓값을 구하시오.

각 물건은 하나씩만 존재하며, 물건을 쪼개서 넣을 수 없다 (0/1 배낭 문제).

## 입력 형식

- 첫째 줄에 물건의 개수 N과 배낭의 최대 용량 K가 주어진다
- 둘째 줄부터 N개의 줄에 각 물건의 무게 W와 가치 V가 주어진다

## 출력 형식

- 배낭에 담을 수 있는 물건들의 가치 합의 최댓값을 출력한다

## 제약 조건

- 1 ≤ N ≤ 100
- 1 ≤ K ≤ 100,000
- 1 ≤ W ≤ 100,000
- 1 ≤ V ≤ 1,000
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
4 7
6 13
4 8
3 6
5 12
```

### 예제 출력 1

```
14
```

### 예제 입력 2

```
3 10
5 10
4 40
6 30
```

### 예제 출력 2

```
70
```
