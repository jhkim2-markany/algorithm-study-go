# 19. 동적 프로그래밍

복잡한 문제를 작은 하위 문제로 나누어 풀고, 그 결과를 저장하여 동일한 하위 문제를 반복 계산하지 않도록 하는 알고리즘 기법이다. 최적 부분 구조(Optimal Substructure)와 중복 부분 문제(Overlapping Subproblems) 성질을 가진 문제에 적용하며, 탑다운(메모이제이션)과 바텀업(타뷸레이션) 두 가지 방식으로 구현할 수 있다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
