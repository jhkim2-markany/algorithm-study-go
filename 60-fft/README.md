# 60. FFT (고속 푸리에 변환)

다항식 곱셈을 O(N log N)에 수행하는 알고리즘이다. 이산 푸리에 변환(DFT)과 역변환(IDFT)을 분할 정복으로 빠르게 계산하며, 큰 수 곱셈, 합성곱(Convolution) 등 다양한 문제에 활용된다. BOJ 플래티넘 이상 문제에서 자주 등장하는 상위권 차별화 유형이다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
