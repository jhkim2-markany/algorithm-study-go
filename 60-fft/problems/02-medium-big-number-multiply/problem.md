# 큰 수 곱셈

**난이도:** 중

## 문제 설명

매우 큰 두 양의 정수 A와 B가 주어진다. 두 수의 곱 A × B를 구하시오.

일반적인 정수 자료형으로는 표현할 수 없는 큰 수이므로, 문자열로 입력받아 FFT를 활용하여 곱셈을 수행해야 한다.

## 입력 형식

- 첫째 줄에 양의 정수 A가 문자열로 주어진다
- 둘째 줄에 양의 정수 B가 문자열로 주어진다

## 출력 형식

- 첫째 줄에 A × B의 결과를 출력한다

## 제약 조건

- 1 ≤ A, B의 자릿수 ≤ 100,000
- A, B는 양의 정수이다 (선행 0 없음)
- 시간 제한: 2초

## 예제

### 예제 입력 1

```
123
456
```

### 예제 출력 1

```
56088
```

### 예제 입력 2

```
999
999
```

### 예제 출력 2

```
998001
```

### 예제 입력 3

```
1
99999
```

### 예제 출력 3

```
99999
```
