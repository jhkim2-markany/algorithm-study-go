# 합성곱 카운팅

**난이도:** 상

## 문제 설명

1부터 N까지의 정수로 이루어진 배열 A와 배열 B가 주어진다. 배열 A에서 하나의 원소 a를, 배열 B에서 하나의 원소 b를 골랐을 때, 합 a + b = S가 되는 쌍의 개수를 모든 가능한 S 값에 대해 구하시오.

즉, 각 S에 대해 A[i] + B[j] = S를 만족하는 (i, j) 쌍의 개수를 구한다.

## 입력 형식

- 첫째 줄에 배열의 크기 N이 주어진다 (1 ≤ N ≤ 200,000)
- 둘째 줄에 배열 A의 원소 N개가 공백으로 구분되어 주어진다
- 셋째 줄에 배열 B의 원소 N개가 공백으로 구분되어 주어진다

## 출력 형식

- 가능한 합 S의 최솟값부터 최댓값까지, 각 S에 대한 쌍의 개수를 공백으로 구분하여 출력한다
- 쌍의 개수가 0인 S도 포함하여 연속적으로 출력한다

## 제약 조건

- 1 ≤ N ≤ 200,000
- 1 ≤ A[i], B[j] ≤ N
- 시간 제한: 3초

## 예제

### 예제 입력 1

```
3
1 2 3
1 2 3
```

### 예제 출력 1

```
1 2 3 2 1
```

### 예제 입력 2

```
4
1 1 2 2
3 3 4 4
```

### 예제 출력 2

```
4 8 4
```
