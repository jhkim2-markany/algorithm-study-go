# 다항식 곱셈 기본

**난이도:** 하

## 문제 설명

두 다항식 A(x)와 B(x)가 주어진다. 두 다항식의 곱 C(x) = A(x) × B(x)의 계수를 구하시오.

다항식 A(x) = a₀ + a₁x + a₂x² + ... + aₙx^n 은 계수 배열 [a₀, a₁, ..., aₙ]으로 표현된다.

## 입력 형식

- 첫째 줄에 다항식 A의 차수 N이 주어진다 (0 ≤ N ≤ 100,000)
- 둘째 줄에 A의 계수 a₀, a₁, ..., aₙ이 공백으로 구분되어 주어진다
- 셋째 줄에 다항식 B의 차수 M이 주어진다 (0 ≤ M ≤ 100,000)
- 넷째 줄에 B의 계수 b₀, b₁, ..., bₘ이 공백으로 구분되어 주어진다

## 출력 형식

- 첫째 줄에 결과 다항식 C의 계수 c₀, c₁, ..., cₙ₊ₘ을 공백으로 구분하여 출력한다

## 제약 조건

- 0 ≤ N, M ≤ 100,000
- -1,000 ≤ aᵢ, bⱼ ≤ 1,000
- 시간 제한: 2초

## 예제

### 예제 입력 1

```
2
1 2 3
1
4 5
```

### 예제 출력 1

```
4 13 22 15
```

### 예제 입력 2

```
0
7
0
3
```

### 예제 출력 2

```
21
```
