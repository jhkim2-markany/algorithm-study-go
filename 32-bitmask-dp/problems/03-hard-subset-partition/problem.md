# 집합 분할 최소 차이

**난이도:** 상

## 문제 설명

N개의 양의 정수로 이루어진 집합이 주어진다. 이 집합을 공집합이 아닌 두 개의 부분집합으로 분할하여, 두 부분집합의 원소 합의 차이의 절댓값이 최소가 되도록 하시오.

모든 원소는 정확히 하나의 부분집합에 속해야 하며, 두 부분집합 모두 최소 하나의 원소를 포함해야 한다.

## 입력 형식

- 첫째 줄에 원소의 수 N이 주어진다
- 둘째 줄에 N개의 양의 정수가 공백으로 구분되어 주어진다

## 출력 형식

- 두 부분집합의 합 차이의 최솟값을 출력한다

## 제약 조건

- 2 ≤ N ≤ 20
- 1 ≤ 각 원소 ≤ 1,000,000
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
4
1 6 11 5
```

### 예제 출력 1

```
1
```

### 예제 입력 2

```
3
1 2 3
```

### 예제 출력 2

```
0
```

### 예제 입력 3

```
5
3 1 4 2 2
```

### 예제 출력 3

```
0
```
