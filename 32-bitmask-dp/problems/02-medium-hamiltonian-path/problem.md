# 최소 비용 해밀턴 경로

**난이도:** 중

## 문제 설명

N개의 도시와 도시 간 이동 비용이 주어진다. 임의의 도시에서 출발하여 모든 도시를 정확히 한 번씩 방문하는 경로(해밀턴 경로) 중 최소 비용을 구하시오.

외판원 문제(TSP)와 달리 출발 도시로 돌아올 필요가 없으며, 출발 도시도 자유롭게 선택할 수 있다.

## 입력 형식

- 첫째 줄에 도시의 수 N이 주어진다
- 다음 N개의 줄에 N×N 비용 행렬이 주어진다
- i번째 줄의 j번째 값은 도시 i에서 도시 j로 이동하는 비용이다
- 비용이 0이면 해당 경로로 이동할 수 없다

## 출력 형식

- 모든 도시를 정확히 한 번씩 방문하는 경로의 최소 비용을 출력한다
- 불가능한 경우 -1을 출력한다

## 제약 조건

- 2 ≤ N ≤ 16
- 0 ≤ 비용 ≤ 1,000,000
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
4
0 10 15 20
10 0 35 25
15 35 0 30
20 25 30 0
```

### 예제 출력 1

```
50
```

### 예제 입력 2

```
3
0 5 0
3 0 7
0 2 0
```

### 예제 출력 2

```
5
```
