# 작업 배정 문제

**난이도:** 하

## 문제 설명

N명의 작업자와 N개의 작업이 있다. 각 작업자가 각 작업을 수행하는 데 드는 비용이 주어진다. 모든 작업자에게 정확히 하나의 작업을 배정하고, 모든 작업이 정확히 한 명의 작업자에게 배정되도록 할 때, 총 비용의 최솟값을 구하시오.

## 입력 형식

- 첫째 줄에 작업자와 작업의 수 N이 주어진다
- 다음 N개의 줄에 N×N 비용 행렬이 주어진다
- i번째 줄의 j번째 값은 작업자 i가 작업 j를 수행하는 비용이다

## 출력 형식

- 모든 작업을 배정했을 때의 최소 총 비용을 출력한다

## 제약 조건

- 1 ≤ N ≤ 20
- 1 ≤ 비용 ≤ 1,000
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
3
3 2 7
5 1 3
2 7 2
```

### 예제 출력 1

```
6
```

### 예제 입력 2

```
4
9 2 7 8
6 4 3 7
5 8 1 8
7 6 9 4
```

### 예제 출력 2

```
13
```
