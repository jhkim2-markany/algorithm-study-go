# 32. 비트필드 DP

비트마스크를 이용한 상태 압축과 동적 프로그래밍을 결합하여, 집합의 부분집합에 대한 최적화 문제를 효율적으로 해결하는 알고리즘 유형이다. 방문 상태, 선택 상태, 배정 상태 등을 비트로 표현하여 DP 테이블의 인덱스로 활용한다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
