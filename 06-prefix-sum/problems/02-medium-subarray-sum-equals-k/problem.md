# 합이 K인 부분배열

**난이도:** 중

## 문제 설명

N개의 정수로 이루어진 배열과 정수 K가 주어진다. 연속된 부분배열의 합이 정확히 K가 되는 부분배열의 개수를 구하시오.

부분배열이란 배열에서 연속된 하나 이상의 원소로 이루어진 부분을 말한다.

## 입력 형식

- 첫째 줄에 배열의 크기 N과 정수 K가 공백으로 구분되어 주어진다
- 둘째 줄에 N개의 정수가 공백으로 구분되어 주어진다

## 출력 형식

- 합이 K인 연속 부분배열의 개수를 출력한다

## 제약 조건

- 1 ≤ N ≤ 200,000
- -1,000,000,000 ≤ K ≤ 1,000,000,000
- -10,000 ≤ 배열 원소 ≤ 10,000
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5 5
1 2 3 4 5
```

### 예제 출력 1

```
2
```

### 예제 입력 2

```
4 0
1 -1 1 -1
```

### 예제 출력 2

```
4
```

### 예제 입력 3

```
3 3
3 3 3
```

### 예제 출력 3

```
3
```
