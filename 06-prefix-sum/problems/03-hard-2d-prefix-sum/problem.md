# 2차원 누적합

**난이도:** 상

## 문제 설명

N×M 크기의 정수 행렬과 Q개의 쿼리가 주어진다. 각 쿼리는 네 정수 r1, c1, r2, c2로 이루어져 있으며, (r1, c1)부터 (r2, c2)까지의 부분 행렬에 포함된 모든 원소의 합을 구해야 한다. (1-indexed)

## 입력 형식

- 첫째 줄에 행렬의 행 수 N, 열 수 M, 쿼리 수 Q가 공백으로 구분되어 주어진다
- 다음 N개의 줄에 각각 M개의 정수가 공백으로 구분되어 주어진다
- 다음 Q개의 줄에 각각 네 정수 r1, c1, r2, c2가 공백으로 구분되어 주어진다

## 출력 형식

- 각 쿼리에 대해 부분 행렬의 합을 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ N, M ≤ 1,000
- 1 ≤ Q ≤ 100,000
- -1,000 ≤ 행렬 원소 ≤ 1,000
- 1 ≤ r1 ≤ r2 ≤ N
- 1 ≤ c1 ≤ c2 ≤ M
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
3 3 3
1 2 3
4 5 6
7 8 9
1 1 2 2
1 1 3 3
2 2 3 3
```

### 예제 출력 1

```
12
45
28
```

### 예제 입력 2

```
2 4 2
1 -1 2 -2
-3 3 -4 4
1 1 2 4
1 2 2 3
```

### 예제 출력 2

```
0
0
```
