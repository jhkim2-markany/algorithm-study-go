# 구간 합

**난이도:** 하

## 문제 설명

N개의 정수로 이루어진 배열과 M개의 구간 합 쿼리가 주어진다. 각 쿼리는 두 정수 l, r로 이루어져 있으며, 배열의 l번째 원소부터 r번째 원소까지의 합을 구해야 한다. (1-indexed)

## 입력 형식

- 첫째 줄에 배열의 크기 N과 쿼리의 수 M이 공백으로 구분되어 주어진다
- 둘째 줄에 N개의 정수가 공백으로 구분되어 주어진다
- 다음 M개의 줄에 각각 두 정수 l, r이 공백으로 구분되어 주어진다

## 출력 형식

- 각 쿼리에 대해 구간 합을 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- 1 ≤ M ≤ 100,000
- -1,000 ≤ 배열 원소 ≤ 1,000
- 1 ≤ l ≤ r ≤ N
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5 3
5 4 3 2 1
1 3
2 4
1 5
```

### 예제 출력 1

```
12
9
15
```

### 예제 입력 2

```
4 2
1 -1 1 -1
1 4
2 3
```

### 예제 출력 2

```
0
0
```
