# 06. 누적합

배열의 구간 합을 효율적으로 계산하기 위해 미리 합을 누적해두는 알고리즘 유형이다. 전처리를 통해 임의의 구간 합을 O(1)에 구할 수 있으며, 구간 합 쿼리가 반복되는 문제에서 큰 성능 향상을 제공한다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
