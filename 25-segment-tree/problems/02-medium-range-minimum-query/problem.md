# 구간 최솟값 쿼리

**난이도:** 중

## 문제 설명

길이가 N인 정수 배열이 주어진다. 다음 두 가지 연산을 M번 수행하시오.

- `1 i v`: 배열의 i번째 원소를 v로 변경한다.
- `2 l r`: 배열의 l번째 원소부터 r번째 원소까지의 최솟값을 출력한다.

## 입력 형식

- 첫째 줄에 배열의 크기 N과 연산의 수 M이 주어진다 (1 ≤ N, M ≤ 100,000)
- 둘째 줄에 배열의 초기값 N개가 공백으로 구분되어 주어진다 (각 원소는 1 이상 1,000,000,000 이하)
- 셋째 줄부터 M개의 줄에 연산이 주어진다

## 출력 형식

- 연산 2가 주어질 때마다 구간 최솟값을 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ N, M ≤ 100,000
- 1 ≤ 배열 원소 ≤ 1,000,000,000
- 1 ≤ i ≤ N
- 1 ≤ l ≤ r ≤ N
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5 5
5 1 4 2 3
2 2 4
1 2 6
2 2 4
2 1 5
1 4 1
```

### 예제 출력 1

```
1
2
2
```

### 예제 입력 2

```
3 3
10 20 30
2 1 3
1 2 5
2 1 3
```

### 예제 출력 2

```
10
5
```
