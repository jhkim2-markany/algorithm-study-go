# 느리게 갱신되는 세그먼트 트리

**난이도:** 상

## 문제 설명

길이가 N인 정수 배열이 주어진다. 다음 두 가지 연산을 M번 수행하시오.

- `1 l r v`: 배열의 l번째 원소부터 r번째 원소까지 모두 v를 더한다.
- `2 l r`: 배열의 l번째 원소부터 r번째 원소까지의 합을 출력한다.

## 입력 형식

- 첫째 줄에 배열의 크기 N과 연산의 수 M이 주어진다 (1 ≤ N, M ≤ 100,000)
- 둘째 줄에 배열의 초기값 N개가 공백으로 구분되어 주어진다 (각 원소는 -1,000,000 이상 1,000,000 이하)
- 셋째 줄부터 M개의 줄에 연산이 주어진다

## 출력 형식

- 연산 2가 주어질 때마다 구간 합을 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ N, M ≤ 100,000
- -1,000,000 ≤ 배열 원소 ≤ 1,000,000
- -1,000,000 ≤ v ≤ 1,000,000
- 1 ≤ l ≤ r ≤ N
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5 4
1 2 3 4 5
1 2 4 3
2 1 5
1 1 3 -2
2 2 4
```

### 예제 출력 1

```
24
14
```

### 예제 입력 2

```
4 3
0 0 0 0
1 1 4 5
1 2 3 3
2 1 4
```

### 예제 출력 2

```
26
```
