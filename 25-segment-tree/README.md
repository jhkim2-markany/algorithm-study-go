# 25. 세그먼트 트리

배열의 구간에 대한 질의(query)와 갱신(update)을 효율적으로 처리하는 트리 기반 자료구조이다. 구간 합, 구간 최솟값/최댓값 등의 질의를 O(log N)에 수행할 수 있으며, 느리게 갱신되는 세그먼트 트리(Lazy Propagation)를 활용하면 구간 갱신도 O(log N)에 처리할 수 있다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
