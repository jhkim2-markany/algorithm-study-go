# 커스텀 정렬

**난이도:** 중

## 문제 설명

N명의 학생 정보가 주어진다. 각 학생은 이름, 국어 점수, 영어 점수, 수학 점수를 가진다. 다음 조건에 따라 학생들을 정렬하여 이름을 출력하시오.

1. 국어 점수가 높은 순서대로 (내림차순)
2. 국어 점수가 같으면 영어 점수가 낮은 순서대로 (오름차순)
3. 국어 점수와 영어 점수가 같으면 수학 점수가 높은 순서대로 (내림차순)
4. 모든 점수가 같으면 이름이 사전순으로 앞선 순서대로 (오름차순)

## 입력 형식

- 첫째 줄에 학생 수 N이 주어진다 (1 ≤ N ≤ 100,000)
- 다음 N개의 줄에 각 학생의 이름, 국어 점수, 영어 점수, 수학 점수가 공백으로 구분되어 주어진다
- 이름은 알파벳 소문자로만 이루어져 있으며 길이는 1 이상 10 이하이다
- 점수는 0 이상 100 이하의 정수이다

## 출력 형식

- N개의 줄에 걸쳐 정렬된 순서대로 학생의 이름을 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- 이름: 알파벳 소문자, 길이 1~10
- 0 ≤ 점수 ≤ 100
- 시간 제한: 1초
- 메모리 제한: 256MB
