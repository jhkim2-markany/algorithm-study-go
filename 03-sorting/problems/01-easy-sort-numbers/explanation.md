# 숫자 정렬 - 해설

## 접근 방식

1. N개의 정수를 배열에 입력받는다
2. Go 표준 라이브러리의 `sort.Ints` 함수를 사용하여 오름차순 정렬한다
3. 정렬된 배열을 공백으로 구분하여 출력한다

## 핵심 아이디어

Go의 `sort.Ints` 함수는 내부적으로 인트로소트(Introsort) 변형을 사용하여 평균 O(N log N)의 시간 복잡도를 보장한다. N이 최대 1,000,000이므로 O(N log N) 알고리즘이면 시간 내에 충분히 처리할 수 있다.

대량의 입출력이 발생하므로 `bufio`를 사용하여 입출력 성능을 최적화하는 것이 중요하다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N log N) |
| 공간 복잡도 | O(N) |

N개의 원소를 정렬하는 데 O(N log N), 배열 저장에 O(N)의 공간이 필요하다.

## 대안적 접근

- **직접 구현**: 병합 정렬이나 퀵 정렬을 직접 구현할 수 있다. 정렬 알고리즘의 동작 원리를 학습하는 데 도움이 된다.
- **카운팅 정렬**: 값의 범위가 제한적이라면 O(N + K) 시간에 정렬할 수 있지만, 이 문제에서는 값의 범위가 넓어 적합하지 않다.
