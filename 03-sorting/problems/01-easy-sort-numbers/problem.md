# 숫자 정렬

**난이도:** 하

## 문제 설명

N개의 정수가 주어졌을 때, 이를 오름차순으로 정렬하여 출력하시오.

## 입력 형식

- 첫째 줄에 정수의 개수 N이 주어진다 (1 ≤ N ≤ 1,000,000)
- 둘째 줄에 N개의 정수가 공백으로 구분되어 주어진다
- 각 정수의 범위는 -1,000,000,000 이상 1,000,000,000 이하이다

## 출력 형식

- 한 줄에 N개의 정수를 오름차순으로 정렬하여 공백으로 구분하여 출력한다

## 제약 조건

- 1 ≤ N ≤ 1,000,000
- -1,000,000,000 ≤ 각 정수 ≤ 1,000,000,000
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5
5 3 1 4 2
```

### 예제 출력 1

```
1 2 3 4 5
```

### 예제 입력 2

```
3
-1 0 1
```

### 예제 출력 2

```
-1 0 1
```
