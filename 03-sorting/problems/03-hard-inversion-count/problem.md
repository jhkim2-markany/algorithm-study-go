# 역전 카운트

**난이도:** 상

## 문제 설명

크기가 N인 배열 A가 주어진다. 배열에서 역전(Inversion)이란 i < j이면서 A[i] > A[j]인 쌍 (i, j)를 말한다. 배열에 존재하는 역전의 총 개수를 구하시오.

예를 들어, 배열 [3, 1, 2]에서 역전은 (3, 1), (3, 2)로 총 2개이다.

## 입력 형식

- 첫째 줄에 배열의 크기 N이 주어진다 (1 ≤ N ≤ 500,000)
- 둘째 줄에 N개의 정수가 공백으로 구분되어 주어진다
- 각 정수의 범위는 1 이상 1,000,000,000 이하이다

## 출력 형식

- 역전의 총 개수를 출력한다

## 제약 조건

- 1 ≤ N ≤ 500,000
- 1 ≤ A[i] ≤ 1,000,000,000
- 시간 제한: 2초
- 메모리 제한: 256MB
- 역전의 개수는 int64 범위를 초과하지 않는다

## 예제

### 예제 입력 1

```
5
3 1 2 5 4
```

### 예제 출력 1

```
3
```

### 예제 입력 2

```
3
3 2 1
```

### 예제 출력 2

```
3
```
