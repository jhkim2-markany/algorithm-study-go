# 사이클 판별

**난이도:** 중

## 문제 설명

N개의 정점과 M개의 간선으로 이루어진 방향 그래프가 주어진다. 이 그래프에 사이클이 존재하는지 판별하시오.

사이클이란 어떤 정점에서 출발하여 간선을 따라 이동했을 때 다시 출발 정점으로 돌아올 수 있는 경로를 말한다.

## 입력 형식

- 첫째 줄에 정점의 수 N과 간선의 수 M이 주어진다 (1 ≤ N ≤ 100,000, 0 ≤ M ≤ 200,000)
- 다음 M개의 줄에 각각 두 정수 u, v가 주어진다. 이는 정점 u에서 정점 v로 향하는 방향 간선이 있음을 의미한다 (1 ≤ u, v ≤ N)

## 출력 형식

- 사이클이 존재하면 "Yes"를, 존재하지 않으면 "No"를 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- 0 ≤ M ≤ 200,000
- 자기 자신으로의 간선이 존재할 수 있다
- 같은 방향의 중복 간선은 주어지지 않는다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
4 4
1 2
2 3
3 4
4 2
```

### 예제 출력 1

```
Yes
```

### 예제 입력 2

```
4 3
1 2
1 3
3 4
```

### 예제 출력 2

```
No
```

### 예제 입력 3

```
2 1
1 1
```

### 예제 출력 3

```
Yes
```
