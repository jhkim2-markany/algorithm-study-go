# 단절점 찾기

**난이도:** 상

## 문제 설명

N개의 정점과 M개의 간선으로 이루어진 무방향 연결 그래프가 주어진다. 이 그래프에서 단절점(Articulation Point)을 모두 찾아 오름차순으로 출력하시오.

단절점이란 해당 정점을 제거했을 때 그래프가 두 개 이상의 연결 요소로 분리되는 정점을 말한다.

## 입력 형식

- 첫째 줄에 정점의 수 N과 간선의 수 M이 주어진다 (2 ≤ N ≤ 100,000, 1 ≤ M ≤ 500,000)
- 다음 M개의 줄에 각각 두 정수 u, v가 주어진다. 이는 정점 u와 정점 v 사이에 간선이 있음을 의미한다 (1 ≤ u, v ≤ N, u ≠ v)

## 출력 형식

- 첫째 줄에 단절점의 개수를 출력한다
- 둘째 줄에 단절점의 번호를 오름차순으로 공백으로 구분하여 출력한다
- 단절점이 없으면 첫째 줄에 0만 출력한다

## 제약 조건

- 2 ≤ N ≤ 100,000
- 1 ≤ M ≤ 500,000
- 그래프는 연결 그래프이다
- 같은 간선이 중복으로 주어지지 않는다
- 자기 자신으로의 간선은 주어지지 않는다
- 시간 제한: 2초
- 메모리 제한: 256MB
