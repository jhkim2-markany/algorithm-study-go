# 연결 요소의 개수

**난이도:** 하

## 문제 설명

N개의 정점과 M개의 간선으로 이루어진 무방향 그래프가 주어진다. 이 그래프의 연결 요소(Connected Component)의 개수를 구하시오.

연결 요소란 그래프 내에서 서로 연결된 정점들의 최대 집합이다. 같은 연결 요소에 속한 임의의 두 정점 사이에는 경로가 존재한다.

## 입력 형식

- 첫째 줄에 정점의 수 N과 간선의 수 M이 주어진다 (1 ≤ N ≤ 1,000, 0 ≤ M ≤ N×(N-1)/2)
- 다음 M개의 줄에 각각 두 정수 u, v가 주어진다. 이는 정점 u와 정점 v 사이에 간선이 있음을 의미한다 (1 ≤ u, v ≤ N, u ≠ v)

## 출력 형식

- 연결 요소의 개수를 출력한다

## 제약 조건

- 1 ≤ N ≤ 1,000
- 0 ≤ M ≤ N×(N-1)/2
- 같은 간선이 중복으로 주어지지 않는다
- 자기 자신으로의 간선은 주어지지 않는다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
6 5
1 2
2 5
5 1
3 4
4 6
```

### 예제 출력 1

```
2
```

### 예제 입력 2

```
4 0
```

### 예제 출력 2

```
4
```
