# 15. 그래프 탐색 DFS

그래프의 깊이를 우선으로 탐색하는 알고리즘이다. 한 경로를 끝까지 탐색한 뒤 되돌아와 다른 경로를 탐색하는 방식으로, 재귀 호출 또는 스택을 사용하여 구현한다. 연결 요소 탐색, 사이클 판별, 위상 정렬 등 다양한 그래프 문제의 기반이 된다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
