# 로봇 시뮬레이션

**난이도:** 상

## 문제 설명

R×C 크기의 격자 위에 여러 대의 로봇이 놓여 있다. 각 로봇은 위치와 방향(상, 하, 좌, 우)을 가진다. 주어진 명령 목록을 순서대로 수행한 뒤, 각 로봇의 최종 위치와 방향을 출력하시오.

명령은 다음과 같다:
- `L`: 현재 방향에서 왼쪽(반시계 방향)으로 90도 회전
- `R`: 현재 방향에서 오른쪽(시계 방향)으로 90도 회전
- `F`: 현재 방향으로 한 칸 전진

로봇이 격자 밖으로 나가려 하면 해당 이동은 무시한다. 로봇이 다른 로봇이 있는 칸으로 이동하려 하면 해당 이동은 무시한다.

명령은 모든 로봇에게 동시에 적용되지 않고, 로봇 번호 순서대로 각 로봇이 모든 명령을 수행한 뒤 다음 로봇이 명령을 수행한다.

## 입력 형식

- 첫째 줄에 격자의 행 수 R과 열 수 C가 공백으로 구분되어 주어진다
- 둘째 줄에 로봇의 수 N과 명령의 수 M이 공백으로 구분되어 주어진다
- 다음 N개의 줄에 각 로봇의 초기 행 r, 열 c, 방향 d가 공백으로 구분되어 주어진다
  - 방향: 1(상), 2(우), 3(하), 4(좌)
  - 행과 열은 1부터 시작한다
- 다음 줄에 M개의 명령이 공백 없이 주어진다

## 출력 형식

- N개의 줄에 각 로봇의 최종 행, 열, 방향을 공백으로 구분하여 출력한다
- 방향은 입력과 동일한 형식(1: 상, 2: 우, 3: 하, 4: 좌)으로 출력한다

## 제약 조건

- 1 ≤ R, C ≤ 100
- 1 ≤ N ≤ 100
- 1 ≤ M ≤ 1000
- 초기 상태에서 두 로봇이 같은 위치에 있는 경우는 없다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5 5
2 4
1 1 2
3 3 3
FFLF
```

### 예제 출력 1

```
1 3 1
5 4 2
```

### 예제 입력 2

```
3 3
1 3
2 2 2
FFF
```

### 예제 출력 2

```
2 3 2
```
