# 행렬 회전 - 해설

## 접근 방식

1. N×N 행렬을 입력받아 2차원 배열에 저장한다
2. 시계 방향 90도 회전의 좌표 변환 규칙을 파악한다
3. 새로운 배열에 변환된 좌표에 따라 값을 배치한다
4. 결과 행렬을 출력한다

## 핵심 아이디어

시계 방향 90도 회전에서 원래 위치 `(i, j)`의 원소는 새로운 위치 `(j, N-1-i)`로 이동한다. 이를 역으로 생각하면, 회전된 행렬의 `(i, j)` 위치에는 원래 행렬의 `(N-1-j, i)` 위치 값이 들어간다.

이 좌표 변환 공식만 정확히 적용하면 간단하게 풀 수 있는 기본적인 구현 문제이다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N²) |
| 공간 복잡도 | O(N²) |

N×N 행렬의 모든 원소를 한 번씩 처리하므로 시간 복잡도는 O(N²)이다. 회전 결과를 저장할 새로운 N×N 배열이 필요하므로 공간 복잡도도 O(N²)이다.

## 대안적 접근

- **제자리 회전(In-place rotation)**: 추가 배열 없이 4개 원소를 순환 교환하는 방식으로 O(1) 추가 공간으로 회전할 수 있다. 다만 구현이 다소 복잡해진다.
- **전치 + 좌우 반전**: 행렬을 전치(transpose)한 뒤 각 행을 좌우 반전하면 시계 방향 90도 회전과 동일한 결과를 얻는다.
