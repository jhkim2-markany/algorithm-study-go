# 행렬 회전

**난이도:** 하

## 문제 설명

N×N 크기의 정수 행렬이 주어진다. 이 행렬을 시계 방향으로 90도 회전한 결과를 출력하시오.

시계 방향 90도 회전이란, 행렬의 첫 번째 행이 마지막 열이 되고, 두 번째 행이 마지막에서 두 번째 열이 되는 식으로 변환하는 것을 말한다.

## 입력 형식

- 첫째 줄에 행렬의 크기 N이 주어진다 (1 ≤ N ≤ 100)
- 다음 N개의 줄에 각각 N개의 정수가 공백으로 구분되어 주어진다
- 각 정수의 범위는 -1000 이상 1000 이하이다

## 출력 형식

- N개의 줄에 걸쳐 회전된 행렬을 출력한다
- 각 줄의 정수는 공백으로 구분한다

## 제약 조건

- 1 ≤ N ≤ 100
- -1000 ≤ 행렬 원소 ≤ 1000
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
3
1 2 3
4 5 6
7 8 9
```

### 예제 출력 1

```
7 4 1
8 5 2
9 6 3
```

### 예제 입력 2

```
2
1 2
3 4
```

### 예제 출력 2

```
3 1
4 2
```
