# 나선형 행렬

**난이도:** 중

## 문제 설명

양의 정수 N이 주어졌을 때, N×N 크기의 행렬을 나선형(Spiral) 순서로 1부터 N²까지의 수로 채운 결과를 출력하시오.

나선형 순서란, 행렬의 왼쪽 위 모서리에서 시작하여 오른쪽 → 아래 → 왼쪽 → 위 방향으로 바깥에서 안쪽으로 돌아가며 채우는 순서를 말한다.

## 입력 형식

- 첫째 줄에 양의 정수 N이 주어진다 (1 ≤ N ≤ 100)

## 출력 형식

- N개의 줄에 걸쳐 나선형으로 채워진 행렬을 출력한다
- 각 줄의 정수는 공백으로 구분한다

## 제약 조건

- 1 ≤ N ≤ 100
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
3
```

### 예제 출력 1

```
1 2 3
8 9 4
7 6 5
```

### 예제 입력 2

```
4
```

### 예제 출력 2

```
1 2 3 4
12 13 14 5
11 16 15 6
10 9 8 7
```
