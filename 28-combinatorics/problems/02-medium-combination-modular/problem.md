# 조합의 수 (모듈러 연산)

**난이도:** 중

## 문제 설명

자연수 N과 정수 R이 주어질 때, 이항 계수 C(N, R)을 1,000,000,007 (10⁹ + 7)로 나눈 나머지를 구하시오.

N이 매우 클 수 있으므로, 팩토리얼 전처리와 모듈러 역원을 활용하여 효율적으로 계산해야 한다.

## 입력 형식

- 첫째 줄에 테스트 케이스의 수 T가 주어진다 (1 ≤ T ≤ 100,000)
- 다음 T개의 줄에 각각 두 정수 N과 R이 공백으로 구분되어 주어진다

## 출력 형식

- 각 테스트 케이스에 대해 C(N, R) mod 1,000,000,007의 값을 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ T ≤ 100,000
- 0 ≤ R ≤ N ≤ 1,000,000
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
3
5 2
1000000 500000
10 0
```

### 예제 출력 1

```
10
149033233
1
```
