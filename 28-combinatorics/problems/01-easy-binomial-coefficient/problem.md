# 이항 계수 계산

**난이도:** 하

## 문제 설명

자연수 N과 정수 R이 주어질 때, 이항 계수 C(N, R)의 값을 구하시오. 이항 계수 C(N, R)은 N개의 원소에서 R개를 순서 없이 선택하는 경우의 수이다.

## 입력 형식

- 첫째 줄에 테스트 케이스의 수 T가 주어진다 (1 ≤ T ≤ 100)
- 다음 T개의 줄에 각각 두 정수 N과 R이 공백으로 구분되어 주어진다

## 출력 형식

- 각 테스트 케이스에 대해 C(N, R)의 값을 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ T ≤ 100
- 0 ≤ R ≤ N ≤ 30
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
3
5 2
10 3
6 0
```

### 예제 출력 1

```
10
120
1
```
