# 이항 계수 계산 - 해설

## 접근 방식

1. 파스칼의 삼각형을 이용하여 이항 계수를 전처리한다
2. dp[i][0] = dp[i][i] = 1로 초기화한다
3. 점화식 dp[i][j] = dp[i-1][j-1] + dp[i-1][j]로 나머지를 채운다
4. 각 쿼리에 대해 dp[n][r] 값을 바로 출력한다

## 핵심 아이디어

파스칼의 삼각형은 이항 계수의 점화식 C(n, r) = C(n-1, r-1) + C(n-1, r)을 2차원 배열로 구현한 것이다. n개에서 r개를 선택할 때, 특정 원소를 포함하는 경우 C(n-1, r-1)와 포함하지 않는 경우 C(n-1, r)로 나누어 생각하면 이 점화식이 성립한다.

N ≤ 30이므로 전처리 비용이 매우 작고, 오버플로우 없이 int64로 충분히 표현할 수 있다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N² + T) |
| 공간 복잡도 | O(N²) |

파스칼의 삼각형 전처리에 O(N²), 각 쿼리 응답에 O(1)이므로 전체 시간 복잡도는 O(N² + T)이다.

## 대안적 접근

- 팩토리얼을 직접 계산하여 nCr = n! / (r! × (n-r)!)로 구할 수 있으나, 나눗셈 과정에서 정수 오버플로우나 정밀도 문제가 발생할 수 있다. 파스칼의 삼각형은 덧셈만 사용하므로 이런 문제가 없다.
