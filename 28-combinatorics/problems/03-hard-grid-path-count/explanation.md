# 격자 경로의 수 - 해설

## 접근 방식

1. 장애물이 없는 경우 (1,1)에서 (N,M)까지의 경로 수는 C(N+M-2, N-1)이다
2. 장애물이 있는 경우, 포함-배제 원리를 적용한다
3. 장애물을 행, 열 순으로 정렬한다
4. dp[i]를 "(1,1)에서 i번째 장애물까지 다른 장애물을 지나지 않는 경로 수"로 정의한다
5. 각 장애물에 대해, 이전 장애물을 경유하는 경로를 빼준다
6. 도착점 (N,M)을 마지막 장애물로 추가하여 최종 답을 구한다

## 핵심 아이디어

격자에서 오른쪽과 아래쪽으로만 이동할 때, (r1,c1)에서 (r2,c2)까지의 경로 수는 조합 C(dr+dc, dr)이다 (dr = r2-r1, dc = c2-c1). 이는 총 dr+dc번의 이동 중 dr번을 아래쪽으로 선택하는 경우의 수와 같다.

장애물이 있을 때는 DP + 포함-배제를 사용한다. dp[i]를 계산할 때, (1,1)에서 장애물 i까지의 전체 경로 수에서 중간에 다른 장애물 j를 반드시 지나는 경로 수(dp[j] × pathCount(j→i))를 빼준다. 장애물을 정렬해두면 j < i인 장애물만 고려하면 된다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N + K² + K log K) |
| 공간 복잡도 | O(N + K) |

팩토리얼 전처리 O(N+M), 장애물 정렬 O(K log K), DP 계산 O(K²)이다. K ≤ 2,000이므로 K²은 최대 4,000,000으로 충분히 빠르다.

## 대안적 접근

- 장애물이 없는 경우 단순히 C(N+M-2, N-1)로 O(1)에 구할 수 있다.
- 격자 크기가 작으면 (N, M ≤ 1,000) 2차원 DP로 직접 경로 수를 세는 방법도 가능하지만, N, M이 최대 100,000이므로 조합론적 접근이 필수이다.
