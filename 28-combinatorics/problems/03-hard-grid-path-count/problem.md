# 격자 경로의 수

**난이도:** 상

## 문제 설명

N×M 크기의 격자가 있다. 왼쪽 위 (1, 1)에서 오른쪽 아래 (N, M)까지 이동하려고 한다. 이동은 오른쪽 또는 아래쪽으로만 가능하다.

격자 위에 K개의 장애물이 놓여 있어 해당 칸은 지나갈 수 없다. 장애물을 피하면서 (1, 1)에서 (N, M)까지 도달하는 경로의 수를 1,000,000,007 (10⁹ + 7)로 나눈 나머지를 구하시오.

## 입력 형식

- 첫째 줄에 세 정수 N, M, K가 공백으로 구분되어 주어진다
- 다음 K개의 줄에 각각 장애물의 위치 r, c가 주어진다 (r행 c열)

## 출력 형식

- (1, 1)에서 (N, M)까지의 경로 수를 1,000,000,007로 나눈 나머지를 출력한다

## 제약 조건

- 1 ≤ N, M ≤ 100,000
- 0 ≤ K ≤ 2,000
- 장애물은 (1, 1)과 (N, M)에 놓이지 않는다
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
3 3 1
2 2
```

### 예제 출력 1

```
2
```

### 예제 입력 2

```
4 4 0
```

### 예제 출력 2

```
20
```
