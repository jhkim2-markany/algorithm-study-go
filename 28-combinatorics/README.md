# 28. 조합론

주어진 집합에서 원소를 선택하거나 배열하는 경우의 수를 계산하는 알고리즘 유형이다. 순열, 조합, 이항 계수, 파스칼의 삼각형 등이 핵심이다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이

## 문제 목록

| 번호 | 문제 | 난이도 |
| --- | --- | --- |
| 01 | 이항 계수 계산 | 하 |
| 02 | 조합의 수 (모듈러 연산) | 중 |
| 03 | 격자 경로의 수 | 상 |
