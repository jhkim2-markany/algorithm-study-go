# 모듈러 거듭제곱 - 해설

## 접근 방식

1. A, B, M을 입력받는다
2. 분할 정복 기반의 빠른 거듭제곱(Binary Exponentiation)을 적용한다
3. B의 이진 표현을 이용하여 A^B mod M을 O(log B)에 계산한다
4. 결과를 출력한다

## 핵심 아이디어

빠른 거듭제곱은 지수 B를 이진수로 분해하는 아이디어에 기반한다. 예를 들어 A^13을 계산할 때, 13 = 1101(2)이므로:

- A^13 = A^8 × A^4 × A^1

이렇게 하면 13번 곱하는 대신 A, A^2, A^4, A^8만 계산하면 된다.

알고리즘의 핵심 루프:
- B가 홀수이면 현재 결과에 A를 곱한다
- A를 제곱하고 B를 반으로 나눈다
- B가 0이 될 때까지 반복한다

모든 곱셈 후에 mod M을 적용하여 중간 결과가 오버플로되지 않도록 한다. 이는 모듈러 연산의 성질 `(a * b) % m = ((a % m) * (b % m)) % m`에 의해 정당하다.

B=0인 경우 A^0 = 1이므로 루프에 진입하지 않고 초기값 1이 반환된다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(log B) |
| 공간 복잡도 | O(1) |

B를 매 단계마다 반으로 나누므로 반복 횟수는 O(log B)이다. B가 10^9일 때도 약 30번의 곱셈만 수행한다.

## 대안적 접근

- **재귀적 분할 정복**: 반복문 대신 재귀로 구현할 수 있다. `power(a, n) = power(a, n/2)^2 * (n이 홀수면 a)` 형태이다. 다만 재귀 깊이가 O(log B)이므로 스택 오버플로 위험은 낮지만, 반복문 방식이 더 효율적이다.
- **페르마 소정리 활용**: M이 소수이고 A와 M이 서로소일 때, A^(M-1) ≡ 1 (mod M)을 이용하여 지수를 줄일 수 있다. 이는 모듈러 역원 계산에 자주 활용된다.
