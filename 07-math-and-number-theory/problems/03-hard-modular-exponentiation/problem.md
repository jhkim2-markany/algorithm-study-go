# 모듈러 거듭제곱

**난이도:** 상

## 문제 설명

세 자연수 A, B, M이 주어졌을 때, A^B mod M의 값을 구하시오.

A^B는 A를 B번 곱한 값이다. B가 매우 클 수 있으므로 직접 곱셈을 반복하면 시간 초과가 발생한다. 효율적인 거듭제곱 알고리즘을 사용해야 한다.

## 입력 형식

- 첫째 줄에 세 자연수 A, B, M이 공백으로 구분되어 주어진다
- 1 ≤ A ≤ 1,000,000,000
- 0 ≤ B ≤ 1,000,000,000
- 2 ≤ M ≤ 1,000,000,000

## 출력 형식

- 첫째 줄에 A^B mod M의 값을 출력한다

## 제약 조건

- 1 ≤ A ≤ 1,000,000,000
- 0 ≤ B ≤ 1,000,000,000
- 2 ≤ M ≤ 1,000,000,000
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
2 10 1000
```

### 예제 출력 1

```
24
```

### 예제 입력 2

```
3 13 1000000007
```

### 예제 출력 2

```
1594323
```

### 예제 입력 3

```
5 0 7
```

### 예제 출력 3

```
1
```
