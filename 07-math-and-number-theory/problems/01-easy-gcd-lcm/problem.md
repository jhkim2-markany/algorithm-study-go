# GCD/LCM 계산

**난이도:** 하

## 문제 설명

두 자연수 A와 B가 주어졌을 때, A와 B의 최대공약수(GCD)와 최소공배수(LCM)를 구하시오.

최대공약수는 두 수를 동시에 나눌 수 있는 가장 큰 양의 정수이고, 최소공배수는 두 수의 공통 배수 중 가장 작은 양의 정수이다.

## 입력 형식

- 첫째 줄에 두 자연수 A와 B가 공백으로 구분되어 주어진다 (1 ≤ A, B ≤ 1,000,000)

## 출력 형식

- 첫째 줄에 A와 B의 최대공약수를 출력한다
- 둘째 줄에 A와 B의 최소공배수를 출력한다

## 제약 조건

- 1 ≤ A, B ≤ 1,000,000
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
12 8
```

### 예제 출력 1

```
4
24
```

### 예제 입력 2

```
48 18
```

### 예제 출력 2

```
6
144
```
