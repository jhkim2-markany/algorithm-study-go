# GCD/LCM 계산 - 해설

## 접근 방식

1. 두 자연수 A, B를 입력받는다
2. 유클리드 호제법으로 최대공약수(GCD)를 구한다
3. GCD를 이용하여 최소공배수(LCM)를 계산한다
4. 결과를 출력한다

## 핵심 아이디어

유클리드 호제법은 `GCD(a, b) = GCD(b, a % b)`라는 성질을 이용한다. b가 0이 될 때까지 나머지 연산을 반복하면 마지막 a 값이 GCD이다.

LCM은 `A * B / GCD(A, B)` 공식으로 구한다. 오버플로를 방지하기 위해 `A / GCD * B` 순서로 계산하는 것이 안전하다.

예를 들어 A=12, B=8인 경우:
- GCD(12, 8) → GCD(8, 4) → GCD(4, 0) → 4
- LCM = 12 / 4 * 8 = 24

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(log(min(A, B))) |
| 공간 복잡도 | O(1) |

유클리드 호제법은 매 단계마다 나머지가 절반 이하로 줄어들므로 O(log(min(A, B)))에 수렴한다.

## 대안적 접근

- **소인수분해 방식**: 두 수를 소인수분해한 뒤 공통 소인수의 최소 지수 곱으로 GCD를 구할 수 있다. 다만 소인수분해 자체가 O(√N)이므로 유클리드 호제법보다 느리다.
