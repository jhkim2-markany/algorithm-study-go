# 에라토스테네스의 체 - 해설

## 접근 방식

1. 크기 N+1의 불리언 배열을 생성하고 2 이상의 인덱스를 true로 초기화한다
2. 2부터 √N까지 순회하며, 현재 수가 소수이면 그 배수를 모두 false로 표시한다
3. 배수 제거 시 i*i부터 시작한다 (i*2, i*3, ... 은 이미 이전 단계에서 처리됨)
4. 배열에서 true인 인덱스를 모아 소수 목록을 만든다

## 핵심 아이디어

에라토스테네스의 체는 "합성수는 반드시 √N 이하의 소인수를 가진다"는 성질을 이용한다. 따라서 √N까지만 체를 수행하면 N 이하의 모든 소수를 구할 수 있다.

배수 제거를 i*i부터 시작하는 이유는, i*2, i*3, ..., i*(i-1)은 이미 2, 3, ..., (i-1)의 배수로서 이전 단계에서 제거되었기 때문이다. 이 최적화로 불필요한 연산을 줄인다.

예를 들어 N=10인 경우:
- i=2: 4, 6, 8, 10을 제거
- i=3: 9를 제거 (6은 이미 제거됨)
- √10 ≈ 3.16이므로 여기서 종료
- 남은 소수: 2, 3, 5, 7

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N log log N) |
| 공간 복잡도 | O(N) |

에라토스테네스의 체의 시간 복잡도는 조화급수의 소수 부분합에 의해 O(N log log N)이다. 이는 N이 100만일 때도 충분히 빠르다.

## 대안적 접근

- **단순 소수 판별 반복**: 각 수마다 √N까지 나누어보는 방식은 O(N√N)으로 N이 클 때 비효율적이다.
- **세그먼트 체**: N이 매우 클 때(10^12 이상) 메모리를 절약하기 위해 구간별로 체를 수행하는 방법이 있다.
