# 07. 수학과 정수론

코딩테스트에서 자주 등장하는 수학적 개념과 정수론 알고리즘을 다루는 유형이다. GCD/LCM, 소수 판별, 에라토스테네스의 체, 모듈러 연산 등 핵심 기법을 학습한다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
