# 차선 최소 신장 트리

**난이도:** 상

## 문제 설명

N개의 정점과 M개의 간선으로 이루어진 가중치 무방향 그래프가 주어진다. 이 그래프의 차선 최소 신장 트리(Second-best MST)의 가중치를 구하시오.

차선 최소 신장 트리란, 최소 신장 트리(MST)를 제외한 모든 신장 트리 중에서 가중치 합이 가장 작은 신장 트리이다. 차선 MST의 가중치는 MST의 가중치와 같을 수 있다.

차선 MST는 MST에서 간선 하나를 제거하고 다른 간선 하나를 추가하여 얻을 수 있다. 즉, MST와 정확히 하나의 간선만 다르다.

## 입력 형식

- 첫째 줄에 정점의 수 N과 간선의 수 M이 주어진다
- 둘째 줄부터 M개의 줄에 각 간선의 정보 u, v, w가 주어진다

## 출력 형식

- 차선 최소 신장 트리의 가중치를 출력한다

## 제약 조건

- 2 ≤ N ≤ 1,000
- N-1 ≤ M ≤ 100,000
- 1 ≤ u, v ≤ N (u ≠ v)
- 1 ≤ w ≤ 1,000,000
- 그래프는 연결 그래프이다
- 간선이 N개 이상이므로 차선 MST가 반드시 존재한다
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
4 5
1 2 1
2 3 2
3 4 3
1 3 3
2 4 5
```

### 예제 출력 1

```
7
```

### 예제 입력 2

```
3 3
1 2 1
2 3 2
1 3 3
```

### 예제 출력 2

```
4
```
