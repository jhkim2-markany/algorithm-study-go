# 최소 비용 연결

**난이도:** 하

## 문제 설명

N개의 도시와 M개의 도로 정보가 주어진다. 각 도로는 두 도시를 양방향으로 연결하며 건설 비용이 있다. 모든 도시를 연결하는 데 필요한 최소 비용을 구하시오.

모든 도시가 직접 또는 간접적으로 연결되어야 한다. 즉, 임의의 두 도시 사이에 경로가 존재해야 한다.

## 입력 형식

- 첫째 줄에 도시의 수 N과 도로의 수 M이 주어진다
- 둘째 줄부터 M개의 줄에 각 도로의 정보 u, v, w가 주어진다 (u번 도시와 v번 도시를 연결하는 비용 w)

## 출력 형식

- 모든 도시를 연결하는 최소 비용을 출력한다

## 제약 조건

- 1 ≤ N ≤ 10,000
- 1 ≤ M ≤ 100,000
- 1 ≤ u, v ≤ N
- 1 ≤ w ≤ 1,000,000
- 도시 번호는 1부터 시작한다
- 모든 도시를 연결할 수 있는 입력만 주어진다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
4 5
1 2 1
1 3 4
2 3 2
2 4 5
3 4 3
```

### 예제 출력 1

```
6
```

### 예제 입력 2

```
3 3
1 2 10
2 3 20
1 3 30
```

### 예제 출력 2

```
30
```
