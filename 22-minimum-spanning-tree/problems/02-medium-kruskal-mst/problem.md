# 크루스칼 MST

**난이도:** 중

## 문제 설명

N개의 정점과 M개의 간선으로 이루어진 가중치 무방향 그래프가 주어진다. 크루스칼 알고리즘을 사용하여 최소 신장 트리(MST)를 구하고, MST의 총 가중치와 MST에 포함된 간선들을 출력하시오.

MST에 포함된 간선은 선택된 순서대로 출력한다. 즉, 가중치가 작은 간선부터 출력한다. 가중치가 같은 간선이 여러 개 있을 경우, 간선의 첫 번째 정점 번호가 작은 것을 먼저 선택하고, 첫 번째 정점 번호도 같으면 두 번째 정점 번호가 작은 것을 먼저 선택한다.

## 입력 형식

- 첫째 줄에 정점의 수 N과 간선의 수 M이 주어진다
- 둘째 줄부터 M개의 줄에 각 간선의 정보 u, v, w가 주어진다 (u번 정점과 v번 정점을 연결하는 가중치 w)
- 각 간선에서 u < v가 보장된다

## 출력 형식

- 첫째 줄에 MST의 총 가중치를 출력한다
- 둘째 줄부터 MST에 포함된 간선을 선택 순서대로 한 줄에 하나씩 "u v w" 형식으로 출력한다

## 제약 조건

- 2 ≤ N ≤ 10,000
- N-1 ≤ M ≤ 100,000
- 1 ≤ u < v ≤ N
- 1 ≤ w ≤ 1,000,000
- 그래프는 연결 그래프이다
- 같은 두 정점을 연결하는 간선은 최대 하나이다
- 시간 제한: 1초
- 메모리 제한: 256MB
