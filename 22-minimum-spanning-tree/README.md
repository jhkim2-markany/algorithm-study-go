# 22. 최소 신장 트리

가중치 무방향 그래프에서 모든 정점을 연결하면서 간선 가중치의 합이 최소인 트리를 찾는 알고리즘이다. Kruskal 알고리즘은 간선을 정렬하여 Union-Find로 사이클을 판별하고, Prim 알고리즘은 우선순위 큐로 인접 정점을 확장하며 MST를 구성한다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
