# 64. 볼록 껍질 트릭 (Convex Hull Trick)

DP 점화식에서 일차 함수(선형 함수)들의 최솟값 또는 최댓값을 빠르게 구하는 최적화 기법이다. O(N²) DP를 O(N log N) 또는 O(N)으로 줄일 수 있으며, 일차 함수 집합을 볼록 껍질 형태로 관리하여 쿼리에 효율적으로 응답한다. Li Chao Tree를 사용하면 쿼리 순서에 제약 없이 처리할 수 있다.
