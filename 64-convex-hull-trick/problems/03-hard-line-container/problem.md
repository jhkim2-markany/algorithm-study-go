# 직선 컨테이너

**난이도:** 상

## 문제 설명

직선 추가와 최솟값 쿼리가 임의의 순서로 섞여서 주어진다.

각 직선은 y = mᵢx + bᵢ 형태이며, 각 쿼리는 정수 x에 대해 현재까지 추가된 모든 직선 중 y값이 최소인 값을 구하는 것이다.

## 입력 형식

- 첫째 줄에 연산의 수 Q가 주어진다.
- 다음 Q개의 줄에 연산이 주어진다:
  - `1 m b`: 직선 y = mx + b를 추가한다
  - `2 x`: 현재까지 추가된 직선들 중 x에서의 최솟값을 출력한다

## 출력 형식

- 각 타입 2 쿼리에 대해 최솟값을 한 줄에 하나씩 출력한다.

## 제약 조건

- 1 ≤ Q ≤ 300,000
- -1,000,000 ≤ m, b ≤ 1,000,000
- -1,000,000 ≤ x ≤ 1,000,000
- 타입 2 쿼리 전에 최소 하나의 직선이 추가되어 있음이 보장된다

## 예제

### 예제 입력 1

```
7
1 1 0
1 -1 4
2 1
2 3
1 0 1
2 2
2 0
```

### 예제 출력 1

```
1
1
1
0
```

### 예제 입력 2

```
4
1 2 1
1 -3 10
2 3
2 0
```

### 예제 출력 2

```
1
1
```

## 힌트

직선 추가와 쿼리가 임의 순서로 섞여 있으므로, 기울기 정렬이 불가능하다. Li Chao Tree를 사용하면 삽입과 쿼리 모두 O(log X)에 처리할 수 있다.

예제 1 설명:
- 직선 y=x, y=-x+4 추가 후:
  - x=1 쿼리: min(1, 3) = 1
  - x=3 쿼리: min(3, 1) = 1
- 직선 y=1 추가 후:
  - x=2 쿼리: min(2, 2, 1) = 1
  - x=0 쿼리: min(0, 4, 1) = 0
