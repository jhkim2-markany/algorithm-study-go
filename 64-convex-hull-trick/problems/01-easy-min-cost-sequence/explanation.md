# 최소 비용 수열 분할 - 해설

## 접근 방식

1. 누적합 sum[i]를 구한다
2. DP 점화식을 세운다: `dp[i] = min(dp[j] + (sum[i] - sum[j])²)` (0 ≤ j < i)
3. 점화식을 전개하여 일차 함수 형태로 변환한다
4. 볼록 껍질 트릭을 적용하여 O(N)에 해결한다

## 핵심 아이디어

점화식을 전개하면:

```
dp[i] = min(dp[j] + sum[i]² - 2·sum[i]·sum[j] + sum[j]²)
      = sum[i]² + min((-2·sum[j])·sum[i] + (dp[j] + sum[j]²))
```

여기서 `f_j(x) = (-2·sum[j])·x + (dp[j] + sum[j]²)`로 놓으면, x = sum[i]에서의 최솟값을 구하는 문제가 된다.

기울기 `-2·sum[j]`는 j가 증가하면 단조 감소하고, 쿼리 x = sum[i]는 i가 증가하면 단조 증가한다. 따라서 단조 CHT를 적용하여 포인터 방식으로 O(N)에 처리할 수 있다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N) |
| 공간 복잡도 | O(N) |

나이브 DP는 O(N²)이지만, CHT를 적용하면 각 직선이 최대 한 번 추가되고 한 번 제거되며, 포인터도 전체적으로 O(N)만 전진하므로 총 O(N)이다.
