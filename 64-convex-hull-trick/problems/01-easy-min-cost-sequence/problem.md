# 최소 비용 수열 분할

**난이도:** 하

## 문제 설명

길이 N인 양의 정수 수열 a₁, a₂, ..., aₙ이 주어진다. 이 수열을 연속된 구간들로 분할하려고 한다.

한 구간 [l, r]의 비용은 (구간 합)²으로 정의한다. 즉, (aₗ + aₗ₊₁ + ... + aᵣ)²이다.

수열 전체를 빠짐없이 분할할 때, 모든 구간 비용의 합의 최솟값을 구하라.

## 입력 형식

- 첫째 줄에 수열의 길이 N이 주어진다.
- 둘째 줄에 수열의 원소 a₁, a₂, ..., aₙ이 공백으로 구분되어 주어진다.

## 출력 형식

- 최소 비용을 출력한다.

## 제약 조건

- 1 ≤ N ≤ 100,000
- 1 ≤ aᵢ ≤ 1,000

## 예제

### 예제 입력 1

```
5
3 1 4 1 5
```

### 예제 출력 1

```
52
```

### 예제 입력 2

```
3
1 1 1
```

### 예제 출력 2

```
3
```

### 예제 입력 3

```
1
10
```

### 예제 출력 3

```
100
```

## 힌트

예제 1에서 최적 분할은 [3,1,4,1] [5]이며, 비용은 9² + 5² = 81... 이 아니라, [3] [1] [4] [1] [5]로 각각 분할하면 9+1+16+1+25=52가 된다. 각 원소를 개별 구간으로 만드는 것이 항상 최적은 아니며, 문제의 변형에서 구간 수에 대한 추가 비용이 붙을 수 있다.

이 기본 문제에서는 구간 수 제한이 없으므로, CHT를 연습하기 위한 기본 형태로 풀어보자.
