# 다리 놓기

**난이도:** 중

## 문제 설명

일직선 위에 N개의 기둥이 세워져 있다. i번째 기둥의 높이는 hᵢ이고, 무게는 wᵢ이다.

일부 기둥을 제거하고 남은 기둥들 사이에 다리를 놓으려고 한다. 인접한 두 기둥 i, j (i < j) 사이에 다리를 놓는 비용은 (hᵢ - hⱼ)²이다.

기둥을 제거하는 비용은 해당 기둥의 무게 wᵢ이다. 첫 번째 기둥과 마지막 기둥은 반드시 남겨야 한다.

남은 기둥들 사이의 다리 비용 합 + 제거된 기둥들의 무게 합의 최솟값을 구하라.

## 입력 형식

- 첫째 줄에 기둥의 수 N이 주어진다.
- 둘째 줄에 각 기둥의 높이 h₁, h₂, ..., hₙ이 공백으로 구분되어 주어진다.
- 셋째 줄에 각 기둥의 무게 w₁, w₂, ..., wₙ이 공백으로 구분되어 주어진다.

## 출력 형식

- 최소 총 비용을 출력한다.

## 제약 조건

- 2 ≤ N ≤ 100,000
- 1 ≤ hᵢ ≤ 1,000,000
- 1 ≤ wᵢ ≤ 1,000,000

## 예제

### 예제 입력 1

```
6
3 8 7 1 6 2
5 1 7 4 3 2
```

### 예제 출력 1

```
16
```

### 예제 입력 2

```
2
10 20
5 5
```

### 예제 출력 2

```
100
```

### 예제 입력 3

```
3
1 1 1
100 1 100
```

### 예제 출력 3

```
0
```

## 힌트

예제 1에서 기둥 2, 4, 5를 제거하면 (비용 1+4+3=8), 남은 기둥은 [3, 7, 2]이고 다리 비용은 (3-7)² + (7-2)² = 16+25 = 41로 총 49이다. 기둥 4를 제거하면 (비용 4), 남은 기둥은 [3, 8, 7, 6, 2]이고 다리 비용은 25+1+1+16=43으로 총 47이다. 최적 해는 기둥 2, 3, 5를 제거하여 (비용 1+7+3=11), 남은 기둥 [3, 1, 2]로 다리 비용 4+1=5, 총 16이다.

O(N²) DP를 세운 뒤, CHT로 O(N log N)에 최적화할 수 있다.
