# 09. 파라메트릭 서치

최적화 문제를 결정 문제로 변환하여 이진 탐색으로 해결하는 알고리즘 유형이다. "최솟값의 최댓값" 또는 "최댓값의 최솟값"과 같은 문제에서 답을 이진 탐색의 대상으로 놓고, 조건 만족 여부를 판별하여 최적해를 찾는다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
