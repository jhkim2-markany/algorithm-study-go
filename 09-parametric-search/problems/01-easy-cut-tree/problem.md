# 나무 자르기

**난이도:** 하

## 문제 설명

상근이는 나무 M미터가 필요하다. 근처에 나무를 구할 곳이 없기 때문에, 벌목기를 이용하여 나무를 자르려고 한다.

벌목기는 다음과 같이 동작한다. 먼저, 절단기에 높이 H를 설정한다. 그러면 한 줄에 연속해 있는 나무를 모두 절단한다. 높이가 H보다 큰 나무는 H 위의 부분이 잘리고, 낮은 나무는 잘리지 않는다.

예를 들어, 한 줄에 나무의 높이가 20, 15, 10, 17이고 절단기 높이를 15로 설정하면, 잘린 나무의 길이는 각각 5, 0, 0, 2이다. 총 7미터의 나무를 가져갈 수 있다.

적어도 M미터의 나무를 집에 가져가기 위해서 절단기에 설정할 수 있는 높이의 최댓값을 구하시오.

## 입력 형식

- 첫째 줄에 나무의 수 N과 필요한 나무의 길이 M이 주어진다 (1 ≤ N ≤ 100,000, 1 ≤ M ≤ 1,000,000,000)
- 둘째 줄에 각 나무의 높이가 공백으로 구분되어 주어진다
- 나무의 높이는 1,000,000,000 이하의 양의 정수이다

## 출력 형식

- 적어도 M미터의 나무를 가져가기 위해 설정할 수 있는 높이의 최댓값을 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- 1 ≤ M ≤ 1,000,000,000
- 나무 높이 ≤ 1,000,000,000
- 항상 M미터를 얻을 수 있는 입력만 주어진다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
4 7
20 15 10 17
```

### 예제 출력 1

```
15
```

### 예제 입력 2

```
5 20
4 42 40 26 46
```

### 예제 출력 2

```
36
```
