# 37. 배낭 문제

주어진 용량 제한 내에서 물건들의 가치 합을 최대화하는 최적화 문제이다. 0/1 배낭, 완전 배낭, 분할 배낭 등 다양한 변형이 있으며, 동적 프로그래밍의 대표적인 응용 분야이다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
