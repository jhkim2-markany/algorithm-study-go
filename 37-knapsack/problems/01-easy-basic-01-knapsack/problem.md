# 기본 0/1 배낭

**난이도:** 하

## 문제 설명

N개의 물건이 있다. 각 물건은 무게 W_i와 가치 V_i를 가진다. 용량이 K인 배낭에 물건들을 담아 가치의 합을 최대화하라. 각 물건은 최대 1개만 담을 수 있다.

## 입력 형식

첫째 줄에 물건의 수 N과 배낭의 용량 K가 주어진다. (1 ≤ N ≤ 100, 1 ≤ K ≤ 100,000)

둘째 줄부터 N개의 줄에 각 물건의 무게 W_i와 가치 V_i가 주어진다. (1 ≤ W_i ≤ 100,000, 0 ≤ V_i ≤ 1,000)

## 출력 형식

배낭에 담을 수 있는 물건들의 최대 가치 합을 출력한다.

## 제약 조건

- 1 ≤ N ≤ 100
- 1 ≤ K ≤ 100,000
- 1 ≤ W_i ≤ 100,000
- 0 ≤ V_i ≤ 1,000

## 예제

### 예제 입력 1

```
4 7
6 13
4 8
3 6
5 12
```

### 예제 출력 1

```
14
```
