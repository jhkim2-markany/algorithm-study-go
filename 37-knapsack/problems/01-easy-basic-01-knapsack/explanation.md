# 기본 0/1 배낭 - 해설

## 접근 방식

1. 1차원 DP 배열을 사용하여 공간을 최적화한다
2. 각 물건에 대해 용량을 역순으로 순회한다
3. `dp[w] = max(dp[w], dp[w-weight[i]] + value[i])`로 갱신한다
4. 역순 순회를 통해 각 물건이 최대 1번만 선택되도록 보장한다

## 핵심 아이디어

0/1 배낭 문제의 핵심은 용량을 역순으로 순회하는 것이다. 정순으로 순회하면 같은 물건이 여러 번 선택될 수 있지만, 역순으로 순회하면 이전 물건까지의 결과만 참조하게 되어 각 물건을 최대 1번만 사용하는 것이 보장된다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N × K) |
| 공간 복잡도 | O(K) |
