# 완전 배낭 - 해설

## 접근 방식

1. 1차원 DP 배열을 사용한다
2. 각 물건에 대해 용량을 정순으로 순회한다
3. `dp[w] = max(dp[w], dp[w-weight[i]] + value[i])`로 갱신한다
4. 정순 순회를 통해 같은 물건을 여러 번 선택할 수 있게 한다

## 핵심 아이디어

0/1 배낭과 완전 배낭의 차이는 용량 순회 방향에 있다. 0/1 배낭은 역순으로 순회하여 각 물건을 1번만 사용하지만, 완전 배낭은 정순으로 순회하여 같은 물건을 여러 번 사용할 수 있게 한다. 정순 순회 시 `dp[w-weight[i]]`가 이미 현재 물건을 포함한 값일 수 있으므로 중복 선택이 자연스럽게 허용된다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N × K) |
| 공간 복잡도 | O(K) |
