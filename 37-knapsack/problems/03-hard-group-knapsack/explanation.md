# 그룹 배낭 - 해설

## 접근 방식

1. 1차원 DP 배열을 사용한다
2. 각 그룹에 대해 용량을 역순으로 순회한다
3. 각 용량에서 그룹 내 모든 물건을 시도하여 최대 가치를 갱신한다
4. 역순 순회와 그룹 단위 처리로 각 그룹에서 최대 1개만 선택되도록 보장한다

## 핵심 아이디어

그룹 배낭의 핵심은 0/1 배낭의 역순 순회를 그룹 단위로 확장하는 것이다. 용량을 역순으로 순회하면서 각 용량에서 그룹 내 모든 물건을 시도한다. 이때 dp 배열이 아직 현재 그룹의 물건을 반영하지 않은 상태이므로, 각 그룹에서 최대 1개의 물건만 선택되는 것이 보장된다.

일반 0/1 배낭에서는 물건 → 용량 순서로 순회하지만, 그룹 배낭에서는 그룹 → 용량 → 그룹 내 물건 순서로 순회한다는 점이 다르다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(K × ΣM_i) |
| 공간 복잡도 | O(K) |
