# 그룹 배낭

**난이도:** 상

## 문제 설명

N개의 그룹이 있고, 각 그룹에는 여러 개의 물건이 있다. 각 그룹에서 최대 1개의 물건만 선택할 수 있다. 용량이 K인 배낭에 물건들을 담아 가치의 합을 최대화하라.

## 입력 형식

첫째 줄에 그룹의 수 N과 배낭의 용량 K가 주어진다. (1 ≤ N ≤ 100, 1 ≤ K ≤ 10,000)

이후 각 그룹에 대해:
- 첫 줄에 그룹 내 물건의 수 M_i가 주어진다. (1 ≤ M_i ≤ 100)
- 다음 M_i개의 줄에 각 물건의 무게 W와 가치 V가 주어진다. (1 ≤ W ≤ 10,000, 1 ≤ V ≤ 10,000)

## 출력 형식

배낭에 담을 수 있는 물건들의 최대 가치 합을 출력한다.

## 제약 조건

- 1 ≤ N ≤ 100
- 1 ≤ K ≤ 10,000
- 1 ≤ M_i ≤ 100
- 1 ≤ W ≤ 10,000
- 1 ≤ V ≤ 10,000

## 예제

### 예제 입력 1

```
2 10
3
2 3
3 5
4 7
2
3 4
5 8
```

### 예제 출력 1

```
15
```
