# 35. 오프라인 쿼리

쿼리를 입력 순서대로 처리하지 않고, 효율적인 순서로 재배열하여 처리하는 알고리즘 유형이다. Mo's 알고리즘, 쿼리 정렬, 제곱근 분할(sqrt decomposition) 등이 핵심이다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
