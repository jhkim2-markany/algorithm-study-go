# 구간 서로 다른 수 개수

**난이도:** 하

## 문제 설명

길이 N인 정수 배열 A가 주어진다. Q개의 쿼리가 주어지며, 각 쿼리는 구간 [L, R]이다. 각 쿼리에 대해 A[L]부터 A[R]까지의 구간에서 서로 다른 수의 개수를 구하시오.

## 입력 형식

- 첫째 줄에 배열의 크기 N과 쿼리의 수 Q가 주어진다
- 둘째 줄에 배열의 원소 A[1], A[2], ..., A[N]이 공백으로 구분되어 주어진다
- 다음 Q개의 줄에 각각 L, R이 주어진다 (1-indexed)

## 출력 형식

- 각 쿼리에 대해 구간 [L, R]에서 서로 다른 수의 개수를 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- 1 ≤ Q ≤ 100,000
- 1 ≤ A[i] ≤ 1,000,000
- 1 ≤ L ≤ R ≤ N
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
8 5
1 2 1 3 2 1 4 3
1 4
2 6
3 8
1 8
5 7
```

### 예제 출력 1

```
3
3
4
4
3
```
