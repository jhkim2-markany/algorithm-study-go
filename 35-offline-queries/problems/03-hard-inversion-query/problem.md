# 구간 역전 쌍 카운팅

**난이도:** 상

## 문제 설명

길이 N인 정수 배열 A가 주어진다. Q개의 쿼리가 주어지며, 각 쿼리는 구간 [L, R]이다. 각 쿼리에 대해 A[L]부터 A[R]까지의 구간에서 역전 쌍(inversion pair)의 개수를 구하시오.

역전 쌍이란 L ≤ i < j ≤ R이면서 A[i] > A[j]인 쌍 (i, j)를 말한다.

## 입력 형식

- 첫째 줄에 배열의 크기 N과 쿼리의 수 Q가 주어진다
- 둘째 줄에 배열의 원소 A[1], A[2], ..., A[N]이 공백으로 구분되어 주어진다
- 다음 Q개의 줄에 각각 L, R이 주어진다 (1-indexed)

## 출력 형식

- 각 쿼리에 대해 구간 [L, R]에서의 역전 쌍 개수를 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- 1 ≤ Q ≤ 100,000
- 1 ≤ A[i] ≤ 100,000
- 1 ≤ L ≤ R ≤ N
- 시간 제한: 3초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5 3
3 1 4 1 5
1 3
1 5
2 4
```

### 예제 출력 1

```
1
3
1
```
