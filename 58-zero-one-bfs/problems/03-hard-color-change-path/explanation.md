# 색상 변경 최소 비용 경로 - 해설

## 접근 방식

1. 상태를 (노드, 현재 선택 색상)으로 정의한다. 색상은 R(0) 또는 B(1)이다
2. 같은 색 간선을 따라 이동하면 비용 0, 다른 색 간선을 따라 이동하면 비용 1 (색상 변경)
3. 가중치가 0 또는 1이므로 0-1 BFS를 적용한다
4. 시작 상태: (1, R)과 (1, B) 모두 비용 0으로 시작 (출발 시 색상 선택)
5. 도착: dist[N][0]과 dist[N][1] 중 최솟값이 답이다

## 핵심 아이디어

이 문제의 핵심은 **상태 공간 확장**이다. 단순히 노드만으로는 최적 경로를 결정할 수 없다. 같은 노드에 도달하더라도 현재 선택한 색상에 따라 이후 비용이 달라지기 때문이다.

상태를 (노드, 색상)으로 확장하면, 총 2N개의 상태가 생기고 각 상태 간 전이 비용이 0 또는 1이 된다. 이를 0-1 BFS로 풀면 O(N+M)에 해결할 수 있다.

```
상태 전이:
  (v, R) --R간선--> (u, R): 비용 0 (같은 색)
  (v, R) --B간선--> (u, B): 비용 1 (색상 변경)
  (v, B) --B간선--> (u, B): 비용 0 (같은 색)
  (v, B) --R간선--> (u, R): 비용 1 (색상 변경)
```

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N + M) |
| 공간 복잡도 | O(N + M) |

상태 수가 2N, 전이 수가 2M이므로 0-1 BFS의 시간 복잡도는 O(N+M)이다.

## 대안적 접근

- 다익스트라: 상태 공간 (노드, 색상)에서 다익스트라를 적용하면 O((N+M) log N)으로 풀 수 있다
- BFS 레이어링: 색상 변경 횟수를 레이어로 두고 BFS를 반복하는 방식도 가능하지만, 0-1 BFS가 가장 효율적이다
