# 색상 변경 최소 비용 경로

**난이도:** 상

## 문제 설명

N개의 노드와 M개의 간선으로 이루어진 무방향 그래프가 주어진다. 각 간선에는 색상이 칠해져 있으며, 색상은 'R'(빨강) 또는 'B'(파랑) 중 하나이다.

노드 1에서 출발하여 노드 N까지 이동하려고 한다. 출발할 때 하나의 색상을 선택한다. 선택한 색상의 간선을 따라 이동하는 것은 비용이 0이다. 하지만 다른 색상의 간선을 따라 이동하려면 색상을 변경해야 하며, 이때 비용이 1이 든다. 색상을 변경하면 이후에는 변경된 색상의 간선을 비용 0으로 이동할 수 있다.

노드 1에서 노드 N까지 이동하는 최소 비용을 구하라. 도달할 수 없으면 -1을 출력한다.

## 입력 형식

- 첫째 줄에 노드의 수 N과 간선의 수 M이 주어진다.
- 둘째 줄부터 M개의 줄에 간선 정보 u, v, c가 주어진다. 이는 노드 u와 노드 v를 잇는 색상 c의 간선이 있음을 의미한다. c는 'R' 또는 'B'이다.

## 출력 형식

- 노드 1에서 노드 N까지의 최소 비용을 출력한다. 도달할 수 없으면 -1을 출력한다.

## 제약 조건

- 2 ≤ N ≤ 100,000
- 1 ≤ M ≤ 200,000
- 1 ≤ u, v ≤ N
- c는 'R' 또는 'B'
- 자기 자신으로의 간선은 없다
- 같은 두 노드 사이에 여러 간선이 있을 수 있다 (색상이 다를 수 있음)

## 예제

### 예제 입력 1

```
4 5
1 2 R
2 3 R
3 4 R
1 3 B
3 4 B
```

### 예제 출력 1

```
0
```

### 예제 입력 2

```
4 4
1 2 R
2 3 B
3 4 R
1 3 R
```

### 예제 출력 2

```
1
```

### 예제 입력 3

```
3 2
1 2 R
1 2 B
```

### 예제 출력 3

```
-1
```
