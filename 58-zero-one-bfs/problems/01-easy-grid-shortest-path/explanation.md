# 격자 최단 경로 - 해설

## 접근 방식

1. 격자를 그래프로 모델링한다. 각 칸이 노드이고, 인접한 칸 사이에 간선이 있다
2. 빈 칸으로 이동하는 간선의 가중치는 0, 벽으로 이동하는 간선의 가중치는 1이다
3. 가중치가 0 또는 1이므로 0-1 BFS를 적용한다
4. 덱을 사용하여 가중치 0 간선은 앞에, 가중치 1 간선은 뒤에 추가한다
5. (0,0)에서 시작하여 (N-1,M-1)까지의 최단 거리를 구한다

## 핵심 아이디어

격자 문제를 0-1 BFS로 변환하는 가장 기본적인 형태이다. 각 칸의 종류(빈 칸/벽)에 따라 이동 비용이 0 또는 1로 결정되므로, 덱 기반 0-1 BFS를 직접 적용할 수 있다.

좌표를 `r*m + c`로 인코딩하여 덱에 정수 하나로 저장하면 메모리와 코드가 간결해진다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N × M) |
| 공간 복잡도 | O(N × M) |

각 칸을 최대 한 번 처리하므로 O(N×M)이다.

## 대안적 접근

- 다익스트라: 우선순위 큐를 사용하면 O(NM log(NM))으로 풀 수 있지만, 0-1 BFS가 더 빠르고 간단하다
- 일반 BFS: 가중치가 다르므로 일반 BFS로는 최단 경로를 보장할 수 없다
