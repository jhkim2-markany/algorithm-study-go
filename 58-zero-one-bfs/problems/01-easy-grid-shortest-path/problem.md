# 격자 최단 경로

**난이도:** 하

## 문제 설명

N×M 격자가 주어진다. 각 칸은 빈 칸(`.`) 또는 벽(`#`)이다. 왼쪽 위 (1,1)에서 오른쪽 아래 (N,M)까지 이동하려고 한다.

상하좌우 인접한 칸으로 이동할 수 있으며, 빈 칸으로 이동하는 비용은 0이고 벽으로 이동하는 비용은 1이다.

(1,1)에서 (N,M)까지의 최소 비용을 구하라.

## 입력 형식

- 첫째 줄에 격자의 크기 N, M이 주어진다.
- 둘째 줄부터 N개의 줄에 격자 정보가 주어진다. 각 줄은 M개의 문자로 이루어져 있으며, `.`은 빈 칸, `#`은 벽을 나타낸다.

## 출력 형식

- (1,1)에서 (N,M)까지의 최소 비용을 출력한다.

## 제약 조건

- 1 ≤ N, M ≤ 1,000
- 격자의 (1,1)과 (N,M)은 항상 빈 칸이다

## 예제

### 예제 입력 1

```
4 4
..#.
.#..
.##.
....
```

### 예제 출력 1

```
0
```

### 예제 입력 2

```
3 3
.#.
###
.#.
```

### 예제 출력 2

```
2
```
