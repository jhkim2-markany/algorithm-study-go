# 벽 부수기 최단 경로 - 해설

## 접근 방식

1. 격자를 그래프로 모델링한다. 각 칸이 노드이고, 인접한 칸 사이에 간선이 있다
2. 빈 칸(0)으로 이동하는 간선의 가중치는 0, 벽(1)으로 이동하는 간선의 가중치는 1이다
3. 0-1 BFS를 적용하여 (0,0)에서 (N-1,M-1)까지의 최단 거리를 구한다
4. 최단 거리가 곧 부숴야 하는 벽의 최소 개수이다

## 핵심 아이디어

이 문제의 핵심은 "벽을 부수는 횟수의 최솟값"을 구하는 것이다. 빈 칸 이동은 비용 0, 벽 부수기는 비용 1이므로 0-1 BFS의 전형적인 적용 사례이다.

격자의 각 칸 값이 곧 이동 비용이 되므로, `grid[nr][nc] - '0'`으로 가중치를 바로 구할 수 있다.

구조체 `Pos`를 사용하여 덱에 좌표를 저장하면 코드가 직관적이다. 정수 인코딩(`r*m+c`)을 사용해도 되지만, 구조체가 가독성이 좋다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N × M) |
| 공간 복잡도 | O(N × M) |

각 칸을 최대 한 번 처리하므로 O(N×M)이다.

## 대안적 접근

- 다익스트라: O(NM log(NM))으로 풀 수 있지만 0-1 BFS가 더 효율적이다
- BFS + 이진 탐색: 벽을 K개까지 부술 수 있을 때 도달 가능한지 판별하는 방식도 가능하지만, 0-1 BFS가 직접적이고 간단하다
