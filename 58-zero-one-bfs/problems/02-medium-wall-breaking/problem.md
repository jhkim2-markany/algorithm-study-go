# 벽 부수기 최단 경로

**난이도:** 중

## 문제 설명

N×M 격자가 주어진다. 각 칸은 빈 칸(0) 또는 벽(1)이다. 왼쪽 위 (1,1)에서 오른쪽 아래 (N,M)까지 이동하려고 한다.

상하좌우 인접한 칸으로 이동할 수 있다. 빈 칸으로 이동하는 것은 자유롭지만, 벽을 부수고 이동하려면 비용이 1 든다.

(1,1)에서 (N,M)까지 이동하는 데 부숴야 하는 벽의 최소 개수를 구하라.

## 입력 형식

- 첫째 줄에 격자의 크기 N, M이 주어진다.
- 둘째 줄부터 N개의 줄에 격자 정보가 주어진다. 각 줄에 M개의 정수가 공백 없이 주어지며, 0은 빈 칸, 1은 벽을 나타낸다.

## 출력 형식

- (1,1)에서 (N,M)까지 이동하는 데 부숴야 하는 벽의 최소 개수를 출력한다.

## 제약 조건

- 1 ≤ N, M ≤ 1,000
- 격자의 (1,1)과 (N,M)은 항상 빈 칸(0)이다
- (1,1)에서 (N,M)까지 항상 도달할 수 있다

## 예제

### 예제 입력 1

```
5 5
00100
01010
01010
01010
00010
```

### 예제 출력 1

```
1
```

### 예제 입력 2

```
3 3
000
000
000
```

### 예제 출력 2

```
0
```

### 예제 입력 3

```
1 1
0
```

### 예제 출력 3

```
0
```
