# 58. 0-1 BFS

가중치가 0 또는 1인 그래프에서 덱(Deque)을 사용하여 최단 경로를 구하는 알고리즘이다. 다익스트라보다 간단하고 빠르게 O(V+E)에 최단 경로를 구할 수 있다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
