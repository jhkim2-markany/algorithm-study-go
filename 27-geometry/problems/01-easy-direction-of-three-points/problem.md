# 세 점의 방향 판별

**난이도:** 하

## 문제 설명

2차원 평면 위에 세 점 P₁(x₁, y₁), P₂(x₂, y₂), P₃(x₃, y₃)이 주어진다. 세 점을 P₁ → P₂ → P₃ 순서로 이었을 때, 이 경로가 반시계 방향인지, 시계 방향인지, 일직선인지 판별하시오.

## 입력 형식

- 첫째 줄에 테스트 케이스의 수 T가 주어진다 (1 ≤ T ≤ 100)
- 다음 T개의 줄에 각각 6개의 정수 x₁, y₁, x₂, y₂, x₃, y₃이 공백으로 구분되어 주어진다

## 출력 형식

- 각 테스트 케이스에 대해 한 줄에 하나씩 결과를 출력한다
  - 반시계 방향이면 `1`
  - 시계 방향이면 `-1`
  - 일직선이면 `0`

## 제약 조건

- 1 ≤ T ≤ 100
- -10,000 ≤ x₁, y₁, x₂, y₂, x₃, y₃ ≤ 10,000
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
3
0 0 4 0 2 3
0 0 2 3 4 0
0 0 2 2 4 4
```

### 예제 출력 1

```
1
-1
0
```
