# 세 점의 방향 판별 - 해설

## 접근 방식

1. 세 점 P₁, P₂, P₃의 좌표를 입력받는다
2. P₁에서 P₂로 향하는 벡터와 P₁에서 P₃로 향하는 벡터를 구한다
3. 두 벡터의 외적(Cross Product)을 계산한다
4. 외적의 부호로 방향을 판별하여 출력한다

## 핵심 아이디어

CCW(Counter-Clockwise) 판별은 기하 알고리즘의 가장 기본적인 연산이다. 두 벡터의 외적 공식을 사용한다:

```
외적 = (x₂ - x₁) × (y₃ - y₁) - (x₃ - x₁) × (y₂ - y₁)
```

외적 값의 부호가 양수이면 반시계 방향(CCW), 음수이면 시계 방향(CW), 0이면 세 점이 일직선 위에 있다. 이 연산은 선분 교차 판정, 볼록 껍질, 다각형 넓이 계산 등 거의 모든 기하 알고리즘의 기초가 된다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(T) |
| 공간 복잡도 | O(1) |

각 테스트 케이스당 외적 계산은 O(1)이므로, 전체 시간 복잡도는 O(T)이다. 추가 메모리는 사용하지 않는다.

## 대안적 접근

- 기울기 비교 방식으로도 방향을 판별할 수 있으나, 나눗셈이 필요하여 부동소수점 오차가 발생할 수 있다. 외적을 사용하면 정수 연산만으로 정확한 판별이 가능하다.
