# 다각형 넓이 계산

**난이도:** 상

## 문제 설명

2차원 평면 위에 N개의 꼭짓점으로 이루어진 단순 다각형이 주어진다. 꼭짓점은 다각형의 둘레를 따라 순서대로 주어지며, 시계 방향 또는 반시계 방향일 수 있다. 이 다각형의 넓이를 구하시오.

단순 다각형이란 변끼리 교차하지 않는 다각형을 말한다.

## 입력 형식

- 첫째 줄에 꼭짓점의 수 N이 주어진다 (3 ≤ N ≤ 100,000)
- 다음 N개의 줄에 각각 꼭짓점의 좌표 xᵢ, yᵢ가 공백으로 구분되어 주어진다
- 꼭짓점은 다각형의 둘레를 따라 순서대로 주어진다

## 출력 형식

- 다각형의 넓이를 소수점 첫째 자리까지 출력한다

## 제약 조건

- 3 ≤ N ≤ 100,000
- -1,000,000 ≤ xᵢ, yᵢ ≤ 1,000,000
- 좌표는 정수로 주어진다
- 다각형은 단순 다각형이다 (변끼리 교차하지 않음)
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
4
0 0
4 0
4 3
0 3
```

### 예제 출력 1

```
12.0
```

### 예제 입력 2

```
3
0 0
4 0
2 3
```

### 예제 출력 2

```
6.0
```

### 예제 입력 3

```
5
1 0
3 0
4 2
2 4
0 2
```

### 예제 출력 3

```
10.0
```
