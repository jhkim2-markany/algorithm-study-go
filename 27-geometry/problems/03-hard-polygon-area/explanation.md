# 다각형 넓이 계산 - 해설

## 접근 방식

1. N개의 꼭짓점 좌표를 순서대로 입력받는다
2. 신발끈 공식(Shoelace Formula)을 적용하여 넓이를 계산한다
3. 인접한 두 꼭짓점에 대해 외적 값을 누적한다
4. 마지막 꼭짓점과 첫 번째 꼭짓점도 연결하여 계산한다
5. 누적 합의 절댓값을 2로 나누어 최종 넓이를 구한다

## 핵심 아이디어

신발끈 공식(Shoelace Formula)은 다각형의 꼭짓점 좌표만으로 넓이를 계산하는 공식이다. 원리는 다음과 같다:

각 변에 대해 원점과 이루는 삼각형의 부호 있는 넓이를 누적한다. 반시계 방향으로 순회하면 양수, 시계 방향이면 음수가 되므로, 절댓값을 취하면 방향에 관계없이 올바른 넓이를 얻는다.

```
넓이 = |Σᵢ (xᵢ × yᵢ₊₁ - xᵢ₊₁ × yᵢ)| / 2
```

이 공식은 볼록 다각형뿐 아니라 오목 다각형에도 적용된다. 단, 변끼리 교차하지 않는 단순 다각형이어야 한다.

좌표가 정수이므로 int64를 사용하여 오버플로를 방지한다. 좌표 범위가 ±10⁶이므로 외적 값은 최대 2×10¹²이 되어 int64로 충분히 처리할 수 있다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N) |
| 공간 복잡도 | O(N) |

N개의 꼭짓점을 한 번 순회하며 외적을 누적하므로 시간 복잡도는 O(N)이다. 좌표를 저장하는 배열이 필요하므로 공간 복잡도도 O(N)이다.

## 대안적 접근

- 삼각형 분할법: 다각형을 여러 삼각형으로 분할하여 각 삼각형의 넓이를 합산하는 방법이다. 신발끈 공식이 사실상 이 방법을 수학적으로 정리한 것이다.
- 적분 기반 접근: 각 변을 따라 적분하여 넓이를 구하는 방법도 있으나, 구현이 복잡하고 신발끈 공식과 동일한 결과를 준다.
