# 선분 교차 판정 - 해설

## 접근 방식

1. 두 선분 AB와 CD의 좌표를 입력받는다
2. CCW를 이용하여 선분 AB에 대한 점 C, D의 방향을 각각 구한다
3. CCW를 이용하여 선분 CD에 대한 점 A, B의 방향을 각각 구한다
4. 일반적인 교차 조건과 일직선 위의 특수 케이스를 모두 확인한다

## 핵심 아이디어

두 선분이 교차하려면, 한 선분의 양 끝점이 다른 선분을 기준으로 서로 반대편에 있어야 한다. 이를 CCW로 판별한다:

- 선분 AB 기준: CCW(A,B,C)와 CCW(A,B,D)의 부호가 다르면 C와 D는 AB의 양쪽에 위치
- 선분 CD 기준: CCW(C,D,A)와 CCW(C,D,B)의 부호가 다르면 A와 B는 CD의 양쪽에 위치
- 두 조건이 모두 만족하면 교차한다

특수 케이스로 세 점 이상이 일직선 위에 있는 경우가 있다. 이때는 CCW 값이 0이 되므로, 해당 점이 선분 위에 실제로 놓이는지 좌표 범위를 확인해야 한다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(T) |
| 공간 복잡도 | O(1) |

각 테스트 케이스당 CCW 계산 4회와 상수 시간 비교 연산만 수행하므로 O(1)이다. 전체 시간 복잡도는 O(T)이다.

## 대안적 접근

- 연립방정식을 풀어 교점을 직접 구하는 방법이 있으나, 부동소수점 오차 문제가 발생할 수 있다. CCW 기반 방법은 정수 연산만으로 정확한 판정이 가능하여 더 안정적이다.
