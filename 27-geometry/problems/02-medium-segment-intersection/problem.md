# 선분 교차 판정

**난이도:** 중

## 문제 설명

2차원 평면 위에 두 선분이 주어진다. 첫 번째 선분은 점 A(x₁, y₁)와 점 B(x₂, y₂)를 잇고, 두 번째 선분은 점 C(x₃, y₃)와 점 D(x₄, y₄)를 잇는다. 두 선분이 교차하는지 판정하시오.

두 선분이 한 점에서 만나거나, 일부 구간이 겹치는 경우도 교차로 판정한다.

## 입력 형식

- 첫째 줄에 테스트 케이스의 수 T가 주어진다 (1 ≤ T ≤ 100)
- 다음 T개의 줄에 각각 8개의 정수 x₁, y₁, x₂, y₂, x₃, y₃, x₄, y₄가 공백으로 구분되어 주어진다

## 출력 형식

- 각 테스트 케이스에 대해 한 줄에 하나씩 결과를 출력한다
  - 두 선분이 교차하면 `1`
  - 교차하지 않으면 `0`

## 제약 조건

- 1 ≤ T ≤ 100
- -100,000 ≤ 모든 좌표 ≤ 100,000
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
3
0 0 4 4 0 4 4 0
0 0 2 2 3 3 5 5
0 0 1 1 2 2 3 3
```

### 예제 출력 1

```
1
0
0
```
