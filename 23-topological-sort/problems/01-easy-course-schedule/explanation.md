# 수강 순서 - 해설

## 접근 방식

1. 선수 과목 관계를 방향 그래프로 모델링한다. 과목 A → 과목 B 간선은 "A를 먼저 수강해야 B를 수강 가능"을 의미한다.
2. 모든 과목을 수강할 수 있는지는 그래프에 사이클이 존재하는지 여부로 판별할 수 있다.
3. Kahn 알고리즘(BFS 기반 위상 정렬)을 수행하여 처리된 정점 수가 전체 과목 수와 같은지 확인한다.
4. 처리된 정점 수가 N이면 사이클이 없으므로 "Yes", 그렇지 않으면 사이클이 존재하므로 "No"를 출력한다.

## 핵심 아이디어

위상 정렬의 가장 기본적인 적용 문제이다. 핵심은 "DAG에서만 위상 정렬이 가능하다"는 성질을 이용하여 사이클 존재 여부를 판별하는 것이다.

Kahn 알고리즘에서 진입 차수가 0인 정점부터 처리하면, 사이클에 포함된 정점은 진입 차수가 절대 0이 되지 않으므로 큐에 들어가지 못한다. 따라서 처리된 정점 수가 전체 정점 수보다 적으면 사이클이 존재한다고 판단할 수 있다.

예제 1에서 과목 1은 선수 과목이 없으므로 먼저 수강하고, 이후 과목 2, 3을 수강한 뒤 과목 4를 수강할 수 있다. 예제 2에서는 1→2→3→1 순환이 있어 어떤 과목도 먼저 수강할 수 없다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N + M) |
| 공간 복잡도 | O(N + M) |

모든 정점과 간선을 한 번씩 처리하므로 O(N + M)이다. 인접 리스트에 O(N + M), 진입 차수 배열과 큐에 O(N)의 공간이 필요하다.

## 대안적 접근

- DFS 기반 위상 정렬로도 사이클을 판별할 수 있다. 방문 상태를 미방문/방문 중/방문 완료로 구분하여, 방문 중인 정점을 다시 만나면 사이클이다.
- 단순히 사이클 판별만 필요하므로 DFS로 백 엣지(back edge)를 탐지하는 방법도 가능하다.
