# 수강 순서

**난이도:** 하

## 문제 설명

N개의 과목과 M개의 선수 과목 관계가 주어진다. 각 선수 과목 관계는 "과목 A를 수강해야 과목 B를 수강할 수 있다"를 의미한다. 모든 과목을 수강할 수 있는지 판별하시오.

모든 과목을 수강할 수 있으면 "Yes"를, 선수 과목 관계에 순환이 있어 모든 과목을 수강할 수 없으면 "No"를 출력한다.

## 입력 형식

- 첫째 줄에 과목의 수 N과 선수 과목 관계의 수 M이 주어진다
- 둘째 줄부터 M개의 줄에 각 선수 과목 관계 A B가 주어진다 (A를 먼저 수강해야 B를 수강 가능)

## 출력 형식

- 모든 과목을 수강할 수 있으면 "Yes", 아니면 "No"를 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- 0 ≤ M ≤ 200,000
- 1 ≤ A, B ≤ N
- A ≠ B
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
4 4
1 2
1 3
2 4
3 4
```

### 예제 출력 1

```
Yes
```

### 예제 입력 2

```
3 3
1 2
2 3
3 1
```

### 예제 출력 2

```
No
```
