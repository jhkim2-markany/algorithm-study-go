# DAG 최장 경로

**난이도:** 상

## 문제 설명

N개의 정점과 M개의 가중치 간선으로 이루어진 방향 비순환 그래프(DAG)가 주어진다. 시작 정점 S에서 출발하여 도달할 수 있는 정점까지의 최장 경로의 길이를 각각 구하시오.

경로의 길이는 경로에 포함된 간선 가중치의 합으로 정의한다. 시작 정점에서 도달할 수 없는 정점의 경우 -1을 출력한다.

## 입력 형식

- 첫째 줄에 정점의 수 N, 간선의 수 M, 시작 정점 S가 주어진다
- 둘째 줄부터 M개의 줄에 각 간선의 정보 u v w가 주어진다 (u에서 v로 가는 가중치 w의 간선)

## 출력 형식

- N개의 줄에 걸쳐 시작 정점 S에서 각 정점까지의 최장 경로 길이를 출력한다
- 도달할 수 없는 정점은 -1을 출력한다
- 시작 정점 자신까지의 거리는 0이다

## 제약 조건

- 1 ≤ N ≤ 100,000
- 0 ≤ M ≤ 200,000
- 1 ≤ S ≤ N
- 1 ≤ u, v ≤ N
- 1 ≤ w ≤ 1,000
- 그래프는 DAG임이 보장된다 (사이클 없음)
- 시간 제한: 1초
- 메모리 제한: 256MB
