# 23. 위상 정렬

방향 비순환 그래프(DAG)에서 정점들을 선행 관계를 만족하는 순서로 나열하는 알고리즘이다. Kahn 알고리즘은 진입 차수(in-degree)를 이용한 BFS 방식으로, DFS 기반 위상 정렬은 후위 순회의 역순으로 정렬을 수행한다. 작업 스케줄링, 수강 순서, 빌드 의존성 해결 등에 활용된다.

- `examples/` - 기본 구현 예시 코드
