# 62. 오일러 경로 테크닉 (Euler Tour Technique)

트리를 DFS로 순회하면서 각 노드의 방문 시작 시각(in-time)과 종료 시각(out-time)을 기록하여, 트리를 1차원 배열로 평탄화(flatten)하는 기법이다. 이를 통해 서브트리 질의나 경로 갱신 같은 트리 연산을 세그먼트 트리, 펜윅 트리 등 구간 자료구조로 효율적으로 처리할 수 있다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
