# 서브트리 합 구하기

**난이도:** 하

## 문제 설명

N개의 노드로 이루어진 트리가 주어진다. 노드는 1번부터 N번까지 번호가 매겨져 있고, 루트는 1번 노드이다. 각 노드에는 정수 값이 하나씩 부여되어 있다.

Q개의 질의가 주어지며, 각 질의는 다음 두 종류 중 하나이다:

- `1 v x`: 노드 v의 값을 x로 변경한다
- `2 v`: 노드 v를 루트로 하는 서브트리에 속하는 모든 노드의 값의 합을 출력한다

## 입력 형식

- 첫째 줄에 노드의 수 N과 질의의 수 Q가 주어진다.
- 둘째 줄에 각 노드의 초기 값 a₁, a₂, ..., aₙ이 공백으로 구분되어 주어진다.
- 셋째 줄부터 N-1개의 줄에 트리의 간선 정보 u, v가 주어진다.
- 이후 Q개의 줄에 질의가 주어진다.

## 출력 형식

- 각 2번 질의에 대해 서브트리 합을 한 줄에 하나씩 출력한다.

## 제약 조건

- 1 ≤ N, Q ≤ 100,000
- 1 ≤ aᵢ ≤ 1,000
- 1 ≤ u, v ≤ N
- 1 ≤ x ≤ 1,000
- 주어지는 그래프는 항상 트리이다
