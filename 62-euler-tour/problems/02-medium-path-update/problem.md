# 경로 값 갱신

**난이도:** 중

## 문제 설명

N개의 노드로 이루어진 트리가 주어진다. 노드는 1번부터 N번까지 번호가 매겨져 있고, 루트는 1번 노드이다. 모든 노드의 초기 값은 0이다.

Q개의 질의가 주어지며, 각 질의는 다음 두 종류 중 하나이다:

- `1 v x`: 루트(1번 노드)에서 노드 v까지의 경로에 있는 모든 노드의 값에 x를 더한다
- `2 v`: 노드 v의 현재 값을 출력한다

루트에서 노드 v까지의 경로란, 루트에서 출발하여 v에 도달하는 유일한 경로에 포함된 모든 노드를 의미한다 (루트와 v 포함).

## 입력 형식

- 첫째 줄에 노드의 수 N과 질의의 수 Q가 주어진다.
- 둘째 줄부터 N-1개의 줄에 트리의 간선 정보 u, v가 주어진다.
- 이후 Q개의 줄에 질의가 주어진다.

## 출력 형식

- 각 2번 질의에 대해 노드의 현재 값을 한 줄에 하나씩 출력한다.

## 제약 조건

- 1 ≤ N, Q ≤ 100,000
- 1 ≤ v ≤ N
- 1 ≤ x ≤ 1,000
- 주어지는 그래프는 항상 트리이다

## 예제

### 예제 입력 1

```
5 5
1 2
1 3
2 4
2 5
1 4 10
1 5 20
2 1
2 2
2 4
```

### 예제 출력 1

```
30
30
10
```

### 예제 설명 1

```
트리 구조:
      1
     / \
    2   3
   / \
  4   5

질의 1: 루트→4 경로 (1,2,4)에 10을 더한다
  노드 값: [_, 10, 10, 0, 10, 0]

질의 2: 루트→5 경로 (1,2,5)에 20을 더한다
  노드 값: [_, 30, 30, 0, 10, 20]

질의 3: 노드 1의 값 = 30
질의 4: 노드 2의 값 = 30
질의 5: 노드 4의 값 = 10
```

### 예제 입력 2

```
3 3
1 2
2 3
1 3 5
1 2 3
2 2
```

### 예제 출력 2

```
8
```
