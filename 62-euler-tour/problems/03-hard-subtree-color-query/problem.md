# 서브트리 색상 종류 쿼리

**난이도:** 상

## 문제 설명

N개의 노드로 이루어진 트리가 주어진다. 노드는 1번부터 N번까지 번호가 매겨져 있고, 루트는 1번 노드이다. 각 노드에는 1 이상 C 이하의 색상 번호가 부여되어 있다.

Q개의 질의가 주어지며, 각 질의는 다음 두 종류 중 하나이다:

- `1 v c`: 노드 v의 색상을 c로 변경한다
- `2 v`: 노드 v를 루트로 하는 서브트리에 존재하는 서로 다른 색상의 종류 수를 출력한다

## 입력 형식

- 첫째 줄에 노드의 수 N, 색상의 수 C, 질의의 수 Q가 주어진다.
- 둘째 줄에 각 노드의 초기 색상 c₁, c₂, ..., cₙ이 공백으로 구분되어 주어진다.
- 셋째 줄부터 N-1개의 줄에 트리의 간선 정보 u, v가 주어진다.
- 이후 Q개의 줄에 질의가 주어진다.

## 출력 형식

- 각 2번 질의에 대해 서브트리 내 서로 다른 색상의 종류 수를 한 줄에 하나씩 출력한다.

## 제약 조건

- 1 ≤ N ≤ 50,000
- 1 ≤ C ≤ 50,000
- 1 ≤ Q ≤ 50,000
- 1 ≤ cᵢ ≤ C
- 1 ≤ u, v ≤ N
- 주어지는 그래프는 항상 트리이다

```
  (색2) \
    / \   6(색1) 7(색3)
   4   5

질의 1: 노드 1의 서브트리 = 전체 → 색상 {1,2,3} → 3종류

질의 3: 노드 4의 색상을 1로 변경
질의 4: 노드 2의 서브트리 = {2,4,5} → 색상 {2,1,2} → 2종류
질의 5: 노드 3의 서브트리 = {3,6,7} → 색상 {1,1,3} → 2종류
```
