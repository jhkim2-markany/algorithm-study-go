# 최소 버텍스 커버

**난이도:** 상

## 문제 설명

N개의 행과 M개의 열로 이루어진 격자판이 있다. 격자판의 일부 칸에 장애물이 놓여 있다. 장애물이 없는 칸에 감시 카메라를 설치할 수 있으며, 각 카메라는 자신이 위치한 행 전체와 열 전체를 감시한다 (장애물에 의해 차단되지 않는다고 가정).

모든 장애물이 없는 칸을 최소한 하나의 카메라가 감시하도록 하려면, 최소 몇 대의 카메라가 필요한지 구하시오.

이 문제는 이분 그래프의 최소 버텍스 커버로 모델링할 수 있다. 행을 왼쪽 그룹, 열을 오른쪽 그룹으로 하고, 장애물이 없는 각 칸 (r, c)를 행 r과 열 c를 잇는 간선으로 표현하면, 최소 버텍스 커버가 곧 최소 카메라 수이다.

## 입력 형식

- 첫째 줄에 행 수 N, 열 수 M, 장애물 수 K가 공백으로 구분되어 주어진다.
- 둘째 줄부터 K개의 줄에 장애물의 위치 (r, c)가 주어진다 (1-indexed).

## 출력 형식

- 모든 빈 칸을 감시하기 위한 최소 카메라 수를 출력한다.

## 제약 조건

- 1 ≤ N, M ≤ 500
- 0 ≤ K ≤ N × M
- 장애물 위치는 격자판 범위 내이며 중복되지 않는다
- 시간 제한: 2초

## 예제

### 예제 입력 1

```
3 3 1
2 2
```

### 예제 출력 1

```
3
```

### 예제 입력 2

```
2 2 0
```

### 예제 출력 2

```
2
```

### 예제 입력 3

```
3 3 4
1 1
1 3
3 1
3 3
```

### 예제 출력 3

```
2
```
