# 기본 이분 매칭 - 해설

## 접근 방식

1. 학생을 왼쪽 그룹, 동아리를 오른쪽 그룹으로 하는 이분 그래프를 구성한다.
2. 각 학생이 가입 가능한 동아리를 간선으로 연결한다.
3. 헝가리안 알고리즘(Kuhn's Algorithm)을 사용하여 최대 매칭을 구한다.
4. 각 학생에 대해 DFS로 증가 경로를 찾아 매칭을 확장한다.

## 핵심 아이디어

헝가리안 알고리즘은 왼쪽 정점을 하나씩 순회하면서, 각 정점에서 DFS로 증가 경로를 찾는다. 증가 경로란 미매칭 정점에서 시작하여 매칭/비매칭 간선을 번갈아 거쳐 다른 미매칭 정점에 도달하는 경로이다.

DFS 과정에서 오른쪽 정점 v가 이미 매칭되어 있더라도, v의 현재 매칭 상대가 다른 오른쪽 정점으로 "양보"할 수 있으면 매칭을 재조정한다. 이 재귀적 양보가 알고리즘의 핵심이다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N × E) |
| 공간 복잡도 | O(N + M + E) |

N명의 학생 각각에 대해 DFS를 수행하며, 각 DFS는 최대 O(E)의 간선을 탐색한다. E는 전체 간선 수(학생-동아리 연결 수)이다.

## 대안적 접근

이분 매칭은 최대 유량 문제로 변환할 수 있다. 가상의 소스에서 모든 학생에게 용량 1의 간선을, 모든 동아리에서 가상의 싱크로 용량 1의 간선을 연결하면 Edmonds-Karp 등의 유량 알고리즘으로 풀 수 있다. 다만 이 문제 규모에서는 헝가리안 알고리즘이 더 간결하고 효율적이다.
