# 기본 이분 매칭

**난이도:** 하

## 문제 설명

N명의 학생과 M개의 동아리가 있다. 각 학생은 가입하고 싶은 동아리 목록을 가지고 있으며, 각 동아리는 최대 1명의 학생만 받을 수 있다. 또한 각 학생은 최대 1개의 동아리에만 가입할 수 있다.

최대 몇 명의 학생이 동아리에 가입할 수 있는지 구하시오.

## 입력 형식

- 첫째 줄에 학생 수 N과 동아리 수 M이 공백으로 구분되어 주어진다.
- 둘째 줄부터 N개의 줄에 각 학생이 가입하고 싶은 동아리 정보가 주어진다.
  - 각 줄의 첫 번째 수는 해당 학생이 가입 가능한 동아리 수 K이다.
  - 이어서 K개의 동아리 번호가 공백으로 구분되어 주어진다 (1-indexed).

## 출력 형식

- 최대 매칭 수(동아리에 가입할 수 있는 최대 학생 수)를 출력한다.

## 제약 조건

- 1 ≤ N, M ≤ 1,000
- 0 ≤ K ≤ M
- 시간 제한: 1초

## 예제

### 예제 입력 1

```
4 4
2 1 2
1 1
2 2 3
2 3 4
```

### 예제 출력 1

```
4
```

### 예제 입력 2

```
3 2
1 1
1 1
1 2
```

### 예제 출력 2

```
2
```
