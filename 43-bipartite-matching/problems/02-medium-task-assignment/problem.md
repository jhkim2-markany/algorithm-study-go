# 업무 배정 최적화

**난이도:** 중

## 문제 설명

회사에 N명의 직원과 M개의 프로젝트가 있다. 각 직원은 자신이 수행할 수 있는 프로젝트 목록을 가지고 있다. 각 프로젝트는 정확히 1명의 직원에게만 배정되어야 하며, 각 직원은 최대 1개의 프로젝트만 담당할 수 있다.

회사는 최대한 많은 프로젝트를 배정하고 싶다. 최대 매칭을 구한 후, 각 프로젝트가 어떤 직원에게 배정되었는지 출력하시오. 매칭되지 않은 프로젝트는 0을 출력한다.

## 입력 형식

- 첫째 줄에 직원 수 N과 프로젝트 수 M이 공백으로 구분되어 주어진다.
- 둘째 줄부터 N개의 줄에 각 직원이 수행 가능한 프로젝트 정보가 주어진다.
  - 각 줄의 첫 번째 수는 해당 직원이 수행 가능한 프로젝트 수 K이다.
  - 이어서 K개의 프로젝트 번호가 공백으로 구분되어 주어진다 (1-indexed).

## 출력 형식

- 첫째 줄에 최대 매칭 수를 출력한다.
- 둘째 줄에 M개의 정수를 공백으로 구분하여 출력한다. i번째 정수는 프로젝트 i에 배정된 직원 번호(1-indexed)이며, 배정되지 않았으면 0이다.

## 제약 조건

- 1 ≤ N, M ≤ 1,000
- 0 ≤ K ≤ M
- 시간 제한: 1초

## 예제

### 예제 입력 1

```
5 5
2 1 2
2 1 3
2 2 4
1 3
2 4 5
```

### 예제 출력 1

```
5
1 3 2 5 0
```

### 예제 입력 2

```
3 4
1 1
2 1 2
1 3
```

### 예제 출력 2

```
3
1 2 3 0
```
