# 모듈러 분수의 합

**난이도:** 상

## 문제 설명

N개의 분수 a₁/b₁, a₂/b₂, ..., aₙ/bₙ이 주어진다. 이 분수들의 합 S = a₁/b₁ + a₂/b₂ + ... + aₙ/bₙ을 소수 M = 10⁹+7로 나눈 나머지를 구하시오.

분수의 합 S = P/Q (기약분수)일 때, 답은 P × Q⁻¹ mod M이다. 단, 이 문제에서는 각 분수를 모듈러 역원을 이용하여 직접 합산하면 된다.

즉, 답 = Σ(aᵢ × bᵢ⁻¹) mod M이다.

## 입력 형식

- 첫째 줄에 분수의 개수 N이 주어진다
- 다음 N개의 줄에 각각 정수 aᵢ와 bᵢ가 공백으로 구분되어 주어진다

## 출력 형식

- 분수들의 합을 M = 10⁹+7로 나눈 나머지를 출력한다

## 제약 조건

- 1 ≤ N ≤ 500,000
- 1 ≤ aᵢ ≤ 10⁹
- 1 ≤ bᵢ ≤ 10⁹
- M = 10⁹ + 7 (소수)
- gcd(bᵢ, M) = 1 (모든 bᵢ는 M의 배수가 아님)
- 시간 제한: 2초
- 메모리 제한: 256MB
