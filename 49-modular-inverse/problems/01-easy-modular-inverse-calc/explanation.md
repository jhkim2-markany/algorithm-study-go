# 모듈러 역원 계산 - 해설

## 접근 방식

1. M이 소수이므로 페르마 소정리를 적용할 수 있다
2. 페르마 소정리: A^(M-1) ≡ 1 (mod M) → A^(M-2) ≡ A⁻¹ (mod M)
3. 빠른 거듭제곱(분할 정복 거듭제곱)으로 A^(M-2) mod M을 계산한다

## 핵심 아이디어

페르마 소정리에 의해 소수 M에 대해 A^(M-1) ≡ 1 (mod M)이 성립한다. 양변을 A로 나누면 A^(M-2) ≡ A⁻¹ (mod M)이 된다. 따라서 모듈러 역원 문제는 빠른 거듭제곱 문제로 환원된다.

빠른 거듭제곱은 지수를 이진수로 분해하여 O(log M)에 계산한다. 예를 들어 3^5 = 3^(101₂) = 3^4 × 3^1로 분해하여 곱셈 횟수를 줄인다.

예제 검증: 3의 역원 (mod 7)
- 3^(7-2) = 3^5 = 243
- 243 mod 7 = 5
- 검증: 3 × 5 = 15 ≡ 1 (mod 7) ✓

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(T × log M) |
| 공간 복잡도 | O(1) |

각 테스트 케이스마다 빠른 거듭제곱 O(log M)을 수행한다. 추가 메모리는 상수만 사용한다.

## 대안적 접근

- 확장 유클리드 알고리즘: M이 소수가 아닌 경우에도 gcd(A, M) = 1이면 역원을 구할 수 있다. 시간 복잡도는 동일하게 O(log M)이다.
