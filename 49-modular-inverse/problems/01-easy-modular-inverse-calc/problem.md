# 모듈러 역원 계산

**난이도:** 하

## 문제 설명

정수 A와 소수 M이 주어진다. A의 모듈러 곱셈 역원, 즉 A × X ≡ 1 (mod M)을 만족하는 X (0 ≤ X < M)를 구하시오.

페르마 소정리에 의해 M이 소수이고 A가 M의 배수가 아니면, A의 역원은 A^(M-2) mod M이다.

## 입력 형식

- 첫째 줄에 테스트 케이스의 수 T가 주어진다
- 다음 T개의 줄에 각각 정수 A와 소수 M이 공백으로 구분되어 주어진다

## 출력 형식

- 각 테스트 케이스에 대해 A의 모듈러 역원 X를 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ T ≤ 100,000
- 1 ≤ A < M
- 2 ≤ M ≤ 10⁹ + 7
- M은 소수이다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
3
3 7
5 1000000007
2 13
```

### 예제 출력 1

```
5
400000003
7
```

### 예제 입력 2

```
2
1 1000000007
999999999 1000000007
```

### 예제 출력 2

```
1
500000004
```
