# 이항 계수 모듈러 계산

**난이도:** 중

## 문제 설명

자연수 N, R이 주어질 때 이항 계수 C(N, R) = N! / (R! × (N-R)!)을 소수 M = 10⁹+7로 나눈 나머지를 구하시오.

여러 쿼리가 주어지므로 팩토리얼과 역팩토리얼을 전처리하여 각 쿼리를 O(1)에 처리해야 한다.

## 입력 형식

- 첫째 줄에 쿼리의 수 Q가 주어진다
- 다음 Q개의 줄에 각각 정수 N과 R이 공백으로 구분되어 주어진다

## 출력 형식

- 각 쿼리에 대해 C(N, R) mod (10⁹+7)을 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ Q ≤ 200,000
- 0 ≤ R ≤ N ≤ 1,000,000
- M = 10⁹ + 7 (소수)
- 시간 제한: 2초
- 메모리 제한: 256MB
