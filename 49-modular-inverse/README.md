# 49. 모듈로 곱셈 역원 (Modular Inverse)

정수 a에 대해 a × x ≡ 1 (mod M)을 만족하는 x를 구하는 기법이다. 페르마 소정리, 확장 유클리드 알고리즘을 활용하여 모듈러 나눗셈과 이항 계수의 모듈러 계산을 수행한다.
