# 네트워크 연결

**난이도:** 중

## 문제 설명

N개의 컴퓨터와 M개의 네트워크 케이블 정보가 주어진다. 각 케이블은 두 컴퓨터를 연결하며 설치 비용이 있다. 모든 컴퓨터가 직접 또는 간접적으로 서로 통신할 수 있도록 케이블을 선택할 때, 필요한 최소 비용을 구하시오.

이미 연결된 컴퓨터끼리는 다시 연결할 필요가 없다. 만약 모든 컴퓨터를 연결하는 것이 불가능하면 -1을 출력한다.

## 입력 형식

- 첫째 줄에 컴퓨터의 수 N과 케이블의 수 M이 주어진다 (1 ≤ N ≤ 100,000, 0 ≤ M ≤ 200,000)
- 둘째 줄부터 M개의 줄에 각 케이블의 정보 u, v, w가 주어진다 (1 ≤ u, v ≤ N, 1 ≤ w ≤ 1,000,000)
  - u번과 v번 컴퓨터를 연결하는 케이블의 비용이 w이다

## 출력 형식

- 모든 컴퓨터를 연결하는 최소 비용을 출력한다
- 모든 컴퓨터를 연결할 수 없으면 -1을 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- 0 ≤ M ≤ 200,000
- 1 ≤ u, v ≤ N
- 1 ≤ w ≤ 1,000,000
- 같은 두 컴퓨터를 연결하는 케이블이 여러 개 있을 수 있다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
4 5
1 2 1
1 3 4
2 3 2
2 4 5
3 4 3
```

### 예제 출력 1

```
6
```

### 예제 입력 2

```
4 1
1 2 3
```

### 예제 출력 2

```
-1
```
