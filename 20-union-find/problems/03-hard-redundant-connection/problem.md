# 여분의 간선

**난이도:** 상

## 문제 설명

N개의 노드로 이루어진 트리에 간선이 하나 추가되어 정확히 하나의 사이클이 생겼다. 즉, N개의 노드와 N개의 간선으로 이루어진 무방향 그래프가 주어진다.

사이클을 제거하기 위해 간선을 정확히 하나 제거하려고 한다. 제거할 수 있는 간선이 여러 개라면, 입력에서 가장 마지막에 주어진 간선을 제거한다.

제거해야 할 간선의 두 끝 노드를 출력하시오.

## 입력 형식

- 첫째 줄에 노드의 수 N이 주어진다 (3 ≤ N ≤ 100,000)
- 둘째 줄부터 N개의 줄에 간선의 양 끝 노드 u, v가 주어진다 (1 ≤ u, v ≤ N)

## 출력 형식

- 제거해야 할 간선의 두 끝 노드를 공백으로 구분하여 출력한다

## 제약 조건

- 3 ≤ N ≤ 100,000
- 간선의 수는 정확히 N개이다
- 그래프에는 정확히 하나의 사이클이 존재한다
- 1 ≤ u, v ≤ N
- 자기 자신으로의 간선은 없다
- 제거 가능한 간선이 여러 개이면 입력 순서가 가장 뒤인 간선을 선택한다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
3
1 2
1 3
2 3
```

### 예제 출력 1

```
2 3
```

### 예제 입력 2

```
5
1 2
2 3
3 4
4 5
3 5
```

### 예제 출력 2

```
3 5
```
