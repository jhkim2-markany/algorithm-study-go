# 20. 유니온 파인드

서로소 집합(Disjoint Set)을 효율적으로 관리하는 자료구조이다. 두 원소가 같은 집합에 속하는지 판별(Find)하고, 두 집합을 하나로 합치는(Union) 연산을 거의 상수 시간에 수행한다. 그래프의 연결 요소 판별, 사이클 탐지, 네트워크 연결 등에 활용된다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
