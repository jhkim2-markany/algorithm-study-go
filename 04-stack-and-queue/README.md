# 04. 스택과 큐

후입선출(LIFO) 구조의 스택과 선입선출(FIFO) 구조의 큐를 활용하는 알고리즘 유형이다. 데이터의 삽입과 삭제 순서가 핵심이며, 괄호 검사, 히스토그램, 프로세스 스케줄링 등 다양한 문제에 적용된다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이

| 02 | 주식 가격 | 중 |
