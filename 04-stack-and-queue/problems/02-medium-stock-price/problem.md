# 주식 가격

**난이도:** 중

## 문제 설명

N일 동안의 주식 가격이 순서대로 주어진다. 각 날짜에 대해, 해당 날짜의 주식 가격이 떨어지지 않은 기간이 며칠인지 구하시오.

i번째 날의 가격이 떨어지지 않은 기간이란, i번째 날부터 시작하여 가격이 처음으로 떨어지기 직전까지의 일수를 말한다. 만약 끝까지 가격이 떨어지지 않으면 남은 전체 일수가 답이 된다.

## 입력 형식

- 첫째 줄에 날짜 수 N이 주어진다 (1 ≤ N ≤ 100,000)
- 둘째 줄에 N개의 주식 가격이 공백으로 구분되어 주어진다

## 출력 형식

- N개의 정수를 공백으로 구분하여 출력한다
- i번째 정수는 i번째 날의 가격이 떨어지지 않은 기간(일수)이다

## 제약 조건

- 1 ≤ N ≤ 100,000
- 1 ≤ 주식 가격 ≤ 10,000
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5
1 2 3 2 3
```

### 예제 출력 1

```
4 3 1 1 0
```

### 예제 입력 2

```
6
5 4 3 2 1 1
```

### 예제 출력 2

```
1 1 1 1 1 0
```
