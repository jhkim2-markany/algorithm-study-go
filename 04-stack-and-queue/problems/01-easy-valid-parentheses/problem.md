# 괄호 검사

**난이도:** 하

## 문제 설명

소괄호 `(`, `)`, 중괄호 `{`, `}`, 대괄호 `[`, `]`로만 이루어진 문자열이 주어진다. 이 문자열의 괄호가 올바르게 짝지어져 있는지 판별하시오.

올바른 괄호 문자열이란 다음 조건을 모두 만족하는 문자열이다:
- 여는 괄호는 같은 종류의 닫는 괄호로 닫혀야 한다
- 여는 괄호는 올바른 순서로 닫혀야 한다 (가장 나중에 열린 괄호가 가장 먼저 닫혀야 한다)
- 모든 닫는 괄호에는 대응하는 여는 괄호가 있어야 한다

## 입력 형식

- 첫째 줄에 테스트 케이스의 수 T가 주어진다 (1 ≤ T ≤ 100)
- 다음 T개의 줄에 각각 괄호 문자열이 주어진다

## 출력 형식

- 각 테스트 케이스에 대해 올바른 괄호 문자열이면 `YES`, 아니면 `NO`를 출력한다

## 제약 조건

- 1 ≤ T ≤ 100
- 1 ≤ 문자열 길이 ≤ 10,000
- 문자열은 `(`, `)`, `{`, `}`, `[`, `]`로만 구성된다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5
(){}[]
({[]})
(]
([)]
{[()]}
```

### 예제 출력 1

```
YES
YES
NO
NO
YES
```
