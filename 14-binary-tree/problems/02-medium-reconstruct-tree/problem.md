# 이진 트리 복원

**난이도:** 중

## 문제 설명

이진 트리의 전위 순회(Preorder)와 중위 순회(Inorder) 결과가 주어진다. 이 두 순회 결과를 이용하여 원래 이진 트리를 복원하고, 후위 순회(Postorder) 결과를 출력하시오.

트리의 노드는 1부터 N까지의 서로 다른 정수로 이루어져 있다.

## 입력 형식

- 첫째 줄에 노드의 수 N이 주어진다 (1 ≤ N ≤ 100,000)
- 둘째 줄에 전위 순회 결과가 공백으로 구분되어 주어진다
- 셋째 줄에 중위 순회 결과가 공백으로 구분되어 주어진다

## 출력 형식

- 후위 순회 결과를 공백으로 구분하여 한 줄에 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- 노드 값은 1부터 N까지의 서로 다른 정수이다
- 주어지는 순회 결과는 항상 유효한 이진 트리를 구성한다
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
7
1 2 4 7 5 3 6
7 4 2 5 1 3 6
```

### 예제 출력 1

```
7 4 5 2 6 3 1
```

### 예제 입력 2

```
3
1 2 3
2 1 3
```

### 예제 출력 2

```
2 3 1
```
