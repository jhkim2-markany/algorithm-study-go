# 이진 트리의 최대 경로 합

**난이도:** 상

## 문제 설명

N개의 노드로 이루어진 이진 트리가 주어진다. 각 노드에는 정수 값이 부여되어 있으며, 값은 음수일 수도 있다.

이진 트리에서 경로란, 임의의 노드에서 시작하여 부모-자식 관계를 따라 임의의 노드에서 끝나는 연속된 노드의 열이다. 경로는 반드시 아래로만 내려갈 필요는 없으며, 한 노드를 꼭짓점으로 하여 왼쪽 서브트리에서 올라와 오른쪽 서브트리로 내려가는 경로도 허용된다. 단, 같은 노드를 두 번 방문할 수는 없다.

경로에 포함된 노드 값의 합이 최대가 되는 경로를 찾고, 그 합을 출력하시오.

## 입력 형식

- 첫째 줄에 노드의 수 N이 주어진다 (1 ≤ N ≤ 100,000)
- 둘째 줄에 각 노드의 값 v₁, v₂, ..., vₙ이 공백으로 구분되어 주어진다 (-1,000 ≤ vᵢ ≤ 1,000)
- 다음 N개의 줄에 각각 세 정수 node, left, right가 주어진다. node는 현재 노드 번호, left는 왼쪽 자식 번호, right는 오른쪽 자식 번호이다. 자식이 없으면 -1이 주어진다.

## 출력 형식

- 최대 경로 합을 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- -1,000 ≤ 노드 값 ≤ 1,000
- 루트는 항상 1번 노드이다
- 경로에는 최소 1개의 노드가 포함되어야 한다
- 시간 제한: 2초
- 메모리 제한: 256MB


```
12
```

```
29
```

