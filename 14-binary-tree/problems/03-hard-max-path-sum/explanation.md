# 이진 트리의 최대 경로 합 - 해설

## 접근 방식

1. 후위 순회 방식으로 리프 노드부터 루트까지 올라가며 각 노드에서의 최대 이득을 계산한다
2. 각 노드에서 왼쪽/오른쪽 서브트리로부터 얻을 수 있는 최대 이득을 구한다
3. 이득이 음수인 경우 해당 서브트리를 포함하지 않는 것이 유리하므로 0으로 처리한다
4. 현재 노드를 꼭짓점으로 하는 경로(왼쪽 이득 + 현재 값 + 오른쪽 이득)로 전역 최댓값을 갱신한다
5. 부모에게는 현재 노드 + 왼쪽/오른쪽 중 더 큰 이득만 반환한다

## 핵심 아이디어

이 문제의 핵심은 "각 노드를 꼭짓점으로 하는 경로"와 "부모에게 전달하는 경로"를 구분하는 것이다.

각 노드에서 두 가지를 동시에 계산한다:
- 꼭짓점 경로: 왼쪽 서브트리에서 올라와 현재 노드를 거쳐 오른쪽 서브트리로 내려가는 경로. 이 경로는 더 이상 위로 올라갈 수 없으므로 전역 최댓값 갱신에만 사용한다.
- 전달 경로: 현재 노드에서 왼쪽 또는 오른쪽 중 한쪽으로만 내려가는 경로. 부모 노드가 이 경로를 연장할 수 있으므로 반환값으로 사용한다.

음수 이득을 0으로 처리하는 것은 "해당 서브트리를 경로에 포함하지 않는다"는 의미이다. 이를 통해 음수 값이 많은 서브트리를 자연스럽게 제외할 수 있다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N) |
| 공간 복잡도 | O(N) |

모든 노드를 정확히 한 번씩 방문하므로 O(N)이다. 노드 값과 자식 배열 저장에 O(N) 공간이 필요하며, 재귀 호출 스택은 트리의 높이에 비례한다.

## 대안적 접근

- 모든 경로를 열거하는 브루트포스 방법은 O(N²)이다. 각 노드를 시작점으로 DFS를 수행하여 모든 경로의 합을 계산한다. N이 작을 때 사용할 수 있다.
- 트리의 지름 문제와 유사한 구조이다. 지름은 간선 수의 최대, 이 문제는 노드 값 합의 최대라는 차이가 있지만, "각 노드를 꼭짓점으로 하는 최대 경로"를 구하는 패턴은 동일하다.
