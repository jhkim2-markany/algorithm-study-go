# 이진 트리 순회

**난이도:** 하

## 문제 설명

N개의 노드로 이루어진 이진 트리가 주어진다. 각 노드는 1부터 N까지의 번호가 매겨져 있으며, 각 노드의 왼쪽 자식과 오른쪽 자식의 번호가 주어진다. 이 이진 트리의 전위 순회, 중위 순회, 후위 순회 결과를 각각 출력하시오.

## 입력 형식

- 첫째 줄에 노드의 수 N이 주어진다 (1 ≤ N ≤ 100,000)
- 다음 N개의 줄에 각각 세 정수 node, left, right가 주어진다. node는 현재 노드 번호, left는 왼쪽 자식 번호, right는 오른쪽 자식 번호이다. 자식이 없으면 -1이 주어진다.

## 출력 형식

- 첫째 줄에 전위 순회 결과를 공백으로 구분하여 출력한다
- 둘째 줄에 중위 순회 결과를 공백으로 구분하여 출력한다
- 셋째 줄에 후위 순회 결과를 공백으로 구분하여 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- 루트는 항상 1번 노드이다
- 자식이 없는 경우 -1로 표시한다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
7
1 2 3
2 4 5
3 -1 6
4 7 -1
5 -1 -1
6 -1 -1
7 -1 -1
```

### 예제 출력 1

```
1 2 4 7 5 3 6
7 4 2 5 1 3 6
7 4 5 2 6 3 1
```

### 예제 입력 2

```
1
1 -1 -1
```

### 예제 출력 2

```
1
1
1
```
