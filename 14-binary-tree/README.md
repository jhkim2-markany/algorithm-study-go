# 14. 이진 트리

각 노드가 최대 두 개의 자식(왼쪽, 오른쪽)을 가지는 트리 자료구조이다. 이진 탐색 트리, 힙, 세그먼트 트리 등 다양한 자료구조의 기반이 되며, 중위/전위/후위 순회와 재귀적 구조를 활용하여 문제를 해결한다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
