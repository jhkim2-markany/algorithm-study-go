# 10. 투 포인터와 슬라이딩 윈도우

배열이나 문자열에서 두 개의 포인터 또는 윈도우를 활용하여 구간 탐색을 효율적으로 수행하는 알고리즘 유형이다. 브루트포스 O(N²)를 O(N)으로 최적화하는 핵심 기법이다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이

## 문제 목록

| 번호 | 문제 | 난이도 |
| --- | --- | --- |
| 01 | 정렬된 배열의 두 수의 합 | 하 |
| 02 | 가장 긴 고유 문자 부분 문자열 | 중 |
| 03 | 최소 윈도우 부분 문자열 | 상 |
