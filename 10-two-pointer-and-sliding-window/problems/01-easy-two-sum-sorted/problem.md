# 정렬된 배열의 두 수의 합

**난이도:** 하

## 문제 설명

오름차순으로 정렬된 정수 배열이 주어진다. 배열에서 두 수를 골라 합이 정확히 목표값 T가 되는 쌍을 찾아, 두 수의 인덱스를 출력하시오. 인덱스는 1부터 시작한다.

조건을 만족하는 쌍이 반드시 하나 존재한다고 가정한다.

## 입력 형식

- 첫째 줄에 배열의 크기 N과 목표값 T가 공백으로 구분되어 주어진다 (2 ≤ N ≤ 100,000, 1 ≤ T ≤ 200,000,000)
- 둘째 줄에 N개의 정수가 오름차순으로 공백으로 구분되어 주어진다

## 출력 형식

- 합이 T가 되는 두 수의 인덱스를 공백으로 구분하여 출력한다 (1-based)

## 제약 조건

- 2 ≤ N ≤ 100,000
- 1 ≤ 배열 원소 ≤ 100,000,000
- 1 ≤ T ≤ 200,000,000
- 배열은 오름차순으로 정렬되어 있다
- 정답이 반드시 하나 존재한다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5 9
1 3 5 7 9
```

### 예제 출력 1

```
1 4
```

### 예제 입력 2

```
4 6
1 2 3 4
```

### 예제 출력 2

```
2 3
```
