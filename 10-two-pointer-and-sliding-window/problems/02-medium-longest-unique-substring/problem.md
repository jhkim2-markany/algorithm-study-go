# 가장 긴 고유 문자 부분 문자열

**난이도:** 중

## 문제 설명

영문 소문자로 이루어진 문자열 S가 주어진다. 문자열 S에서 같은 문자가 두 번 이상 등장하지 않는 연속 부분 문자열 중 가장 긴 것의 길이를 구하시오.

## 입력 형식

- 첫째 줄에 문자열 S가 주어진다 (1 ≤ |S| ≤ 1,000,000)
- S는 영문 소문자로만 이루어져 있다

## 출력 형식

- 중복 문자가 없는 가장 긴 연속 부분 문자열의 길이를 출력한다

## 제약 조건

- 1 ≤ |S| ≤ 1,000,000
- S는 영문 소문자('a' ~ 'z')로만 구성된다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
abcabcbb
```

### 예제 출력 1

```
3
```

### 예제 입력 2

```
bbbbb
```

### 예제 출력 2

```
1
```

### 예제 입력 3

```
pwwkew
```

### 예제 출력 3

```
3
```
