# 최소 윈도우 부분 문자열

**난이도:** 상

## 문제 설명

영문 대소문자로 이루어진 문자열 S와 T가 주어진다. 문자열 S에서 T의 모든 문자를 포함하는 가장 짧은 연속 부분 문자열의 길이를 구하시오.

T에 같은 문자가 여러 번 등장하면, 부분 문자열에도 해당 문자가 그만큼 포함되어야 한다. 조건을 만족하는 부분 문자열이 없으면 0을 출력한다.

## 입력 형식

- 첫째 줄에 문자열 S가 주어진다 (1 ≤ |S| ≤ 1,000,000)
- 둘째 줄에 문자열 T가 주어진다 (1 ≤ |T| ≤ |S|)
- S와 T는 영문 대소문자로만 이루어져 있다

## 출력 형식

- T의 모든 문자를 포함하는 S의 가장 짧은 연속 부분 문자열의 길이를 출력한다
- 조건을 만족하는 부분 문자열이 없으면 0을 출력한다

## 제약 조건

- 1 ≤ |S| ≤ 1,000,000
- 1 ≤ |T| ≤ |S|
- S, T는 영문 대소문자('a'~'z', 'A'~'Z')로만 구성된다
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
ADOBECODEBANC
ABC
```

### 예제 출력 1

```
4
```

### 예제 입력 2

```
a
a
```

### 예제 출력 2

```
1
```

### 예제 입력 3

```
a
aa
```

### 예제 출력 3

```
0
```
