# 경로 최대 간선 가중치

**난이도:** 상

## 문제 설명

N개의 노드로 이루어진 트리가 주어진다. 노드는 1번부터 N번까지 번호가 매겨져 있고, 각 간선에는 양의 정수 가중치가 부여되어 있다.

Q개의 연산이 주어지며, 각 연산은 다음 두 종류 중 하나이다:

- `1 i w`: i번째 간선의 가중치를 w로 변경한다. (간선은 입력 순서대로 1번부터 번호가 매겨진다)
- `2 u v`: 노드 u에서 노드 v까지의 경로에 포함된 간선들의 가중치 중 최댓값을 출력한다.

## 입력 형식

- 첫째 줄에 노드의 수 N이 주어진다.
- 둘째 줄부터 N-1개의 줄에 간선 정보 a, b, w가 주어진다. 이는 노드 a와 노드 b를 잇는 간선의 가중치가 w임을 의미한다.
- 다음 줄에 연산의 수 Q가 주어진다.
- 이후 Q개의 줄에 연산이 주어진다.

## 출력 형식

- 연산 2에 대해 경로 상 최대 간선 가중치를 한 줄에 하나씩 출력한다.

## 제약 조건

- 2 ≤ N ≤ 100,000
- 1 ≤ Q ≤ 100,000
- 1 ≤ w ≤ 1,000,000
- 1 ≤ u, v ≤ N, u ≠ v
- 주어지는 그래프는 항상 트리이다
