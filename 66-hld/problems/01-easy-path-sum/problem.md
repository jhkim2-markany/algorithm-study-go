# 경로 합 구하기

**난이도:** 하

## 문제 설명

N개의 노드로 이루어진 트리가 주어진다. 노드는 1번부터 N번까지 번호가 매겨져 있고, 각 노드에는 정수 값이 부여되어 있다. 루트는 1번 노드이다.

Q개의 질의가 주어지며, 각 질의는 두 노드 u, v가 주어진다. 노드 u에서 노드 v까지의 경로에 포함된 모든 노드의 값의 합을 구하라.

## 입력 형식

- 첫째 줄에 노드의 수 N과 질의의 수 Q가 주어진다.
- 둘째 줄에 각 노드의 값 v₁, v₂, ..., vₙ이 공백으로 구분되어 주어진다.
- 셋째 줄부터 N-1개의 줄에 트리의 간선 정보 a, b가 주어진다. 이는 노드 a와 노드 b가 간선으로 연결되어 있음을 의미한다.
- 이후 Q개의 줄에 질의 u, v가 주어진다.

## 출력 형식

- 각 질의에 대해 경로 합을 한 줄에 하나씩 출력한다.

## 제약 조건

- 1 ≤ N ≤ 100,000
- 1 ≤ Q ≤ 100,000
- 1 ≤ vᵢ ≤ 1,000
- 1 ≤ u, v ≤ N
- 주어지는 그래프는 항상 트리이다
