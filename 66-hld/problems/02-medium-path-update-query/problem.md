# 경로 갱신과 질의

**난이도:** 중

## 문제 설명

N개의 노드로 이루어진 트리가 주어진다. 노드는 1번부터 N번까지 번호가 매겨져 있고, 각 노드의 초기 값은 0이다. 루트는 1번 노드이다.

Q개의 연산이 주어지며, 각 연산은 다음 두 종류 중 하나이다:

- `1 u v w`: 노드 u에서 노드 v까지의 경로에 포함된 모든 노드의 값에 w를 더한다.
- `2 u v`: 노드 u에서 노드 v까지의 경로에 포함된 모든 노드의 값의 합을 출력한다.

## 입력 형식

- 첫째 줄에 노드의 수 N과 연산의 수 Q가 주어진다.
- 둘째 줄부터 N-1개의 줄에 트리의 간선 정보 a, b가 주어진다.
- 이후 Q개의 줄에 연산이 주어진다.

## 출력 형식

- 연산 2에 대해 경로 합을 한 줄에 하나씩 출력한다.

## 제약 조건

- 1 ≤ N ≤ 100,000
- 1 ≤ Q ≤ 100,000
- 1 ≤ u, v ≤ N
- 1 ≤ w ≤ 1,000
- 모든 연산 결과는 64비트 정수 범위를 넘지 않는다
