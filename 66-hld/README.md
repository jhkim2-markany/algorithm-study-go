# 66. Heavy-Light Decomposition (HLD)

트리의 경로를 Heavy Chain과 Light Edge로 분류하여, 임의의 두 노드 사이의 경로를 O(log N)개의 연속 구간으로 분할하는 기법이다. 분할된 구간에 세그먼트 트리를 결합하면 경로 질의(합, 최댓값 등)와 경로 갱신을 O(log²N)에 처리할 수 있다.
