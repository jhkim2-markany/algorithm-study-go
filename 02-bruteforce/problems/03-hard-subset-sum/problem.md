# 부분집합의 합

**난이도:** 상

## 문제 설명

N개의 양의 정수로 이루어진 배열이 주어진다. 이 배열의 부분집합(빈 집합 제외) 중에서 원소의 합이 정확히 S가 되는 부분집합의 개수를 구하시오.

부분집합이란 배열의 원소 중 일부(또는 전부)를 선택한 것을 말한다. 같은 값을 가진 원소라도 서로 다른 위치에 있으면 다른 원소로 취급한다.

## 입력 형식

- 첫째 줄에 배열의 크기 N과 목표 합 S가 공백으로 구분되어 주어진다
- 둘째 줄에 N개의 양의 정수가 공백으로 구분되어 주어진다

## 출력 형식

- 합이 S가 되는 부분집합의 개수를 출력한다

## 제약 조건

- 1 ≤ N ≤ 20
- 1 ≤ S ≤ 2,000,000
- 1 ≤ 배열 원소 ≤ 100,000
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5 10
1 2 3 4 5
```

### 예제 출력 1

```
3
```

### 예제 입력 2

```
3 6
2 2 2
```

### 예제 출력 2

```
1
```
