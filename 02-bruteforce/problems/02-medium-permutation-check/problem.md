# 순열 검사

**난이도:** 중

## 문제 설명

N개의 서로 다른 정수로 이루어진 배열이 주어진다. 이 배열의 원소들을 재배열하여 만들 수 있는 모든 순열 중에서, 인접한 두 원소의 차이의 절댓값이 모두 K 이하인 순열이 존재하는지 판별하시오.

존재하면 해당 순열을 출력하고, 존재하지 않으면 "NO"를 출력한다. 답이 여러 개인 경우 사전순으로 가장 앞서는 순열을 출력한다.

## 입력 형식

- 첫째 줄에 배열의 크기 N과 정수 K가 공백으로 구분되어 주어진다
- 둘째 줄에 N개의 서로 다른 정수가 공백으로 구분되어 주어진다

## 출력 형식

- 조건을 만족하는 순열이 존재하면 해당 순열을 공백으로 구분하여 출력한다
- 존재하지 않으면 "NO"를 출력한다

## 제약 조건

- 1 ≤ N ≤ 8
- 1 ≤ K ≤ 100
- -100 ≤ 배열 원소 ≤ 100
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
4 3
1 5 3 7
```

### 예제 출력 1

```
1 3 5 7
```

### 예제 입력 2

```
3 1
1 5 10
```

### 예제 출력 2

```
NO
```
