# 두 수의 합

**난이도:** 하

## 문제 설명

N개의 정수로 이루어진 배열이 주어진다. 이 배열에서 두 수를 선택하여 합이 정확히 M이 되는 쌍이 존재하는지 판별하시오. 존재하면 그 쌍을 출력하고, 존재하지 않으면 "NO"를 출력한다.

같은 원소를 두 번 선택할 수 없다. 답이 여러 개인 경우 가장 먼저 발견되는 쌍을 출력한다. 쌍은 배열에서의 인덱스가 작은 원소부터 출력한다.

## 입력 형식

- 첫째 줄에 배열의 크기 N과 목표 합 M이 공백으로 구분되어 주어진다
- 둘째 줄에 N개의 정수가 공백으로 구분되어 주어진다

## 출력 형식

- 합이 M이 되는 두 수가 존재하면 두 수를 공백으로 구분하여 출력한다
- 존재하지 않으면 "NO"를 출력한다

## 제약 조건

- 2 ≤ N ≤ 1000
- -100,000 ≤ 배열 원소 ≤ 100,000
- -200,000 ≤ M ≤ 200,000
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5 9
1 3 5 7 2
```

### 예제 출력 1

```
7 2
```

### 예제 입력 2

```
4 100
1 2 3 4
```

### 예제 출력 2

```
NO
```
