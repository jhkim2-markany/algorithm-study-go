# 확장 유클리드와 모듈러 역원

**난이도:** 상

## 문제 설명

Q개의 쿼리가 주어진다. 각 쿼리는 두 정수 A와 M으로 구성되며, A의 모듈러 M에 대한 곱셈 역원을 구해야 한다.

모듈러 역원이란, A × X ≡ 1 (mod M)을 만족하는 정수 X (0 ≤ X < M)를 말한다. 모듈러 역원은 GCD(A, M) = 1일 때만 존재한다.

확장 유클리드 알고리즘을 사용하여 모듈러 역원을 구하시오. 역원이 존재하지 않는 경우 -1을 출력한다.

## 입력 형식

- 첫째 줄에 쿼리의 수 Q가 주어진다.
- 다음 Q개의 줄에 각각 두 정수 A와 M이 공백으로 구분되어 주어진다.

## 출력 형식

- 각 쿼리에 대해 A의 모듈러 M 역원을 한 줄에 하나씩 출력한다.
- 역원이 존재하지 않으면 -1을 출력한다.

## 제약 조건

- 1 ≤ Q ≤ 100,000
- 1 ≤ A ≤ 1,000,000,000
- 2 ≤ M ≤ 1,000,000,000

## 예제

### 예제 입력 1

```
4
3 7
10 17
6 9
2 1000000007
```

### 예제 출력 1

```
5
12
-1
500000004
```
