# 분수의 덧셈

**난이도:** 중

## 문제 설명

N개의 분수가 주어졌을 때, 이 분수들의 합을 기약분수로 나타내시오.

기약분수란 분자와 분모의 최대공약수가 1인 분수를 말한다. 예를 들어 2/4는 기약분수가 아니지만, 1/2는 기약분수이다.

분수의 합이 정수인 경우에도 분모를 1로 하여 출력한다.

## 입력 형식

- 첫째 줄에 분수의 개수 N이 주어진다.
- 다음 N개의 줄에 각 분수의 분자와 분모가 공백으로 구분되어 주어진다.

## 출력 형식

- 첫째 줄에 분수의 합을 기약분수로 나타낸 분자와 분모를 공백으로 구분하여 출력한다.

## 제약 조건

- 1 ≤ N ≤ 100
- 1 ≤ 분모 ≤ 10,000
- -10,000 ≤ 분자 ≤ 10,000
- 결과 분모는 항상 양수이다

## 예제

### 예제 입력 1

```
3
1 6
2 3
1 4
```

### 예제 출력 1

```
13 12
```

### 예제 입력 2

```
2
1 2
1 2
```

### 예제 출력 2

```
1 1
```
