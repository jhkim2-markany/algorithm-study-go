# 41. 유클리드 호제법

두 수의 최대공약수(GCD)를 효율적으로 구하는 알고리즘이다. 나머지 연산을 반복하여 GCD를 계산하며, 확장 유클리드 알고리즘을 통해 모듈러 역원까지 구할 수 있다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
