# 부분집합 열거 - 해설

## 접근 방식

1. N개의 원소로 이루어진 집합의 부분집합은 총 2^N개이다
2. 0부터 2^N - 1까지의 정수를 비트마스크로 사용한다
3. 각 비트마스크에서 i번째 비트가 1이면 원소 (i+1)이 부분집합에 포함된다
4. 비트마스크 값이 작은 순서대로 출력하면 사전순이 된다

## 핵심 아이디어

N개의 원소가 있을 때, 각 원소는 "포함" 또는 "미포함" 두 가지 상태를 가진다. 이를 비트 0과 1로 대응시키면, N비트 정수 하나가 하나의 부분집합을 나타낸다. 예를 들어 N=3일 때 비트마스크 5(이진수 101)는 1번과 3번 원소를 포함하는 부분집합 {1, 3}을 의미한다.

0부터 2^N - 1까지 순회하면 공집합(0)부터 전체 집합(2^N - 1)까지 모든 부분집합을 빠짐없이 열거할 수 있다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N × 2^N) |
| 공간 복잡도 | O(1) |

2^N개의 부분집합 각각에 대해 N개의 비트를 확인하므로 O(N × 2^N)이다. N ≤ 15이므로 최대 약 49만 번의 연산으로 충분히 빠르다.

## 대안적 접근

- 재귀를 이용한 백트래킹으로도 부분집합을 열거할 수 있다. 각 원소에 대해 포함/미포함을 결정하며 재귀 호출하는 방식이다. 비트마스킹 방식이 코드가 더 간결하고 비트 연산으로 빠르게 동작한다.
