# 부분집합 열거

**난이도:** 하

## 문제 설명

1부터 N까지의 자연수로 이루어진 집합이 주어진다. 이 집합의 모든 부분집합을 사전순으로 출력하시오. 각 부분집합은 원소를 오름차순으로 나열하며, 공집합도 포함한다.

## 입력 형식

- 첫째 줄에 자연수 N이 주어진다

## 출력 형식

- 모든 부분집합을 한 줄에 하나씩 출력한다
- 각 부분집합의 원소는 공백으로 구분하여 오름차순으로 출력한다
- 공집합은 빈 줄로 출력한다
- 부분집합은 사전순으로 출력한다 (비트마스크 값이 작은 순서)

## 제약 조건

- 1 ≤ N ≤ 15
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
3
```

### 예제 출력 1

```

1
2
1 2
3
1 3
2 3
1 2 3
```
