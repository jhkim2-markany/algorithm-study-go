# 비트마스크 집합 연산

**난이도:** 중

## 문제 설명

1부터 N까지의 자연수로 이루어진 전체 집합 U = {1, 2, ..., N}이 있다. 처음에 빈 집합 S가 주어지며, 다음 Q개의 연산을 순서대로 수행한 후 최종 집합 S의 원소를 출력하시오.

연산의 종류는 다음과 같다:

- `add x`: 원소 x를 S에 추가한다 (이미 있으면 무시)
- `remove x`: 원소 x를 S에서 제거한다 (없으면 무시)
- `toggle x`: 원소 x가 S에 있으면 제거하고, 없으면 추가한다
- `union A`: 집합 A와의 합집합으로 S를 갱신한다 (A는 공백으로 구분된 원소 목록)
- `intersect A`: 집합 A와의 교집합으로 S를 갱신한다

## 입력 형식

- 첫째 줄에 두 정수 N과 Q가 공백으로 구분되어 주어진다
- 다음 Q개의 줄에 각각 연산이 주어진다
- `union`과 `intersect` 연산의 경우, 연산 이름 뒤에 원소의 개수 K와 K개의 원소가 주어진다

## 출력 형식

- 최종 집합 S의 원소를 오름차순으로 공백으로 구분하여 출력한다
- 집합이 비어 있으면 `empty`를 출력한다

## 제약 조건

- 1 ≤ N ≤ 20
- 1 ≤ Q ≤ 100,000
- 1 ≤ x ≤ N
- union, intersect 연산의 원소는 모두 1 이상 N 이하이다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5 6
add 1
add 3
add 5
toggle 3
union 2 2 4
intersect 3 1 2 5
```

### 예제 출력 1

```
1 2 5
```
