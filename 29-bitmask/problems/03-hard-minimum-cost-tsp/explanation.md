# 최소 비용 외판원 문제 - 해설

## 접근 방식

1. 방문한 도시의 집합을 비트마스크로 표현한다
2. dp[mask][i]를 "방문한 도시 집합이 mask이고 현재 도시가 i일 때의 최소 비용"으로 정의한다
3. 초기 상태: dp[1][0] = 0 (도시 0에서 출발, 도시 0만 방문)
4. 모든 상태에 대해, 아직 방문하지 않은 도시로 이동하는 전이를 수행한다
5. 모든 도시를 방문한 상태(mask = 2^N - 1)에서 도시 0으로 돌아오는 비용을 더해 최솟값을 구한다

## 핵심 아이디어

외판원 문제(TSP)는 N!개의 순열을 모두 탐색하면 시간 초과가 발생한다. 비트마스크 DP를 사용하면 "어떤 도시들을 방문했는가"와 "현재 어디에 있는가"만으로 상태를 정의할 수 있어, 상태 수를 2^N × N으로 줄일 수 있다.

핵심 관찰은 "이미 방문한 도시의 집합과 현재 위치가 같으면, 방문 순서에 관계없이 이후 최적 경로가 동일하다"는 것이다. 이 최적 부분 구조 덕분에 DP가 적용 가능하다.

비트마스크의 각 비트는 해당 번호의 도시 방문 여부를 나타낸다. 예를 들어 N=4일 때 mask=0b1011(=11)은 도시 0, 1, 3을 방문한 상태이다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(2^N × N²) |
| 공간 복잡도 | O(2^N × N) |

상태 수가 2^N × N이고, 각 상태에서 N개의 도시로 전이를 시도하므로 O(2^N × N²)이다. N ≤ 16일 때 약 2^16 × 16² ≈ 1,677만으로 시간 내에 충분하다.

## 대안적 접근

- 브루트포스로 모든 순열을 탐색하면 O(N!)이므로 N=16일 때 약 2×10¹³으로 시간 초과이다. 비트마스크 DP가 사실상 유일한 효율적 풀이이다.
- 메모이제이션을 사용한 탑다운 방식으로도 구현할 수 있다. 재귀 함수 solve(mask, cur)를 정의하고, 이미 계산된 상태는 캐시에서 반환하는 방식이다.
