# 29. 비트마스킹

정수의 이진 표현을 활용하여 집합의 상태를 효율적으로 표현하고 조작하는 알고리즘 유형이다. AND, OR, XOR, 시프트 등 비트 연산을 통해 부분집합 관리, 상태 압축, 플래그 처리 등을 수행한다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
