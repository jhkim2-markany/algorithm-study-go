# 최근접 점 쌍 - 해설

## 접근 방식

1. 모든 점을 x좌표 기준으로 정렬한다.
2. 점들을 중간 지점을 기준으로 왼쪽과 오른쪽으로 나눈다 (분할).
3. 왼쪽 부분과 오른쪽 부분에서 각각 최근접 점 쌍의 거리를 재귀적으로 구한다 (정복).
4. 두 결과 중 작은 값을 d라 하면, 중간선에서 d 이내에 있는 점들만 모아 교차 영역을 검사한다 (결합).
5. 교차 영역의 점들을 y좌표 기준으로 정렬하고, 각 점에 대해 y좌표 차이가 d 이내인 점들만 비교한다.

## 핵심 아이디어

분할 정복의 핵심은 결합 단계의 효율성이다. 중간선 근처의 교차 영역에서 최근접 점 쌍을 찾을 때, y좌표 기준으로 정렬된 점들에 대해 각 점은 최대 6~7개의 점만 비교하면 된다는 것이 수학적으로 증명되어 있다.

이는 d × 2d 크기의 직사각형 안에 서로 거리가 d 이상인 점이 최대 8개까지만 존재할 수 있기 때문이다. 따라서 교차 영역 검사는 O(N)에 수행되며, 전체 알고리즘은 O(N log² N) 또는 최적화 시 O(N log N)이 된다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N log² N) |
| 공간 복잡도 | O(N) |

초기 x좌표 정렬에 O(N log N), 각 재귀 단계에서 교차 영역의 y좌표 정렬에 O(N log N)이 걸린다. T(N) = 2T(N/2) + O(N log N)이므로 O(N log² N)이다. y좌표 정렬을 미리 해두면 O(N log N)으로 최적화할 수 있다.

## 대안적 접근

- 브루트포스로 모든 쌍을 확인하면 O(N²)이지만, N ≤ 100,000에서는 시간 초과가 발생한다.
- 랜덤화 알고리즘을 사용하면 기대 시간 복잡도 O(N)에 해결할 수 있지만, 구현이 복잡하다.
- y좌표 정렬을 병합 정렬의 병합 과정에서 함께 수행하면 O(N log N)으로 최적화할 수 있다.
