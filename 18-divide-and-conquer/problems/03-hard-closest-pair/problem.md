# 최근접 점 쌍

**난이도:** 상

## 문제 설명

2차원 평면 위에 N개의 점이 주어진다. 이 중 가장 가까운 두 점 사이의 거리를 구하시오.

두 점 (x1, y1)과 (x2, y2) 사이의 거리는 유클리드 거리 √((x1-x2)² + (y1-y2)²)이다.

## 입력 형식

- 첫째 줄에 점의 개수 N이 주어진다 (2 ≤ N ≤ 100,000)
- 다음 N개의 줄에 각 점의 x좌표와 y좌표가 공백으로 구분되어 주어진다 (-1,000,000 ≤ x, y ≤ 1,000,000)

## 출력 형식

- 가장 가까운 두 점 사이의 거리를 소수점 아래 6자리까지 출력한다

## 제약 조건

- 2 ≤ N ≤ 100,000
- -1,000,000 ≤ x, y ≤ 1,000,000
- 모든 점의 좌표는 정수이다
- 같은 위치에 두 개 이상의 점이 존재할 수 있다
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
4
0 0
10 10
1 1
5 5
```

### 예제 출력 1

```
1.414214
```

### 예제 입력 2

```
3
0 0
1 0
0 1
```

### 예제 출력 2

```
1.000000
```
