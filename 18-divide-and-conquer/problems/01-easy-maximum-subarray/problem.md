# 최대 부분 배열 합

**난이도:** 하

## 문제 설명

N개의 정수로 이루어진 배열이 주어진다. 연속된 부분 배열의 합 중 최댓값을 구하시오.

부분 배열은 배열에서 연속된 하나 이상의 원소로 이루어진 부분이다.

## 입력 형식

- 첫째 줄에 배열의 크기 N이 주어진다 (1 ≤ N ≤ 100,000)
- 둘째 줄에 N개의 정수가 공백으로 구분되어 주어진다 (-1,000 ≤ 각 원소 ≤ 1,000)

## 출력 형식

- 최대 부분 배열 합을 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- -1,000 ≤ 각 원소 ≤ 1,000
- 부분 배열은 최소 1개의 원소를 포함해야 한다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
9
-2 1 -3 4 -1 2 1 -5 4
```

### 예제 출력 1

```
6
```

### 예제 입력 2

```
5
-1 -2 -3 -4 -5
```

### 예제 출력 2

```
-1
```
