# 최대 부분 배열 합 - 해설

## 접근 방식

1. 배열을 중간 지점을 기준으로 왼쪽과 오른쪽으로 나눈다.
2. 최대 부분 배열은 다음 세 가지 경우 중 하나에 해당한다:
   - 왼쪽 부분에만 존재하는 경우
   - 오른쪽 부분에만 존재하는 경우
   - 중간 지점을 걸치는 경우
3. 왼쪽과 오른쪽은 재귀적으로 해결하고, 중간을 걸치는 경우는 중간에서 양쪽으로 확장하며 최대 합을 구한다.
4. 세 값 중 최댓값이 답이 된다.

## 핵심 아이디어

분할 정복의 기본 패턴을 적용한 문제이다. 핵심은 중간을 걸치는 부분 배열의 최대 합을 O(N)에 구하는 것이다.

중간 지점에서 왼쪽으로 확장하며 최대 합을 구하고, 오른쪽으로 확장하며 최대 합을 구한 뒤, 두 값을 더하면 중간을 걸치는 최대 부분 배열 합이 된다. 이 과정이 O(N)이고 재귀 깊이가 O(log N)이므로 전체 시간 복잡도는 O(N log N)이다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N log N) |
| 공간 복잡도 | O(log N) |

T(N) = 2T(N/2) + O(N) 점화식을 따르며, 마스터 정리에 의해 O(N log N)이다. 공간 복잡도는 재귀 호출 스택의 깊이인 O(log N)이다.

## 대안적 접근

- 카데인 알고리즘(Kadane's Algorithm)을 사용하면 O(N) 시간에 해결할 수 있다. 동적 프로그래밍 기반으로 현재까지의 최대 부분 배열 합을 갱신하는 방식이다.
- 브루트포스로 모든 부분 배열을 확인하면 O(N²) 또는 O(N³)이 걸린다.
