# 병합 정렬 역전 카운트

**난이도:** 중

## 문제 설명

N개의 정수로 이루어진 배열이 주어진다. 배열에서 역전(inversion)의 수를 구하시오.

역전이란 i < j이면서 arr[i] > arr[j]인 쌍 (i, j)을 말한다. 즉, 앞에 있는 원소가 뒤에 있는 원소보다 큰 경우의 수이다.

## 입력 형식

- 첫째 줄에 배열의 크기 N이 주어진다 (1 ≤ N ≤ 500,000)
- 둘째 줄에 N개의 정수가 공백으로 구분되어 주어진다 (1 ≤ 각 원소 ≤ 1,000,000)

## 출력 형식

- 역전의 수를 출력한다

## 제약 조건

- 1 ≤ N ≤ 500,000
- 1 ≤ 각 원소 ≤ 1,000,000
- 모든 원소는 서로 다르다
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5
2 3 8 6 1
```

### 예제 출력 1

```
5
```

### 예제 입력 2

```
3
1 2 3
```

### 예제 출력 2

```
0
```
