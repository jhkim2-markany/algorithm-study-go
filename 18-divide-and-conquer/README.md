# 18. 분할 정복

문제를 더 작은 부분 문제로 나누고(Divide), 각 부분 문제를 재귀적으로 해결한 뒤(Conquer), 결과를 합쳐(Combine) 원래 문제의 해를 구하는 알고리즘 기법이다. 병합 정렬, 퀵 정렬, 최근접 점 쌍 등 다양한 문제에 활용된다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
