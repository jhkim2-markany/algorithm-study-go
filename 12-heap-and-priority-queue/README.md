# 12. 힙과 우선순위 큐

완전 이진 트리 기반의 자료구조로, 최솟값 또는 최댓값을 O(log N)에 삽입·삭제할 수 있다. Go의 `container/heap` 패키지를 활용하여 우선순위 큐를 구현하고 다양한 문제에 적용한다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이

## 문제 목록

| 번호 | 문제 | 난이도 |
| --- | --- | --- |
| 01 | K번째 큰 수 | 하 |
| 02 | 정렬된 리스트 합치기 | 중 |
| 03 | 데이터 스트림의 중앙값 | 상 |
