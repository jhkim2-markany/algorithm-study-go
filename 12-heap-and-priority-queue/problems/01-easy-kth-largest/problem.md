# K번째 큰 수

**난이도:** 하

## 문제 설명

N개의 정수가 주어진다. 이 중에서 K번째로 큰 수를 구하시오.

예를 들어, 정수 배열이 [3, 1, 4, 1, 5, 9, 2, 6]이고 K가 3이면, 큰 순서대로 9, 6, 5이므로 3번째로 큰 수는 5이다.

## 입력 형식

- 첫째 줄에 정수의 개수 N과 K가 공백으로 구분되어 주어진다 (1 ≤ K ≤ N ≤ 100,000)
- 둘째 줄에 N개의 정수가 공백으로 구분되어 주어진다

## 출력 형식

- K번째로 큰 수를 출력한다

## 제약 조건

- 1 ≤ K ≤ N ≤ 100,000
- -1,000,000,000 ≤ 각 정수 ≤ 1,000,000,000
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
8 3
3 1 4 1 5 9 2 6
```

### 예제 출력 1

```
5
```

### 예제 입력 2

```
5 1
10 20 30 40 50
```

### 예제 출력 2

```
50
```
