# 데이터 스트림의 중앙값

**난이도:** 상

## 문제 설명

정수가 하나씩 순서대로 주어진다. 각 정수가 추가될 때마다 지금까지 입력된 모든 정수의 중앙값을 구하시오.

중앙값은 정렬된 수열의 가운데 값이다. 수열의 길이가 홀수이면 정확히 가운데 값이고, 짝수이면 가운데 두 값 중 작은 값을 중앙값으로 한다.

## 입력 형식

- 첫째 줄에 정수의 개수 N이 주어진다 (1 ≤ N ≤ 100,000)
- 다음 N개의 줄에 각각 하나의 정수가 주어진다

## 출력 형식

- N개의 줄에 걸쳐, 각 정수가 추가된 시점의 중앙값을 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- -10,000 ≤ 각 정수 ≤ 10,000
- 시간 제한: 1초
- 메모리 제한: 256MB
