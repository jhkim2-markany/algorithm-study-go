# 음수 사이클 탐지

**난이도:** 상

## 문제 설명

N개의 정점과 M개의 간선으로 이루어진 방향 가중치 그래프가 주어진다. 간선의 가중치는 음수일 수 있다.

이 그래프에 음수 사이클(사이클을 이루는 간선 가중치의 합이 음수인 사이클)이 존재하는지 판별하시오.

음수 사이클이 존재하면 "YES"를, 존재하지 않으면 "NO"를 출력한다.

## 입력 형식

- 첫째 줄에 테스트 케이스의 수 T가 주어진다 (1 ≤ T ≤ 5)
- 각 테스트 케이스의 첫째 줄에 정점의 수 N과 간선의 수 M이 주어진다 (1 ≤ N ≤ 500, 1 ≤ M ≤ 5,000)
- 다음 M개의 줄에 간선 정보 u, v, w가 주어진다 (u에서 v로 가는 가중치 w)
- 1 ≤ u, v ≤ N, -10,000 ≤ w ≤ 10,000

## 출력 형식

- 각 테스트 케이스마다 음수 사이클이 존재하면 "YES", 존재하지 않으면 "NO"를 출력한다

## 제약 조건

- 1 ≤ T ≤ 5
- 1 ≤ N ≤ 500
- 1 ≤ M ≤ 5,000
- 1 ≤ u, v ≤ N
- -10,000 ≤ w ≤ 10,000
- 자기 자신으로의 간선이 존재할 수 있다
- 중복 간선이 존재할 수 있다
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
2
3 3
1 2 1
2 3 2
3 1 3
3 3
1 2 1
2 3 -5
3 1 2
```

### 예제 출력 1

```
NO
YES
```

### 예제 입력 2

```
1
4 4
1 2 -1
2 3 -1
3 4 -1
4 2 -1
```

### 예제 출력 2

```
YES
```
