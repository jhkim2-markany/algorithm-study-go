# 다익스트라 경로 추적

**난이도:** 중

## 문제 설명

N개의 도시와 M개의 도로가 있다. 각 도로는 단방향이며 이동 비용이 주어진다.

도시 1에서 도시 N까지의 최단 거리와 그 경로를 구하시오. 최단 경로가 여러 개인 경우 아무거나 하나를 출력하면 된다.

## 입력 형식

- 첫째 줄에 도시의 수 N과 도로의 수 M이 주어진다 (2 ≤ N ≤ 100,000, 1 ≤ M ≤ 200,000)
- 다음 M개의 줄에 도로 정보 u, v, w가 주어진다 (u에서 v로 가는 비용 w)
- 1 ≤ u, v ≤ N, 1 ≤ w ≤ 100,000

## 출력 형식

- 첫째 줄에 도시 1에서 도시 N까지의 최단 거리를 출력한다
- 둘째 줄에 최단 경로에 포함된 도시의 수를 출력한다
- 셋째 줄에 최단 경로를 이루는 도시 번호를 순서대로 공백으로 구분하여 출력한다
- 도달할 수 없는 경우 첫째 줄에 -1을 출력한다

## 제약 조건

- 2 ≤ N ≤ 100,000
- 1 ≤ M ≤ 200,000
- 1 ≤ u, v ≤ N
- 1 ≤ w ≤ 100,000
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5 6
1 2 2
1 3 3
2 3 4
2 4 5
3 4 1
4 5 2
```

### 예제 출력 1

```
6
4
1 3 4 5
```

### 예제 입력 2

```
3 1
1 2 5
```

### 예제 출력 2

```
-1
```
