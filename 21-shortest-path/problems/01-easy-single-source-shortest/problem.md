# 단일 출발점 최단 경로

**난이도:** 하

## 문제 설명

N개의 도시와 M개의 도로가 있다. 각 도로는 두 도시를 양방향으로 연결하며, 모든 도로의 이동 비용은 동일하게 1이다.

도시 1에서 출발하여 각 도시까지의 최단 거리를 구하시오. 도달할 수 없는 도시는 -1을 출력한다.

## 입력 형식

- 첫째 줄에 도시의 수 N과 도로의 수 M이 주어진다 (1 ≤ N ≤ 100,000, 0 ≤ M ≤ 200,000)
- 다음 M개의 줄에 도로로 연결된 두 도시 번호 u, v가 주어진다 (1 ≤ u, v ≤ N)

## 출력 형식

- N개의 줄에 걸쳐 도시 1에서 도시 i까지의 최단 거리를 출력한다
- 도달할 수 없는 경우 -1을 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- 0 ≤ M ≤ 200,000
- 1 ≤ u, v ≤ N
- 자기 자신으로의 간선은 없다
- 중복 간선이 존재할 수 있다
- 시간 제한: 1초
- 메모리 제한: 256MB
