# 단일 출발점 최단 경로 - 해설

## 접근 방식

1. 모든 간선의 가중치가 1인 비가중치 그래프이므로 BFS를 사용한다.
2. 도시 1을 시작점으로 BFS를 수행하며, 각 도시까지의 최단 거리를 기록한다.
3. BFS는 레벨 단위로 탐색하므로, 처음 방문하는 시점의 거리가 곧 최단 거리이다.
4. 방문하지 못한 도시는 -1로 출력한다.

## 핵심 아이디어

비가중치 그래프에서의 최단 경로는 BFS로 구할 수 있다. BFS는 출발점에서 가까운 정점부터 순서대로 방문하므로, 각 정점을 처음 방문할 때의 거리가 최단 거리가 된다.

거리 배열을 -1로 초기화하면 방문 여부와 도달 불가 판별을 동시에 처리할 수 있다. 출발점의 거리를 0으로 설정한 후, 인접 정점의 거리가 -1이면 현재 거리 + 1로 갱신하고 큐에 넣는다.

Dijkstra 알고리즘을 사용해도 되지만, 모든 가중치가 동일한 경우 BFS가 더 간단하고 효율적이다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N + M) |
| 공간 복잡도 | O(N + M) |

BFS는 모든 정점과 간선을 한 번씩 방문하므로 O(N + M)이다. 인접 리스트와 거리 배열에 O(N + M) 공간이 필요하다.

## 대안적 접근

- Dijkstra 알고리즘을 사용할 수 있지만, 비가중치 그래프에서는 BFS가 O(N + M)으로 더 효율적이다.
- 가중치가 0 또는 1인 그래프에서는 0-1 BFS(덱 사용)를 적용할 수 있다.
