# 42. 볼록 껍질

주어진 점 집합을 모두 포함하는 가장 작은 볼록 다각형을 구하는 알고리즘이다. Graham Scan, Andrew's Monotone Chain 등의 방법으로 CCW를 활용하여 볼록 껍질을 구성한다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이

## 문제 목록

| 번호 | 문제 | 난이도 |
| --- | --- | --- |
| 01 | 볼록 껍질 구하기 | 하 |
| 02 | 볼록 껍질 둘레 계산 | 중 |
| 03 | 가장 먼 두 점 (회전하는 캘리퍼스) | 상 |
