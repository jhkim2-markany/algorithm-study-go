# 볼록 껍질 구하기

**난이도:** 하

## 문제 설명

2차원 평면 위에 N개의 점이 주어진다. 이 점들의 볼록 껍질을 이루는 꼭짓점의 개수를 구하시오.

볼록 껍질이란 주어진 모든 점을 포함하는 가장 작은 볼록 다각형이다. 볼록 껍질의 변 위에 있는 점은 꼭짓점으로 포함하지 않는다.

## 입력 형식

- 첫째 줄에 점의 개수 N이 주어진다.
- 둘째 줄부터 N개의 줄에 각 점의 x좌표와 y좌표가 공백으로 구분되어 주어진다.

## 출력 형식

- 볼록 껍질을 이루는 꼭짓점의 개수를 출력한다.

## 제약 조건

- 3 ≤ N ≤ 100,000
- -1,000,000 ≤ x, y ≤ 1,000,000
- 모든 점이 일직선 위에 있는 경우는 주어지지 않는다

## 예제

### 예제 입력 1

```
8
0 0
4 0
4 4
0 4
2 1
1 2
3 3
2 2
```

### 예제 출력 1

```
4
```

### 예제 입력 2

```
5
0 0
1 0
2 0
1 1
0 2
```

### 예제 출력 2

```
4
```
