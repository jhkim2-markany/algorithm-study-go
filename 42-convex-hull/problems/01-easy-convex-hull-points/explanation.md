# 볼록 껍질 구하기 - 해설

## 접근 방식

1. N개의 점을 x좌표 기준으로 정렬한다 (같으면 y좌표 기준).
2. Andrew's Monotone Chain 알고리즘을 사용하여 볼록 껍질을 구성한다.
3. 왼쪽에서 오른쪽으로 순회하며 하부 껍질을 만든다.
4. 오른쪽에서 왼쪽으로 순회하며 상부 껍질을 만든다.
5. 두 껍질을 합쳐 볼록 껍질의 꼭짓점 개수를 출력한다.

## 핵심 아이디어

Andrew's Monotone Chain은 정렬된 점을 순서대로 처리하면서, CCW(외적)를 이용해 볼록성을 유지한다. 새로운 점을 추가할 때 마지막 두 점과의 방향이 우회전(시계 방향)이거나 일직선이면 마지막 점을 제거한다. 이렇게 하면 항상 좌회전만 남아 볼록 다각형이 된다.

CCW 값이 0인 경우(일직선)도 제거 대상에 포함하여, 변 위의 점은 꼭짓점에 포함하지 않는다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N log N) |
| 공간 복잡도 | O(N) |

정렬에 O(N log N), 껍질 구성에 O(N)이 소요된다. 각 점은 최대 한 번 추가되고 한 번 제거되므로 전체 O(N)이다.

## 대안적 접근

Graham Scan을 사용할 수도 있다. 기준점에서 각도 정렬 후 스택으로 처리하는 방식이며, 같은 O(N log N) 복잡도를 가진다. 다만 각도 정렬 시 atan2를 사용하면 부동소수점 오차가 발생할 수 있어, 정수 좌표에서는 Monotone Chain이 더 안정적이다.
