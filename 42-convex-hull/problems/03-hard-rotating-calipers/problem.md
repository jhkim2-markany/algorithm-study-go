# 가장 먼 두 점

**난이도:** 상

## 문제 설명

2차원 평면 위에 N개의 점이 주어진다. 이 중 가장 먼 두 점 사이의 거리의 제곱을 구하시오.

## 입력 형식

- 첫째 줄에 점의 개수 N이 주어진다.
- 둘째 줄부터 N개의 줄에 각 점의 x좌표와 y좌표가 공백으로 구분되어 주어진다.

## 출력 형식

- 가장 먼 두 점 사이의 거리의 제곱을 출력한다.

## 제약 조건

- 2 ≤ N ≤ 100,000
- -1,000,000 ≤ x, y ≤ 1,000,000
- 모든 점의 좌표가 같은 경우는 주어지지 않는다

## 예제

### 예제 입력 1

```
5
0 0
4 0
3 4
1 3
0 2
```

### 예제 출력 1

```
25
```

### 예제 입력 2

```
4
0 0
10 0
10 10
0 10
```

### 예제 출력 2

```
200
```
