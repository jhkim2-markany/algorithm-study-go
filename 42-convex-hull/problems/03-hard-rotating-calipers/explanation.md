# 가장 먼 두 점 - 해설

## 접근 방식

1. N개의 점에 대해 볼록 껍질을 구한다.
2. 가장 먼 두 점은 반드시 볼록 껍질의 꼭짓점이므로, 껍질 위의 점만 고려하면 된다.
3. 회전하는 캘리퍼스(Rotating Calipers) 기법을 사용하여 O(N)에 최대 거리를 구한다.
4. 볼록 껍질의 각 변에 대해 대척점(가장 먼 점)을 추적하며 최대 거리를 갱신한다.

## 핵심 아이디어

가장 먼 두 점은 반드시 볼록 껍질의 꼭짓점이다. 이 성질을 이용하면 내부 점을 무시하고 껍질 위의 점만 탐색할 수 있다.

회전하는 캘리퍼스는 볼록 껍질의 변을 순회하면서, 각 변에 대한 대척점(antipodal point)을 추적하는 기법이다. 핵심 관찰은 변을 순서대로 이동할 때 대척점도 같은 방향으로만 이동한다는 것이다. 따라서 두 포인터처럼 대척점을 한 방향으로만 전진시키면 전체 O(N)에 모든 대척점 쌍을 확인할 수 있다.

단순히 볼록 껍질의 모든 점 쌍을 비교하면 O(H²)이지만, 회전하는 캘리퍼스를 사용하면 O(H)로 줄일 수 있다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N log N) |
| 공간 복잡도 | O(N) |

볼록 껍질 구성에 O(N log N), 회전하는 캘리퍼스에 O(H)가 소요된다. H ≤ N이므로 전체 시간 복잡도는 O(N log N)이다.

## 대안적 접근

볼록 껍질의 모든 꼭짓점 쌍을 비교하는 O(H²) 방법도 있다. H가 충분히 작으면(예: H = O(√N)) 실용적일 수 있지만, 최악의 경우 H = N이므로 O(N²)이 된다. 회전하는 캘리퍼스가 항상 효율적이다.
