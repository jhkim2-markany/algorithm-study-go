# 거스름돈

**난이도:** 하

## 문제 설명

편의점 점원이 손님에게 거스름돈을 줘야 한다. 거스름돈으로 사용할 수 있는 동전의 종류는 500원, 100원, 50원, 10원으로 총 4가지이다. 각 동전의 개수는 무한하다고 가정할 때, 거슬러 줘야 할 금액이 주어지면 최소 동전 개수를 구하시오.

거슬러 줘야 할 금액은 항상 10의 배수이다.

## 입력 형식

- 첫째 줄에 거슬러 줘야 할 금액 N이 주어진다 (10 ≤ N ≤ 1,000,000)
- N은 항상 10의 배수이다

## 출력 형식

- 최소 동전 개수를 출력한다

## 제약 조건

- 10 ≤ N ≤ 1,000,000
- N은 10의 배수
- 동전 종류: 500원, 100원, 50원, 10원
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
1260
```

### 예제 출력 1

```
6
```

### 예제 입력 2

```
2680
```

### 예제 출력 2

```
11
```
