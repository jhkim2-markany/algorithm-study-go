# 활동 선택

**난이도:** 중

## 문제 설명

한 개의 회의실에서 N개의 회의를 진행하려고 한다. 각 회의는 시작 시간과 종료 시간이 정해져 있으며, 한 회의가 끝나는 시간과 다른 회의가 시작하는 시간이 같은 경우 두 회의를 연속으로 진행할 수 있다.

회의실을 사용할 수 있는 회의의 최대 개수를 구하시오.

## 입력 형식

- 첫째 줄에 회의의 수 N이 주어진다 (1 ≤ N ≤ 100,000)
- 다음 N개의 줄에 각 회의의 시작 시간 S와 종료 시간 E가 공백으로 구분되어 주어진다

## 출력 형식

- 회의실을 사용할 수 있는 최대 회의 수를 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- 0 ≤ S < E ≤ 2,000,000,000
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5
1 4
3 5
0 6
5 7
3 9
```

### 예제 출력 1

```
3
```

### 예제 입력 2

```
3
1 3
2 4
3 5
```

### 예제 출력 2

```
2
```
