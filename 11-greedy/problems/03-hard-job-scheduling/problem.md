# 작업 스케줄링

**난이도:** 상

## 문제 설명

N개의 작업이 있다. 각 작업은 마감 기한과 보상이 정해져 있다. 하루에 하나의 작업만 수행할 수 있으며, 각 작업은 정확히 하루가 걸린다. 작업은 마감 기한 이전(마감 기한 당일 포함)에 완료해야 보상을 받을 수 있다.

1일차부터 시작하여 작업을 수행할 때, 얻을 수 있는 최대 보상의 합을 구하시오.

## 입력 형식

- 첫째 줄에 작업의 수 N이 주어진다 (1 ≤ N ≤ 100,000)
- 다음 N개의 줄에 각 작업의 마감 기한 D와 보상 P가 공백으로 구분되어 주어진다

## 출력 형식

- 얻을 수 있는 최대 보상의 합을 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- 1 ≤ D ≤ N
- 1 ≤ P ≤ 10,000
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5
2 50
1 10
2 20
1 30
3 40
```

### 예제 출력 1

```
120
```

### 예제 입력 2

```
4
1 100
1 200
1 300
2 250
```

### 예제 출력 2

```
550
```
