# 팩토리얼과 피보나치

**난이도:** 하

## 문제 설명

두 가지 재귀 함수를 구현하시오.

1. 자연수 N이 주어졌을 때, N!의 값을 구하시오.
2. 자연수 M이 주어졌을 때, 피보나치 수열의 M번째 값을 구하시오.

피보나치 수열은 다음과 같이 정의된다:
- F(0) = 0
- F(1) = 1
- F(n) = F(n-1) + F(n-2) (n ≥ 2)

## 입력 형식

- 첫째 줄에 자연수 N이 주어진다.
- 둘째 줄에 자연수 M이 주어진다.

## 출력 형식

- 첫째 줄에 N!의 값을 출력한다.
- 둘째 줄에 피보나치 수열의 M번째 값을 출력한다.

## 제약 조건

- 0 ≤ N ≤ 20
- 0 ≤ M ≤ 40

## 예제

### 예제 입력 1

```
5
10
```

### 예제 출력 1

```
120
55
```
