# 하노이의 탑

**난이도:** 중

## 문제 설명

하노이의 탑 문제는 세 개의 기둥(A, B, C)과 N개의 원판으로 구성된다. 처음에 모든 원판은 기둥 A에 크기 순서대로 쌓여 있다 (가장 큰 원판이 아래, 가장 작은 원판이 위).

다음 규칙을 지키면서 모든 원판을 기둥 A에서 기둥 C로 옮기시오:

1. 한 번에 하나의 원판만 옮길 수 있다.
2. 큰 원판을 작은 원판 위에 놓을 수 없다.
3. 기둥의 맨 위에 있는 원판만 옮길 수 있다.

총 이동 횟수와 각 이동 과정을 출력하시오.

## 입력 형식

- 첫째 줄에 원판의 수 N이 주어진다.

## 출력 형식

- 첫째 줄에 총 이동 횟수를 출력한다.
- 둘째 줄부터 각 이동을 "X Y" 형식으로 출력한다 (기둥 X에서 기둥 Y로 이동).
- 기둥은 1, 2, 3으로 표기한다 (A=1, B=2, C=3).

## 제약 조건

- 1 ≤ N ≤ 20
- N이 20 이하이므로 이동 과정을 모두 출력한다

## 예제

### 예제 입력 1

```
3
```

### 예제 출력 1

```
7
1 3
1 2
3 2
1 3
2 1
2 3
1 3
```
