# 모든 부분집합 생성

**난이도:** 상

## 문제 설명

1부터 N까지의 자연수로 이루어진 집합이 주어진다. 이 집합의 모든 부분집합을 사전순으로 출력하시오.

각 부분집합은 원소를 오름차순으로 정렬하여 출력하며, 부분집합들은 다음 기준으로 사전순 정렬한다:
1. 부분집합의 크기가 작은 것이 먼저 온다.
2. 크기가 같으면 원소를 사전순으로 비교한다.

빈 집합도 포함하여 출력한다.

## 입력 형식

- 첫째 줄에 자연수 N이 주어진다.

## 출력 형식

- 첫째 줄에 부분집합의 총 개수를 출력한다.
- 둘째 줄부터 각 부분집합을 한 줄에 하나씩 출력한다.
- 빈 집합은 빈 줄로 출력한다.
- 각 부분집합의 원소는 공백으로 구분한다.

## 제약 조건

- 1 ≤ N ≤ 15
- 부분집합의 총 개수는 2^N개이다

## 예제

### 예제 입력 1

```
3
```

### 예제 출력 1

```
8

1
2
3
1 2
1 3
2 3
1 2 3
```
