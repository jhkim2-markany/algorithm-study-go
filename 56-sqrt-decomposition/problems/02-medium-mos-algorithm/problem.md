# 구간 서로 다른 수의 개수 (Mo's Algorithm)

**난이도:** 중

## 문제 설명

N개의 정수로 이루어진 배열 A가 주어진다.

Q개의 쿼리가 주어지며, 각 쿼리는 구간 [l, r]이다. 각 쿼리에 대해 A[l]부터 A[r]까지 서로 다른 수의 개수를 구하라. (1-indexed, 양 끝 포함)

## 입력 형식

- 첫째 줄에 배열의 크기 N이 주어진다.
- 둘째 줄에 배열 A의 원소 N개가 공백으로 구분되어 주어진다.
- 셋째 줄에 쿼리의 수 Q가 주어진다.
- 다음 Q줄에 각각 l, r이 주어진다.

## 출력 형식

- 각 쿼리에 대해 구간 내 서로 다른 수의 개수를 한 줄에 하나씩 출력한다.

## 제약 조건

- 1 ≤ N ≤ 100,000
- 1 ≤ Q ≤ 100,000
- 1 ≤ A[i] ≤ 1,000,000
- 1 ≤ l ≤ r ≤ N

## 예제

### 예제 입력 1

```

```
3

```

```

1
```

### 예제 입력 3

```
