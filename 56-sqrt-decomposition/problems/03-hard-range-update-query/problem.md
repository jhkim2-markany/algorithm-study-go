# 구간 갱신과 구간 합 쿼리

**난이도:** 상

## 문제 설명

N개의 정수로 이루어진 배열 A가 주어진다.

다음 두 종류의 연산을 Q번 수행하라.

- `1 l r v`: A[l]부터 A[r]까지 모든 원소에 v를 더한다 (1-indexed, 양 끝 포함)
- `2 l r`: A[l]부터 A[r]까지의 합을 출력한다 (1-indexed, 양 끝 포함)

## 입력 형식

- 첫째 줄에 배열의 크기 N과 연산의 수 Q가 주어진다.
- 둘째 줄에 배열 A의 원소 N개가 공백으로 구분되어 주어진다.
- 다음 Q줄에 각각 연산이 주어진다.

## 출력 형식

- 각 2번 연산에 대해 구간 합을 한 줄에 하나씩 출력한다.

## 제약 조건

- 1 ≤ N ≤ 100,000
- 1 ≤ Q ≤ 100,000
- 1 ≤ A[i] ≤ 1,000,000
- 1 ≤ v ≤ 10,000
- 1 ≤ l ≤ r ≤ N

## 예제

### 예제 입력 1

```
8 6
1 3 5 2 7 6 3 8
2 1 8
1 2 5 3
2 1 8
2 2 5
1 1 8 1
2 1 8
```

```
35
47
25
55
```

### 예제 입력 2

```
3 4

1 1 3 5
2 1 3

```

### 예제 출력 2

```
75
```
