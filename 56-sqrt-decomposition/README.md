# 56. 제곱근 분할법 (Sqrt Decomposition)

배열을 √N 크기의 블록으로 분할하여 구간 쿼리와 갱신을 효율적으로 처리하는 기법이다. 세그먼트 트리보다 구현이 간단하며, Mo's Algorithm과 결합하면 오프라인 구간 쿼리를 O((N+Q)√N)에 처리할 수 있다.
