# LIS 경로 역추적

**난이도:** 중

## 문제 설명

N개의 정수로 이루어진 수열 A가 주어진다.

이 수열에서 가장 긴 순증가 부분 수열의 길이와 그 부분 수열을 구하라.

가장 긴 순증가 부분 수열이 여러 개인 경우, 아무거나 하나를 출력하면 된다.

## 입력 형식

- 첫째 줄에 수열의 길이 N이 주어진다.
- 둘째 줄에 수열 A의 원소 N개가 공백으로 구분되어 주어진다.

## 출력 형식

- 첫째 줄에 가장 긴 순증가 부분 수열의 길이를 출력한다.
- 둘째 줄에 그 부분 수열을 공백으로 구분하여 출력한다.

## 제약 조건

- 1 ≤ N ≤ 100,000
- 1 ≤ A[i] ≤ 1,000,000

## 예제

### 예제 입력 1

```
6
10 20 10 30 20 50
```

### 예제 출력 1

```
4
10 20 30 50
```

### 예제 입력 2

```
5
5 4 3 2 1
```

### 예제 출력 2

```
1
1
```

### 예제 입력 3

```
1
7
```

### 예제 출력 3

```
1
7
```
