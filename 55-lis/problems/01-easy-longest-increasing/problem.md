# 최장 증가 부분 수열 길이

**난이도:** 하

## 문제 설명

N개의 정수로 이루어진 수열 A가 주어진다.

이 수열에서 가장 긴 순증가 부분 수열의 길이를 구하라.

부분 수열이란 수열에서 일부 원소를 골라 원래 순서를 유지한 수열이며, 순증가란 선택한 원소들이 앞에서 뒤로 갈수록 값이 순수하게 증가하는 것을 말한다.

## 입력 형식

- 첫째 줄에 수열의 길이 N이 주어진다.
- 둘째 줄에 수열 A의 원소 N개가 공백으로 구분되어 주어진다.

## 출력 형식

- 가장 긴 순증가 부분 수열의 길이를 출력한다.

## 제약 조건

- 1 ≤ N ≤ 100,000
- 1 ≤ A[i] ≤ 1,000,000

## 예제

### 예제 입력 1

```
6
10 20 10 30 20 50
```

### 예제 출력 1

```
4
```

### 예제 입력 2

```
5
5 4 3 2 1
```

### 예제 출력 2

```
1
```

### 예제 입력 3

```
1
42
```

### 예제 출력 3

```
1
```
