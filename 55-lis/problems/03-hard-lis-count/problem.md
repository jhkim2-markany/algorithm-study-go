# LIS 개수 세기

**난이도:** 상

## 문제 설명

N개의 정수로 이루어진 수열 A가 주어진다.

이 수열에서 가장 긴 순증가 부분 수열(LIS)의 길이와, 그러한 LIS의 개수를 구하라.

LIS의 개수가 매우 클 수 있으므로, 개수를 1,000,000,007로 나눈 나머지를 출력한다.

## 입력 형식

- 첫째 줄에 수열의 길이 N이 주어진다.
- 둘째 줄에 수열 A의 원소 N개가 공백으로 구분되어 주어진다.

## 출력 형식

- 첫째 줄에 LIS의 길이를 출력한다.
- 둘째 줄에 LIS의 개수를 1,000,000,007로 나눈 나머지를 출력한다.

## 제약 조건

- 1 ≤ N ≤ 5,000
- 1 ≤ A[i] ≤ 1,000,000

## 예제

### 예제 입력 1

```
6
1 3 2 4 3 5
```

### 예제 출력 1

```
4
3
```

### 예제 입력 2

```
5
1 2 3 4 5
```

### 예제 출력 2

```
5
1
```

### 예제 입력 3

```
6
10 20 10 30 20 50
```

### 예제 출력 3

```
4
1
```
