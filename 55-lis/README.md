# 55. LIS (최장 증가 부분 수열)

수열에서 원소의 순서를 유지하면서 값이 순증가하는 가장 긴 부분 수열을 구하는 알고리즘이다. O(N²) DP 방식과 O(N log N) 이분 탐색 방식이 있으며, 코딩테스트에서는 이분 탐색 기반 풀이가 필수이다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
