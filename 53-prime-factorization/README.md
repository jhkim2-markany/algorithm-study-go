# 53. 소인수분해 (Prime Factorization)

주어진 자연수를 소수의 곱으로 분해하는 알고리즘 유형이다. 시행 나눗셈(Trial Division), 최적화된 소인수분해, 최소 소인수(SPF) 체를 이용한 전처리 기반 소인수분해 등이 핵심이다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
