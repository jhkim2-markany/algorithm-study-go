# 소인수분해 출력 - 해설

## 접근 방식

1. 가장 작은 소수인 2부터 시작하여 N을 나누어 본다
2. N이 d로 나누어 떨어지면, 나누어 떨어지지 않을 때까지 반복하여 나누고 매번 d를 출력한다
3. d를 1 증가시키며 d² ≤ N인 동안 반복한다
4. 반복 종료 후 N > 1이면 N 자체가 마지막 소인수이므로 출력한다

## 핵심 아이디어

시행 나눗셈(Trial Division)을 사용한다. N이 합성수라면 √N 이하의 소인수를 반드시 가지므로, d² ≤ N인 범위까지만 탐색하면 된다. 작은 소인수부터 순서대로 나누기 때문에 결과는 자동으로 오름차순이 된다.

d가 소수가 아닌 합성수일 때는 이미 그보다 작은 소인수로 N을 모두 나눈 상태이므로, N % d == 0이 성립하지 않아 자연스럽게 건너뛰게 된다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(√N) |
| 공간 복잡도 | O(1) |

## 대안적 접근

- 짝수를 별도 처리하고 홀수만 탐색하면 상수 시간을 절반으로 줄일 수 있다.
- N이 최대 10^7이므로 SPF 체를 전처리하면 O(log N)에 소인수분해할 수 있지만, 단일 수에 대해서는 시행 나눗셈이 더 간단하고 충분히 빠르다.
