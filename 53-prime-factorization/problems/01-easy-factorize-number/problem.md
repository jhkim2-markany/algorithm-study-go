# 소인수분해 출력

**난이도:** 하

## 문제 설명

자연수 N이 주어졌을 때, N을 소인수분해한 결과를 출력하라. 소인수를 오름차순으로 한 줄에 하나씩 출력한다. 같은 소인수가 여러 번 곱해지면 그 횟수만큼 반복하여 출력한다.

## 입력 형식

첫째 줄에 자연수 N이 주어진다. (2 ≤ N ≤ 10,000,000)

## 출력 형식

N의 소인수를 오름차순으로 한 줄에 하나씩 출력한다. 같은 소인수는 중복하여 출력한다.

## 제약 조건

- 2 ≤ N ≤ 10,000,000
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
12
```

### 예제 출력 1

```
2
2
3
```

### 예제 입력 2

```
60
```

### 예제 출력 2

```
2
2
3
5
```

### 예제 입력 3

```
97
```

### 예제 출력 3

```
97
```
