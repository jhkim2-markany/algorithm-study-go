# 구간 소인수 합

**난이도:** 상

## 문제 설명

자연수 L과 R이 주어졌을 때, L 이상 R 이하의 모든 자연수에 대해 각 수의 소인수의 합을 구하라. 이때 같은 소인수가 여러 번 곱해지면 그 횟수만큼 더한다.

예를 들어 12 = 2² × 3이므로 소인수의 합은 2 + 2 + 3 = 7이다. 구간 [2, 5]에서 각 수의 소인수 합은 다음과 같다:
- 2 → 2
- 3 → 3
- 4 = 2² → 2 + 2 = 4
- 5 → 5

따라서 전체 합은 2 + 3 + 4 + 5 = 14이다.

## 입력 형식

첫째 줄에 두 자연수 L과 R이 공백으로 구분되어 주어진다.

## 출력 형식

L 이상 R 이하의 모든 자연수에 대해 소인수 합의 총합을 출력한다.

## 제약 조건

- 2 ≤ L ≤ R ≤ 5,000,000
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
2 5
```

### 예제 출력 1

```
14
```

### 예제 입력 2

```
10 12
```

### 예제 출력 2

```
25
```

### 예제 입력 3

```
100 100
```

### 예제 출력 3

```
14
```
