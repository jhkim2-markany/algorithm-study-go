# 구간 소인수 합 - 해설

## 접근 방식

1. R까지의 SPF(최소 소인수) 체를 구축한다
2. 구간 [L, R]의 각 수에 대해 SPF 배열을 이용하여 소인수분해한다
3. 소인수분해 과정에서 각 소인수를 지수만큼 더하여 소인수 합을 구한다
4. 모든 수의 소인수 합을 누적하여 총합을 출력한다

## 핵심 아이디어

구간의 크기가 최대 5,000,000이고 각 수를 시행 나눗셈으로 소인수분해하면 O((R-L+1) × √R)로 시간 초과가 발생할 수 있다. SPF 체를 전처리하면 각 수의 소인수분해를 O(log N)에 수행할 수 있어 전체 시간 복잡도가 크게 개선된다.

SPF 체의 핵심은 에라토스테네스의 체를 변형하여 각 합성수에 대해 가장 작은 소인수만 기록하는 것이다. 이후 소인수분해 시 spf[x]로 최소 소인수를 바로 알 수 있으므로, x를 해당 소인수로 나누고 다음 최소 소인수로 넘어가는 과정을 반복하면 된다.

결과가 int 범위를 초과할 수 있으므로 int64를 사용하여 누적한다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(R log log R + (R-L+1) log R) |
| 공간 복잡도 | O(R) |

- SPF 체 구축: O(R log log R)
- 구간 소인수 합 계산: O((R-L+1) × log R)

## 대안적 접근

- SPF 체 대신 에라토스테네스의 체를 변형하여 각 소수 p의 배수에 대해 p를 직접 더하는 방식도 가능하다. 이 경우 소인수분해 단계 없이 O(R log log R)에 모든 수의 소인수 합을 구할 수 있다.
- R이 매우 크고 구간이 좁은 경우, 세그먼트 체(Segmented Sieve)를 응용하여 메모리를 절약할 수 있다.
