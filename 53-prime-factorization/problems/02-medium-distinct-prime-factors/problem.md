# 서로 다른 소인수 개수 쿼리

**난이도:** 중

## 문제 설명

자연수 N과 Q개의 쿼리가 주어진다. 각 쿼리마다 자연수 X가 주어지면, X의 서로 다른 소인수의 개수를 출력하라.

예를 들어 12 = 2² × 3이므로 서로 다른 소인수는 2와 3으로 2개이다. 60 = 2² × 3 × 5이므로 서로 다른 소인수는 3개이다.

## 입력 형식

첫째 줄에 쿼리의 최대 범위 N과 쿼리의 수 Q가 공백으로 구분되어 주어진다.

둘째 줄부터 Q개의 줄에 각각 자연수 X가 주어진다.

## 출력 형식

각 쿼리마다 X의 서로 다른 소인수의 개수를 한 줄에 하나씩 출력한다.

## 제약 조건

- 2 ≤ N ≤ 1,000,000
- 1 ≤ Q ≤ 200,000
- 2 ≤ X ≤ N
- 시간 제한: 2초
- 메모리 제한: 256MB
