# 08. 이진 탐색

정렬된 배열에서 탐색 범위를 절반씩 줄여가며 원하는 값을 효율적으로 찾는 알고리즘이다. O(log N) 시간 복잡도로 대규모 데이터에서도 빠르게 탐색할 수 있다.
