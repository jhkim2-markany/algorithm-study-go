# Lower/Upper Bound

**난이도:** 중

## 문제 설명

N개의 정수가 오름차순으로 정렬된 배열이 주어진다. 이후 M개의 질의가 주어지며, 각 질의마다 정수 X가 주어진다. 배열에서 X가 등장하는 횟수를 출력하시오.

## 입력 형식

- 첫째 줄에 배열의 크기 N과 질의의 수 M이 공백으로 구분되어 주어진다 (1 ≤ N, M ≤ 100,000)
- 둘째 줄에 N개의 정수가 오름차순으로 공백으로 구분되어 주어진다 (중복 가능)
- 셋째 줄부터 M개의 줄에 각각 찾고자 하는 정수 X가 주어진다

## 출력 형식

- M개의 줄에 각 질의에 대해 X가 배열에 등장하는 횟수를 출력한다

## 제약 조건

- 1 ≤ N, M ≤ 100,000
- -1,000,000,000 ≤ 배열 원소, X ≤ 1,000,000,000
- 배열은 오름차순으로 정렬되어 있다 (중복 원소 가능)
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
8 3
1 2 2 2 3 3 5 7
2
3
4
```

### 예제 출력 1

```
3
2
0
```

### 예제 입력 2

```
5 2
1 1 1 1 1
1
2
```

### 예제 출력 2

```
5
0
```
