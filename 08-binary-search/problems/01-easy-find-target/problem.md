# 값 찾기

**난이도:** 하

## 문제 설명

N개의 정수가 오름차순으로 정렬된 배열이 주어진다. 이후 M개의 질의가 주어지며, 각 질의마다 특정 정수가 배열에 존재하는지 확인하여 존재하면 해당 인덱스(0-based)를, 존재하지 않으면 -1을 출력하시오.

## 입력 형식

- 첫째 줄에 배열의 크기 N과 질의의 수 M이 공백으로 구분되어 주어진다 (1 ≤ N, M ≤ 100,000)
- 둘째 줄에 N개의 정수가 오름차순으로 공백으로 구분되어 주어진다
- 셋째 줄부터 M개의 줄에 각각 찾고자 하는 정수가 주어진다

## 출력 형식

- M개의 줄에 각 질의에 대한 결과를 출력한다
- 배열에 값이 존재하면 해당 인덱스(0-based)를, 존재하지 않으면 -1을 출력한다

## 제약 조건

- 1 ≤ N, M ≤ 100,000
- -1,000,000,000 ≤ 배열 원소 ≤ 1,000,000,000
- 배열의 모든 원소는 서로 다르다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5 3
1 3 5 7 9
5
1
6
```

### 예제 출력 1

```
2
0
-1
```

### 예제 입력 2

```
3 2
-10 0 10
0
5
```

### 예제 출력 2

```
1
-1
```
