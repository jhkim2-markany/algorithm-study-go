# 행렬에서 K번째 수

**난이도:** 상

## 문제 설명

N×N 크기의 행렬이 주어진다. 이 행렬의 각 행과 각 열은 오름차순으로 정렬되어 있다. 행렬의 모든 원소를 오름차순으로 나열했을 때 K번째 수를 구하시오.

## 입력 형식

- 첫째 줄에 행렬의 크기 N이 주어진다 (1 ≤ N ≤ 1,500)
- 다음 N개의 줄에 각각 N개의 정수가 공백으로 구분되어 주어진다
- 마지막 줄에 K가 주어진다 (1 ≤ K ≤ N²)

## 출력 형식

- 행렬의 모든 원소를 오름차순으로 나열했을 때 K번째 수를 출력한다

## 제약 조건

- 1 ≤ N ≤ 1,500
- -1,000,000,000 ≤ 행렬 원소 ≤ 1,000,000,000
- 각 행은 왼쪽에서 오른쪽으로 오름차순이다
- 각 열은 위에서 아래로 오름차순이다
- 1 ≤ K ≤ N²
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
3
1 3 7
2 5 8
4 6 9
7
```

### 예제 출력 1

```
7
```

### 예제 입력 2

```
2
1 2
3 4
3
```

### 예제 출력 2

```
3
```
