# 최소 비용 할당

**난이도:** 하

## 문제 설명

N명의 작업자와 N개의 작업이 있다. 각 작업자를 정확히 하나의 작업에 배정해야 하며, 각 작업에는 정확히 한 명의 작업자만 배정할 수 있다. 작업자 i를 작업 j에 배정할 때 드는 비용이 `cost[i][j]`로 주어진다.

모든 작업자를 작업에 1:1로 배정할 때, 총 비용의 최솟값을 구하라.

## 입력 형식

- 첫째 줄에 작업자와 작업의 수 N이 주어진다.
- 둘째 줄부터 N개의 줄에 비용 행렬이 주어진다. i번째 줄의 j번째 정수는 작업자 i를 작업 j에 배정하는 비용이다.

## 출력 형식

- 모든 작업자를 작업에 1:1 배정할 때의 최소 총 비용을 출력한다.

## 제약 조건

- 1 ≤ N ≤ 100
- 1 ≤ cost[i][j] ≤ 1,000

## 예제

### 예제 입력 1

```
3
3 8 5
5 2 7
6 4 1
```

### 예제 출력 1

```
6
```

### 예제 입력 2

```
2
1 2
3 4
```

### 예제 출력 2

```
5
```

## 힌트

- 예제 1: 작업자1→작업1(3), 작업자2→작업2(2), 작업자3→작업3(1)으로 배정하면 총 비용 = 6
- 예제 2: 작업자1→작업1(1), 작업자2→작업2(4)로 배정하면 총 비용 = 5
