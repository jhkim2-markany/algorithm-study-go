# 최소 비용 수송

**난이도:** 중

## 문제 설명

P개의 공장과 W개의 창고가 있다. 각 공장 i는 `supply[i]`개의 물건을 생산하고, 각 창고 j는 `demand[j]`개의 물건을 필요로 한다. 공장 i에서 창고 j로 물건 1개를 운송하는 비용은 `cost[i][j]`이다.

모든 창고의 수요를 만족시키면서 총 운송 비용을 최소화하라. 전체 공급량의 합은 전체 수요량의 합 이상임이 보장된다.

## 입력 형식

- 첫째 줄에 공장 수 P와 창고 수 W가 주어진다.
- 둘째 줄에 각 공장의 공급량 `supply[1], supply[2], ..., supply[P]`가 공백으로 구분되어 주어진다.
- 셋째 줄에 각 창고의 수요량 `demand[1], demand[2], ..., demand[W]`가 공백으로 구분되어 주어진다.
- 넷째 줄부터 P개의 줄에 비용 행렬이 주어진다. i번째 줄의 j번째 정수는 공장 i에서 창고 j로의 단위 운송 비용이다.

## 출력 형식

- 모든 창고의 수요를 만족시키는 최소 총 운송 비용을 출력한다.

## 제약 조건

- 1 ≤ P, W ≤ 50
- 1 ≤ supply[i], demand[j] ≤ 100
- 1 ≤ cost[i][j] ≤ 1,000
- 전체 공급량 ≥ 전체 수요량

## 예제

### 예제 입력 1

```
2 3
10 8
5 6 4
2 3 5
4 1 3
```

### 예제 출력 1

```
32
```

### 예제 입력 2

```
2 2
5 5
5 5
1 3
2 4
```

### 예제 출력 2

```
15
```

## 힌트

- 예제 1: 공장1→창고1(5개, 비용10), 공장1→창고3(4개, 비용20), 공장2→창고3(0개), 공장2→창고2(6개, 비용6) 등 여러 조합 중 최소 비용을 찾는다
- 공장에서 남는 물건은 운송하지 않아도 된다
