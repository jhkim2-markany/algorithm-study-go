# 63. 최소 비용 최대 유량 (MCMF - Minimum Cost Maximum Flow)

네트워크에서 최대 유량을 보내되, 그 비용을 최소화하는 알고리즘이다. 최대 유량 알고리즘에서 증가 경로를 찾을 때 SPFA(Bellman-Ford 기반)를 사용하여 최소 비용 경로를 우선 선택한다. 할당 문제, 수송 문제, 프로젝트 선택 등 다양한 최적화 문제에 활용된다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
