# 57. 중간에서 만나기 (Meet in the Middle)

탐색 공간을 반으로 나누어 각각 브루트포스로 탐색한 뒤, 두 결과를 합쳐서 답을 구하는 최적화 기법이다. 전체 탐색이 O(2^N)일 때 O(2^(N/2))로 줄일 수 있어, N이 40 정도인 부분집합 문제에서 핵심적으로 사용된다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이

## 문제 목록

| 번호 | 문제 | 난이도 |
| --- | --- | --- |
| 01 | 부분집합 합 존재 판정 | 하 |
| 02 | 목표값에 가장 가까운 부분집합 합 | 중 |
| 03 | 합이 0인 네 값 조합 수 세기 | 상 |
