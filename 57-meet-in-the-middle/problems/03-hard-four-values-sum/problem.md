# 합이 0인 네 값 조합 수 세기

**난이도:** 상

## 문제 설명

네 개의 정수 배열 A, B, C, D가 주어진다. 각 배열에서 하나씩 원소를 선택하여 네 원소의 합이 0이 되는 조합의 수를 구하라.

즉, A[i] + B[j] + C[k] + D[l] = 0을 만족하는 (i, j, k, l) 쌍의 수를 구한다.

## 입력 형식

- 첫째 줄에 배열의 크기 N이 주어진다 (네 배열의 크기는 모두 동일).
- 둘째 줄에 배열 A의 N개의 정수가 공백으로 구분되어 주어진다.
- 셋째 줄에 배열 B의 N개의 정수가 공백으로 구분되어 주어진다.
- 넷째 줄에 배열 C의 N개의 정수가 공백으로 구분되어 주어진다.
- 다섯째 줄에 배열 D의 N개의 정수가 공백으로 구분되어 주어진다.

## 출력 형식

- A[i] + B[j] + C[k] + D[l] = 0을 만족하는 조합의 수를 출력한다.

## 제약 조건

- 1 ≤ N ≤ 4,000
- -10,000,000 ≤ 각 원소 ≤ 10,000,000

## 예제

### 예제 입력 1

```
3
1 -1 0
2 -2 0
3 -3 0
-6 6 0
```

### 예제 출력 1

```
16
```

### 예제 입력 2

```
2
1 2
3 4
-5 -6
1 0
```

### 예제 출력 2

```
2
```
