# 목표값에 가장 가까운 부분집합 합

**난이도:** 중

## 문제 설명

N개의 양의 정수로 이루어진 배열이 주어진다. 이 배열에서 일부 원소를 선택하여 그 합이 목표값 S에 가장 가까운 부분집합을 찾고, 그 합을 출력하라.

여러 부분집합의 합이 동일한 차이를 가지면, 그 중 아무 값이나 출력해도 된다. 빈 부분집합(합 = 0)도 유효한 선택이다.

## 입력 형식

- 첫째 줄에 배열의 크기 N과 목표값 S가 공백으로 구분되어 주어진다.
- 둘째 줄에 N개의 양의 정수가 공백으로 구분되어 주어진다.

## 출력 형식

- 목표값 S에 가장 가까운 부분집합의 합을 출력한다.

## 제약 조건

- 1 ≤ N ≤ 40
- 1 ≤ 각 원소 ≤ 1,000,000
- 1 ≤ S ≤ 20,000,000

## 예제

### 예제 입력 1

```
6 15
3 1 4 1 5 9
```

### 예제 출력 1

```
15
```

### 예제 입력 2

```
4 10
7 3 2 5
```

### 예제 출력 2

```
10
```

### 예제 입력 3

```
3 100
1 2 3
```

### 예제 출력 3

```
6
```
