# 부분집합 합 존재 판정

**난이도:** 하

## 문제 설명

N개의 정수로 이루어진 배열이 주어진다. 이 배열에서 일부 원소를 선택하여 그 합이 정확히 S가 되는 부분집합이 존재하는지 판정하라.

빈 부분집합(아무것도 선택하지 않는 경우)의 합은 0이다.

## 입력 형식

- 첫째 줄에 배열의 크기 N과 목표 합 S가 공백으로 구분되어 주어진다.
- 둘째 줄에 N개의 정수가 공백으로 구분되어 주어진다.

## 출력 형식

- 합이 S인 부분집합이 존재하면 "Yes", 존재하지 않으면 "No"를 출력한다.

## 제약 조건

- 1 ≤ N ≤ 40
- -1,000,000 ≤ 각 원소 ≤ 1,000,000
- -20,000,000 ≤ S ≤ 20,000,000

## 예제

### 예제 입력 1

```
6 10
3 1 4 1 5 9
```

### 예제 출력 1

```
Yes
```

### 예제 입력 2

```
5 100
1 2 3 4 5
```

### 예제 출력 2

```
No
```

### 예제 입력 3

```
3 0
1 -1 2
```

### 예제 출력 3

```
Yes
```
