# 부분집합 합 존재 판정 - 해설

## 접근 방식

1. 배열을 전반부와 후반부로 반씩 나눈다
2. 전반부의 모든 부분집합 합을 비트마스크로 열거한다 (sumA)
3. 후반부의 모든 부분집합 합을 비트마스크로 열거한다 (sumB)
4. sumB를 정렬한다
5. sumA의 각 원소 a에 대해, sumB에서 S - a를 이분 탐색한다
6. 찾으면 "Yes", 모두 실패하면 "No"를 출력한다

## 핵심 아이디어

N ≤ 40이므로 완전 탐색 O(2^40)은 시간 초과이다. 중간에서 만나기(MITM)를 적용하면 O(2^20 × 20) ≈ 2천만으로 충분히 해결된다.

배열을 반으로 나누면 각 절반의 부분집합 수는 최대 2^20 ≈ 100만 개이다. 한쪽을 정렬해두면 다른 쪽의 각 원소에 대해 이분 탐색으로 보완값을 O(log N)에 찾을 수 있다.

빈 부분집합(mask=0)도 유효한 부분집합이므로 합 0이 sumA와 sumB에 모두 포함된다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(2^(N/2) × N) |
| 공간 복잡도 | O(2^(N/2)) |

## 대안적 접근

- N ≤ 20이면 단순 비트마스크 브루트포스 O(2^N)으로 해결 가능하다
- 합의 범위가 작으면 (|S| ≤ 10^7) DP로 O(N × S)에 해결할 수 있다
