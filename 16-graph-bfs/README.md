# 16. 그래프 탐색 BFS

그래프의 너비를 우선으로 탐색하는 알고리즘이다. 시작 정점에서 가까운 정점부터 차례대로 방문하며, 큐를 사용하여 구현한다. 최단 거리 탐색, 레벨별 탐색, 이분 그래프 판별 등 다양한 그래프 문제에 활용된다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
