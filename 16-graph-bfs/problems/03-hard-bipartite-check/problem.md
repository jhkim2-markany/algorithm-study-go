# 이분 그래프 판별

**난이도:** 상

## 문제 설명

N개의 정점과 M개의 간선으로 이루어진 무방향 그래프가 주어진다. 이 그래프가 이분 그래프(Bipartite Graph)인지 판별하시오.

이분 그래프란 그래프의 모든 정점을 두 그룹으로 나눌 수 있되, 같은 그룹에 속한 정점끼리는 간선으로 연결되지 않는 그래프를 말한다. 즉, 모든 간선의 양 끝 정점이 서로 다른 그룹에 속해야 한다.

그래프가 비연결 그래프일 수 있으며, 각 연결 요소에 대해 독립적으로 이분 그래프 여부를 확인해야 한다.

## 입력 형식

- 첫째 줄에 테스트 케이스의 수 T가 주어진다 (1 ≤ T ≤ 5)
- 각 테스트 케이스의 첫째 줄에 정점의 수 N과 간선의 수 M이 주어진다 (1 ≤ N ≤ 20,000, 0 ≤ M ≤ 200,000)
- 다음 M개의 줄에 각각 두 정수 u, v가 주어진다 (1 ≤ u, v ≤ N, u ≠ v)

## 출력 형식

- 각 테스트 케이스에 대해 이분 그래프이면 "YES", 아니면 "NO"를 출력한다

## 제약 조건

- 1 ≤ T ≤ 5
- 1 ≤ N ≤ 20,000
- 0 ≤ M ≤ 200,000
- 같은 간선이 중복으로 주어질 수 있다
- 자기 자신으로의 간선은 주어지지 않는다
- 시간 제한: 2초
- 메모리 제한: 256MB
