# 이분 그래프 판별 - 해설

## 접근 방식

1. 그래프를 인접 리스트로 표현한다.
2. 모든 정점에 대해 미방문이면 BFS를 시작한다 (비연결 그래프 처리).
3. BFS를 수행하며 각 정점에 두 가지 색(1 또는 2) 중 하나를 칠한다.
4. 시작 정점에 색 1을 칠하고, 인접 정점에는 반대 색을 칠한다.
5. 인접한 정점이 이미 같은 색으로 칠해져 있으면 이분 그래프가 아니다.
6. 모든 연결 요소에서 충돌 없이 2-색칠이 완료되면 이분 그래프이다.

## 핵심 아이디어

이분 그래프 판별은 그래프 2-색칠(2-coloring) 문제와 동치이다. 그래프의 정점을 두 색으로 칠할 때, 인접한 정점이 항상 다른 색이 되도록 칠할 수 있으면 이분 그래프이다.

BFS는 레벨 단위로 탐색하므로 2-색칠에 자연스럽게 적합하다. 시작 정점을 색 1로 칠하면, BFS에서 거리가 홀수인 정점은 색 2, 짝수인 정점은 색 1이 된다. 탐색 중 같은 색의 정점이 인접해 있으면 홀수 길이의 사이클이 존재한다는 뜻이며, 이는 이분 그래프가 아님을 의미한다.

비연결 그래프의 경우 각 연결 요소를 독립적으로 검사해야 한다. 하나의 연결 요소라도 이분 그래프가 아니면 전체 그래프도 이분 그래프가 아니다.

중복 간선이 주어질 수 있으므로 인접 리스트에 같은 정점이 여러 번 들어갈 수 있지만, 2-색칠 로직에는 영향을 주지 않는다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N + M) per 테스트 케이스 |
| 공간 복잡도 | O(N + M) |

각 정점과 간선을 한 번씩 확인하므로 O(N + M)이다. 인접 리스트에 O(N + M), 색칠 배열과 큐에 O(N) 공간이 필요하다.

## 대안적 접근

- DFS를 사용한 2-색칠도 동일한 시간 복잡도로 가능하다. 재귀 DFS에서 현재 정점과 반대 색을 인접 정점에 칠하는 방식이다.
- 유니온 파인드를 활용하여 각 정점과 그 "반대 그룹"을 관리하는 방법도 있다.
