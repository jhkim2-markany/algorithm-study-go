# 미로 최단 거리

**난이도:** 하

## 문제 설명

N×M 크기의 미로가 주어진다. 미로에서 1은 이동할 수 있는 칸, 0은 이동할 수 없는 벽을 나타낸다. (1, 1) 위치에서 출발하여 (N, M) 위치로 이동할 때, 지나야 하는 최소 칸 수를 구하시오. 시작 칸과 도착 칸도 포함하여 센다.

이동은 상하좌우 4방향으로만 가능하며, 미로 밖으로는 이동할 수 없다.

## 입력 형식

- 첫째 줄에 미로의 행 수 N과 열 수 M이 주어진다 (2 ≤ N, M ≤ 100)
- 다음 N개의 줄에 M개의 숫자가 공백 없이 주어진다. 각 숫자는 0 또는 1이다

## 출력 형식

- (1, 1)에서 (N, M)까지 이동하는 데 지나야 하는 최소 칸 수를 출력한다

## 제약 조건

- 2 ≤ N, M ≤ 100
- (1, 1)과 (N, M)은 항상 1이다
- (1, 1)에서 (N, M)으로 이동할 수 있는 경로가 항상 존재한다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
4 6
101111
101010
101011
111011
```

### 예제 출력 1

```
15
```

### 예제 입력 2

```
2 2
11
11
```

### 예제 출력 2

```
3
```
