# 추후 추가할 알고리즘 목록

현재 프로젝트에 포함되지 않은 알고리즘들을 BOJ 분류 기준 문제 수와 함께 정리한다.
추후 학습 콘텐츠 확장 시 우선순위 참고용으로 사용한다.

> 기준일: 2026-03-23
> 현재 프로젝트: 67개 알고리즘 유형 (01~67번 폴더)

---

## 독립 폴더로 분리 예정

아래 항목들은 현재 다른 폴더에 포함되어 있지만, 추후 별도 폴더로 분리하여 이론·예시·문제를 체계적으로 구성할 예정입니다.

| 항목 | 현재 위치 | 현재 커버 수준 |
| --- | --- | --- |
| SCC (강한 연결 요소) | 24-graph-advanced | 이론+예시+문제 완비 (Kosaraju, Tarjan) |
| Mo's Algorithm | 35-offline-queries | 이론+예시 포함 |
| 스프라그-그런디 정리 | 30-game-theory | 이론+예시 포함 |
| 라빈-카프 | 48-hashing | 이론+예시 포함 |
| 밀러-라빈 | 34-primality-test | 이론+예시 포함 |
| 외판원 순회 (TSP) | 32-bitmask-dp | 대표 예시로 포함 |
| KMP | 26-string-algorithm | 이론에서 다룸 |
| 벨만-포드 | 21-shortest-path | 예시 코드 포함 |
| 플로이드-워셜 | 50-floyd-warshall | 독립 폴더로 분리 완료 |

## 추가 가능한 알고리즘 (미구현)

### Tier 3: 대회 준비 (플래티넘~다이아)

ICPC, BOJ 플래티넘 이상, 대회 준비용이다.
코테만 준비한다면 후순위로 둬도 된다.

| 번호 | 알고리즘 | BOJ 문제 수 | 설명 |
| --- | --- | --- | --- |
| 68 | 오일러 경로/회로 (Eulerian Path) | 87 | 모든 간선을 한 번씩 방문하는 경로/회로 |
| 69 | 단절점과 단절선 (Articulation Points & Bridges) | 82 | 그래프 연결성 분석. DFS 기반 |
| 70 | 2-SAT | 75 | 논리식 만족 가능성. SCC 활용 |
| 71 | 퍼시스턴트 세그먼트 트리 | 64 | 버전 관리 가능한 세그먼트 트리 |
| 72 | 아호-코라식 (Aho-Corasick) | 34 | 다중 패턴 매칭. 트라이 + KMP |
| 73 | 매내처 (Manacher's) | 35 | 최장 회문 부분 문자열 O(N) |
| 74 | Slope Trick | 34 | 볼록 함수 기울기 추적 DP 최적화 |

### Tier 4: 고급 대회 (다이아 이상, 선택적)

알면 좋지만 코테에서는 거의 나오지 않는다. 대회 입상 목표 시 학습한다.

| 번호 | 알고리즘 | BOJ 문제 수 | 설명 |
| --- | --- | --- | --- |
| 75 | 스플레이 트리 (Splay Tree) | 47 | 자가 균형 이진 탐색 트리 |
| 76 | 병렬 이분 탐색 (Parallel Binary Search) | 40 | 여러 쿼리를 동시에 이분 탐색 |
| 77 | 회전하는 캘리퍼스 (Rotating Calipers) | 39 | 볼록 껍질 위 최원점 쌍 |
| 78 | 폴라드 로 (Pollard Rho) | 34 | 큰 수의 소인수분해 |
| 79 | 뫼비우스 반전 (Möbius Inversion) | 31 | 정수론적 함수 반전 |
| 80 | 링크/컷 트리 (Link-Cut Tree) | 28 | 동적 트리 연결/분리 |
| 81 | 담금질 기법 (Simulated Annealing) | 26 | 메타휴리스틱 최적화 |
| 82 | 리-차오 트리 (Li Chao Tree) | 24 | 일차 함수 삽입/최솟값 쿼리 |
| 83 | Aliens 트릭 | 24 | 라그랑주 완화 기반 DP 최적화 |
| 84 | 세그먼트 트리 비츠 (Segment Tree Beats) | 20 | 구간 chmin/chmax 연산 |
