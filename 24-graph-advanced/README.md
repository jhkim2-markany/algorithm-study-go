# 24. 그래프 알고리즘 기타

이분 그래프 판별, 강한 연결 요소(SCC), 네트워크 플로우 등 기본 그래프 탐색을 넘어서는 고급 그래프 알고리즘을 다룬다. 이분 그래프는 BFS/DFS 색칠로 판별하고, SCC는 Kosaraju 또는 Tarjan 알고리즘으로 구하며, 네트워크 플로우는 Ford-Fulkerson/Edmonds-Karp 알고리즘으로 최대 유량을 계산한다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
