# 강한 연결 요소

**난이도:** 중

## 문제 설명

N개의 정점과 M개의 간선으로 이루어진 방향 그래프가 주어진다. 이 그래프의 강한 연결 요소(Strongly Connected Component, SCC)의 개수를 구하시오.

강한 연결 요소란 그 안의 임의의 두 정점 u, v에 대해 u에서 v로, v에서 u로 모두 도달할 수 있는 최대 정점 집합이다.

## 입력 형식

- 첫째 줄에 정점의 수 N과 간선의 수 M이 주어진다
- 둘째 줄부터 M개의 줄에 각 간선의 시작 정점 U와 끝 정점 V가 주어진다 (U → V)

## 출력 형식

- 첫째 줄에 SCC의 개수를 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- 0 ≤ M ≤ 200,000
- 1 ≤ U, V ≤ N
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
6 8
1 2
2 3
3 1
2 4
4 5
5 6
6 4
5 1
```

### 예제 출력 1

```
2
```

### 예제 입력 2

```
4 4
1 2
2 3
3 4
4 1
```

### 예제 출력 2

```
1
```
