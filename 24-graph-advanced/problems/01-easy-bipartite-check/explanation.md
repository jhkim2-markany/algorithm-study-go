# 이분 그래프 판별 - 해설

## 접근 방식

1. 무방향 그래프를 인접 리스트로 구성한다.
2. 각 정점을 두 가지 색(0, 1) 중 하나로 칠하되, 인접한 정점은 반드시 다른 색으로 칠한다.
3. BFS를 사용하여 시작 정점에 색 0을 칠하고, 인접 정점에 반대 색을 칠하며 탐색한다.
4. 탐색 중 인접한 정점이 이미 같은 색으로 칠해져 있으면 이분 그래프가 아니다.
5. 비연결 그래프를 고려하여 모든 연결 요소에 대해 BFS를 수행한다.

## 핵심 아이디어

이분 그래프 판별의 핵심은 "그래프 2-색칠(2-coloring)" 문제로 변환하는 것이다. 이분 그래프는 홀수 길이의 사이클이 존재하지 않는 그래프와 동치이다.

BFS로 정점을 레벨별로 탐색하면, 같은 레벨의 정점은 같은 색, 인접 레벨의 정점은 다른 색으로 자연스럽게 칠해진다. 만약 같은 레벨(같은 색)의 두 정점이 간선으로 연결되어 있다면, 이는 홀수 길이 사이클이 존재함을 의미하므로 이분 그래프가 아니다.

예제 1에서 정점 1,3을 그룹 A, 정점 2,4를 그룹 B로 나누면 모든 간선이 그룹 간에만 존재한다. 예제 2에서 삼각형(1-2-3-1)은 홀수 사이클이므로 2-색칠이 불가능하다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N + M) |
| 공간 복잡도 | O(N + M) |

모든 정점과 간선을 한 번씩 처리하므로 O(N + M)이다. 인접 리스트에 O(N + M), 색칠 배열과 BFS 큐에 O(N)의 공간이 필요하다.

## 대안적 접근

- DFS로도 동일하게 색칠하며 판별할 수 있다. 재귀적으로 인접 정점에 반대 색을 칠하고, 충돌이 발생하면 이분 그래프가 아니다.
- Union-Find를 활용하여 각 정점의 인접 정점들을 같은 그룹으로 묶고, 정점과 인접 정점이 같은 그룹에 속하면 이분 그래프가 아닌 것으로 판별하는 방법도 있다.
