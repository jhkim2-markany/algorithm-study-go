# 이분 그래프 판별

**난이도:** 하

## 문제 설명

N개의 정점과 M개의 간선으로 이루어진 무방향 그래프가 주어진다. 이 그래프가 이분 그래프인지 판별하시오.

이분 그래프란 그래프의 정점을 두 그룹으로 나누어, 같은 그룹에 속한 정점끼리는 간선으로 연결되지 않도록 할 수 있는 그래프이다.

이분 그래프이면 "Yes"를, 아니면 "No"를 출력한다.

## 입력 형식

- 첫째 줄에 정점의 수 N과 간선의 수 M이 주어진다
- 둘째 줄부터 M개의 줄에 각 간선의 양 끝 정점 U V가 주어진다

## 출력 형식

- 이분 그래프이면 "Yes", 아니면 "No"를 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- 0 ≤ M ≤ 200,000
- 1 ≤ U, V ≤ N
- U ≠ V
- 그래프는 비연결 그래프일 수 있다
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
4 4
1 2
2 3
3 4
4 1
```

### 예제 출력 1

```
Yes
```

### 예제 입력 2

```
3 3
1 2
2 3
3 1
```

### 예제 출력 2

```
No
```
