# 네트워크 플로우

**난이도:** 상

## 문제 설명

N개의 정점과 M개의 간선으로 이루어진 방향 그래프가 주어진다. 각 간선에는 용량(capacity)이 있다. 소스 정점 S에서 싱크 정점 T로 보낼 수 있는 최대 유량(maximum flow)을 구하시오.

유량은 다음 조건을 만족해야 한다:
- 각 간선의 유량은 0 이상이고 용량 이하이다 (용량 제한)
- 소스와 싱크를 제외한 모든 정점에서 들어오는 유량의 합과 나가는 유량의 합이 같다 (유량 보존)

## 입력 형식

- 첫째 줄에 정점의 수 N, 간선의 수 M, 소스 S, 싱크 T가 주어진다
- 둘째 줄부터 M개의 줄에 각 간선의 시작 정점 U, 끝 정점 V, 용량 C가 주어진다

## 출력 형식

- 소스에서 싱크로의 최대 유량을 출력한다

## 제약 조건

- 2 ≤ N ≤ 500
- 1 ≤ M ≤ 5,000
- 1 ≤ S, T ≤ N
- S ≠ T
- 1 ≤ U, V ≤ N
- 1 ≤ C ≤ 1,000
- 같은 방향의 중복 간선이 존재할 수 있다
- 시간 제한: 2초
- 메모리 제한: 256MB

5
```

```
