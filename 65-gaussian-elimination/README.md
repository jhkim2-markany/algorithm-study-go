# 65. 가우스 소거법 (Gaussian Elimination)

연립일차방정식을 행렬의 행 연산을 통해 체계적으로 푸는 알고리즘이다. 전진 소거(Forward Elimination)로 상삼각 행렬을 만든 뒤, 후진 대입(Back Substitution)으로 해를 구한다. 행렬의 랭크, 행렬식(Determinant) 계산, GF(2) 위의 XOR 연립방정식 등 다양한 선형대수 문제에 활용된다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이

| --- | --- | --- |
