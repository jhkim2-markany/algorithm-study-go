# XOR 연립방정식

**난이도:** 중

## 문제 설명

N개의 미지수 x₁, x₂, ..., xₙ (각각 0 또는 1)과 M개의 XOR 방정식이 주어진다. 각 방정식은 일부 미지수들의 XOR 결과가 0 또는 1이라는 조건이다.

모든 방정식을 동시에 만족하는 해가 존재하는지 판별하고, 존재한다면 아무 해 하나를 출력하라.

## 입력 형식

- 첫째 줄에 미지수의 수 N과 방정식의 수 M이 주어진다.
- 둘째 줄부터 M개의 줄에 각 방정식의 정보가 주어진다.
  - 각 줄의 첫 번째 수 K는 해당 방정식에 포함된 미지수의 개수이다.
  - 이어서 K개의 미지수 번호 i₁, i₂, ..., iₖ가 주어진다 (1-indexed).
  - 마지막으로 결과값 b (0 또는 1)가 주어진다.
  - 이는 x_{i₁} XOR x_{i₂} XOR ... XOR x_{iₖ} = b 를 의미한다.

## 출력 형식

- 해가 존재하면 첫째 줄에 x₁, x₂, ..., xₙ의 값을 공백으로 구분하여 출력한다.
- 해가 존재하지 않으면 "IMPOSSIBLE"을 출력한다.

## 제약 조건

- 1 ≤ N ≤ 500
- 1 ≤ M ≤ 500
- 1 ≤ K ≤ N
- 각 미지수 번호는 1 이상 N 이하이다
- b는 0 또는 1이다

## 예제

### 예제 입력 1

```
3 3
2 1 2 1
2 2 3 0
2 1 3 1
```

### 예제 출력 1

```
1 0 0
```

### 예제 입력 2

```
2 2
1 1 1
1 1 0
```

### 예제 출력 2

```
IMPOSSIBLE
```
