# 연립방정식 풀기

**난이도:** 하

## 문제 설명

N개의 미지수와 N개의 방정식으로 이루어진 연립일차방정식이 주어진다. 이 연립방정식의 유일한 해를 구하라.

모든 계수와 상수는 실수이며, 주어지는 연립방정식은 항상 유일한 해를 가진다고 보장된다.

## 입력 형식

- 첫째 줄에 미지수의 수 N이 주어진다.
- 둘째 줄부터 N개의 줄에 각 방정식의 계수 a₁, a₂, ..., aₙ과 상수 b가 공백으로 구분되어 주어진다.
- 즉, i번째 줄은 a_{i,1} x₁ + a_{i,2} x₂ + ... + a_{i,n} xₙ = bᵢ 를 나타낸다.

## 출력 형식

- x₁, x₂, ..., xₙ의 값을 공백으로 구분하여 소수점 아래 6자리까지 출력한다.

## 제약 조건

- 1 ≤ N ≤ 100
- 모든 계수와 상수의 절댓값은 10⁶ 이하이다
- 연립방정식은 항상 유일한 해를 가진다

## 예제

### 예제 입력 1

```
3
2 1 -1 8
-3 -1 2 -11
-2 1 2 -3
```

### 예제 출력 1

```
2.000000 3.000000 -1.000000
```

### 예제 입력 2

```
2
1 2 5
3 -1 1
```

### 예제 출력 2

```
1.000000 2.000000
```
