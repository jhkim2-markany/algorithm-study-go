# 연립방정식 풀기 - 해설

## 접근 방식

1. N×(N+1) 확대 행렬 [A|b]를 구성한다
2. 전진 소거로 상삼각 행렬을 만든다
3. 부분 피벗팅으로 수치 안정성을 확보한다
4. 후진 대입으로 마지막 미지수부터 역순으로 해를 구한다

## 핵심 아이디어

가우스 소거법의 가장 기본적인 형태이다. 유일해가 보장되므로 해의 존재 여부 판별 없이 전진 소거와 후진 대입만 수행하면 된다.

부분 피벗팅은 실수 연산의 오차를 줄이기 위해 각 단계에서 절댓값이 가장 큰 원소를 피벗으로 선택하는 기법이다.

```
전진 소거: row[i] -= (a[i][col] / a[pivot][col]) × row[pivot]
후진 대입: x[i] = (b[i] - Σ a[i][j]×x[j]) / a[i][i]  (j = i+1 ~ n-1)
```

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N³) |
| 공간 복잡도 | O(N²) |

전진 소거에서 각 열에 대해 O(N)개의 행을 O(N) 연산으로 소거하므로 O(N³)이다.
