# 행렬 랭크 구하기

**난이도:** 상

## 문제 설명

N×M 크기의 정수 행렬이 주어진다. 이 행렬의 랭크(Rank)를 구하라.

행렬의 랭크란 선형 독립인 행(또는 열)의 최대 개수이다. 모든 연산은 소수 P = 998244353에 대한 모듈러 연산으로 수행한다.

## 입력 형식

- 첫째 줄에 행의 수 N과 열의 수 M이 주어진다.
- 둘째 줄부터 N개의 줄에 행렬의 각 행이 M개의 정수로 주어진다.

## 출력 형식

- 행렬의 랭크를 출력한다.

## 제약 조건

- 1 ≤ N, M ≤ 500
- 0 ≤ 각 원소 ≤ 998244352
- 모든 연산은 mod 998244353으로 수행한다

## 예제

### 예제 입력 1

```
3 3
1 2 3
4 5 6
7 8 9
```

### 예제 출력 1

```
2
```

### 예제 입력 2

```
3 3
1 0 0
0 1 0
0 0 1
```

### 예제 출력 2

```
3
```

### 예제 입력 3

```
2 4
1 2 3 4
2 4 6 8
```

### 예제 출력 3

```
1
```
