# 행렬 랭크 구하기 - 해설

## 접근 방식

1. N×M 행렬을 입력받는다
2. 모듈러 가우스 소거법으로 행 사다리꼴(REF)을 만든다
3. 피벗이 존재하는 행의 수가 곧 랭크이다

## 핵심 아이디어

행렬의 랭크는 가우스 소거법으로 행 사다리꼴을 만들었을 때 영이 아닌 행의 수와 같다.

모듈러 연산에서의 핵심:

- 나눗셈 대신 모듈러 역원을 사용한다: a/b mod p = a × b⁻¹ mod p
- 페르마 소정리: b⁻¹ ≡ b^(p-2) mod p (p가 소수일 때)
- 실수 오차가 없으므로 부분 피벗팅 대신 0이 아닌 아무 행이나 피벗으로 선택해도 된다

```
소거 과정:
1. 각 열에 대해 0이 아닌 피벗 행을 찾는다
2. 피벗 행을 정규화한다 (피벗 원소를 1로)
3. 다른 모든 행에서 해당 열을 소거한다
4. 피벗이 존재하는 열의 수 = 랭크
```

예제 1에서 행렬 [[1,2,3],[4,5,6],[7,8,9]]의 세 번째 행은 첫 번째와 두 번째 행의 선형 결합이므로 (row3 = 2×row2 - row1), 소거 후 영행이 되어 랭크는 2이다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N × M × min(N, M)) |
| 공간 복잡도 | O(N × M) |

각 피벗 열에 대해 O(N)개의 행을 O(M) 연산으로 소거하고, 피벗 열은 최대 min(N, M)개이다. 모듈러 역원 계산은 O(log P)이지만 전체 복잡도에 비해 무시할 수 있다.
