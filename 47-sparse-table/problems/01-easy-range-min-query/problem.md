# 구간 최솟값 쿼리

**난이도:** 하

## 문제 설명

길이 N인 정수 배열이 주어진다. M개의 쿼리가 주어지며, 각 쿼리는 구간 [L, R]에서 최솟값을 구하는 것이다. 배열의 값은 변경되지 않는다.

## 입력 형식

- 첫째 줄에 배열의 크기 N과 쿼리의 수 M이 주어진다
- 둘째 줄에 배열의 원소 N개가 공백으로 구분되어 주어진다
- 다음 M개의 줄에 각 쿼리의 L, R이 주어진다 (1-indexed)

## 출력 형식

- 각 쿼리에 대해 구간 [L, R]의 최솟값을 한 줄에 하나씩 출력한다

## 제약 조건

- 1 ≤ N ≤ 100,000
- 1 ≤ M ≤ 100,000
- 1 ≤ L ≤ R ≤ N
- -10⁹ ≤ 배열 원소 ≤ 10⁹
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5 3
5 1 4 2 3
1 3
2 5
1 5
```

### 예제 출력 1

```
1
1
1
```

### 예제 입력 2

```
7 4
3 7 2 8 1 6 4
1 1
3 5
2 6
1 7
```

### 예제 출력 2

```
3
1
1
1
```
