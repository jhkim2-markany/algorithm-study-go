# 슬라이딩 윈도우 최솟값

**난이도:** 상

## 문제 설명

길이 N인 정수 배열과 윈도우 크기 K가 주어진다. 크기 K인 슬라이딩 윈도우가 배열의 왼쪽 끝에서 오른쪽 끝까지 한 칸씩 이동할 때, 각 위치에서 윈도우 내의 최솟값과 최댓값을 모두 구하라.

즉, 모든 i (1 ≤ i ≤ N-K+1)에 대해 구간 [i, i+K-1]의 최솟값과 최댓값을 구해야 한다.

## 입력 형식

- 첫째 줄에 배열의 크기 N과 윈도우 크기 K가 주어진다
- 둘째 줄에 배열의 원소 N개가 공백으로 구분되어 주어진다

## 출력 형식

- 첫째 줄에 각 윈도우 위치에서의 최솟값을 공백으로 구분하여 출력한다
- 둘째 줄에 각 윈도우 위치에서의 최댓값을 공백으로 구분하여 출력한다

## 제약 조건

- 1 ≤ K ≤ N ≤ 1,000,000
- -10⁹ ≤ 배열 원소 ≤ 10⁹
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
8 3
1 3 -1 -3 5 3 6 7
```

### 예제 출력 1

```
-1 -3 -3 -3 3 3
3 3 5 5 6 7
```

### 예제 입력 2

```
5 2
5 1 4 2 3
```

### 예제 출력 2

```
1 1 2 2
5 4 4 3
```
