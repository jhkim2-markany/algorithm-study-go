# 47. 희소 배열 (Sparse Table)

정적 배열에서 구간 최솟값/최댓값 등 멱등 연산에 대한 쿼리를 O(1)에 응답하는 자료구조이다. 전처리 O(N log N), 쿼리 O(1)이 핵심이다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
