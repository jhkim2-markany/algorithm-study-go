# 조건부 확률과 기댓값

**난이도:** 상

## 문제 설명

N종류의 아이템이 있는 뽑기 기계가 있다. 각 뽑기에서 i번째 아이템이 나올 확률은 pᵢ이다 (p₁ + p₂ + ... + pₙ = 1). 뽑기를 반복하여 모든 N종류의 아이템을 최소 1개씩 모으려고 한다.

모든 종류의 아이템을 모으기 위해 필요한 뽑기 횟수의 기댓값을 구하시오. 소수점 아래 6자리까지 출력한다.

## 입력 형식

- 첫째 줄에 아이템 종류의 수 N이 주어진다
- 둘째 줄에 각 아이템의 확률 p₁, p₂, ..., pₙ이 공백으로 구분되어 주어진다 (소수점 형태)

## 출력 형식

- 기댓값을 소수점 아래 6자리까지 출력한다

## 제약 조건

- 1 ≤ N ≤ 20
- 0 < pᵢ ≤ 1
- p₁ + p₂ + ... + pₙ = 1
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
2
0.5 0.5
```

### 예제 출력 1

```
3.000000
```

### 예제 입력 2

```
3
0.5 0.3 0.2
```

### 예제 출력 2

```
8.644444
```

### 예제 입력 3

```
1
1.0
```

### 예제 출력 3

```
1.000000
```
