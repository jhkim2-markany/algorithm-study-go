# 주사위 확률 계산 - 해설

## 접근 방식

1. 주사위를 N번 던져서 합이 S가 되는 경우의 수를 DP로 구한다
2. 전체 경우의 수 6^N을 분모로 사용한다
3. 경우의 수와 전체 경우의 수의 최대공약수로 나누어 기약분수를 만든다

## 핵심 아이디어

확률 DP의 기본 형태이다. dp[i][j]를 "주사위를 i번 던져서 합이 j가 되는 경우의 수"로 정의한다. i번째 주사위에서 눈 f(1~6)가 나오면 dp[i][j] += dp[i-1][j-f]로 전이한다.

부동소수점 오차를 피하기 위해 확률을 직접 계산하지 않고, 경우의 수를 정수로 세어 기약분수로 출력한다. 최대공약수(GCD)를 구해 분자와 분모를 약분하면 기약분수가 된다.

예제 1에서 주사위 2번으로 합 7을 만드는 경우는 (1,6), (2,5), (3,4), (4,3), (5,2), (6,1)의 6가지이고, 전체 경우의 수는 36이므로 확률은 6/36 = 1/6이다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N × S × 6) |
| 공간 복잡도 | O(N × S) |

주사위 횟수 N, 목표 합 S에 대해 각 상태에서 6가지 전이를 확인하므로 O(N × S × 6)이다. N ≤ 8이므로 충분히 빠르다.

## 대안적 접근

- 조합론적 접근: 포함-배제 원리를 사용하여 합이 S가 되는 경우의 수를 직접 계산할 수 있다. 하지만 DP 방식이 더 직관적이고 구현이 간단하다.
