# 주사위 확률 계산

**난이도:** 하

## 문제 설명

공정한 6면 주사위를 N번 던진다. 주사위를 N번 던져서 나온 눈의 합이 정확히 S가 될 확률을 구하시오. 확률은 기약분수 형태 P/Q로 출력한다.

## 입력 형식

- 첫째 줄에 주사위를 던지는 횟수 N과 목표 합 S가 공백으로 구분되어 주어진다

## 출력 형식

- 확률을 기약분수 P/Q 형태로 출력한다
- 확률이 0이면 "0/1"을 출력한다

## 제약 조건

- 1 ≤ N ≤ 8
- N ≤ S ≤ 6×N
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
2 7
```

### 예제 출력 1

```
1/6
```

### 예제 입력 2

```
1 3
```

### 예제 출력 2

```
1/6
```

### 예제 입력 3

```
2 2
```

### 예제 출력 3

```
1/36
```
