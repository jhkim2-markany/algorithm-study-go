# 기댓값 계산 DP

**난이도:** 중

## 문제 설명

1번 칸에서 시작하여 N번 칸에 도착하는 보드 게임이 있다. 매 턴마다 공정한 6면 주사위를 던져서 나온 눈의 수만큼 앞으로 이동한다. 현재 위치에서 주사위 눈만큼 이동했을 때 N번 칸을 넘어가면 이동하지 않고 제자리에 머문다. 정확히 N번 칸에 도착하면 게임이 끝난다.

1번 칸에서 시작하여 N번 칸에 도착하기까지 주사위를 던지는 횟수의 기댓값을 구하시오. 소수점 아래 6자리까지 출력한다.

## 입력 형식

- 첫째 줄에 보드의 마지막 칸 번호 N이 주어진다

## 출력 형식

- 기댓값을 소수점 아래 6자리까지 출력한다

## 제약 조건

- 2 ≤ N ≤ 1,000
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
2
```

### 예제 출력 1

```
6.000000
```

### 예제 입력 2

```
7
```

### 예제 출력 2

```
1.000000
```

### 예제 입력 3

```
8
```

### 예제 출력 3

```
2.521626
```
