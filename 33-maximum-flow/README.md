# 33. 최대 유량

네트워크에서 소스(source)에서 싱크(sink)로 보낼 수 있는 최대 유량을 구하는 알고리즘 유형이다. Ford-Fulkerson 방법, Edmonds-Karp 알고리즘, 최대 유량-최소 컷 정리 등이 핵심이다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
