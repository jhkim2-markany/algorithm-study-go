# 도시 간 물자 수송 - 해설

## 접근 방식

1. 주어진 네트워크를 인접 리스트로 구성하고 Edmonds-Karp 알고리즘으로 최대 유량을 구한다
2. 최대 유량-최소 컷 정리에 의해 최대 수송량과 최소 차단 비용은 같다
3. 최소 컷을 구하기 위해, 최대 유량 계산 후 잔여 그래프에서 소스로부터 BFS를 수행한다
4. 소스에서 도달 가능한 정점 집합 S와 나머지 집합 T를 구분한다
5. S에서 T로 가는 원본 간선들이 최소 컷에 포함되는 간선이다

## 핵심 아이디어

최대 유량-최소 컷 정리가 이 문제의 핵심이다:

- 최대 유량 알고리즘이 종료되면, 잔여 그래프에서 소스로부터 싱크에 도달할 수 없다
- 이때 소스에서 도달 가능한 정점 집합 S와 나머지 T가 최소 컷을 형성한다
- S에서 T로 가는 간선들은 모두 포화 상태(유량 = 용량)이며, 이들의 용량 합이 최대 유량과 같다
- 같은 두 도시 사이에 여러 도로가 있을 수 있으므로, 각 간선을 독립적으로 처리해야 한다

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(V × E²) |
| 공간 복잡도 | O(V + E) |

Edmonds-Karp 알고리즘의 시간 복잡도는 O(V × E²)이다. 최소 컷을 구하는 BFS는 O(V + E)로 추가 비용이 적다. N ≤ 200, M ≤ 2,000이므로 시간 내에 충분하다.

## 대안적 접근

- Dinic 알고리즘을 사용하면 O(V² × E)로 더 빠르게 최대 유량을 구할 수 있다
- 최소 컷만 필요한 경우에도 최대 유량을 먼저 구해야 하므로, 유량 알고리즘은 필수적이다
