# 프로젝트 선택 문제

**난이도:** 상

## 문제 설명

회사에 N개의 프로젝트가 있다. 각 프로젝트 i를 수행하면 이익 p_i를 얻는다 (p_i가 음수이면 비용이 든다).

프로젝트 간에 M개의 의존 관계가 있다. 의존 관계 (a, b)는 "프로젝트 a를 수행하려면 반드시 프로젝트 b도 수행해야 한다"는 의미이다.

의존 관계를 모두 만족하면서 수행할 프로젝트 집합을 선택할 때, 얻을 수 있는 최대 순이익을 구하시오. 아무 프로젝트도 선택하지 않으면 순이익은 0이다.

## 입력 형식

- 첫째 줄에 프로젝트의 수 N과 의존 관계의 수 M이 주어진다
- 둘째 줄에 각 프로젝트의 이익 p_1, p_2, ..., p_N이 공백으로 구분되어 주어진다
- 다음 M개의 줄에 의존 관계 a b가 주어진다 (프로젝트 a는 프로젝트 b에 의존)

## 출력 형식

- 최대 순이익을 출력한다

## 제약 조건

- 1 ≤ N ≤ 200
- 0 ≤ M ≤ N × (N - 1) / 2
- -10,000 ≤ p_i ≤ 10,000
- 1 ≤ a, b ≤ N, a ≠ b
- 의존 관계에 순환은 없다
- 시간 제한: 2초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
5 3
10 -3 8 -5 4
1 2
3 4
3 2
```

### 예제 출력 1

```
14
```

### 예제 입력 2

```
3 2
5 -10 3
1 2
3 2
```

### 예제 출력 2

```
5
```
