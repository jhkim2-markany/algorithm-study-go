# 기본 네트워크 유량

**난이도:** 하

## 문제 설명

N개의 정점과 M개의 간선으로 이루어진 방향 그래프가 주어진다. 각 간선에는 용량이 있으며, 소스(정점 1)에서 싱크(정점 N)로 보낼 수 있는 최대 유량을 구하시오.

유량은 다음 조건을 만족해야 한다:
- 각 간선의 유량은 0 이상이고 용량 이하이다
- 소스와 싱크를 제외한 모든 정점에서 들어오는 유량과 나가는 유량이 같다

## 입력 형식

- 첫째 줄에 정점의 수 N과 간선의 수 M이 주어진다
- 다음 M개의 줄에 각 간선의 시작 정점 u, 끝 정점 v, 용량 c가 주어진다

## 출력 형식

- 소스(정점 1)에서 싱크(정점 N)로의 최대 유량을 출력한다

## 제약 조건

- 2 ≤ N ≤ 100
- 1 ≤ M ≤ 1,000
- 1 ≤ u, v ≤ N
- 1 ≤ c ≤ 1,000
- 시간 제한: 1초
- 메모리 제한: 256MB

## 예제

### 예제 입력 1

```
4 5
1 2 3
1 3 2
2 3 1
2 4 2
3 4 3
```

### 예제 출력 1

```
5
```

### 예제 입력 2

```
2 1
1 2 10
```

### 예제 출력 2

```
10
```
