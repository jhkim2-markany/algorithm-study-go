# 기본 네트워크 유량 - 해설

## 접근 방식

1. 주어진 방향 그래프를 인접 리스트로 구성한다
2. 각 간선에 대해 역방향 간선도 함께 추가한다 (용량 0)
3. Edmonds-Karp 알고리즘(BFS 기반)으로 증가 경로를 반복적으로 찾는다
4. 증가 경로의 병목 용량만큼 유량을 갱신한다
5. 더 이상 증가 경로가 없을 때 총 유량이 최대 유량이다

## 핵심 아이디어

Edmonds-Karp 알고리즘은 Ford-Fulkerson 방법에서 증가 경로를 BFS로 찾는 구체적 구현이다. 핵심 포인트는 다음과 같다:

- 역방향 간선을 통해 이전에 보낸 유량을 취소할 수 있다. 이를 통해 탐욕적으로 잘못된 경로를 선택하더라도 나중에 수정이 가능하다
- BFS로 최단 증가 경로를 찾으므로 시간 복잡도가 O(V × E²)로 보장된다
- 간선을 추가할 때 순방향과 역방향 간선이 서로의 인덱스를 저장하여, 유량 갱신 시 O(1)에 역방향 간선에 접근할 수 있다

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(V × E²) |
| 공간 복잡도 | O(V + E) |

BFS 한 번에 O(E)이고, 증가 경로는 최대 O(V × E)번 찾을 수 있으므로 전체 O(V × E²)이다. N ≤ 100, M ≤ 1,000이므로 충분히 빠르다.

## 대안적 접근

- DFS 기반 Ford-Fulkerson은 구현이 간단하지만, 정수 용량이 아닌 경우 종료가 보장되지 않을 수 있다
- Dinic 알고리즘은 O(V² × E)로 더 빠르지만, 이 문제의 제약 조건에서는 Edmonds-Karp로 충분하다
