# 54. 트리 DP (Tree DP)

트리 구조에서 동적 프로그래밍을 적용하는 알고리즘 유형이다. 서브트리 단위로 부분 문제를 정의하고, DFS를 통해 리프에서 루트 방향으로 값을 합산하거나, 리루팅(Rerooting) 기법으로 모든 노드를 루트로 삼은 결과를 효율적으로 구한다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
