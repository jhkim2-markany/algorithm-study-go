# 트리의 지름

**난이도:** 중

## 문제 설명

N개의 노드로 이루어진 트리가 주어진다. 각 간선에는 양의 가중치가 있다.

트리의 지름이란 트리에서 가장 먼 두 노드 사이의 거리(경로 위 간선 가중치의 합)를 말한다. 트리의 지름을 구하라.

## 입력 형식

- 첫째 줄에 노드의 수 N이 주어진다.
- 둘째 줄부터 N-1개의 줄에 간선 정보 u, v, w가 주어진다. 이는 노드 u와 노드 v가 가중치 w인 간선으로 연결되어 있음을 의미한다.

## 출력 형식

- 트리의 지름을 출력한다.

## 제약 조건

- 2 ≤ N ≤ 100,000
- 1 ≤ u, v ≤ N
- 1 ≤ w ≤ 1,000
- 주어지는 그래프는 항상 트리이다

## 예제

### 예제 입력 1

```
5
1 2 3
1 3 2
2 4 4
2 5 1
```

### 예제 출력 1

```
9
```

### 예제 입력 2

```
2
1 2 5
```

### 예제 출력 2

```
5
```
