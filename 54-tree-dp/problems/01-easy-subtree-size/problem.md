# 서브트리 크기 구하기

**난이도:** 하

## 문제 설명

N개의 노드로 이루어진 트리가 주어진다. 노드는 1번부터 N번까지 번호가 매겨져 있고, 루트는 1번 노드이다.

각 노드에 대해 해당 노드를 루트로 하는 서브트리에 포함된 노드의 수를 구하라.

## 입력 형식

- 첫째 줄에 노드의 수 N이 주어진다.
- 둘째 줄부터 N-1개의 줄에 트리의 간선 정보 u, v가 주어진다. 이는 노드 u와 노드 v가 간선으로 연결되어 있음을 의미한다.

## 출력 형식

- 1번 노드부터 N번 노드까지, 각 노드의 서브트리 크기를 공백으로 구분하여 출력한다.

## 제약 조건

- 1 ≤ N ≤ 100,000
- 1 ≤ u, v ≤ N
- 주어지는 그래프는 항상 트리이다

## 예제

### 예제 입력 1

```
6
1 2
1 3
2 4
2 5
3 6
```

### 예제 출력 1

```
6 3 2 1 1 1
```

### 예제 입력 2

```
1
```

### 예제 출력 2

```
1
```
