# 서브트리 크기 구하기 - 해설

## 접근 방식

1. 트리를 인접 리스트로 저장한다
2. 루트(1번 노드)에서 DFS 후위 순회를 수행한다
3. 리프 노드의 서브트리 크기는 1이다
4. 내부 노드의 서브트리 크기는 자기 자신(1) + 모든 자식의 서브트리 크기 합이다
5. DFS에서 부모 노드를 인자로 전달하여 역방향 탐색을 방지한다

## 핵심 아이디어

트리 DP의 가장 기본적인 형태이다. 점화식은 다음과 같다:

```
sz[v] = 1 + Σ sz[child]  (child는 v의 모든 자식)
```

DFS 후위 순회를 사용하면 자식 노드의 값이 먼저 계산되므로, 부모 노드에서 자식의 결과를 바로 사용할 수 있다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N) |
| 공간 복잡도 | O(N) |

각 노드와 간선을 정확히 한 번씩 방문하므로 O(N)이다.
