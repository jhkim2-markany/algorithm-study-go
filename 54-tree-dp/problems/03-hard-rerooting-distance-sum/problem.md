# 모든 노드의 거리 합

**난이도:** 상

## 문제 설명

N개의 노드로 이루어진 트리가 주어진다. 간선의 가중치는 모두 1이다.

각 노드 i에 대해, 노드 i에서 다른 모든 노드까지의 거리의 합을 구하라.

두 노드 사이의 거리는 두 노드를 잇는 경로 위의 간선 수이다.

## 입력 형식

- 첫째 줄에 노드의 수 N이 주어진다.
- 둘째 줄부터 N-1개의 줄에 간선 정보 u, v가 주어진다. 이는 노드 u와 노드 v가 간선으로 연결되어 있음을 의미한다.

## 출력 형식

- N개의 정수를 공백으로 구분하여 출력한다. i번째 정수는 노드 i에서 다른 모든 노드까지의 거리 합이다.

## 제약 조건

- 1 ≤ N ≤ 200,000
- 1 ≤ u, v ≤ N
- 주어지는 그래프는 항상 트리이다

## 예제

### 예제 입력 1

```
4
1 2
2 3
2 4
```

### 예제 출력 1

```
5 3 5 5
```

### 예제 입력 2

```
6
1 2
1 3
2 4
2 5
3 6
```

### 예제 출력 2

```
9 7 9 11 11 13
```

### 예제 입력 3

```
1
```

### 예제 출력 3

```
0
```
