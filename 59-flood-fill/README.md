# 59. 플러드 필 (Flood Fill)

격자(grid) 위에서 연결된 영역을 탐색하고 채우는 알고리즘이다. BFS 또는 DFS를 기반으로 같은 속성을 가진 인접 칸들을 탐색하며, 삼성 SW 역량테스트와 카카오 코딩테스트에서 단골로 출제되는 유형이다.

## 구성

- `theory.md` - 이론 및 개념 설명
- `examples/` - 기본 구현 예시 코드
- `problems/` - 연습 문제 및 풀이
