# 색칠 쿼리 처리

**난이도:** 상

## 문제 설명

N×M 격자가 주어진다. 각 칸은 1 이상 100,000 이하의 색상 번호를 가진다. 상하좌우로 인접하고 같은 색상인 칸들은 하나의 영역을 이룬다.

Q개의 쿼리가 주어진다. 각 쿼리는 `r c x` 형태로, (r, c) 칸이 속한 영역 전체의 색상을 x로 변경하라는 의미이다. 색상이 변경되면 인접한 같은 색상의 영역과 합쳐질 수 있다.

모든 쿼리를 처리한 후 격자의 상태를 출력하라.

## 입력 형식

- 첫째 줄에 격자의 크기 N, M과 쿼리 수 Q가 주어진다.
- 둘째 줄부터 N개의 줄에 격자 정보가 주어진다. 각 줄에 M개의 정수가 공백으로 구분되어 주어진다.
- 다음 Q개의 줄에 각 쿼리 `r c x`가 주어진다. (r, c)는 1-indexed이다.

## 출력 형식

- 모든 쿼리를 처리한 후의 격자를 N개의 줄에 출력한다. 각 줄에 M개의 정수를 공백으로 구분하여 출력한다.

## 제약 조건

- 1 ≤ N, M ≤ 500
- 1 ≤ Q ≤ 10,000
- 1 ≤ 격자의 색상 번호, x ≤ 100,000
- 1 ≤ r ≤ N, 1 ≤ c ≤ M

## 예제

### 예제 입력 1

```
3 3 2
1 1 2
1 2 2
3 3 2
1 1 2
2 2 1
```

### 예제 출력 1

```
1 1 1
1 1 1
3 3 1
```

### 예제 입력 2

```
2 2 1
1 2
3 4
1 1 5
```

### 예제 출력 2

```
5 2
3 4
```
