# 영역 개수 세기 - 해설

## 접근 방식

1. 격자의 모든 칸을 순회한다
2. 값이 1(땅)이고 아직 방문하지 않은 칸을 발견하면, 새로운 섬이다
3. 해당 칸에서 BFS(플러드 필)를 수행하여 연결된 모든 땅을 방문 처리한다
4. BFS를 시작한 횟수가 곧 섬의 개수이다

## 핵심 아이디어

플러드 필의 가장 기본적인 활용이다. 격자에서 연결 요소(Connected Component)의 개수를 세는 문제로, 미방문 칸을 발견할 때마다 BFS를 수행하면 된다.

방문 처리는 큐에 넣는 시점에 해야 한다. 큐에서 꺼내는 시점에 방문 처리하면 같은 칸이 여러 번 큐에 들어가 비효율적이다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N × M) |
| 공간 복잡도 | O(N × M) |

각 칸을 최대 한 번 방문하므로 O(N×M)이다. 방문 배열과 큐에 O(N×M) 공간이 필요하다.

## 대안적 접근

- DFS 재귀: 구현이 더 간단하지만, 격자가 클 때(1000×1000) 스택 오버플로우 위험이 있다
- 유니온 파인드: 인접한 1끼리 Union 연산 후 집합 수를 세는 방법도 가능하다
