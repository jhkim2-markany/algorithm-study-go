# 가장 큰 섬의 크기 - 해설

## 접근 방식

1. 격자의 모든 섬에 고유 번호를 부여하고 각 섬의 크기를 구한다 (플러드 필)
2. 모든 바다 칸(0)을 순회하면서, 해당 칸을 땅으로 바꿨을 때 인접한 섬들의 크기 합을 계산한다
3. 인접한 섬 번호가 중복되지 않도록 집합(map)으로 관리한다
4. 모든 바다 칸에 대해 계산한 값 중 최댓값이 답이다

## 핵심 아이디어

바다 칸마다 매번 플러드 필을 수행하면 O(N²M²)으로 시간 초과가 발생한다. 대신 **사전 처리**로 각 섬의 크기를 미리 구해두면, 바다 칸마다 O(1)에 인접 섬 크기를 합산할 수 있다.

핵심은 섬에 고유 번호를 부여하는 것이다. 같은 섬에 속한 칸이 바다 칸의 여러 방향에서 인접할 수 있으므로, 섬 번호로 중복을 제거해야 정확한 크기를 구할 수 있다.

## 복잡도 분석

| 구분 | 복잡도 |
| --- | --- |
| 시간 복잡도 | O(N × M) |
| 공간 복잡도 | O(N × M) |

1단계 플러드 필: O(N×M). 2단계 바다 칸 순회: O(N×M), 각 칸에서 4방향만 확인하므로 O(1). 전체 O(N×M)이다.

## 대안적 접근

- 유니온 파인드: 인접한 1끼리 Union 후, 바다 칸마다 인접 집합의 크기를 합산하는 방법도 가능하다
- 브루트포스: 바다 칸마다 플러드 필을 수행하면 O(N²M²)으로 작은 격자에서만 가능하다
